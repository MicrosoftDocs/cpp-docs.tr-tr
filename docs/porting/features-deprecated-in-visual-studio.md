# <a name="c-features-deprecated-in-visual-studio-2019"></a>C++Visual Studio 2019 ' de kullanım dışı Özellikler

- Windows XP geliştirme desteği

  Visual Studio 2019 ve üzeri, v142 araç takımını kullanarak Windows XP için kod oluşturmaya yönelik destek içermez. Visual Studio 2017 ' de gönderilen v141 araç takımını kullanarak Windows XP geliştirme desteği Visual Studio Yükleyicisi isteğe bağlı bir bileşen olarak sunulmaktadır. V141 Windows XP platform araç takımını nasıl yükleyeceğiniz hakkında bilgi için bkz. [WINDOWS XP için programları yapılandırma](../build/configuring-programs-for-windows-xp.md).
