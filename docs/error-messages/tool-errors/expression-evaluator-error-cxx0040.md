# <a name="expression-evaluator-error-cxx0040"></a>İfade Değerlendirici Hatası CXX0040

işlev örtük dönüştürme gerektiriyor

C ifade değerlendiricisi, Oluşturucu çağrılarını içeren örtük dönüştürmeleri desteklemez.

Bu hata CAN0040 ile aynıdır.
