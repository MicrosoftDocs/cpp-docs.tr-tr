# <a name="resource-compiler-error-rc2015"></a>Kaynak Derleyicisi Hatası RC2015
çok fazla karakter içinde sabiti  
  
 Karakter sabiti ikiden fazla karakter içeriyor.  
  
 Karakter sabitleri, tek karakter (standart karakter sabitleri) ya da iki karakter (uzun karakter sabitleri) sınırlıdır.  
  
 Kaçış dizisi (örneğin, sekme \t) için tek bir karakter dönüştürülür unutmayın.
