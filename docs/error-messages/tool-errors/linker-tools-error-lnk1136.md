# <a name="linker-tools-error-lnk1136"></a>Bağlayıcı Araçları Hatası LNK1136

dosya geçersiz veya bozuk

Giriş dosyası bozuk bir başlık ya da sıfır veya anormal olarak küçük boyut.

Dosya yanlış bir ek bağımlılık olarak belirtilmemişse LNK1136 da kaynaklanıyor olabilir.
