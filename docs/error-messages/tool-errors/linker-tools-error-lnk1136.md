# <a name="linker-tools-error-lnk1136"></a>Bağlayıcı Araçları Hatası LNK1136

dosya geçersiz veya bozuk

Giriş dosyası bozuk bir üstbilgiye sahip veya sıfır boyutunda ya da anormal küçük.

LNK1136 Ayrıca dosya yanlış bir ek bağımlılık olarak belirtilmiş olabilir.
