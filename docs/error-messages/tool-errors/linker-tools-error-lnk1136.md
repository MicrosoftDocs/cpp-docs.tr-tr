# <a name="linker-tools-error-lnk1136"></a>Bağlayıcı Araçları Hatası LNK1136
geçersiz veya bozuk dosya  
  
 Giriş dosyası bozuk üstbilgisi veya sıfır veya anormal olarak küçük boyut.  
  
 Dosya yanlış ek bir bağımlılık olarak belirtildiyse, LNK1136 da kaynaklanıyor olabilir.
