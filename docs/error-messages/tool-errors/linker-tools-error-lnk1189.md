# <a name="linker-tools-error-lnk1189"></a>Bağlayıcı Araçları Hatası LNK1189

LIBTOOMANYMEMBERS:: sayı nesnelerinin kitaplık sınırı aşıldı

65535 nesne veya bir kitaplıktaki üye sınırı aşıldı.

Bu sorunu onarmak için, kitaplığınızı daha az nesne veya üye ile iki veya daha fazla kitaplığa ayırın.
