# <a name="expression-evaluator-error-cxx0066"></a>İfade Değerlendirici Hatası CXX0066

statik üye yok

Bir sınıfın statik üyesi bulunamadı veya tanımlı değil. Bu hata, tanımlanmış ancak tanımlanmamış bir statik sınıf üyesinin oluşmasına neden olabilir veya yalnızca hata ayıklama bilgilerini içermeyen modüllerde tanımlanabilir ve başvurulur.

Bu hata CAN0066 ile aynıdır.
