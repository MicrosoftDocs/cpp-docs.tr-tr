# <a name="resource-compiler-error-rc2152"></a>Kaynak Derleyicisi Hatası RC2152

Geçersiz denetim karakteri

**Hızlandırıcılar** deyimindeki bir denetim karakteri geçersiz. Geçerli bir denetim karakteri bir harf (yalnızca) işaretinden (yalnızca) oluşur **^** .
