# <a name="linker-tools-warning-lnk4216"></a>Bağlayıcı Araçları Uyarısı LNK4216

Dışarı aktarılan giriş noktası girişi

DLL'den dışarı bir giriş noktası. Bir DLL giriş noktası aktarılması gerekmez.
