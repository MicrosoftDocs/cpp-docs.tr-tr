# <a name="linker-tools-warning-lnk4216"></a>Bağlayıcı Araçları Uyarısı LNK4216
Dışarı aktarılan giriş noktası girdisi  
  
 Bir giriş noktası DLL'den dışarı aktarılmış. DLL giriş noktasını aktarılması gerekmez.
