# <a name="linker-tools-warning-lnk4216"></a>Bağlayıcı Araçları Uyarısı LNK4216

Giriş noktası girişi verildi

Bir DLL 'den bir giriş noktası aktarılmış. DLL 'nin giriş noktasının verilmesi gerekmez.
