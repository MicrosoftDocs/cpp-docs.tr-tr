# <a name="linker-tools-warning-lnk4076"></a>Bağlayıcı Araçları Uyarısı LNK4076
Geçersiz artımlı durum dosyası 'filename'; nonincrementally bağlama  
  
 BAĞLANTI artımlı durum (.ilk) dosyasına yazamıyor. Her iki `filename` bozuk veya artımlı bir bağlama veritabanı değil. Dosya ve yeniden bağlama kaldırın.
