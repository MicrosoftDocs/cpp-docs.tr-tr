# <a name="nmake-fatal-error-u1097"></a>NMAKE Önemli Hatası U1097

filename-Parts söz dizimi bağımlı gerektirir

Geçerli bağımlılıkta açık veya örtük bir bağımlı yok. Yüzde () belirticisini kullanan filename-Parts söz dizimi `%` , geçerli hedefin ilk bağımlılığının bileşenlerini temsil eder.
