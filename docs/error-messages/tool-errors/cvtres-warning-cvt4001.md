# <a name="cvtres-warning-cvt4001"></a>CVTRES Uyarısı CVT4001
**makine türü belirtilmemiş; kabul edilir**   
 ***türü***  
  
 CVTRES makine belirtimi bulamadı. Belirtilen makine türü varsayılır. Varsayılan /MACHINE seçeneğini kullanarak hatalı, yeniden çalıştır CVTRES ise.
