# <a name="expression-evaluator-error-cxx0051"></a>İfade Değerlendirici Hatası CXX0051

kesme noktasında birden fazla aşırı yüklenmiş sembol belirtilmiş

İfade değerlendirici, bir kesme noktası olarak kullanılacak birden fazla aşırı yüklenmiş simgenin hangisi olduğunu belirleyemedi.

Bu hata CAN0051 ile aynıdır.
