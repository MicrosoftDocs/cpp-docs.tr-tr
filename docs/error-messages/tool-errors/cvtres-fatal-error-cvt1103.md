# <a name="cvtres-fatal-error-cvt1103"></a>CVTRES Önemli Hatası CVT1103
Dosya adı okunamıyor  
  
 CVTRES verilen dosyasını okumaya çalışırken kurtarılamaz bir hata oluştu.  
  
 Bu hata, dosya bozulması nedeniyle oluşabilir.
