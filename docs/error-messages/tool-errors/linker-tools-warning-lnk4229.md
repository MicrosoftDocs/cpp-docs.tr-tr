# <a name="linker-tools-warning-lnk4229"></a>Bağlayıcı Araçları Uyarısı LNK4229

Geçersiz yönerge /directive bulundu; yoksayıldı

Aracılığıyla bir yönerge geçirilen [yorum](../../preprocessor/comment-c-cpp.md) pragma geçerli değildi. Bağlayıcı yok sayar `/directive`.
