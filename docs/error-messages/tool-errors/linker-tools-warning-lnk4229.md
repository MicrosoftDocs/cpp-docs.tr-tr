# <a name="linker-tools-warning-lnk4229"></a>Bağlayıcı Araçları Uyarısı LNK4229

Pragma [açıklaması](../../preprocessor/comment-c-cpp.md) aracılığıyla geçirilen bir yönerge geçerli değil. Bağlayıcı `/directive`yoksayar.
