# <a name="linker-tools-error-lnk1287"></a>Bağlayıcı Araçları Hatası LNK1287

Geçersiz yönetilen giriş noktası işlevi

Giriş noktası, yönetilen bir görüntü için geçerli değil. Yönetilen giriş noktası işlevinin dönüş türü yalnızca `void` veya `int`olabilir. Yönetilen giriş noktası işlevinin parametresinin türü yalnızca `void` veya `String []`olabilir.
