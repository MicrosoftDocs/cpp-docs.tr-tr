# <a name="nmake-fatal-error-u1078"></a>NMAKE Önemli Hatası U1078
'ifadesi' konumundaki sabit taşması  
  
 Verilen ifade aralığı - 2,147,483,648 için 2.147.483.647 aşan bir sabit içeriyor. Sabit aşağıdaki durumlardan biri sunulmuştur:  
  
-   Önişlem yönergesi ile belirtilen deyim  
  
-   Çizgi ile belirtilen bir hata düzeyi (**-**) komutu değiştiricisi
