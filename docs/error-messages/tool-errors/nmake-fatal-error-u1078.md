# <a name="nmake-fatal-error-u1078"></a>NMAKE Önemli Hatası U1078

' expression ' üzerinde sabit taşma

Verilen ifade, 2.147.483.648 ile 2.147.483.647 arasında bir sabit içeriyordu. Sabit, aşağıdaki durumlardan birinde göründü:

- Ön işleme yönergesi ile belirtilen ifade

- Dash ( **-** ) komut değiştiricisi ile belirtilen bir hata düzeyi
