# <a name="nmake-fatal-error-u1078"></a>NMAKE Önemli Hatası U1078

'expression' üzerinde sabit taşma

Verilen ifade aralık - 2.147.483.648 için 2.147.483.647 aşan bir sabit içeriyor. Sabit aşağıdaki durumlardan birinde görüntülendi:

- Bir ön işleme yönergesi ile belirtilen deyim

- Tire ile belirtilen bir hata düzeyi (**-**) komut değiştiricisi
