# <a name="resource-compiler-error-rc2169"></a>Kaynak Derleyicisi Hatası RC2169

kaynak dosyası dosya adının 2.03 biçimde değil.

Belirtilen kaynak 2.03 sürümden önceki bir biçimi kullanılır. Kaynak dosyası dönüştürülmelidir veya 3.00 veya sonraki bir sürümü için biçimi kullanılarak yeniden oluşturuluyor.
