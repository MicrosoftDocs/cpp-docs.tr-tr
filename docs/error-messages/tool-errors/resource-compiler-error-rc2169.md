# <a name="resource-compiler-error-rc2169"></a>Kaynak Derleyicisi Hatası RC2169
Kaynak dosya filename 2.03 biçiminde değil  
  
 Belirtilen kaynak sürüm 2.03'den önceki bir biçim kullanılır. Kaynak dosyanın dönüştürülmesi gerekir veya 3.00 veya sonraki bir sürümü için aşağıdaki biçimi kullanarak yeniden oluşturulmaz.
