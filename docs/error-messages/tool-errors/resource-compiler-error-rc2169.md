# <a name="resource-compiler-error-rc2169"></a>Kaynak Derleyicisi Hatası RC2169

kaynak dosyası dosya adı 2,03 biçiminde değil

Belirtilen kaynak 2,03 sürümünden önceki bir biçimi kullandı. Kaynak dosyası, sürüm 3,00 veya üzeri biçimi kullanılarak dönüştürülmelidir veya yeniden oluşturulmalıdır.
