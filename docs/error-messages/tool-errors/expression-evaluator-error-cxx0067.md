# <a name="expression-evaluator-error-cxx0067"></a>İfade Değerlendirici Hatası CXX0067

işlev değerlendirmesi desteklenmiyor

İfade bir işlev çağrısı içeriyordu. Bazı Expression değerlendiricileri işlev çağrılarını desteklemez.

Bu hata CAN0067 ile aynıdır.
