# <a name="expression-evaluator-error-cxx0067"></a>İfade Değerlendirici Hatası CXX0067
Desteklenmeyen işlev değerlendirmesi  
  
 İfade bir işlev çağrısı içeriyor. Bazı ifade değerlendiricisi işlev çağrılarını desteklemez.  
  
 Bu hata için CAN0067 aynıdır.
