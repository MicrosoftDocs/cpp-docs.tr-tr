# <a name="expression-evaluator-error-cxx0023"></a>İfade Değerlendirici Hatası CXX0023

Hatalı taban

C ifade değerlendirici belirtilen taban 'i tanımıyor. Yalnızca ondalık, onaltılık ve sekizlik radyan geçerlidir.

Bu hata CAN0023 ile aynıdır.
