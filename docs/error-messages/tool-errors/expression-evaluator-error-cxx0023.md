# <a name="expression-evaluator-error-cxx0023"></a>İfade Değerlendirici Hatası CXX0023

Geçersiz sayı tabanı

C ifade değerlendiricisi, belirtilen sayı algılamaz. Yalnızca ondalık, onaltılık ve sekizlik radixes geçerlidir.

Bu hata için CAN0023 aynıdır.
