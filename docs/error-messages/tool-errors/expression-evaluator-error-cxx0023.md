# <a name="expression-evaluator-error-cxx0023"></a>İfade Değerlendirici Hatası CXX0023
bozuk radix  
  
 C ifade değerlendiricisi belirtilen sayı tabanını tanımıyor. Yalnızca ondalık, onaltılık ve sekizli radixes geçerlidir.  
  
 Bu hata için CAN0023 aynıdır.
