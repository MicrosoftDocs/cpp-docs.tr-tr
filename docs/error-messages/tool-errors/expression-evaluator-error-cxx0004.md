# <a name="expression-evaluator-error-cxx0004"></a>İfade Değerlendirici Hatası CXX0004
Sözdizimi hatası  
  
 İfadenin sözdizimi geçersiz.  
  
 İfade doğru sözdizimi ile yeniden yazın.  
  
 Bu hata için CAN0004 aynıdır.
