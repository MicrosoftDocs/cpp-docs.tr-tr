# <a name="linker-tools-warning-lnk4199"></a>Bağlayıcı Araçları Uyarısı LNK4199

/DELAYLOAD: DLLAdı yoksayıldı; DLL adı içinden içeri aktarma bulunamadı

Bağlayıcı, `dllname` dışarı aktardığı işlevlerden herhangi birine ihtiyaç duymadığı için `dllname` yoksayar.
