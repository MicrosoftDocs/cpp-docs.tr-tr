# <a name="resource-compiler-error-rc2116"></a>Kaynak Derleyicisi Hatası RC2116

KIMLIK için sayı bekleniyor

`id` **İletişim** deyimindeki bir denetim ifadesinin alanı için bir sayı bekleniyor. Denetim KIMLIĞI için bir sayı veya deyiminiz olduğundan emin olun `#define` .
