# <a name="resource-compiler-fatal-error-rc1121"></a>Kaynak Derleyicisi Önemli Hatası RC1121

Dosya okunurken g/ç hatası

Kaynak derleyicisi bir dosyayı okuyamadı. Dosyayı içeren sürücünün kullanılabilir olduğundan ve dosyanın geçerli olduğundan emin olun.
