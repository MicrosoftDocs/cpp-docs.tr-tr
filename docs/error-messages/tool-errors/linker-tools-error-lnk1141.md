# <a name="linker-tools-error-lnk1141"></a>Bağlayıcı Araçları Hatası LNK1141
dışarı aktarma dosyası derleme sırasında hata oluştu  
  
 [Bağlantı](../../build/reference/linker-command-line-syntax.md) dışarı aktarma (.exp) dosyası oluşturulamadı.  
  
 Bu hatanın nedeni sistem sorunlarını yetersiz bellek ve sözdizimi hataları gibi seçenekler veya modül tanımlama deyimleri ekleyin.  
  
 Bu hata ek bilgi veren başka bir hata tarafından gelmelidir.
