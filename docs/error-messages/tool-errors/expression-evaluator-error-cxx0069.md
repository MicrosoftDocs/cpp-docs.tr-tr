# <a name="expression-evaluator-error-cxx0069"></a>İfade Değerlendirici Hatası CXX0069

değişken, yığın çerçevesine ihtiyaç duyuyor

İfade değerlendirici bir yığın çerçevesinde gerçekleşmediği için değişkeni değerlendiremez. Bunun nedeni, bir satır içi işlevin parçası olarak belirtilen değişkenlerden kaynaklanabilir.
