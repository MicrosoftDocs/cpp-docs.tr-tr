# <a name="resource-compiler-error-rc2127"></a>Kaynak Derleyicisi Hatası RC2127

virgülle ayrılmış sürüm kelimeleri bekleniyor

Sürüm kaynağındaki sürüm numaraları, **sözcük** türünde olmalı ve virgülle ayrılmalıdır.
