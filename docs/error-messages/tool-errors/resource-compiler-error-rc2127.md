# <a name="resource-compiler-error-rc2127"></a>Kaynak Derleyicisi Hatası RC2127

Beklenen virgülle ayırarak sürüm sözcükler

Sürüm numaraları sürüm kaynağında türünden olmalıdır **WORD**ve virgülle ayrılmış.
