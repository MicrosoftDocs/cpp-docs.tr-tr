# <a name="linker-tools-error-lnk1241"></a>Bağlayıcı Araçları Hatası LNK1241

' kaynak dosyası ' kaynak dosyası zaten belirtildi

Bu hata, komut satırından **cvtres** 'yi el ile çalıştırırsanız ve daha sonra elde edilen. obj dosyasını diğer. res dosyalarına ek olarak bağlayıcıya geçirirseniz oluşur.

Birden çok. res dosyası belirtmek için bunları, **cvtres**tarafından oluşturulan. obj dosyaları içinden değil,. res dosyaları olarak bağlayıcıya geçirin.
