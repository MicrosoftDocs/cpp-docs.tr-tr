# <a name="project-build-error-prj0047"></a>Proje Derleme Hatası PRJ0047

Askıya alınan işlem sürdürülemedi.  Derleme başarısız oldu.

Geliştirme ortamında bir hata oluştu.

Bu hatayı çözmek için projeyi kapatın ve yeniden yükleyin. Gerekirse, çıkın ve Visual Studio 'Yu yeniden başlatın.
