# <a name="resource-compiler-fatal-error-rw1004"></a>Kaynak Derleyicisi Önemli Hatası RW1004

Beklenmeyen dosya sonu

Bu hata, bir metin dosyasının son satırında eksik satır akışı ve satır başı karakterlerinden kaynaklanabilir.
