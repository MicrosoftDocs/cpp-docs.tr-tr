# <a name="linker-tools-error-lnk2026"></a>Bağlayıcı Araçları Hatası LNK2026

SAFESEH görüntüsü için modül güvenli değil

[/SafeSEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md) belirtildi, ancak bir modül güvenli özel durum işleme özelliğiyle uyumlu değildi. Bu modülü **/SafeSEH**ile birlikte kullanmak istiyorsanız, modülü yeniden derlemeniz gerekir.
