# <a name="resource-compiler-warning-rw4003"></a>Kaynak Derleyicisi Uyarısı RW4003

VIRTKEY olmadan kullanılan SHIFT veya CONTROL

Bir Hızlandırıcı tablo kaynağında, KAYDıRMA veya DENETIM VIRTKEY gerektirir. SHIFT ve CONTROL, bir VIRTKEY tür hızlandırıcısında bayrak bitleri olarak gösterildiği için, bir VIRTKEY 'ten bağımsız olarak bulunamaz.
