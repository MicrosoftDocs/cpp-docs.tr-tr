# <a name="c-runtime-errors-r6002-through-r6035"></a>C çalışma zamanı hataları R6002-R6035
Bu bölümde, uygulamanızın yüklenmedi veya çalışan olduğunda C çalışma zamanı kitaplığı (CRT) tarafından oluşturulan hatalar başvurusudur. Bu, uygulamanızın kodda bir hata ya da çalışma zamanı kitaplığı işleyemez, düşük bellek gibi bir koşul belirten hatalardır. Son kullanıcılar, uygulamanızın bunları neden olan koşulları önlemek için uygulamanızın kod değil Bu hatalar görebilirsiniz.
