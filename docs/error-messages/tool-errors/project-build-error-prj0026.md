# <a name="project-build-error-prj0026"></a>Proje Derleme Hatası PRJ0026

> '*Dosya*' yanıt dosyası, kullanıcının ANSI kod sayfasına çevrilemeyen Unicode içerikleri içeriyor.
>
> *Dosyanın UNICODE içeriği*

Proje sistemi, kullanıcının geçerli ANSI kod sayfasına doğru çevrilemeyen bir yanıt dosyasında Unicode içeriği buldu.

Bu hatanın çözümü, ANSI kullanmak için yanıt dosyasının içeriğini güncelleştirmek veya kod sayfasını bilgisayarınıza yüklemek ve sistem varsayılanı olarak ayarlamak içindir.
