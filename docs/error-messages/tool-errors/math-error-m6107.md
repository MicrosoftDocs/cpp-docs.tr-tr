# <a name="math-error-m6107"></a>Matematik Hatası M6107

unemulated

Öykünücü tarafından desteklenmiyor veya geçersiz bir eşişlemcisi yönergesi yürütmek için girişimde bulunuldu.

Program çıkış kodu ile 135 sonlandırır.
