# <a name="expression-evaluator-error-cxx0034"></a>İfade Değerlendirici Hatası CXX0034

**türler işleçle uyumsuz**

Belirtilen işlenen türleri işlem için geçerli değil.

Örneğin, bir işaretçi herhangi bir değere göre çarpılamıyor.

İşlenenleri işleç ile uyumlu bir türe atama yazmanız gerekebilir.
