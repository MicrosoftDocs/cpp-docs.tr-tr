# <a name="expression-evaluator-error-cxx0063"></a>İfade Değerlendirici Hatası CXX0063
Aşırı yüklenmiş işleç -> desteklenen  
  
 Aşırı yüklenmiş sınıf üye erişimi işleci ifade kullanılan (**->**).  
  
 Bu hata için CAN0063 aynıdır.
