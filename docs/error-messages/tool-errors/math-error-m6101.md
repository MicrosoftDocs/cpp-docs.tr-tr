# <a name="math-error-m6101"></a>Matematik Hatası M6101

geçersiz

Geçersiz işlem.

Bir işlenen NaN (sayı değil) ya da sonsuz olduğunda bu hata oluşabilir.

Program çıkış kodu ile 129 sonlandırır.
