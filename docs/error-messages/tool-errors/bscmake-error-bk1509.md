# <a name="bscmake-error-bk1509"></a>BSCMAKE Hatası BK1509

yığın alanı yetersiz

BSCMAKE, sanal bellek dahil belleği tüketti.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Aşağıdaki olası çözümleri kullanarak düzeltilmesi için

1. Disk alanı boşaltın.

1. Takas dosyasının boyutunu artırın.

1. Windows takas dosyasının boyutunu artırın.

1. Bazı giriş dosyalarını kaldırmak için/ei veya/es kullanarak BSCMAKE 'in gerektirdiği belleği azaltın ve makro gövdelerini ortadan kaldırın.
