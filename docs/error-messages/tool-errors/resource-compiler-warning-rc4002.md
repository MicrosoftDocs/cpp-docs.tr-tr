# <a name="resource-compiler-warning-rc4002"></a>Kaynak Derleyicisi Uyarısı RC4002

' tanımlayıcı ' makrosunun gerçek parametre sayısı çok fazla

Verilen tanımlayıcıyla belirtilen gerçek parametrelerin sayısı, tanımlayıcının makro tanımında verilen biçimsel parametrelerin sayısından büyüktü.

Ek gerçek parametreler toplandı, ancak makronun genişletilmesi sırasında yoksayıldı.
