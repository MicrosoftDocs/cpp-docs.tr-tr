# <a name="linker-tools-error-lnk1120"></a>Bağlayıcı Araçları Hatası LNK1120
*sayı* çözümlenmemiş externals  
  
Hatası LNK1120 raporları sayısı (*numarası*) bu bağlantısı işlemi için çözümlenmemiş dış simgesi hata sayısı. Çoğu çözümlenmemiş dış simgesi hatalarını tek tek raporlanır [Bağlayıcı araçları hatası LNK2001](../../error-messages/tool-errors/linker-tools-error-lnk2001.md) ve [Bağlayıcı araçları hatası LNK2019](../../error-messages/tool-errors/linker-tools-error-lnk2019.md), her çözümlenmemiş dış için bir kez bu hata iletisini koyun simge bir hata oluştu.  
  
Bu hatayı düzeltmek için tüm diğer çözümlenmemiş dış hataları veya derleme çıktısında koyun diğer bağlayıcı hatalarını düzeltin. Çözümlenmemiş dış hata kaldığında bu hatayı bildirilmedi.
