# <a name="linker-tools-error-lnk1264"></a>Bağlayıcı Araçları Hatası LNK1264

/LTCG: PGıNSTRUMENT belirtildi, ancak kod üretme gerekli değil; izleme başarısız oldu

**/LTCG: PGıNSTRUMENT** belirtildi, ancak [/GL](../../build/reference/gl-whole-program-optimization.md)ile derlenen bir. obj dosyası bulunamadı. İzleme gerçekleşmiyor ve bağlantı başarısız oldu. İzleme gerçekleşebilmesi için **/GL** ile derlenen komut satırında en az bir. obj dosyası olmalıdır.

Profil temelli iyileştirme (PGO) yalnızca 64 bit derleyicilerde kullanılabilir.
