# <a name="resource-compiler-fatal-error-rc1205"></a>Kaynak Derleyicisi Önemli Hatası RC1205

Geçersiz kod sayfası

Kod belirt sayfası (/c) seçeneğinin ardından geçersiz bir kod sayfası vardı. Daha fazla bilgi için *çalışma zamanı kitaplık başvurusundaki* [kod sayfalarına](../../c-runtime-library/code-pages.md) bakın.
