# <a name="resource-compiler-fatal-error-rc1205"></a>Kaynak Derleyicisi Önemli Hatası RC1205

 Kod sayfası belirtin (/ c) seçeneği geçersiz kod sayfası tarafından izlenen. Bkz: [kod sayfaları](../../c-runtime-library/code-pages.md) içinde *çalışma zamanı kitaplığı başvurusu* daha fazla bilgi için.
