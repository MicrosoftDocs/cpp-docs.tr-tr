# <a name="resource-compiler-error-rc2167"></a>Kaynak Derleyicisi Hatası RC2167

Tanınmayan VERSIONıNFO alanı; BAŞLANGıÇ veya virgül bekleniyor

**VersionInfo** yapısı bildiriminin **sabit** bölümünde tanınmayan bir alan bulundu. **VersionInfo** alanı bir VIRGÜLLE ayrılmış DWORD olmalıdır.
