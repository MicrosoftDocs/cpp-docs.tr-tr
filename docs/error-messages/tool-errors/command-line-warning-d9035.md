# <a name="command-line-warning-d9035"></a>Komut Satırı Uyarısı D9035

'seçeneği' seçeneği kullanım dışıdır ve gelecek sürümde kaldırılacak.

Visual C++ derleyicisi, gelecekteki bir sürümde kaldırılacak bir derleme seçeneği belirttiğiniz. Bu uyarı, uyarı tarafından izlenir [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md) için önerilen bir değişiklik varsa *seçeneği*.

Belirtilen seçeneği hala çalışıyor, ancak artık yapı yapılandırmanızı güncelleştirmeniz gerekir. Sonuç olarak, projenizin derleyici yükselttiğinizde oluşturmaya devam daha yüksektir.
