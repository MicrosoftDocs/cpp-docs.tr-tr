# <a name="command-line-warning-d9035"></a>Komut Satırı Uyarısı D9035

> seçenek '*seçeneği*' kullanım dışıdır ve gelecek sürümde kaldırılacak

## <a name="remarks"></a>Açıklamalar

Derleyici gelecekteki bir sürümde kaldırılacak bir derleyici seçeneği belirtildi. Önerilen bir ardılı ise *seçeneği*, bu uyarı, uyarı tarafından izlenir [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md).

Belirtilen seçeneği hala çalışıyor, ancak artık yapı yapılandırmanızı güncelleştirmeniz gerekir. Sonuç olarak, projenizin derleyici yükselttiğinizde oluşturmaya devam daha yüksektir.
