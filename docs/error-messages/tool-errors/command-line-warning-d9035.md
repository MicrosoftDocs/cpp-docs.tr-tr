# <a name="command-line-warning-d9035"></a>Komut Satırı Uyarısı D9035

> '*Option*' seçeneği kullanım dışı bırakıldı ve gelecek sürümde kaldırılacak

## <a name="remarks"></a>Açıklamalar

Derleyicinin gelecek bir sürümünde kaldırılacak bir derleyici seçeneği belirttiniz. *Seçenek* için önerilen bir değiştirme varsa, bu uyarının ardından Uyarı [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md)gelir.

Belirtilen seçenek hala işe yarar, ancak yapı yapılandırmanızı şimdi güncelleştirmelisiniz. Sonuç olarak, derleyicisini yükselttiğinizde, projenizin oluşturmaya devam etme olasılığı yüksektir.
