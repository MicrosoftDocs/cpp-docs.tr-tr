# <a name="linker-tools-error-lnk1240"></a>Bağlayıcı Araçları Hatası LNK1240
IDL içerik derlenemedi  
  
 Katıştırılmış IDL derlemek için MIDL bağlayıcı oluşturdu, ancak bir sorun oluştu. MIDL tarafından belirtilen hataları denetleyin.
