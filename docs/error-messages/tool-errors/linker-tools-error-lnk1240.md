# <a name="linker-tools-error-lnk1240"></a>Bağlayıcı Araçları Hatası LNK1240

IDL içeriği derlenemedi

Bağlayıcı, gömülü IDL derlemek için MıDL 'yi oluşturdu, ancak bir sorun oluştu. MıDL tarafından belirtilen hataları kontrol edin.
