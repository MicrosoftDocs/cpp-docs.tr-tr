# <a name="linker-tools-error-lnk1240"></a>Bağlayıcı Araçları Hatası LNK1240

IDL içeriği derlenemedi

Gömülü IDL'yi derlemek için MIDL bağlayıcı kökenli ancak bir sorun oluştu. MIDL tarafından belirtilen hataları denetleyin.
