# <a name="expression-evaluator-error-cxx0045"></a>İfade Değerlendirici Hatası CXX0045

bir işlev değil

Programda bir işlevin adı olmayan bir sembol için bağımsız değişken listesi sağlandı.

## <a name="example"></a>Örnek

```
queue( alpha, beta )
```

`queue` bir işlev değildir.

Bu hata CAN0045 ile aynıdır.
