# <a name="linker-tools-warning-lnk4096"></a>Bağlayıcı Araçları Uyarısı LNK4096

/BASE değeri "Number", Windows 95 ve Windows 98 için geçersizdir; görüntü çalışmayabilir

Belirttiğiniz temel adres geçersiz. Windows 95 ve Windows 98 yürütülebilir dosyaları, 0x400000 ' dan büyük bir taban adrese sahip olmalıdır. Taban adresler hakkında daha fazla bilgi için bkz. [/Base](../../build/reference/base-base-address.md) bağlayıcı seçeneği.
