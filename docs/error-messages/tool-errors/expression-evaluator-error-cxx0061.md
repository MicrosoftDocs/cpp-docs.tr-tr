# <a name="expression-evaluator-error-cxx0061"></a>İfade Değerlendirici Hatası CXX0061
işlev iç içe geçirme derinliği aşıldı  
  
 İfade sınırdan büyük derinlik iç içe geçmiş işlev içeriyor. İç içe geçirme derinliği azaltmak için ifadeyi değiştirin.  
  
 Bu hata için CAN0061 aynıdır.
