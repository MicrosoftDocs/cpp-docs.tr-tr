# <a name="project-build-error-prj0033"></a>Proje Derleme Hatası PRJ0033
'Ek bağımlılıklar' özelliği için özel derleme adımı için 'çıkışı değerlendiren dosya 'bulunan dosya' makro' 'macro_expansion'.  
  
 Özel derleme adımı bir dosyada ek bağımlılığı makro değerlendirmesi sorun büyük olasılıkla bir hata içeriyor. Bu hata, ayrıca yol hatalı, karakter veya bir dosya yolu geçersiz karakter bileşimlerini içeren biçimlendirildiğinden emin anlamına gelebilir.  
  
 Bu hatayı gidermek için makrosu düzeltin veya yolunu düzeltin. Proje dizininden mutlak bir yol değerlendirilen yoludur.
