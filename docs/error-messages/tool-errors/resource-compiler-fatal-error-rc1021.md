# <a name="resource-compiler-fatal-error-rc1021"></a>Kaynak Derleyicisi Önemli Hatası RC1021

geçersiz önişlemci komutu ' String '

Sayı işaretini () izleyen karakterler **#** geçerli bir Önişlemci yönergesi oluşturmuyor.

Numara işareti, bir Tanımlayıcıdaki ilk karakter olarak kullanılamaz.
