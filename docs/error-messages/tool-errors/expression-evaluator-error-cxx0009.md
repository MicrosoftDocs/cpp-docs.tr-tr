# <a name="expression-evaluator-error-cxx0009"></a>İfade Değerlendirici Hatası CXX0009

' After karakter sabiti eksik

Karakter sabiti sonunda beklenen tek tırnak işareti eksik. Karakteri tek tırnak işaretleri içine alarak ifadeyi yeniden yazın.

Bu hata CAN0009 ile aynıdır.
