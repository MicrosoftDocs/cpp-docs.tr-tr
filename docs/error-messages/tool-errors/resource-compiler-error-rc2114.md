# <a name="resource-compiler-error-rc2114"></a>Kaynak Derleyicisi Hatası RC2114

beklenen denetim sınıfı adı

`class` **İletişim** deyimindeki bir denetim ifadesinin alanı şu türlerden bırı olmalıdır: düğme, ComboBox, düzenleme, LISTBOX, KAYDıRMA çubuğu, statik veya Kullanıcı tanımlı. Sınıfın doğru yazıldığından emin olun.
