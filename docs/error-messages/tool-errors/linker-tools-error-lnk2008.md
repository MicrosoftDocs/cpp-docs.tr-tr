# <a name="linker-tools-error-lnk2008"></a>Bağlayıcı Araçları Hatası LNK2008

Düzeltme hedefi hizalı değil ' symbol_name '

BAĞLANTı, nesne dosyanızda düzgün hizalanmış bir düzeltme hedefi buldu.

Bu hataya özel secton hizalaması (örneğin, #pragma [Pack](../../preprocessor/pack.md)), [Hizalama](../../cpp/align-cpp.md) değiştiricisi veya secton hizalamasını değiştiren derleme dil kodu kullanılarak neden olabilir.

Kodunuz yukarıdakilerden herhangi birini kullanmıyorsa, bu durum derleyicinin nedeni olabilir.
