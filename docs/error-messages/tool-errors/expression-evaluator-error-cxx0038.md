# <a name="expression-evaluator-error-cxx0038"></a>İfade Değerlendirici Hatası CXX0038

işlev bağımsız değişkeni sayısı ve/veya tür uyumsuzluğu

Belirtilen olarak işlev çağrısı işlevin prototipi ile eşleşmiyor.

Çağrıyı doğru sayıda bağımsız değişkenle yeniden yazın. Gerektiğinde, her bağımsız değişkeni prototipi eşleşecek şekilde atayın.

Bu hata CAN0038 ile aynıdır.
