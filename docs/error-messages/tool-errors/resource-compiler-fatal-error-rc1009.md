# <a name="resource-compiler-fatal-error-rc1009"></a>Kaynak Derleyicisi Önemli Hatası RC1009

Derleyici sınırı: makrolar çok derin iç içe ' Macro '

Dosya, makro iç içe geçme için kaynak derleyicisi sınırını aşıyor. Belirtilen kaynak dosyayı, makroları iç içe geçme derinliğini azaltmak için gözden geçirin.
