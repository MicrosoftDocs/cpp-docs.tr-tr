# <a name="resource-compiler-warning-rc4093"></a>Kaynak Derleyicisi Uyarısı RC4093
karakter sabiti etkin olmayan kod içinde atlanmayan yeni satır  
  
 Sabit bir ifade bir `#if`, `#elif`, **#ifdef**, veya **#ifndef** kodu yapmak sıfır olarak değerlendirilen önişlemci yönergesi etkin olmayan izler. İçindeki etkin olmayan bu kod, yeni satır karakteri tek veya çift tırnak işaretleri kümesinde görüldü.  
  
 Sonraki çift tırnak işareti kadar tüm metin içinde karakter sabiti olarak kabul.
