# <a name="linker-tools-error-lnk1158"></a>Bağlayıcı Araçları Hatası LNK1158

' filename ' çalıştırılamıyor

[Bağlama](../../build/reference/linking.md) tarafından çağrılan verilen yürütülebilir dosya, bağlantı içeren DIZINDE veya PATH ortam değişkeninde belirtilen bir dizinde değil.

Örneğin, 32 bitlik bir işletim sistemine sahip bir makinede [/LTCG](../../build/reference/ltcg-link-time-code-generation.md) BAĞLAYıCı SEÇENEĞINE pgoptimize parametresini kullanmayı denerseniz bu hatayı alırsınız.
