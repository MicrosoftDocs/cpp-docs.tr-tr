# <a name="bscmake-error-bk1504"></a>BSCMAKE Hatası BK1504
'filename' dosyasında getirin olamaz [: neden]  
  
 BSCMAKE dosyasındaki bir konuma taşınamadı.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aşağıdaki olası nedenleri kontrol ederek düzeltmek için  
  
1.  Tam disk.  
  
2.  Bir donanım hatası.  
  
3.  Dosya kesilmesi nedeniyle .sbr dosyası oluşturulurken derleyici kesintisi veya disk alanı yetersiz.
