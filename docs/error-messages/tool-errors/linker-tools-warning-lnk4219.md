# <a name="linker-tools-warning-lnk4219"></a>Bağlayıcı Araçları Uyarısı LNK4219

Düzeltme adı düzeltme taşması. Target ' Target symbol Name ' Aralık dışında, dönüştürücü ekleniyor

Bağlayıcı, hedef sembol yönergenin konumundan çok uzakta olduğundan, adresin veya kaydırın verilen yönergeye sığamayacak bir duruma dönüştürücü ekledi.

Ek yöneltme düzeyini önlemek için görüntüyü yeniden sıralamak isteyebilirsiniz (örneğin, [/Order](../../build/reference/order-put-functions-in-order.md) seçeneğini kullanarak).
