# <a name="linker-tools-error-lnk1201"></a>Bağlayıcı Araçları Hatası LNK1201

' filename ' program veritabanına yazılırken hata; yetersiz disk alanı, geçersiz yol veya yetersiz ayrıcalık denetimi yapın

BAĞLANTı, çıkış dosyası için program veritabanına (PDB) yazamadı.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aşağıdaki olası nedenleri denetleyerek onarmak için

1. Dosya bozuk. PDB dosyasını silip yeniden bağlayın.

1. Dosyayı yazmak için yeterli disk alanı yok.

1. Sürücü, muhtemelen bir ağ sorunu nedeniyle kullanılamıyor.

1. Hata ayıklayıcı, bağlamayı denediğiniz program üzerinde etkin.

1. Yığın alanı yetersiz.  Daha fazla bilgi için bkz. [C1060](../../error-messages/compiler-errors-1/fatal-error-c1060.md) .
