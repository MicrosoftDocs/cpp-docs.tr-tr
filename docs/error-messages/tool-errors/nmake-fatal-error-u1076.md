# <a name="nmake-fatal-error-u1076"></a>NMAKE Önemli Hatası U1076

adı çok uzun

Bir dize aşağıdaki sınırlardan birinin aştı:

- Makro adı için 1024 karakter.

- Hedef yol için 256 karakter.

- bir komut için 2048 karakter.
