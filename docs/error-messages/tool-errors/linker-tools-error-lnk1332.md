# <a name="linker-tools-error-lnk1332"></a>Bağlayıcı Araçları Hatası LNK1332
Algılanan\<sayısı > Windows çalışma zamanı türleri içinde bir modülü içeri ve başka bir modülde tanımlanmış  
  
 Geçerli hedef üretilen, bağlayıcı algılanan <`count`> Windows çalışma zamanı türleri, her biri bir modülde içeri aktarılıp, aynı zamanda başka bir modülde tanımlanmış.  
  
### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Her hata iletisi önerisine göre yapı LNK2039 hataları düzeltin.
