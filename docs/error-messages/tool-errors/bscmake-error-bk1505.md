# <a name="bscmake-error-bk1505"></a>BSCMAKE Hatası BK1505

'filename' dosyasından okunamıyor [: neden]

BSCMAKE dosyasından okunamıyor.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aşağıdaki olası nedenleri kontrol ederek düzeltmek için

1. Veri dosyasının bozuk olmasıdır.

1. Disk alanı veya .sbr dosyası oluşturulurken kesintiye bitmeye derleyici kesilmesi dosya.
