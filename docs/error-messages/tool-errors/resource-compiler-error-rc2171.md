# <a name="resource-compiler-error-rc2171"></a>Kaynak Derleyicisi Hatası RC2171

Bilinmeyen DIB üstbilgi biçimi

Bit eşlem üst bilgisi değil bir **BITMAPCOREHEADER** veya **BITMAPINFOHEADER** yapısı.
