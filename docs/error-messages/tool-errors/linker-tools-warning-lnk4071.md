# <a name="linker-tools-warning-lnk4071"></a>Bağlayıcı Araçları Uyarısı LNK4071

sonraki bağlantılarda artımlı olarak bağlanamaz

BAĞLANTı, bir veya daha fazla sembol için birden çok tanım buldu, ancak hata nedeniyle bir çıkış dosyası oluşturmak için [/Force](../../build/reference/force-force-file-output.md) veya **/Force: Multiple** kullanıldı. BAĞLANTı, artımlı durum (. Ilk) dosyasını sildi.
