# <a name="project-build-error-prj0022"></a>Proje Derleme Hatası PRJ0022
Bilinmeyen aracı özelliği 'property' geçersiz bir dosya adı 'dosya' içerir.  
  
 Dosya adı ***dosya***özelliğinde belirtilen ***özelliği***, geçersizdi.  
  
 Bilinmeyen veya geçersiz makrosu kullanmış olabilir.  
  
 Bkz: [Visual C++ proje özelliklerini ayarlama](../../ide/working-with-project-properties.md) erişim projenizin özellikleri hakkında bilgi için.
