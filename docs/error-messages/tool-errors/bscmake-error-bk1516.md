# <a name="bscmake-error-bk1516"></a>BSCMAKE Hatası BK1516
bscfile bozuk; artımlı olarak güncelleştirilemiyor  
  
 .Bsc dosyası, büyük olasılıkla derleme sırasında sistem hatası nedeniyle bozulmuş. .Bsc dosyasını silin, tüm .sbr dosyaları yeniden oluşturduktan sonra .bsc dosyasını yeniden oluşturma.
