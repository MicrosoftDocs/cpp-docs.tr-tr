# <a name="linker-tools-error-lnk1248"></a>Bağlayıcı Araçları Hatası LNK1248

> görüntü boyutu ('*boyutu*') (80000000) izin verilen boyut sınırını aşıyor

Bağlayıcı çıkış dosyasının boyutunu 32-bit program görüntüsü için olası en büyük boyutunu aşacak belirledi. Birden çok DLL'lere programınızı yapmak isteyebilirsiniz.
