# <a name="linker-tools-error-lnk1248"></a>Bağlayıcı Araçları Hatası LNK1248
görüntü boyutu ('boyutu') (80000000) izin verilen boyut sınırı aşıyor  
  
 Bağlayıcı çıkış dosyasının boyutunu bir programın görüntü için olası en büyük boyutunu aşacak belirledi. Birden çok DLL'lere programınızı yapmak isteyebilirsiniz. Program yürütülebilir (PE) hakkında daha fazla bilgi için bkz: "Eşliği içinde PE: A Tur, Win32 taşınabilir yürütülebilir dosya biçimi" makale [MSDN Kitaplığı](http://go.microsoft.com/fwlink/?linkid=556).
