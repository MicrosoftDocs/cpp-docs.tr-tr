# <a name="resource-compiler-fatal-error-rc1022"></a>Kaynak Derleyicisi Önemli Hatası RC1022

' #endif ' bekleniyor

Bir `#if` , **#ifdef** veya **#ifndef** yönergesi bir `#endif` yönergeyle sonlandırılmadı.

`#if`Bu deyimden önce bir, **#ifdef** veya **#ifndef** deyimin etkin olduğundan emin olun.
