# <a name="linker-tools-warning-lnk4039"></a>Bağlayıcı Araçları Uyarısı LNK4039

/SECTION seçeneği ile belirtilen ' name ' bölümü yok

[Dumpbin](../../build/reference/dumpbin-reference.md) veya [editbin](../../build/reference/editbin-reference.md) , giriş dosyasında `name` adlı bir bölüm bulamadı.

Bir dosyadaki bölümleri görmek için, [/Headers](../../build/reference/headers.md) seçeneğiyle dumpbin komutunu çalıştırın.
