# <a name="expression-evaluator-error-cxx0028"></a>İfade Değerlendirici Hatası CXX0028

struct/Union öğesi değil

' Struct. Member ' veya ' pStruct->Member ' formunun bir ifadesi belirtildi, ancak \<member> yapının bir öğesi değil.

İfade doğru şekilde parantez içinde bulunmayabilir.

Bu hata CAN0028 ile aynıdır.
