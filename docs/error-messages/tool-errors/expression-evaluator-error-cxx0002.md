# <a name="expression-evaluator-error-cxx0002"></a>İfade Değerlendirici Hatası CXX0002

Kullanıcı belleğine erişirken hata oluştu

İfade, hata ayıklanan programa ayrılmamış belleğe başvurmasına çalışır.

Bu hata CAN0002 ile aynıdır.
