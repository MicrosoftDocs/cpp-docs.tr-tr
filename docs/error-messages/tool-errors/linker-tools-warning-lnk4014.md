# <a name="linker-tools-warning-lnk4014"></a>Bağlayıcı Araçları Uyarısı LNK4014

"ObjectName" üye nesnesi bulunamıyor

LıB `objectname` kitaplıkta bulunamadı.

**/Remove** ve **/Extract** seçenekleri, silinecek veya bir dosyaya kopyalanacak üye nesnenin tam adını gerektirir. Tam ad, özgün nesne dosyasının yolunu içerir. Bir kitaplıktaki üye nesnelerinin tam adlarını görmek için, DUMPBIN [/ARCHIVEMEMBERS](../../build/reference/archivemembers.md) veya LIB [/list](../../build/reference/managing-a-library.md)kullanın.
