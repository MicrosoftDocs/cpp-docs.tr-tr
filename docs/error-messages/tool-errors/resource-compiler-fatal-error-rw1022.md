# <a name="resource-compiler-fatal-error-rw1022"></a>Kaynak Derleyicisi Önemli Hatası RW1022

**Dosya yazılırken g/ç hatası**

Kaynak derleyicisi bir dosyaya yazamadı.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aşağıdaki olası nedenleri denetleyerek onarmak için

1. Disk alanı yetersiz. Boş alan, oluşturmakta olduğunuz yürütülebilir dosyanın boyutunun en az iki katına eşit olmalıdır.

1. Birim salt okunurdur.

1. Bozuk sektör.

1. Paylaşım ihlali.
