# <a name="resource-compiler-fatal-error-rw1016"></a>Kaynak Derleyicisi Önemli Hatası RW1016

Önişlemci hatalarından sonra sonlandıran RC

Diğer hatalar nedeniyle kaynak derleyicisi durduruldu. Diğer hataları giderip yeniden derleyin.
