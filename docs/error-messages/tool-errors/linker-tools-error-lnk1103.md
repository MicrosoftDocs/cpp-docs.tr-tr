# <a name="linker-tools-error-lnk1103"></a>Bağlayıcı Araçları Hatası LNK1103

hata ayıklama bilgileri bozuk; modülü yeniden derle

Bu hata, geçerli bir nesne dosyası oluşturulmadan önce derlemenin sonlandırıldığı için oluşur.

Verilen nesne dosyasını yeniden derleyin. Yeniden derleme sorunu düzeltmezse hatanın durumunu not edin, sorunu yalıtmak ve tekrarlanabilir bir test çalışması oluşturmak için deneyin. Daha sonra teknik destek yardım dosyasına veya kılavuzlarınızın birindeki teknik destek bölümüne başvurun.
