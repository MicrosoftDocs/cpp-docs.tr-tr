# <a name="project-build-error-prj0005"></a>Proje Derleme Hatası PRJ0005

' Dizin ' dizininde geçici bir dosya oluşturulamıyor.

Geçici dosya oluşturma çağrısı başarısız oldu. Hatanın nedenleri şunlardır:

- Geçici dosya adlarından kalmadı.

- Geçici dizin salt okunurdur.

- Geçici dizin veya TMP ortam değişkeni yok.

- Bilgisayarınızda boş disk alanı kalmadı.
