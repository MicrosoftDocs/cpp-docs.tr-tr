# <a name="linker-tools-error-lnk1164"></a>Bağlayıcı Araçları Hatası LNK1164

Bölüm bölümü hizalaması (sayı)/ALIGN değerinden büyük

Nesne dosyasındaki verilen bölümün hizalama boyutu, [/align](../../build/reference/align-section-alignment.md) seçeneğiyle belirtilen değeri aşıyor. **/ALIGN** değeri 2 ' nin üssü olmalıdır ve nesne dosyasında verilen bölüm hizalamasını eşit veya daha büyük olmalıdır.

Daha küçük bir bölüm hizalaması ile yeniden derleyin ya da **/align** değerini arttırın.
