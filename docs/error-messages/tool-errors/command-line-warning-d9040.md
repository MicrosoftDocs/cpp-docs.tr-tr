# <a name="command-line-warning-d9040"></a>Komut Satırı Uyarısı D9040

> '/Analyze ' seçeneği yoksayılıyor; Kod Analizi uyarıları derleyicinin bu sürümünde kullanılamıyor

**`/analyze`** Komut satırı seçeneği tüm Visual Studio sürümlerinde kullanılamaz. Bu uyarıyı çözmek için, Visual Studio 'nun desteklenen bir sürümüne geçin veya komut satırı seçeneğini kaldırın.
