geçersiz kaçış sırası

Bir kaçış dizisi beklenen yerde göründü.

Bir kaçış sırası — izleyen bir ters eğik çizgi ( **\\** ), yalnızca bir karakter veya dize sabitinde gerçekleşebilir.
