# <a name="linker-tools-error-lnk1200"></a>Bağlayıcı Araçları Hatası LNK1200

' filename ' program veritabanı okunurken hata oluştu

Program veritabanı (PDB) okunamadı.

Bu hata dosya bozulmasından kaynaklanıyor olabilir.

`filename`Bir nesne dosyası IÇIN pdb ise, [/Zi](../../build/reference/z7-zi-zi-debug-information-format.md)kullanarak nesne dosyasını yeniden derleyin.

`filename`Ana çıkış dosyası IÇIN pdb ise ve bu hata bir artımlı bağlantı sırasında oluştuysa, pdb 'yi silin ve yeniden bağlayın.
