# <a name="resource-compiler-fatal-error-rc1105"></a>Kaynak Derleyicisi Önemli Hatası RC1105
Geçersiz anahtar, seçeneği: çok fazla /d anahtarları  
  
 Çok fazla simgeleri tanımlamak simgelerini kullanma tanımlanmadı (/ d) seçeneği. Bazı simgeleri, kaynak ve yeniden derleyebilirsiniz tanımlayın.
