# <a name="resource-compiler-fatal-error-rc1105"></a>Kaynak Derleyicisi Önemli Hatası RC1105

Geçersiz anahtar, seçenek: çok fazla/d anahtar

Sembolleri tanımla (/d) seçeneği kullanılarak çok fazla sayıda sembol tanımlandı. Kaynağınızdaki bazı sembolleri tanımlayın ve yeniden derleyin.
