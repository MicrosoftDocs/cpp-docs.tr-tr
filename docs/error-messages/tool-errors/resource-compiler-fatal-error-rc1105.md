# <a name="resource-compiler-fatal-error-rc1105"></a>Kaynak Derleyicisi Önemli Hatası RC1105

Geçersiz anahtar, seçeneği: çok fazla /d anahtarları

Çok fazla sembol tanımlamak simgeler kullanarak tanımlanmış (/ d) seçeneği. Bazı simgeleri yeniden derleyin ve kaynak tanımlayın.
