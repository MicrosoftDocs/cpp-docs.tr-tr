# <a name="expression-evaluator-error-cxx0059"></a>İfade Değerlendirici Hatası CXX0059
sol işleneni sınıfı işlevi adı değil.  
  
 Bir işlev çağrısı sol işleneni, bir sınıf adı ve bir işlev çağrısı çözümlenemiyor. Üye işlevi bir ifadede adını atlama bu hataya neden olabilir.  
  
 Bu hata için CAN0059 aynıdır.
