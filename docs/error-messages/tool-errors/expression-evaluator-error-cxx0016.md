# <a name="expression-evaluator-error-cxx0016"></a>İfade Değerlendirici Hatası CXX0016

sabit çok büyük

C ifadesi değerlendirici 4.294.967.295 (0FFFFFFFF onaltılı) değerinden büyük işaretsiz bir tamsayı sabiti kabul edemez veya büyüklüğü yaklaşık 1.8 E + 308 ' den büyük olan bir kayan noktalı sabit değer olamaz.

Bu hata CAN0016 ile aynıdır.
