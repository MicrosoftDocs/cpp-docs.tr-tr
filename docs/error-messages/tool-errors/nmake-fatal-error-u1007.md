# <a name="nmake-fatal-error-u1007"></a>NMAKE Önemli Hatası U1007

adda çift tırnak işaretine izin verilmez

Belirtilen hedef adı veya dosya adı çift tırnak işareti (**"**) içeriyordu.

Çift tırnak işaretleri bir dosya adını çevreliyor, ancak içinde yer alamaz.
