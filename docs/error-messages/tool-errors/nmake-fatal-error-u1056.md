# <a name="nmake-fatal-error-u1056"></a>NMAKE Önemli Hatası U1056

komut işlemcisi bulunamıyor

Komut işlemcisi **ComSpec** veya **Path** ortam değişkenlerinde belirtilen yolda değil.

NMAKE komutları yürütürken komut işlemcisi olarak COMMAND.COM veya CMD.EXE kullanır. **ComSpec**'te ayarlanan yol içinde ilk olarak komut işlemcisini arar. **ComSpec** yoksa, NMAKE **yolda** belirtilen dizinleri arar.
