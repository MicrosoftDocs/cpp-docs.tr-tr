# <a name="expression-evaluator-error-cxx0012"></a>İfade Değerlendirici Hatası CXX0012

Sol küme ayracı eksik

İfade, dengesiz bir küme ayracı içeriyor. İfadeyi dengeli küme ayraçları ile yeniden yazın.

Bu hata CAN0012 ile aynıdır.
