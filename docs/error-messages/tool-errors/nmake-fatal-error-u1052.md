# <a name="nmake-fatal-error-u1052"></a>NMAKE Önemli Hatası U1052

> '*filename*' dosyası bulunamadı

NMAKE, aşağıdakilerden biriyle belirtilen dosyayı bulamadı:

- **/F** seçeneği

- **!** Ön işleme yönergesini dahil et

- Bir yanıt dosyası için oturum açma ( **\@** ) tanımlayıcısı

Dosyanın var olduğunu ve dosya adının doğru yazıldığından emin olun.
