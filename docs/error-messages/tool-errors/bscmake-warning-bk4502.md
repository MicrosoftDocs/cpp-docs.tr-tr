# <a name="bscmake-warning-bk4502"></a>BSCMAKE Uyarısı BK4502

kesilemez. SBR dosyası ' filename ' dosya adında değil

Güncelleştirme sırasında. bsc dosyasının ilk parçası olmayan sıfır uzunluklu bir. sbr dosyası belirtildi.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aşağıdaki olası nedenleri denetleyerek onarmak için

1. Yanlış dosya adı belirtildi.

1. Dosya silindi. (Hata [BK1513](../../error-messages/tool-errors/bscmake-error-bk1513.md) sonuçları.)

1. Dosya bozuk, tam derleme yapmak için BSCMAKE gerekli.
