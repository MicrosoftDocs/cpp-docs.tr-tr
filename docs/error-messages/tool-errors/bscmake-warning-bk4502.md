# <a name="bscmake-warning-bk4502"></a>BSCMAKE Uyarısı BK4502
kesirli kısmı. SBR dosya 'filename' filename içinde değil  
  
 Güncelleştirme sırasında başlangıçta .bsc dosyasının bir parçası olmayan sıfır uzunluklu .sbr dosyası belirtilmedi.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aşağıdaki olası nedenleri kontrol ederek düzeltmek için  
  
1.  Hatalı dosya adı belirtildi.  
  
2.  Dosya silinmiş. (Hata [BK1513](../../error-messages/tool-errors/bscmake-error-bk1513.md) sonuçları.)  
  
3.  Tam bir yapı yapmak BSCMAKE gerektiren dosyası bozuk.
