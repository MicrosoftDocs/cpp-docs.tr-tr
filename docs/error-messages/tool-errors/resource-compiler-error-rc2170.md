# <a name="resource-compiler-error-rc2170"></a>Kaynak Derleyicisi Hatası RC2170

bit eşlem dosyası dosya adı 3,00 biçiminde değil

Windows sürüm 2. x biçimini kullanan bit eşlemler sürüm 3. x kaynak dosyalarında kullanılamaz.

Bu sorunu onarmak için bit eşlemi yeniden çizin veya 3. x biçimine dönüştürün.
