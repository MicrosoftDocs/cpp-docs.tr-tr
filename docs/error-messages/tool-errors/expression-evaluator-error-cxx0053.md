# <a name="expression-evaluator-error-cxx0053"></a>İfade Değerlendirici Hatası CXX0053

kesme noktaları bağlama sırasında işlev olmayan sembol eşleşmesi

Kesme noktası olarak kullanılan bir sembol bir işlev değildi. Bir veri üyesini kesme noktası olarak belirtmek bu hataya neden olabilir.

Bu hata CAN0053 ile aynıdır.
