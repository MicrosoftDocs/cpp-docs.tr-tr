# <a name="project-build-error-prj0014"></a>Proje Derleme Hatası PRJ0014

Oluşturulan işlemlerin denetlenmesi için kullanılan iş nesnesi başarısız oldu.  Derleme devam edemiyor.

Geliştirme ortamında bir hata oluştu.

Bu hatayı çözmek için projeyi kapatın ve yeniden yükleyin. Gerekirse, çıkın ve Visual Studio 'Yu yeniden başlatın.
