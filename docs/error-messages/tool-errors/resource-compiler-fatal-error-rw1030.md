# <a name="resource-compiler-fatal-error-rw1030"></a>Kaynak Derleyicisi Önemli Hatası RW1030

Çıkış hatası

Bu hata, kaynak derleyicisi hedef dosyayı oluşturmadığında oluşabilir. Yeterli disk alanı olduğundan ve birimde yazma izniniz olduğundan emin olun.
