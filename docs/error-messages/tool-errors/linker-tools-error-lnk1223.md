# <a name="linker-tools-error-lnk1223"></a>Bağlayıcı Araçları Hatası LNK1223

dosya geçersiz veya bozuk: dosya geçersiz. pdata katkıları içeriyor

PDATA kullanan RıSC platformları için, derleyici sıralanmamış girdilerle bir. pdata bölümü yayınlaması durumunda bu hata oluşur.

Bu sorunu onarmak için [/GL (tüm program iyileştirmesi)](../../error-messages/tool-errors/linker-tools-error-lnk1223.md) etkin olmadan derlemeyi deneyin. Boş işlev gövdeleri, bazı durumlarda da bu hataya neden olabilir.
