# <a name="linker-tools-warning-lnk4010"></a>Bağlayıcı Araçları Uyarısı LNK4010

Geçersiz alt sistem sürüm numarası numarası; Varsayılan alt sistem sürümü varsayıldı

Görüntünün alt sistemi ([/Subsystem](../../build/reference/subsystem-specify-subsystem.md)) için bir sürüm belirtebilirsiniz. Her alt sistemin en düşük sürüm gereksinimi vardır. Belirtilen sürüm en düşük değerden düşükse, bu uyarı oluşur ve bağlayıcı yalnızca varsayılan alt sistemi kullanır.
