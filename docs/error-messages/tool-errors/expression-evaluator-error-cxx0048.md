# <a name="expression-evaluator-error-cxx0048"></a>İfade Değerlendirici Hatası CXX0048

çağırma sırası desteklenmiyor

İfadede belirtilen bir işlev, C ifade değerlendiricisi tarafından desteklenmeyen bir arama sırası kullanıyor. Bu işlevi izleme penceresi ifadesinde çağrılamaz.

Bu hata CAN0048 ile aynıdır.
