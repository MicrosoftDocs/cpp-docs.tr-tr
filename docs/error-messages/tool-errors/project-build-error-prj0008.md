# <a name="project-build-error-prj0008"></a>Proje Derleme Hatası PRJ0008

' Dosya ' dosyası silinemedi.

**Dosyanın başka bir işlem tarafından açık olmadığından ve yazma korumalı olmadığından emin olun.**

Yeniden oluşturma veya temizleme sırasında, Visual C++, derleme için tüm bilinen ara ve çıkış dosyalarını, ayrıca [genel yapılandırma ayarları özellik sayfasında](../../build/reference/general-property-page-project.md) **temiz özellikte silinecek Uzantılardaki** joker karakter belirtimlerini karşılayan dosyaları siler.

Visual C++ bir dosyayı silemeymeyebilir bu hatayı görürsünüz. Hatayı gidermek için, derlemeyi yapan kullanıcı için dosyayı ve dizinini yazılabilir yapın.
