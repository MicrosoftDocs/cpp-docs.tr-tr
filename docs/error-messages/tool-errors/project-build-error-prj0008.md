# <a name="project-build-error-prj0008"></a>Proje Derleme Hatası PRJ0008
Dosya 'dosyası silinemedi.  
  
 **Dosya başka bir işlem tarafından açık değil ve yazma korumalı olmadığından emin olun.**  
  
 Bir yeniden oluşturma sırasında veya temiz, Visual C++ derleme için tüm bilinen Ara ve çıkış dosyalarının yanı sıra, joker karakter belirtimlere tüm dosyaları siler **temiz silmek için Uzantılar** özelliğinde [genel Yapılandırma ayarları özellik sayfası](../../ide/general-property-page-project.md).  
  
 Visual C++ bir dosyayı silmek mümkün değilse, bu hatayı görürsünüz. Hatayı gidermek için dosya ve alt dizin yazılabilir yapı yapan kullanıcı için olun.
