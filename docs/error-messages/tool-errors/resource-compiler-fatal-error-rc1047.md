# <a name="resource-compiler-fatal-error-rc1047"></a>Kaynak Derleyicisi Önemli Hatası RC1047

"çok sayıda seçenek seçenekleri, 'string'"

Belirtilen seçeneği birden çok kez belirtildi. Verilen dize hataya seçeneği bağımsız değişkenidir.
