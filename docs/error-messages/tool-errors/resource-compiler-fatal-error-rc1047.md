# <a name="resource-compiler-fatal-error-rc1047"></a>Kaynak Derleyicisi Önemli Hatası RC1047

"çok fazla seçenek seçeneği, ' dize '"

Verilen seçenek çok fazla belirtildi. Verilen dize, hataya neden olan seçeneğin bağımsız değişkenidir.
