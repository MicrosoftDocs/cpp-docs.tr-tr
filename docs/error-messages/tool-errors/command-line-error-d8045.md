# <a name="command-line-error-d8045"></a>Komut Satırı Hatası D8045

' File ' C dosyası/clr seçeneğiyle derlenemiyor

**/Clr** kullanan bir derlemeye yalnızca C++ kaynak kodu dosyaları geçirilebilir.  . C dosyasını. cpp dosyası olarak derlemek için **/TP** kullanın; daha fazla bilgi için bkz. [/TC,/TP,/TC,/TP (kaynak dosya türünü belirtin)](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md) .

Daha fazla bilgi için bkz. [/clr (ortak dil çalışma zamanı derlemesi)](../../build/reference/clr-common-language-runtime-compilation.md).
