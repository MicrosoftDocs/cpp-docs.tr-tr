# <a name="resource-compiler-error-rc2175"></a>Kaynak Derleyicisi Hatası RC2175

kaynak dosyası dosya adı 3,00 biçiminde değil

Belirtilen kaynak 3,00 sürümünden önceki bir biçimi kullandı. Kaynak dosyası, sürüm 3,00 veya üzeri biçimi kullanılarak dönüştürülmelidir veya yeniden oluşturulmalıdır.
