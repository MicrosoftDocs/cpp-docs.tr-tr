# <a name="resource-compiler-warning-rw4001"></a>Kaynak Derleyicisi Uyarısı RW4001

. EXE işleme seçenekleri (/L/M/P/T/K/E/31 veya/30)

EXE işleme seçeneklerini belirttiniz, ancak işlenecek yürütülebilir dosya yoktu. Bu seçenekleri bir. res dosyası ile kullanmayın.
