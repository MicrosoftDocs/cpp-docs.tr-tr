# <a name="expression-evaluator-error-cxx0041"></a>İfade Değerlendirici Hatası CXX0041

sınıf öğesi statik üye veya üye işlev olmalıdır

Sınıfın (veya yapının ya da Union) statik olmayan bir üyesi kullanılacak sınıfın örneklenmesi belirtilmeden kullanıldı.

Yalnızca statik veri üyeleri veya üye işlevleri bir örnek oluşturma belirtilmeden kullanılabilir.

Bu hata CAN0041 ile aynıdır.
