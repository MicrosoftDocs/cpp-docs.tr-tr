# <a name="expression-evaluator-error-cxx0026"></a>İfade Değerlendirici Hatası CXX0026

hatalı biçim dizesi

Bir biçim dizesi yanlış belirtildi. İfade sözdizimini denetleyin.

Bu hata için CAN0026 aynıdır.

Daha fazla bilgi için [C++ içindeki Biçim belirticileri](/visualstudio/debugger/format-specifiers-in-cpp).
