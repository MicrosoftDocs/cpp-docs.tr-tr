# <a name="expression-evaluator-error-cxx0072"></a>İfade Değerlendirici Hatası CXX0072

.Pch dosyası Adlandır bağlanmadı veya /Zi ile derlenmemiş bir modüldeki bir türe başvuru koduna sahiptir.
