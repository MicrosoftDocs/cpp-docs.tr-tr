# <a name="resource-compiler-error-rc2163"></a>Kaynak Derleyicisi Hatası RC2163

Hızlandırıcı türü gerekli [ASCII veya VIRTKEY]

**Hızlandırıcılar** deyimindeki `type` alanı ASCII ya da VIRTKEY değerini içermelidir.
