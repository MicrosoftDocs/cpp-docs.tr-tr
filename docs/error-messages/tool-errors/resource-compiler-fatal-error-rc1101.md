# <a name="resource-compiler-fatal-error-rc1101"></a>Kaynak Derleyicisi Önemli Hatası RC1101

> Kaynak ikili dosya adı belirtilmedi

Yeniden adlandırma çıkış (/ fo) seçeneği bir dosya adı tarafından takip.

/Fo seçeneği için aşağıdaki sözdizimini kullanın:

> **RC /fo** *outfilename* <em>infilename</em>**.rc**
