# <a name="nmake-fatal-error-u1071"></a>NMAKE Önemli Hatası U1071

' TargetName ' hedefi için bağımlılık ağacında döngüye dön

Belirtilen hedefin bağımlılık ağacında döngüsel bağımlılık var. Verilen hedef, belirtilen hedefin bağımlılığının birine bağımlı. Döngüsel bağımlılıklar geçersiz.
