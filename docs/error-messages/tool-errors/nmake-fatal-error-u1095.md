# <a name="nmake-fatal-error-u1095"></a>NMAKE Önemli Hatası U1095

Genişletilmiş komut satırı ' CommandLine ' çok uzun

Makro genişletmesinden sonra, verilen komut satırı işletim sistemi için komut satırı sınırını aştı.

MS-DOS, komut satırında 128 karaktere kadar izin verir.

Komut, bir dosyadan komut satırı girişi kabul edebilecek bir program için ise, komutunu değiştirin ve bir dosyadaki dosya veya satır içi bir dosya girin. Örneğin, LINK ve LIB bir yanıt dosyasındaki girişi kabul eder.
