# <a name="resource-compiler-fatal-error-rc1017"></a>Kaynak Derleyicisi Önemli Hatası RC1017

Geçersiz tamsayı sabit ifadesi

İfade bir `#if` yönergesi yoktu veya bir sabite değerlendirmez.
