# <a name="project-build-error-prj0020"></a>Proje Derleme Hatası PRJ0020

> '*Tool*' aracı, '*Property*' özelliği geçersiz '*File*' dosya adını içeriyor.

Araç *aracının* özellik *özelliğinde* belirtilen dosya adı *dosyası* geçersizdi.

Bilinmeyen veya geçersiz bir makro kullanmış olabilirsiniz.

Projenizin özelliklerine erişme hakkında bilgi için bkz. [Visual C++ proje özelliklerini ayarlama](../../build/working-with-project-properties.md) .
