# <a name="expression-evaluator-error-cxx0020"></a>İfade Değerlendirici Hatası CXX0020

Bu işlem için işlenen türleri hatalı

İşleç, bu işleç için geçersiz bir türe sahip bir ifadeye uygulandı.

Örneğin, bir kaydın adresini almak veya kayan nokta ifadesiyle bir diziyi alt simge yapmak geçersizdir.

Bu hata CAN0020 ile aynıdır.
