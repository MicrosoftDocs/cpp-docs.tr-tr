# <a name="command-line-warning-d9043"></a>Komut Satırı Uyarısı D9043

' compiler_option ' için geçersiz ' warning_level ' değeri; ' 4999 ' varsayılıyor; Kod Analizi uyarıları, uyarı düzeyleriyle ilişkili değil

## <a name="example"></a>Örnek

Aşağıdaki örnek C9043 oluşturur.

```cpp
// D9043.cpp
// compile with: /analyze /w16001
// D9043 warning expected
int main() {}
```
