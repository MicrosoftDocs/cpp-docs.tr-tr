# <a name="project-build-error-prj0017"></a>Proje Derleme Hatası PRJ0017

Geçerli çalışma dizini geçersiz.

Geçerli çalışma dizininin yolu _MAXPATH daha uzun. Bu hatayı çözmek için, projenizi ayrıntılı bir düzeyde oluşturmayın.
