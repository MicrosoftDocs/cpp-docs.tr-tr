# <a name="expression-evaluator-error-cxx0050"></a>İfade Değerlendirici Hatası CXX0050

:: öğesinin sol tarafı sınıf/yapı/birleşim olmalıdır

Kapsam çözümleme işlecinin () sol tarafındaki simge `::` bir sınıf, yapı veya birleşim değil.

Bu hata CAN0050 ile aynıdır.
