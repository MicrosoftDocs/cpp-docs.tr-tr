# <a name="compiler-error-c2542"></a>Derleyici hatası C2542

' tanımlayıcı ': sınıf nesnesinin başlatma için oluşturucusu yok

Başlatma ile eşleşen bir parametre listesi olan bir Oluşturucu yok.

Bu hata, başlatma sırasında yanlış parametrelerin oluşmasına neden olabilir.
