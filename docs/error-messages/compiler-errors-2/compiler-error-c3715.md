# <a name="compiler-error-c3715"></a>Derleyici Hatası C3715

'işaretçisi': 'class' için bir işaretçi olmalıdır

Bir işaretçi belirtilen [__hook](../../cpp/hook.md) veya [__unhook](../../cpp/unhook.md) , noktası için geçerli bir sınıf. Bu hatayı düzeltmek için emin olun, `__hook` ve `__unhook` çağrılar, geçerli sınıf işaretçileri belirtin.
