# <a name="compiler-error-c3715"></a>Derleyici Hatası C3715
'işaretçi': 'sınıfı' için bir işaretçi olmalıdır  
  
 Bir işaretçi belirtilen [__hook](../../cpp/hook.md) veya [__unhook](../../cpp/unhook.md) , göstermiyor için geçerli bir sınıf. Bu hatayı düzeltmek için emin olun, `__hook` ve `__unhook` çağrıları geçerli sınıf işaretçileri belirtin.
