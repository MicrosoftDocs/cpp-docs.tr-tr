# <a name="compiler-error-c3818"></a>Derleyici Hatası C3818
dizi özellik bildirimi 'property1' Index özelliği 'property2' aşırı yükleme değil  
  
 Bir dizin oluşturucu biridir ve diğer bir dizi özelliği olduğunda bir aşırı özelliklerini mümkün değildir. 
  
 C3818 eski derleyici seçeneği kullanılarak erişilebilir, yalnızca **/clr:oldSyntax**.
