# <a name="compiler-error-c3818"></a>Derleyici Hatası C3818

dizi özelliği bildirimi 'property1' 'property2' dizin özelliğini aşırı yüklemeyecek

Bir dizin oluşturucu ve diğer bir dizi özelliği olduğunda bir aşırı yükleme özellikleri için mümkün değildir.

C3818 eski derleyici seçeneği kullanılarak erişilebilir, yalnızca **/clr:oldSyntax**.
