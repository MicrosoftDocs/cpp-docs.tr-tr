# <a name="compiler-error-c3818"></a>Derleyici hatası C3818

' Property1 ' dizi özelliği bildirimi ' Property2 ' Dizin özelliğini aşırı yüklemeyecek

Bir Dizin Oluşturucu ve diğeri bir dizi özelliği olduğunda, özellikler için aşırı yükleme mümkün değildir.

C3818 yalnızca eski derleyici seçeneği **/clr: oldSyntax** kullanılarak erişilebilir.
