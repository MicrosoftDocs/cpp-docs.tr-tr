# <a name="compiler-error-c3010"></a>Derleyici Hatası C3010
'etiket': OpenMP yapılandırılmış blok izin dışında atlama  
  
 Kod içinde veya dışında bir OpenMP bloğu atlama olamaz.  
  
 Aşağıdaki örnek C3010 oluşturur:  
  
```  
// C3010.c  
// compile with: /openmp  
int main() {  
   #pragma omp parallel   
   {  
      #pragma omp parallel  
      {  
         goto lbl3;  
      }  
   }  
   lbl3:;   // C3010  
}  
```
