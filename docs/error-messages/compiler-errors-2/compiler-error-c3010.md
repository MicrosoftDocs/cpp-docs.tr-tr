# <a name="compiler-error-c3010"></a>Derleyici Hatası C3010

'etiketi': OpenMP yapısal bloğunun izin dışına atlamaya

Kod içine veya bir OpenMP bloğunun dışına atlama olamaz.

Aşağıdaki örnek, C3010 oluşturur:

```
// C3010.c
// compile with: /openmp
int main() {
   #pragma omp parallel
   {
      #pragma omp parallel
      {
         goto lbl3;
      }
   }
   lbl3:;   // C3010
}
```
