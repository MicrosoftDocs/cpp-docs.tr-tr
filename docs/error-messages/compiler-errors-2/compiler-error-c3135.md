# <a name="compiler-error-c3135"></a>Derleyici hatası C3135

' Property ': bir özellik ' const ' veya ' volatile ' türüne sahip olamaz

Özelliklerde [const](../../cpp/const-cpp.md) ve [volatile](../../cpp/volatile-cpp.md) anahtar sözcüklere izin verilmez.
