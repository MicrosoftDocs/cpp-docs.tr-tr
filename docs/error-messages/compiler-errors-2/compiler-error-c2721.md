# <a name="compiler-error-c2721"></a>Derleyici hatası C2721

' belirtici ': işleç anahtar sözcüğü ile tür arasında depolama sınıfı belirtici geçersizdir

Kullanıcı tanımlı tür dönüştürmeleri tüm depolama sınıflarına uygulanır, bu nedenle tür dönüştürmesinde bir depolama sınıfı belirtemezsiniz.
