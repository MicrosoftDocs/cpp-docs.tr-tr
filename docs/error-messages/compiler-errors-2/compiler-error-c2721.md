# <a name="compiler-error-c2721"></a>Derleyici Hatası C2721

'belirticisi': işleç anahtar sözcüğü ile tür arasında geçersiz depolama sınıfı tanımlayıcısı

Bir depolama sınıfı bir tür dönüştürme belirtemezsiniz, kullanıcı tanımlı tür dönüşümleri tüm depolama sınıfları için geçerlidir.
