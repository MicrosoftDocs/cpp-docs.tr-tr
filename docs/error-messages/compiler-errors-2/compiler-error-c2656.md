# <a name="compiler-error-c2656"></a>Derleyici hatası C2656

' function ': işleve bir bit alanı olarak izin verilmiyor

Bir işlev, bir bit alanının üyesi olarak bildirilmiştir.

Bu hata, Oluşturucu başlatıcı listesindeki bir sözdizimi hatasından kaynaklanabilir.
