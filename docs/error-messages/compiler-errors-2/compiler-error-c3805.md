# <a name="compiler-error-c3805"></a>Derleyici hatası C3805

' token ': beklenmeyen belirteç, '} ' veya bir tanımlayıcı bekleniyor

Bir özellik tanımlarken, geçersiz bir belirteçle karşılaşıldı. Geçersiz belirteci kaldırın.
