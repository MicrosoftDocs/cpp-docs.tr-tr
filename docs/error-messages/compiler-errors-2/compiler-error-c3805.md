# <a name="compiler-error-c3805"></a>Derleyici Hatası C3805

'token': beklenmeyen belirteç bekleniyordu '}' veya bir tanımlayıcı

Özellik tanımlama, geçersiz bir belirteç karşılaşıldı. Geçersiz belirteç kaldırın.
