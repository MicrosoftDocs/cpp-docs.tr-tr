# <a name="compiler-error-c3506"></a>Derleyici hatası C3506

' ID ' kitaplık kimliği için kayıtlı bir tür kitaplığı yok

TypeLib doğru şekilde kaydedilmedi. Bir tür kitaplığını kaydetmek için regsvr32.exe kullanın.
