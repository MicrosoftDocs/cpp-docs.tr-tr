# <a name="compiler-error-c2996"></a>Derleyici Hatası C2996

'function': özyinelemeli işlev şablonu tanımı

Bir işlev tanımı kendi kök şablonlu sınıf örneği dener. Yinelenen şablon örneklemeleri izin verilmez.
