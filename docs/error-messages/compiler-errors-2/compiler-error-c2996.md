# <a name="compiler-error-c2996"></a>Derleyici hatası C2996

' function ': özyinelemeli işlev şablonu tanımı

Bir işlev tanımı kök şablonlu sınıfının örneğini oluşturmaya çalışır. Özyinelemeli şablon örneklemelerinden izin verilmez.
