# <a name="compiler-error-c2545"></a>Derleyici Hatası C2545
'işleci': aşırı Bul işleci oluşturulamıyor  
  
 İşleç sağlanan işlenenleri ile birlikte kullanılamaz. Aşırı yüklenmiş bir işleç gerekli işlenen ile sağlamanız gerekir.  
  
 İşlenen türü yanlış varsa bu hataya neden olabilir.  
  
 Bu hata, bir dönüşüm işleci veya tek bir parametre alan oluşturucu tanımlarsanız düzeltilebilir.
