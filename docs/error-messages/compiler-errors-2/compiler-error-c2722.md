# <a name="compiler-error-c2722"></a>Derleyici hatası C2722

':: operator ': şu işleç komutu geçersiz; ' işleç işleci ' kullan

Bir `operator` ifade veya ' i tekrar tanımlar `::new` `::delete` . `new`Ve `delete` işleçleri geneldir, bu nedenle kapsam çözümleme işleci ( `::` ) anlamlı değildir. İşleci kaldırın `::` .
