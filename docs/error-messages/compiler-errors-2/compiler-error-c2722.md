# <a name="compiler-error-c2722"></a>Derleyici hatası C2722

':: operator ': şu işleç komutu geçersiz; ' işleç işleci ' kullan

`operator` bir ifade `::new` veya `::delete`tekrar tanımlar. `new` ve `delete` işleçleri geneldir, bu nedenle kapsam çözümleme işleci (`::`) anlamsız bir şekilde daha düşüktür. `::` işlecini kaldırın.
