# <a name="compiler-error-c3303"></a>Derleyici Hatası C3303

'attribute': öznitelik yalnızca 'kullanımı' kullanılabilir

Burada geçerli değil, bir özniteliği kullanacak şekilde girişimde bulunuldu.
