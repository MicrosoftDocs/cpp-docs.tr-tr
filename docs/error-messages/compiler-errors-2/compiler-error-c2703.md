# <a name="compiler-error-c2703"></a>Derleyici Hatası C2703

Geçersiz __leave deyimi

A `__leave` deyimi içinde olmalıdır bir `__try` blok.

Aşağıdaki örnek, C2703 oluşturur:

```
// C2703.cpp
int main() {
   __leave;   // C2703
   __try {
      // try the following line instead
      __leave;
   }
   __finally {}
}
```
