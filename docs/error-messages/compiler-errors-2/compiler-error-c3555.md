# <a name="compiler-error-c3555"></a>Derleyici Hatası C3555

> 'decltype' için yanlış argüman

`decltype(` *expression* İfade`)` türü belirtimi için bağımsız değişken geçerli bir ifade değildir.

> [!NOTE]
> C3555 gerçekleşmesi olası değildir. Derleyici C3555 yayıyorsa, bunun nedeni büyük olasılıkla bir iç derleyici hatasının oluşmasıdır.
