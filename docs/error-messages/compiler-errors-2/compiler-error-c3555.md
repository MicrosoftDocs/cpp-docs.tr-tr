# <a name="compiler-error-c3555"></a>Derleyici Hatası C3555
  
> 'decltype' yanlış bağımsız değişken  
  
 Bağımsız değişkeni `decltype(` *ifade* `)` tür belirteci geçerli bir ifade değil.  
  
> [!NOTE]
>  C3555 oluştuğu değil. Derleyici C3555 yayar, bir iç derleyici hatası oluştuğundan olabilir.
