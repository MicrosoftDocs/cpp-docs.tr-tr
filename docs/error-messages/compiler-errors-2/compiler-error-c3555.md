# <a name="compiler-error-c3555"></a>Derleyici Hatası C3555

> 'decltype' için geçersiz bağımsız değişken

Bağımsız değişkeni `decltype(` *ifade* `)` tür tanımlayıcısı geçerli bir ifade değil.

> [!NOTE]
>  C3555 oluşma olasılığı değil. Derleyici C3555 yayar, derleyici iç hatası oluştuğundan olabilir.
