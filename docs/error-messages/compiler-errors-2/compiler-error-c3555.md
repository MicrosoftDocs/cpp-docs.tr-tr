# <a name="compiler-error-c3555"></a>Derleyici hatası C3555

> ' decltype ' için geçersiz bağımsız değişken

`decltype(`*ifadesinin* bağımsız değişkeni`)` tür belirleyicisi geçerli bir ifade değil.

> [!NOTE]
>  C3555 olası olabilir. Derleyici C3555 yayar, büyük olasılıkla iç derleyici hatası oluşmuş olabilir.
