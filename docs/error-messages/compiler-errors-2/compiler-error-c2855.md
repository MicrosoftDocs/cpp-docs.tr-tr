# <a name="compiler-error-c2855"></a>Derleyici hatası C2855

' Option ' komut satırı seçeneği önceden derlenmiş üstbilgiyle tutarsız

Verilen komut satırı seçeneği, ön derlenmiş üstbilgiyi oluşturmak için kullanılan seçenekten farklıdır. Belirtilen komut satırı seçeneğiyle ön derlenmiş üstbilgiyi yeniden oluşturun.
