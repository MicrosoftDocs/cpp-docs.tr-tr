# <a name="compiler-error-c2573"></a>Derleyici Hatası C2573

'class': Bu tür; nesnelerin işaretçileri silinemez sınıfı, 'operator delete' için yerleşim olmayan Aşırı yüklenen yok sahiptir.

Sınıfı, bir yerleşim olmayan delete işleci eksik.
