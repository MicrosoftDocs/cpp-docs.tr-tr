# <a name="compiler-error-c2573"></a>Derleyici hatası C2573

' class ': Bu türdeki nesnelerin işaretçileri silinemez; sınıfta ' operator delete ' için yerleştirme olmayan bir aşırı yükleme yok.

Sınıfta, yerleştirme olmayan bir delete işleci eksik.
