# <a name="compiler-error-c2948"></a>Derleyici hatası C2948

açık örnek oluşturma; özelleştirmede depolama sınıfı belirticisine izin verilmez

**`extern`** Daha önce açıkça örneklenen bir şablon sınıfının bir özelleştirmesi içinde depolama sınıfı tanımlayıcılarını (gibi) kullanamazsınız.
