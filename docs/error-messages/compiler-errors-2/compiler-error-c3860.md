# <a name="compiler-error-c3860"></a>Derleyici hatası C3860

tür bağımsız değişken listesi aşağıdaki sınıf türü adı, tür parametre listesinde kullanılan sırada parametreleri listemalıdır

Genel veya şablon bağımsız değişken listesi hatalı biçimlendirilmiş.

Aşağıdaki örnek C3860 oluşturur:

```cpp
// C3860.cpp
// compile with: /LD
template <class T1, class T2>
struct A {
   void f();
};

template <class T2, class T1>
void A<T1, T2>::f() {}   // C3860
```

Olası çözüm:

```cpp
// C3860b.cpp
// compile with: /c
template <class T1, class T2>
struct A {
   void f();
};

template <class T2, class T1>
void A<T2, T1>::f() {}
```

C3860, genel türler kullanılırken de oluşabilir:

```cpp
// C3860c.cpp
// compile with: /clr
generic<class T,class U>
ref struct GC {
   void f();
};

generic<class T, class U>
void GC<T,T>::f() {}   // C3860
```

Olası çözüm:

```cpp
// C3860d.cpp
// compile with: /clr /c
generic<class T,class U>
ref struct GC {
   void f();
};

generic<class T, class U>
void GC<T,U>::f() {}
```
