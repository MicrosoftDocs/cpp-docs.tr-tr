# <a name="compiler-error-c2529"></a>Derleyici Hatası C2529
'name': başvuru için geçersiz  
  
 Bu hata, işaretçi sözdizimi kullanılarak ve bir işaretçi başvuru bildirme düzeltilebilir.  
  
 Aşağıdaki örnek C2529 oluşturur:  
  
```  
// C2529.cpp  
// compile with: /c  
int i;  
int &ri = i;  
int &(&rri) = ri;   // C2529  
```
