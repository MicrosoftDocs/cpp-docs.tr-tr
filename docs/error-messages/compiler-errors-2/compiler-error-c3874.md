# <a name="compiler-error-c3874"></a>Derleyici Hatası C3874
'işlevinin' dönüş türü 'type' yerine ' int' olmalıdır  
  
 Bir işlev derleyici tarafından beklenen dönüş türüne sahip değil.  
  
 Aşağıdaki örnek C3874 oluşturur:  
  
```  
// C3874b.cpp  
double main()  
{   // C3874  
}  
```
