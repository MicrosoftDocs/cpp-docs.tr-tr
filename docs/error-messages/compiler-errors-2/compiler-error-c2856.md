# <a name="compiler-error-c2856"></a>Derleyici hatası C2856

\#pragma hdrstop bir #if bloğunun içinde olamaz

`hdrstop` pragma, koşullu derleme bloğunun gövdesi içine yerleştirilemez.

`#pragma hdrstop` ifadesini `#if/#endif` bloğunda bulunmayan bir alana taşıyın.
