# <a name="compiler-error-c3632"></a>Derleyici hatası C3632

' Event ': yapı için geçersiz olay stili

[__event](../../cpp/event.md) bildirimleri tüm yapılar için geçerli değildir.

C3632 yalnızca eski derleyici seçeneği **/clr: oldSyntax** kullanılarak erişilebilir.
