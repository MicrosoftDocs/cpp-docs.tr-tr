'event': olay yapı için geçersiz stili  
  
 [__Event](../../cpp/event.md) bildirimleri tüm yapılardan geçerli değildir.  
  
 C3632 eski derleyici seçeneği kullanılarak erişilebilir, yalnızca **/clr:oldSyntax**.
