# <a name="compiler-error-c2830"></a>Derleyici hatası C2830

yalnızca ' operator New ' öğesine yönelik yerleştirme parametreleri varsayılan değerlere sahip olabilir

[New işleci](../../standard-library/new-operators.md#op_new) için standart biçimsel parametreler varsayılan değerlere sahip olamaz. Yalnızca Kullanıcı tanımlı yerleştirme parametreleri varsayılan olarak belirtilebilir.
