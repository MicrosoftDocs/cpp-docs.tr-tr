# <a name="compiler-error-c2667"></a>Derleyici Hatası C2667
'function': numara aşırı hiçbirinin en iyi bir dönüştürme yok  
  
 Aşırı yüklenmiş işlev çağrısı belirsiz ve çözümlenemiyor.  
  
 Dönüştürme işlev çağrısı bir aşırı yüklenmiş işlevlerin gerçek parametrelerinde eşleştirmek için gerekli tüm diğer Aşırı yüklenen işlevler tarafından gerekli dönüşümleri daha iyi kesinlikle olması gerekir.  
  
 İşlev şablonlarının kısmi sıralanması Bilgi Bankası makalesi Q240869 daha fazla bilgi için bkz.
