# <a name="compiler-error-c2532"></a>Derleyici hatası C2532

' tanımlayıcı ': başvuru için geçersiz değiştirici

Başvuru değiştirildi. Başvurular başka bir nesneye başvuracak şekilde değiştirilemez. Bunun yerine bir işaretçi kullanın.
