# <a name="compiler-error-c2670"></a>Derleyici hatası C2670

' tanımlayıcı ': işlev şablonu parametre numarasını ' Type ' türünden dönüştüremiyor

Bir parametre gerekli türe dönüştürülemedi.

Bu hata, açık bir dönüştürme oluşturursanız düzeltilebilir.
