# <a name="compiler-error-c2616"></a>Derleyici Hatası C2616

'conversion': bir lvalue olmayan 'type1' iken 'type2' örtük olarak dönüştürülemez sabit değil

Bir başvuru, bir lvalue olmayan başlatılamaz.

Bu bir hatadır altında ANSI uyumluluğu ([/Za](../../build/reference/za-ze-disable-language-extensions.md)) ve Microsoft uzantıları altında bir uyarı (**/Ze**).
