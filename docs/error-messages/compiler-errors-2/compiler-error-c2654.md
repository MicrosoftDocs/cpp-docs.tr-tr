# <a name="compiler-error-c2654"></a>Derleyici hatası C2654

' tanımlayıcı ': üye işlev dışında üyeye erişme girişimi

Bir bildirimde bir bir üyeye erişilir. Üye verilerine yalnızca üye işlevlerinde erişilebilir.

Bu hata, bir bildirimde değişkenleri başlatmaya çalışırken oluşabilir. Bu amaçla bir Oluşturucu kullanın.
