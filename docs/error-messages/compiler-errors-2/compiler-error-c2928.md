# <a name="compiler-error-c2928"></a>Derleyici hatası C2928

> açık örnek oluşturma; '*Identifier*', '*class-name*' şablon sınıfının bir işlevi veya statik veri üyesi değil

İşlev veya değişken olmayan bir *sınıf adı* üyesini açık olarak örnekleyemezsiniz **`static`** .
