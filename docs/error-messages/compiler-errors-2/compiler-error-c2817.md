# <a name="compiler-error-c2817"></a>Derleyici hatası C2817

' operator delete ' için dönüş türü ' void ' olmalıdır

Aşırı yüklenmiş [operator delete](../../standard-library/new-operators.md#op_delete) işlevi bir değer döndüremez.
