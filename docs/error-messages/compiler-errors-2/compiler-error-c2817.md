# <a name="compiler-error-c2817"></a>Derleyici Hatası C2817

'operator delete' için dönüş türü 'void' olmalıdır

Aşırı yüklenmiş bir [delete işleci](../../standard-library/new-operators.md#op_delete) işlev bir değer döndürür.
