# <a name="compiler-error-c2858"></a>Derleyici hatası C2858

'/Yıc (/Fdfilename) ' komut satırı seçeneği önceden derlenmiş üstbilgiyle tutarsız, bu '/Fdfilename ' kullandı

Önceden derlenmiş üst bilgi kullan ([/yu](../../build/reference/yu-use-precompiled-header-file.md)) seçeneği tarafından belirtilen program veritabanı, önceki ön derlenmiş oluşturma üst bilgisi ([/rivc](../../build/reference/yc-create-precompiled-header-file.md)) seçeneği tarafından belirtilen bir değer değil.
