# <a name="compiler-error-c3333"></a>Derleyici hatası C3333

' tür kitaplığı ': bozuk tür kitaplığı #import olamaz

İfadede belirtilen tür kitaplığı, `#import` derleyici tarafından okunamaz. Mümkünse tür kitaplığını yeniden oluşturmak ya da tedarikçinizden yeni bir tür kitaplığı istemek isteyebilirsiniz. Bu konuyla ilgili ne olduğunu görmek üzere tür kitaplığı dosyasını görüntülemek için Visual C++ ile sağlanan OLE görüntüleyicisini kullanmak isteyebilirsiniz.
