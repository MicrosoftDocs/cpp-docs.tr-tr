# <a name="compiler-error-c3815"></a>Derleyici hatası C3815

' get_accessor ' yönteminin dönüş türü bir ayarlayıcının son parametresinin türüyle eşleşmelidir

Özellikleri bildirirken yöntemin dönüş değeri, `get_accessor` set accessor yönteminin bildirimindeki son parametreyle eşleşmelidir.

C3815 yalnızca eski derleyici seçeneği **/clr: oldSyntax** kullanılarak erişilebilir.
