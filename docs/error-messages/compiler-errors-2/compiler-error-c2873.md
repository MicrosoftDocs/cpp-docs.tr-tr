# <a name="compiler-error-c2873"></a>Derleyici Hatası C2873

'symbol': simge bir bildirim kullanımı içinde kullanılamaz

A `using` yönergesi eksik bir [ad alanı](../../cpp/namespaces-cpp.md) anahtar sözcüğü. Bu derleyici kodu olarak hatalı yorumlayan neden olan bir [using bildirimi](../../cpp/using-declaration.md) yerine [using yönergesi](../../cpp/namespaces-cpp.md#using_directives).
