# <a name="compiler-error-c2873"></a>Derleyici Hatası C2873
'simgesi': sembol bir kullanarak-bildiriminde kullanılamaz  
  
 A `using` yönergesi eksik bir [ad alanı](../../cpp/namespaces-cpp.md) anahtar sözcüğü. Bu kodu olarak hatalı yorumlayan derleyici neden olan bir [bildirimi kullanarak](../../cpp/using-declaration.md) yerine [using yönergesi](../../cpp/namespaces-cpp.md#using_directives).
