# <a name="compiler-error-c2873"></a>Derleyici hatası C2873

' symbol ': simge bir using bildiriminde kullanılamaz

Bir **`using`** yönergede bir [Namespace](../../cpp/namespaces-cpp.md) anahtar sözcüğü eksik. Bu, derleyicinin kodu bir [using yönergesi](../../cpp/namespaces-cpp.md#using_directives)yerine [using bildirimi](../../cpp/using-declaration.md) olarak yorumlamasını sağlar.
