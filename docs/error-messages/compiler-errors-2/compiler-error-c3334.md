# <a name="compiler-error-c3334"></a>Derleyici Hatası C3334
#import bozuk kitaplığı yazamazsınız  
  
 Kitaplık dosyasında bir `#import` deyimi bozuk ve derleyici tarafından kullanılamaz.  Bozulma disk bozulması modülü hatalı kuşaktan gelebilir veya ikili dosya varsa şekilde düzenlenmesi ve değiştirdi.  
  
 Daha fazla bilgi için bkz: [#import yönergesi](../../preprocessor/hash-import-directive-cpp.md).
