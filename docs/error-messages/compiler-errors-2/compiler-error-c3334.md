# <a name="compiler-error-c3334"></a>Derleyici Hatası C3334

#import bozuk tür kitaplığı

Kitaplık dosyası içinde bir `#import` deyimi bozuk ve derleyici tarafından kullanılamaz.  Bozulma modülünün disk bozulması, hatalı kuşaktan gelebilir veya ikili dosya varsa şekilde düzenlendi ve değiştirilebilir.

Daha fazla bilgi için [#import yönergesi](../../preprocessor/hash-import-directive-cpp.md).
