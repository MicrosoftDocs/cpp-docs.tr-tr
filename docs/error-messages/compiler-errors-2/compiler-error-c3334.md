# <a name="compiler-error-c3334"></a>Derleyici hatası C3334

tür kitaplığı #import bozuk

Bir deyimindeki kitaplık dosyası `#import` bozuk ve derleyici tarafından tüketilemiyor.  Bozulma, modülün hatalı nesli, disk bozulması veya ikili dosya bir şekilde düzenlenip değiştirilmişse olabilir.

Daha fazla bilgi için bkz. [#import yönergesi](../../preprocessor/hash-import-directive-cpp.md).
