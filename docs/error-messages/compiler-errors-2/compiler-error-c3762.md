# <a name="compiler-error-c3762"></a>Derleyici hatası C3762

' Attribute ' özniteliği işlenemiyor

Öğesinden devralan Kullanıcı tanımlı bir öznitelik `System.Security.Permissions.SecurityAttribute` , bir güvenlik özniteliği tanımlamak için kullanılıyor. Bu tür bir öznitelik, tanımlandığı derlemede kullanılamaz.
