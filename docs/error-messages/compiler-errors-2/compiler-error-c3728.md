# <a name="compiler-error-c3728"></a>Derleyici hatası C3728

' Event ': olayda bir Raise yöntemi yok

C# gibi bir dille oluşturulan meta veriler, bir olayın tanımlandıkları sınıfın dışından tetiklememe, [#using](../../preprocessor/hash-using-directive-cpp.md) yönergesine dahil EDILMIŞTIR ve CLR programlama kullanan bir Visual C++ program, olayı oluşturmaya çalıştı.

C# gibi bir dilde geliştirilen bir programda bir olayı yükseltmek için, olayı içeren sınıfın olayı oluşturan bir ortak yöntem de tanımlanması gerekir.
