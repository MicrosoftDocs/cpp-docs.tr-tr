# <a name="compiler-error-c3550"></a>Derleyici hatası C3550

Bu bağlamda yalnızca düz ' decltype (Auto) ' kullanılabilir

`decltype(auto)`Bir işlevin dönüş türü için bir yer tutucu olarak kullanılırsa, kendisi tarafından kullanılması gerekir. Bir işaretçi bildiriminin ( `decltype(auto*)` ), başvuru bildiriminin () veya başka bir niteliğin bir parçası olarak kullanılamaz `decltype(auto&)` .
