# <a name="compiler-error-c3048"></a>Derleyici Hatası C3048

'#Pragma omp atomic' sonrasındaki ifade hatalı biçimde

Atomik bir yönerge yanlış olarak belirtildi.

Aşağıdaki örnek, C3048 oluşturur:

```
// C3048.cpp
// compile with: /openmp vcomps.lib
#include "omp.h"
#include <stdio.h>

int main() {
   int a[10];
   omp_set_num_threads(4);
   #pragma omp parallel
   {
      #pragma omp atomic
      a[0] = 1;   // C3048
      // try the following line instead
      // a[0] += 1;
   }
}
```
