# <a name="compiler-error-c2834"></a>Derleyici hatası C2834

' işleç işleci ' genel olarak nitelenmelidir

`new` ve `delete` işleçleri bulundukları sınıfa bağlıdır. Kapsam çözümleme, farklı bir sınıftan bir `new` veya `delete` sürümü seçmek için kullanılamaz. `new` veya `delete` işlecinin birden çok biçimini uygulamak için, ek biçimsel parametrelerle işlecin bir sürümünü oluşturun.
