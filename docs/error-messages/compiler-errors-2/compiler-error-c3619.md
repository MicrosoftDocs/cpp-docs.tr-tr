# <a name="compiler-error-c3619"></a>Derleyici hatası C3619

Şablon, yönetilen veya WinRT türü içinde bildirilemez

Yönetilen veya WinRT sınıfında veya arabiriminde sınıf şablonlarına izin verilmez.

C3619 yalnızca eski derleyici seçeneği **/clr: oldSyntax** kullanılarak erişilebilir.
