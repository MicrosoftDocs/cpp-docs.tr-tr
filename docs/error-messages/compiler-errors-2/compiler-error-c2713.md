# <a name="compiler-error-c2713"></a>Derleyici hatası C2713

> Her işlevde yalnızca bir özel durum işleme izni verilir

Aynı işlevde yapılandırılmış özel durum işleme ( **`__try`** / **`__except`** ) ve C++ özel durum işleme ( **`try`** / **`catch`** ) kullanamazsınız.
