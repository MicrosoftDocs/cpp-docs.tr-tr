# <a name="compiler-error-c3130"></a>Derleyici hatası C3130

İç derleyici hatası: eklenen kod bloğu PDB 'ye yazılamadı

Derleyici eklenen bir kod bloğunu. pdb dosyasına yazamadı bu hata oluşur. Hatanın en yaygın nedeni, disk alanının olmamasıdır.
