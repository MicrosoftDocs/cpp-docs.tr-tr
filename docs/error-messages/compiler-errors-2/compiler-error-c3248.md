# <a name="compiler-error-c3248"></a>Derleyici Hatası C3248
'function1': '__sealed' olarak bildirilen işlevi 'function2' tarafından geçersiz kılınamaz  
  
Geçersiz kılmak türetilmiş bir sınıf çalıştı bir **__sealed** sanal yöntemi.  
  
C3248 eski derleyici seçeneği kullanılarak erişilebilir, yalnızca **/clr:oldSyntax**.
