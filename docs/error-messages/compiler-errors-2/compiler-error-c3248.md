# <a name="compiler-error-c3248"></a>Derleyici Hatası C3248

'function1': '__sealed' olarak bildirilen işlev 'function2' tarafından geçersiz kılınamaz

Bir türetilmiş sınıf geçersiz kılma denedi bir **__sealed** sanal yöntem.

C3248 eski derleyici seçeneği kullanılarak erişilebilir, yalnızca **/clr:oldSyntax**.
