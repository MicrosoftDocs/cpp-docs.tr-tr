# <a name="compiler-error-c3248"></a>Derleyici hatası C3248

' işlev1 ': ' __sealed ' olarak belirtilen işlev ' function2 ' tarafından geçersiz kılınamaz

Türetilmiş bir sınıf **__sealed** sanal bir yöntemi geçersiz kılmayı denedi.

C3248 yalnızca eski derleyici seçeneği **/clr: oldSyntax** kullanılarak erişilebilir.
