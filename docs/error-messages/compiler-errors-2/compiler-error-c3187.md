# <a name="compiler-error-c3187"></a>Derleyici Hatası C3187

> ' &#95; &#95; func &#95; &#95;': yalnızca bir işlev gövdesi içinde kullanılabilir

Önceden tanımlanmış makrosu işlev gövdesine dışında kullanılamaz.

Bu hatayı düzeltmek için bir işlev gövdesi içinde makrosu taşıyın.
