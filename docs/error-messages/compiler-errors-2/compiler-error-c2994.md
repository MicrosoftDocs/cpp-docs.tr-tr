# <a name="compiler-error-c2994"></a>Derleyici Hatası C2994
Şablon parametresi listesinde adlandırılmamış sınıfı  
  
 Kullanamazsınız `class` anahtar sözcüğü bir sınıf adı belirtmeden şablon bağımsız değişken olarak.
