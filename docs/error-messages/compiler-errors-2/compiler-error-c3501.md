# <a name="compiler-error-c3501"></a>Derleyici Hatası C3501
ProgID 'ProgID' kayıtlı hiçbir typelib yok  
  
 Sınıf kimliği için bir verilen ProgID ilişkili tür kitaplığı yok. Bu nedenle, bu için ProgID geçiremezsiniz `#import` deyimi.
