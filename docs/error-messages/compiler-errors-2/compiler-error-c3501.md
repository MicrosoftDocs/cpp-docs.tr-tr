# <a name="compiler-error-c3501"></a>Derleyici hatası C3501

ProgID ' ProgID ' için kayıtlı bir tür kitaplığı yok

Belirli bir ProgID 'nin sınıf KIMLIĞINDE ilişkili bir tür kitaplığı yok. Bu nedenle, bu ProgID 'yi `#import` deyime geçiremezsiniz.
