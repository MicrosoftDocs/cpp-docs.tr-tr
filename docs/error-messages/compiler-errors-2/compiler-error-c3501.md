# <a name="compiler-error-c3501"></a>Derleyici Hatası C3501

ProgID 'Program Kimliği' için kayıtlı bir tür kitaplığı yok

Sınıf kimliği için bir verilen ProgID bir ilişkili tür kitaplığı yok. Bu nedenle, bu program kimliği için geçiremezsiniz `#import` deyimi.
