# <a name="compiler-error-c3173"></a>Derleyici hatası C3173

IDL birleştirmede sürüm uyumsuzluğu

Bu hata, bir nesne dosyası derleyicinin önceki bir sürümüyle oluşturulmuş gömülü IDL içerdiğinde oluşur. Derleyici,. obj dosyalarına katıştırılmış IDL içeriğini oluşturmak için aynı derleyicinin aynı zamanda gömülü IDL 'yi birleştirmek için kullanılan derleyicinin aynı olduğundan emin olmak için bir sürüm numarasını kodluyor.

Tüm araçların en C++ son yayınlanan sürümden olması için görsel yüklemenizi güncelleştirin.
