# <a name="compiler-error-c2517"></a>Derleyici Hatası C2517
'tanımlayıcısı': sağında '::' tanımlanmadı  
  
 Kapsam çözümü işleci sağındaki tanımlayıcının (`::`) sınıfı, yapı veya birleşim soldaki tanımlı bir üyesi olmanız gerekir. Hiçbir sınıf, yapı veya birleşim adlandırırsanız sağdaki tanımlayıcı genel kapsamlı bildirilmesi gerekir.
