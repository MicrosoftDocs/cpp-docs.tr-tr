# <a name="compiler-error-c2517"></a>Derleyici hatası C2517

' tanımlayıcı ': ':: ' sağ tarafı tanımsız

Kapsam çözümleme işlecinin () sağ tarafındaki tanımlayıcı, `::` sol taraftaki sınıf, yapı veya birleşimin tanımlı bir üyesi olmalıdır. Hiçbir sınıf, yapı veya birleşim adlandırılmadığında, sağdaki tanımlayıcı genel kapsamla bildirilmelidir.
