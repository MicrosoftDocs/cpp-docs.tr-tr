# <a name="compiler-error-c2773"></a>Derleyici hatası C2773

\#İçeri ve #using yalnızca C++ derleyicisinde kullanılabilir

C derleyicisi `#import` Önişlemci yönergesini tanımıyor. Kaynağı C++ olarak derleyin. Gerekirse [/TP](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md) 'ı kullanın.
