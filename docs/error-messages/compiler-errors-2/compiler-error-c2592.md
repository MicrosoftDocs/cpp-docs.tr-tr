# <a name="compiler-error-c2592"></a>Derleyici hatası C2592

' class ': ' base_class_2 ', ' base_class_1 ' öğesinden devralındı ve yeniden belirtilemez

Yalnızca diğer temel sınıflardan devralma yapan temel sınıfları belirtebilirsiniz. Bu durumda `base_class_1` zaten `base_class_2`devraldığı için, `class` belirtiminde yalnızca `base_class_1` gereklidir.
