# <a name="compiler-error-c3396"></a>Derleyici hatası C3396

' Attribute ': özel öznitelik ' namespace ' içinde bulunamadı

C3396, güncel ortak dil çalışma zamanı kullanmadığınız anlamına gelir.  Çözümlemek için CLR 'nizi yeniden yükleyin.
