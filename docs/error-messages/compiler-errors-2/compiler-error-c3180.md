# <a name="compiler-error-c3180"></a>Derleyici Hatası C3180
'tür adı': ad meta verileri 'sınırı' karakter sınırını aşıyor  
  
 Derleyici meta verilerde yönetilen türü için bir ad kesildi. Kesme türü ile kullanılamaz hale getirir `#using` yönergesi (veya başka bir dilde eşdeğeri).  
  
 Tür adı sınır, tüm ad alanı nitelikleri içerir.
