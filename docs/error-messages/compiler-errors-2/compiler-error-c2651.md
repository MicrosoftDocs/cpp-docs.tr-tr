# <a name="compiler-error-c2651"></a>Derleyici hatası C2651

' işleç ' sol tarafında bir sınıf, yapı veya birleşim olmalıdır

Bir şablon parametresini bir sınıf gibi kullanmak için, sınıf şablonunu integral türü yerine bir sınıf ile özelleştirecek şekilde özelleştirme.
