# <a name="compiler-error-c2568"></a>Derleyici hatası C2568

' Identifier1 ': işlev aşırı yüklemesi çözümlenemiyor

Derleyici, hangi aşırı yüklenmiş işlevin çağrılacağını belirleyemiyor. İşleve geçirilen gerçek parametrelerin, aşırı yüklenmiş bir işlevden birinin biçimsel parametreleriyle eşleşecek şekilde dönüştürülmesi gerekir, ancak hiçbir eşleşme diğerlerinden tamamen daha iyi değildir.
