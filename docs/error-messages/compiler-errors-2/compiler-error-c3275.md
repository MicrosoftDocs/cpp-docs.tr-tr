# <a name="compiler-error-c3275"></a>Derleyici hatası C3275

' enum Member ': Bu simge niteleyici olmadan kullanılamaz

Yönetilen kodu kullanırken ve iki veya daha fazla numaralandırma aynı ada sahip bir tanıtıcı içerdiğinde, doğrudan başvuruları tanımlayıcı olarak nitelemeniz gerekir.

C3275 yalnızca eski derleyici seçeneği **/clr: oldSyntax** kullanılarak erişilebilir.
