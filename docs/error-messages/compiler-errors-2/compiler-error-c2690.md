# <a name="compiler-error-c2690"></a>Derleyici hatası C2690

' operator ': yönetilen veya WinRT dizisinde işaretçi aritmetiği yapılamaz

Yönetilen veya WinRT dizisinde işaretçi aritmetik işlemlerine izin verilmez. Dizide çapraz geçiş yapmak için dizi dizini gösterimini kullanın.
