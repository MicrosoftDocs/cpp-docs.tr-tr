# <a name="compiler-error-c3322"></a>Derleyici hatası C3322

' Property ': ' Attribute ' özniteliği için geçerli bir özellik değil

Bir özniteliğe geçersiz veya tanınmayan bir Özellik (veya parametre) geçirildi. Özniteliğin belgelerini denetleyin.
