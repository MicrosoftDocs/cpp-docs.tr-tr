# <a name="compiler-error-c2634"></a>Derleyici Hatası C2634
' & class::member': başvuru üyesi işaretçidir geçersiz  
  
 Başvuru üyesi için bir işaretçi bildirildi.  
  
 Aşağıdaki örnek C2634 oluşturur:  
  
```  
// C2634.cpp  
int mem;  
struct S {  
   S() : rf(mem) { }  
   int &rf;  
};  
int (S::*pdm) = &S::rf;   // C2634  
```
