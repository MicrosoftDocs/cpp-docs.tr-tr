# <a name="compiler-error-c3456"></a>Derleyici Hatası C3456
[source_annotation_attribute] yapılamaz yönetilen veya WinRT sınıf bildirimi  
  
 source_annotation_attribute Kod Analizi tarafından kullanılmak üzere özel öznitelikleri tanımlamak için kullanılır.  
  
 Bu özel öznitelikler, yalnızca öznitelik anlar karşılık gelen bir eklenti varsa faydalıdır.
