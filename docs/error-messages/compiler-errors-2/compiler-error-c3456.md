# <a name="compiler-error-c3456"></a>Derleyici hatası C3456

[source_annotation_attribute] yönetilen veya WinRT sınıfı bildiriminde kullanılamaz

source_annotation_attribute, kod analizi tarafından kullanılacak özel öznitelikleri tanımlamak için kullanılır.

Bu özel öznitelikler yalnızca özniteliği anlayan ilgili bir eklenti varsa yararlıdır.
