# <a name="compiler-error-c3456"></a>Derleyici Hatası C3456

[source_annotation_attribute] yapılamaz yönetilen veya WinRT sınıf bildirimi

source_annotation_attribute, Kod Analizi tarafından kullanılmak üzere özel öznitelikler tanımlamak için kullanılır.

Bu özel öznitelikleri yalnızca öznitelik anlayan karşılık gelen bir eklenti ise yararlı olur.
