# <a name="compiler-error-c2551"></a>Derleyici Hatası C2551

' void *' türünün açıkça dönüştürülmesi gerekir

A `void` işaretçisi, örtük dönüştürme nonvoid işaretçisi atanır. Açık bir tür dönüştürme kullanmanız gerekir.
