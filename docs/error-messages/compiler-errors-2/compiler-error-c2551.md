# <a name="compiler-error-c2551"></a>Derleyici hatası C2551

' void * ' türü açık tür dönüştürme gerektiriyor

`void` işaretçisi, örtük dönüştürme ile void olmayan bir işaretçiye atanır. Açık bir tür dönüştürme kullanmanız gerekir.
