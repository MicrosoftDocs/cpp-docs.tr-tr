# <a name="compiler-error-c2551"></a>Derleyici Hatası C2551
' void *' türünün açık atama gerekir  
  
 A `void` işaretçi nonvoid işaretçi örtük dönüşüm tarafından atanır. Bir açık atama kullanmanız gerekir.
