# <a name="compiler-error-c2551"></a>Derleyici hatası C2551

' void * ' türü açık tür dönüştürme gerektiriyor

**`void`** Örtük dönüştürme ile void olmayan işaretçiye bir işaretçi atanır. Açık bir tür dönüştürme kullanmanız gerekir.
