# <a name="compiler-error-c2827"></a>Derleyici hatası C2827

Birli formla ' işleç işleci ' genel olarak geçersiz kılınamaz

İşlecin bir nesne dışında birli formu olamaz.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Aşağıdaki olası çözümleri kullanarak düzeltilmesi için

1. Aşırı yüklenmiş operatörü bir nesne için yerel yapın.

1. Aşırı yüklemek için uygun bir birli işleç seçin.
