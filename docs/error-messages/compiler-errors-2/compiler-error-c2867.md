# <a name="compiler-error-c2867"></a>Derleyici Hatası C2867
'tanımlayıcısı': bir ad değil  
  
 A `using` yönergesi için bir ad alanı dışında bir şey uygulanır.  
  
 Aşağıdaki örnek C2867 oluşturur:  
  
```  
// C2867.cpp  
// compile with: /c  
namespace N {  
   class X {};  
}  
using namespace N::X;   // C2867  
```
