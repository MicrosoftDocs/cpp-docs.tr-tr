# <a name="compiler-error-c2839"></a>Derleyici Hatası C2839
geçersiz dönüş türü 'type' için aşırı 'işleci ->'  
  
 `->` Bir sınıf, yapı, veya birleşim veya bir başvuru işleci döndürmesi gerekir.
