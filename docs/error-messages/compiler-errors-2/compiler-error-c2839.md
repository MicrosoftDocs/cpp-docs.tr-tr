# <a name="compiler-error-c2839"></a>Derleyici hatası C2839

aşırı yüklenmiş ' operator-> ' için ' Type ' dönüş türü geçersiz

`->` işleci bir sınıf, yapı veya birleşim ya da bir başvuru döndürmelidir.
