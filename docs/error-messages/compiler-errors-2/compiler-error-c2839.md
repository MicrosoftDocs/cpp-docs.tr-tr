# <a name="compiler-error-c2839"></a>Derleyici Hatası C2839

geçersiz dönüş türü 'type' için aşırı yüklenmiş 'operator ->'

`->` İşleci, bir sınıf, struct veya union veya bir başvuru döndürmelidir.
