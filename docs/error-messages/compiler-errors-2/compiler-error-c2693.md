# <a name="compiler-error-c2693"></a>Derleyici hatası C2693

' operator ': yönetilen veya WinRT dizisine yönelik başvurular için geçersiz karşılaştırma

Her türlü eşitsizlik için yönetilen veya WinRT dizisini test edilemez. Örneğin, yönetilen dizilerin eşit olup olmadığını görmek için test edebilir, ancak bir dizinin başka bir diziden daha büyük veya küçük olup olmadığını görmek için test edebilirsiniz.
