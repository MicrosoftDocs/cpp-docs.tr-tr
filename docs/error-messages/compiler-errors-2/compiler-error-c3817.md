# <a name="compiler-error-c3817"></a>Derleyici hatası C3817

' declaration ': özellik yalnızca bir işleve uygulanabilir

**`property`** Anahtar sözcüğü yalnızca bir işlev tanımına uygulanabilir.

C3817 yalnızca eski derleyici seçeneği **/clr: oldSyntax**kullanılarak erişilebilir.
