# <a name="compiler-error-c3631"></a>Derleyici hatası C3631

' function ': yönetilen veya WinRT olayları aşırı yüklenemez

Yönetilen veya WinRT olayı aşırı yüklenemez.

C3631 yalnızca eski derleyici seçeneği **/clr: oldSyntax** kullanılarak erişilebilir.
