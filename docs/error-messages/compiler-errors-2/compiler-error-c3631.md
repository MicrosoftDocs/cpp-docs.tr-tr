# <a name="compiler-error-c3631"></a>Derleyici Hatası C3631
'function': olamaz yönetilen aşırı veya WinRT olayları  
  
 Bir yönetilen veya WinRT olay olamaz aşırı yüklenebilir.  
  
 C3631 eski derleyici seçeneği kullanılarak erişilebilir, yalnızca **/clr:oldSyntax**.
