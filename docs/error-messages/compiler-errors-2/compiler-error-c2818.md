# <a name="compiler-error-c2818"></a>Derleyici hatası C2818

aşırı yüklenmiş ' operator-> ' uygulaması ' Type ' türü aracılığıyla özyinelemeli

Sınıf üyesi erişim işlecinin yeniden tanımlanması özyinelemeli bir **`return`** ifade içeriyor. `->`İşleci özyineleme ile yeniden tanımlamak için özyinelemeli yordamı işleç geçersiz kılma işlevinden çağrılan ayrı bir işleve taşımanız gerekir.
