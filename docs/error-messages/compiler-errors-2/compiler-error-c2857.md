# <a name="compiler-error-c2857"></a>Derleyici Hatası C2857
' #include ' /Ycfilename komut satırı seçeneği ile belirtilen deyim kaynak dosyasında bulunamadı  
  
 [/Yc](../../build/reference/yc-create-precompiled-header-file.md) seçeneği derlenmiş kaynak dosyasına dahil edilmeyen bir içerme dosyası adını belirtir.  
  
 Bu hatanın nedeni bir `#include` değil derlenmiş bir koşullu derleme blok deyiminde.
