# <a name="compiler-error-c3603"></a>Derleyici hatası C3603

' Symbol ': tür ' Type ' henüz desteklenmiyor

Yönetilen bir nesnede .NET çalışma zamanı tarafından henüz desteklenmeyen bir veri türü kullanmaya çalıştınız.
