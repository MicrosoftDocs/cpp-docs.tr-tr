# <a name="compiler-error-c3800"></a>Derleyici hatası C3800

' declaration ': Özellikler ve olaylar karıştırılamaz

Bir yapıyı hem bir özellik hem de bir olay olarak bildiremezsiniz.

C3800 yalnızca eski derleyici seçeneği **/clr: oldSyntax** kullanılarak erişilebilir.
