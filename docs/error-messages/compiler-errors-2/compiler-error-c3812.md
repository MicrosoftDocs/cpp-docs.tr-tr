# <a name="compiler-error-c3812"></a>Derleyici hatası C3812

' Property ' bir özellik bildirimindeki ilk belirteç olmalıdır

Bir özellik bildirirken, `__property` anahtar sözcüğü satırdaki ilk belirteç olmalıdır.

C3812 yalnızca eski derleyici seçeneği **/clr: oldSyntax** kullanılarak erişilebilir.
