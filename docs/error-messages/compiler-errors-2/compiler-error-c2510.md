# <a name="compiler-error-c2510"></a>Derleyici hatası C2510

' tanımlayıcı ': ':: ' sol tarafında bir sınıf/yapı/birleşim olmalıdır

Bir sınıf, yapı veya birleşim adı, kapsam çözümleme işleci () işlecinin sol tarafında görünmelidir `::` .
