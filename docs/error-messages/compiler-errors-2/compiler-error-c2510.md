# <a name="compiler-error-c2510"></a>Derleyici Hatası C2510

'identifier': sol '::' bir sınıf/yapı/birleşim olmalıdır

Bir sınıf, yapı veya birleşim adı kapsam çözümleme işlecinin sol tarafında yer almalıdır (`::`) işleci.
