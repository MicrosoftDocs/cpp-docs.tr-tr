# <a name="compiler-error-c3855"></a>Derleyici Hatası C3855
'class': 'param' tür parametresi, bildirimiyle uyumlu değil  
  
 Derleyici tür olmayan şablon veya farklı adlara sahip genel parametreler bulundu. Bir şablon uzmanlık tanımında belirtilen şablon parametresi bildiriminden ile uyumsuz olduğunda bu durum oluşabilir.  
  
 Aşağıdaki örnek C3855 oluşturur:  
  
```  
// C3855.cpp  
template <int N>  
struct C {  
   void f();  
};  
  
template <char N>  
void C<N>::f() {}   // C3855  
```  
  
 Olası çözüm:  
  
```  
// C3855b.cpp  
// compile with: /c  
template <int N>  
struct C {  
   void f();  
};  
  
template <int N>  
void C<N>::f() {}  
```  
  
 Ayrıca C3855 genel türler kullanma ortaya çıkabilir:  
  
```  
// C3855c.cpp  
// compile with: /clr  
generic <class T>  
ref struct GC1 {  
   generic <class U>  
   ref struct GC2;  
};  
  
generic <class T>  
generic <class U>  
generic <class V>  
ref struct GC1<T>::GC2 { };   // C3855  
```  
  
 Olası çözüm:  
  
```  
// C3855d.cpp  
// compile with: /clr /c  
generic <class T>  
ref struct GC1 {  
   generic <class U>  
   ref struct GC2;  
};  
  
generic <class T>  
generic <class U>  
ref struct GC1<T>::GC2 { };  
```
