# <a name="fatal-error-c1206"></a>Önemli hata C1206

Uygulama etki alanı başına veriler yüklü olan çalışma zamanı sürümü tarafından desteklenmiyor

Uygulama etki alanı verileri gibi bazı özellikler yalnızca özelliği destekleyen ortak dil çalışma zamanı tarafından desteklenir.

C1206, çalışma zamanının en son sürümünün bilgisayarınızda yüklü olmadığını gösterir. Derleyicinizle birlikte kullanılması amaçlanan ortak dil çalışma zamanı sürümünü yükler.

Daha fazla bilgi için bkz. [AppDomain](../../cpp/appdomain.md) .
