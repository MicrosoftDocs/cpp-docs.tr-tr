# <a name="compiler-error-c2020"></a>Derleyici hatası C2020

' Member ': ' class ' üyesi yeniden tanımı

Temel sınıf veya yapıdan devralınan bir üye yeniden tanımlandı. Devralınan Üyeler temel sınıfta olduğu sürece yeniden tanımlanamaz **`virtual`** .
