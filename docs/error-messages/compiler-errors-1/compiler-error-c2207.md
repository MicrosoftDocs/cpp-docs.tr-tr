# <a name="compiler-error-c2207"></a>Derleyici hatası C2207

' üye ': bir sınıf şablonunun üyesi bir işlev türü alamaz

`member`Sınıf şablonunun daha önce statik olmayan bir veri üyesi olarak ayrıştırılmıştı. Bir üye işlev olarak yeniden tanımlanamaz.
