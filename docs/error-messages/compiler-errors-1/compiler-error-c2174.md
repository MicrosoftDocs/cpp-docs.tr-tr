# <a name="compiler-error-c2174"></a>Derleyici hatası C2174

' function ': gerçek parametrenin türü ' void ': parametre Sayı1, parametre listesi sayı2

Parametre `number1` listesine geçirilen parametre `number2` bir **`void`** parametredir. Parametrelerin türü olamaz **`void`** . **`void*`** Bunun yerine kullanın.
