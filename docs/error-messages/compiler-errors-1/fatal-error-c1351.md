# <a name="fatal-error-c1351"></a>Önemli hata C1351
DLL 'dll' yüklenirken hata oluştu: uyumsuz sürümü  
  
 DLL yanlış sürümü bulundu.  
  
 Bu, yükleme ile ilgili bir sorun olduğunu gösterir ve ürün CD'SİNDEN DLL yeniden yüklemeniz gerekir.
