# <a name="fatal-error-c1051"></a>Önemli hata C1051

Program veritabanı dosyası, 'pdbfile' artık kullanılmıyor biçiminde silin ve yeniden derleyin sahip

Derleyici, daha eski bir sürüm numarasına sahip program veritabanı dosyası güncelleştirilemiyor. Dosyayı silin ve programınızla birlikte derleyin **/zi** veya **/zi**. Daha fazla bilgi için  [ /z7, / zi, /zı (hata ayıklama bilgileri biçimi)](../../build/reference/z7-zi-zi-debug-information-format.md)
