# <a name="fatal-error-c1051"></a>Önemli hata C1051
Program veritabanı dosyası, 'pdbfile' silin ve yeniden derleyin geçersiz bir biçime sahip  
  
 Derleyici daha eski bir sürüm numarası program veritabanı dosyası güncelleştirilemiyor. Dosyayı silin ve yeniden derleyin programınızla birlikte **/zı** veya **/zı**. Daha fazla bilgi için bkz: [/Z7, / zi, /zı (hata ayıklama bilgileri biçimi)](../../build/reference/z7-zi-zi-debug-information-format.md)
