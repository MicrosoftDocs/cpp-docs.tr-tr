# <a name="fatal-error-c1051"></a>Önemli Hata C1051

' pdbdosya ' program veritabanı dosyasında eski bir biçim bulunuyor, silin ve yeniden derleyin

Derleyici, daha eski bir sürüm numarasına sahip program veritabanı dosyasını güncelleştiremiyor. Dosyayı silin ve programınızı **/Zi** veya **/Zi**ile yeniden derleyin. Daha fazla bilgi için bkz.
