# <a name="compiler-error-c2315"></a>Derleyici Hatası C2315

'type1': başvuru tarafından yakalandı, satır numarası 'type2'

Özel durum türü, bir önceki işleyici tarafından işlenir. İkinci yakalama başvuru ilk türü vardır.
