# <a name="compiler-error-c2315"></a>Derleyici hatası C2315

' type1 ': başvuru, satır numarasında ' type2 ' tarafından yakalandı

Özel durum türü, önceki bir işleyici tarafından işlenir. İkinci catch başvurusunun türü birincsahiptir.
