# <a name="compiler-error-c2315"></a>Derleyici Hatası C2315
'type1': satır numarası 'type2' tarafından başvuru yakalandı  
  
 Özel durum türü, bir önceki işleyicisi tarafından işlenir. İkinci yakalama ilk türüne başvuruyor.
