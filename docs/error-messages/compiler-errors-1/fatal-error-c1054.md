# <a name="fatal-error-c1054"></a>Önemli hata C1054
Derleyici sınırı: başlatıcıları çok fazla iç içe  
  
 Kodu başlatıcıları (başlatılmakta türlerinin birleşimi bağlı olarak, 10-15 düzeyleri) iç içe geçirme sınırı aşıyor.  
  
### <a name="to-fix-by-using-the-following-possible-solutions"></a>Aşağıdaki olası çözümlere kullanarak düzeltmek için  
  
1.  İç içe geçme azaltmak için başlatılmakta veri türleri basitleştirin.  
  
2.  Değişkenleri ayrı deyimlerinde sonra bildirimi başlatır.
