# <a name="compiler-error-c2401"></a>Derleyici Hatası C2401

'identifier': yazmaç 'bağlam' içinde taban olmalıdır

Bu bağlamda bir temel kayıt dolaylı bellek işleneni kullanılan kaydı olmalıdır.
