# <a name="compiler-error-c2044"></a>Derleyici Hatası C2044

Geçersiz devam etme

A [devam](../../cpp/continue-statement-cpp.md) yalnızca dahilinde geçerli olan bir `do`, `for`, veya `while` deyimi.
