# <a name="compiler-error-c2372"></a>Derleyici Hatası C2372
'tanımlayıcısı': şemadaki; farklı tür yöneltmesi  
  
 Tanımlayıcı ile farklı türetilmiş bir tür zaten tanımlandı.  
  
 Aşağıdaki örnek C2326 oluşturur:  
  
```  
// C2372.cpp  
// compile with: /c  
extern int *fp;  
extern int fp[];   // C2372  
extern int fp2[];   // OK  
```
