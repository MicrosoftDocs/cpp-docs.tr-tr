# <a name="compiler-error-c2145"></a>Derleyici hatası C2145

sözdizimi hatası: tanımlayıcıdan önce ' token ' eksik

Derleyici beklenen `token` ve tanımlayıcı buldu.

Bu hata, bir bloktaki son bildirimden sonra eksik bir noktalı virgülden kaynaklanıyor olabilir.
