# <a name="fatal-error-c1038"></a>Önemli hata C1038

Derleyici sınırı: işlev: denetim akışı durumu çok karmaşık; basitleşme işlevi

İşlevi derleyicinin işleyebileceğinden daha fazla denetim akışı durumuna sahiptir. Denetim akışını basitleştirecek veya işlevi daha küçük işlevlere böyor.
