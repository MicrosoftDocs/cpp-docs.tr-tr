# <a name="fatal-error-c1038"></a>Önemli hata C1038
Derleyici sınırı: işlevi: denetim akışı durumu çok karmaşık; basitleştirmek işlevi  
  
 İşlev derleyici işleyebileceğinden daha fazla denetim akışı durumlar vardır. Denetim akışı basitleştirin veya daha küçük işlevlerini split işlevi.
