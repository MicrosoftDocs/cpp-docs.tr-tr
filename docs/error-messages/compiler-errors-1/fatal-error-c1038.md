# <a name="fatal-error-c1038"></a>Önemli hata C1038

Derleyici sınırı: işlev: denetim akışı durumu çok karmaşık; işlevi basitleştirin

İşlev derleyici işleyebileceğinden daha fazla denetim akışı durumlar vardır. Denetim akışı basitleştirin veya işlev daha küçük işlevlere bölün.
