# <a name="compiler-error-c2173"></a>Derleyici Hatası C2173
'function': Gerçek parametresi bir işaretçi değil: parametre Sayı1, parametre listesi sayı2  
  
 Parametre `number1` parametre listesine geçirilen `number2` bir işaretçi değil. İşlev işaretçisi bekliyor.
