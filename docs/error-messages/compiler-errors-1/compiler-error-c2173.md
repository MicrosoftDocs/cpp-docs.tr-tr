# <a name="compiler-error-c2173"></a>Derleyici hatası C2173

' function ': gerçek parametre bir işaretçi değil: parametre Sayı1, parametre listesi sayı2

Parametre listesine geçirilen parametre `number1` `number2` bir işaretçi değil. İşlev bir işaretçi bekliyor.
