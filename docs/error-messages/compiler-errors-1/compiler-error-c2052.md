# <a name="compiler-error-c2052"></a>Derleyici Hatası C2052
'type': case deyimi için geçersiz tür  
  
 Case ifadeleri tamsayı olmalıdır.  
  
 Aşağıdaki örnek C2052 oluşturur:  
  
```  
// C2052.cpp  
int main() {  
   int index = 0;  
   switch (index) {  
      case 1:  
         return 24;  
      case 1.0:   // C2052  
      // try the following line instead  
      // case 2:  
         return 23;  
   }  
}  
```
