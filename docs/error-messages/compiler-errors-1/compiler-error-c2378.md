# <a name="compiler-error-c2378"></a>Derleyici Hatası C2378
'tanımlayıcısı': şemadaki; simge ile typedef aşırı yüklenemez  
  
 Tanımlayıcı olarak tanımlandığından bir `typedef`.  
  
 Aşağıdaki örnek C2378 oluşturur:  
  
```  
// C2378.cpp  
// compile with: /c  
int i;  
typedef int i;   // C2378  
typedef int b;   // OK  
```
