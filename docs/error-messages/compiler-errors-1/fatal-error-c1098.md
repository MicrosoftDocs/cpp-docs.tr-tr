# <a name="fatal-error-c1098"></a>Önemli hata C1098

Düzenle ve devam et altyapısında sürüm uyuşmazlığı

Kullanmakta olduğunuz hata ayıklayıcı sürümü, yürütülebilir dosyayı oluşturmak için kullanılan derleyici ile eşleşmiyor. Yeniden derleme sorunu çözmezse, hata ayıklayıcının ve derleyicinin doğru sürümlerine sahip olduğunuzdan emin olmak için Visual C++ yeniden yüklemeniz gerekebilir.
