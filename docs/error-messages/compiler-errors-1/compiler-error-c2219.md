# <a name="compiler-error-c2219"></a>Derleyici hatası C2219

sözdizimi hatası: tür niteleyicisi ' * ' sonunda olmalıdır

Tür niteleyicisi ( **`const`** veya **`volatile`** ) izin verilmeyen yerde görünür.
