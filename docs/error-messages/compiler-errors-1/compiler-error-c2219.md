# <a name="compiler-error-c2219"></a>Derleyici Hatası C2219

sözdizimi hatası: tür niteleyicisi karakterinden son olmalıdır ' *'

Tür niteleyicisi (`const` veya `volatile`) verilmez burada görünür.
