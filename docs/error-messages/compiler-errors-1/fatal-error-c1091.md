# <a name="fatal-error-c1091"></a>Önemli hata C1091

Derleyici sınırı: dize uzunluğu ' length ' baytı aşıyor

Dize sabiti, dizelerin uzunluğu için geçerli sınırı aştı.

Statik dizeyi iki (veya daha fazla) değişkene bölmek ve [strcpy_s](../../c-runtime-library/reference/strcpy-s-wcscpy-s-mbscpy-s.md) kullanarak sonucu bildirimin bir parçası olarak veya çalışma zamanında birleştirmek isteyebilirsiniz.
