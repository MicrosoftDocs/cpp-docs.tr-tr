# <a name="compiler-error-c2022"></a>Derleyici Hatası C2022

'number': karakter için çok büyük

Sekizlik sayı bir ters eğik çizgiden (\\) bir karakter veya dize sabit değeri bir karakteri temsil etmesi için çok büyük.
