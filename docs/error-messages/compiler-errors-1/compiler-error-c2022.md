# <a name="compiler-error-c2022"></a>Derleyici hatası C2022

' number ': karakter için çok büyük

Bir karakter veya dize sabitinde ters eğik çizgi (\\) izleyen sekizlik sayı, bir karakteri temsil etmek için çok büyük.
