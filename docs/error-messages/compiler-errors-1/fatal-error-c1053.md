# <a name="fatal-error-c1053"></a>Önemli hata C1053

' \<identifier> ': işlev çok büyük

İşlev derlemek için çok büyük.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Aşağıdaki olası çözümleri kullanarak düzeltilmesi için

1. İyileştirmeler olmadan derlemeyi deneyin.

1. İşlevi daha küçük işlevlere bölün.

1. Satır içi işlevlere yapılan çağrıları azaltır.
