# <a name="fatal-error-c1008"></a>Önemli hata C1008
Giriş dosyası belirtilmedi  
  
 Derleyici derlemek için C veya C++ kaynak dosyası verilmedi. Denetleme **CL** komut satırı ve filename belirtimleri için ortam değişkeni.
