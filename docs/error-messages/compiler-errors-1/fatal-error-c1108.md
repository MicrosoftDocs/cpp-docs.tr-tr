# <a name="fatal-error-c1108"></a>Önemli hata C1108
DLL bulunamadı: 'dll'  
  
 Belirtilen DLL (***dll***) yolu bulunamadı.  
  
 Bu hatayı gidermek için Visual C++ yeniden yükleyin veya uygun .dll dosyası yükleme CD'SİNDEN bilgisayarınıza kopyalayın.
