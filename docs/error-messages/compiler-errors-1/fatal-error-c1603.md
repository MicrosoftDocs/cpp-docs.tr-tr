# <a name="fatal-error-c1603"></a>Önemli hata C1603

satır içi derleme dalı hedefi, Aralık dışında ' number ' bayt

JCXZ veya JECXZ yönergesi arasındaki hesaplanan uzaklık ve belirtilen hedef etiketi 128 bayttan büyük. Etiketi yönergeye yaklaştırmak için kodunuzu güncelleştirin.
