# <a name="compiler-error-c2206"></a>Derleyici Hatası C2206

'function': typedef işlev tanımı için kullanılamaz

A `typedef` bir işlev türü tanımlamak için kullanılır.

Aşağıdaki örnek, C2206 oluşturur:

```
// C2206.cpp
typedef int functyp();
typedef int MyInt;
functyp func1 {};   // C2206
int main() {
   MyInt i = 0;   // OK
}
```
