# <a name="fatal-error-c1046"></a>Önemli Hata C1046

Derleyici sınırı: yapı çok derin iç içe

Yapı, birleşim veya sınıf, 15 düzey olan iç içe geçmiş sınırı aştı. İç içe geçme düzeyini azaltmak için tanımı yeniden yazın. **`typedef`** İç içe bir veya daha fazla yapı tanımlamak için kullanarak yapıyı, birleşimi veya sınıfı iki veya daha fazla parçaya ayırın.
