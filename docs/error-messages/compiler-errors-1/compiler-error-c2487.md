# <a name="compiler-error-c2487"></a>Derleyici Hatası C2487

'identifier': dll arabirimi sınıfının üyesi dll arabirimi ile bildirilemez

Tüm sınıf veya belirli üyelere, DLL arabirimi olmayan DLL arabirimi sınıfının bildirebilirsiniz. DLL arabirimi ile bir sınıf bildirme ve ardından bir üyesi DLL arabirimi ile söz konusu sınıfın bildirin.
