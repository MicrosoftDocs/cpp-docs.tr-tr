# <a name="compiler-error-c2403"></a>Derleyici Hatası C2403

'identifier': yazmaç 'context' içinde taban/dizin olmalıdır

Bu bağlamda bir temel ya da dizin kaydı dolaylı bellek işleneni kullanılan kaydı olmalıdır.
