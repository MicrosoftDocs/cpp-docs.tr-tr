# <a name="fatal-error-c1023"></a>Önemli hata C1023

' dosya ': beklenmeyen PCH hatası, PCH 'yi yeniden oluşturmayı deneyin

C1023 bazı sorunlardan biri nedeniyle, ön derlenmiş üstbilgi dosyasının yeniden derlenme çözümüdür.

C1023, önceden derlenmiş bir üst bilgi dosyasını, oluşturulduğu sunucudan farklı bir makinede kullanmaya çalıştığınızda oluşmuş olabilir.
