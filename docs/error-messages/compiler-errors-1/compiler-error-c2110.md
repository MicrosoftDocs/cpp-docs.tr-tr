# <a name="compiler-error-c2110"></a>Derleyici Hatası C2110
'+': iki işaretçisi ekleyemezsiniz  
  
 Artı kullanarak iki işaretçi değerleri eklemek için girişimde bulunuldu ( `+` ) işleci.  
  
 Aşağıdaki örnek C2110 oluşturur:  
  
```  
// C2110.cpp  
int main() {  
   int a = 0;  
   int *pa;  
   int *pb;  
   a = pa + pb;   // C2110  
}  
```
