# <a name="compiler-error-c2170"></a>Derleyici hatası C2170

' tanımlayıcı ': bir işlev olarak bildirilmemiş, iç olamaz

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aşağıdaki olası nedenleri denetleyerek onarmak için

1. Pragma `intrinsic`, bir işlev dışında bir öğe için kullanılır.

1. Pragma `intrinsic`, iç formu olmayan bir işlev için kullanılır.
