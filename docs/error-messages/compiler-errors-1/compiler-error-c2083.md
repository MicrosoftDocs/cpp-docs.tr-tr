# <a name="compiler-error-c2083"></a>Derleyici hatası C2083

struct/Union karşılaştırması geçersizdir

Bir yapı veya birleşim, başka bir Kullanıcı tanımlı türle doğrudan karşılaştırılır. Bir karşılaştırma işleci tanımlanmadıysa veya skaler bir türe dönüştürme yoksa buna izin verilmez.
