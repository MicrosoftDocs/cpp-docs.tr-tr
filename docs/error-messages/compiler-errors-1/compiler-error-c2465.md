# <a name="compiler-error-c2465"></a>Derleyici Hatası C2465
Parantez içindeki anonim bir tür tanımlanamıyor  
  
 Anonim yapısı, UNION veya numaralandırılmış türü parantez içinde bir ifade içinde tanımlanır. Tanımı işlevi kapsamında anlamsız olduğundan bu C++'da geçersiz.
