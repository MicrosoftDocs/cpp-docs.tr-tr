# <a name="compiler-error-c2465"></a>Derleyici hatası C2465

parantez içinde anonim bir tür tanımlanamaz

Anonim bir yapı, birleşim veya numaralandırılmış tür, parantez içinde tanımlanır. Bu, tanım işlev kapsamında anlamlı olmadığından, C++ içinde geçersizdir.
