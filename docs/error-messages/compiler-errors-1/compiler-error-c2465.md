# <a name="compiler-error-c2465"></a>Derleyici Hatası C2465

bir anonim tür parantez içinde tanımlanamaz

Bir anonim bir yapı, birlik veya numaralandırılmış tür parantez içinde bir ifade içinde tanımlanır. İşlev kapsamında tanımı anlamsız olduğu için bu c++'ta geçersiz.
