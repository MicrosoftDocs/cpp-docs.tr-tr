# <a name="fatal-error-c1014"></a>Önemli hata C1014

çok fazla içerme dosyası: derinlik = düzey

`#include` yönergelerinin iç içe geçirilmesi çok derin. İç içe geçmiş yönergeler açık dosya içerebilir. Yönergeyi içeren kaynak dosya bir dosya olarak sayılır.
