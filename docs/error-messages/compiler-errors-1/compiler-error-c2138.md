# <a name="compiler-error-c2138"></a>Derleyici Hatası C2138
Numaralandırma herhangi bir üye olmadan tanımlamak için geçersiz  
  
 Numaralandırma en az bir üye olması gerekir, /Za (Microsoft uzantıları devre dışı bırak) seçilidir.
