# <a name="compiler-error-c2138"></a>Derleyici Hatası C2138

Hiç üyesi olmayan bir numaralandırma tanımlamak geçersizdir

Bir sabit listesi en az bir üye olmalıdır, /Za (Microsoft uzantılarını devre dışı) seçilir.
