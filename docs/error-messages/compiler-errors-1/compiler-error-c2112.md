# <a name="compiler-error-c2112"></a>Derleyici Hatası C2112

'-': işaretçi çıkarma integral veya işaretçi işlenen gerekir

Diğer türlere işaret eden işaretçilerin çıkarılması için girişimde bulunuldu.
