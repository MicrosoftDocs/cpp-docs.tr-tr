# <a name="fatal-error-c1602"></a>Önemli hata C1602

Desteklenmeyen iç öğe

Bu hata, bir eksik yükleme nedeniyle oluşmuş olan derleyici. exe dosyalarınızda bir uyuşmazlığın olduğunu gösterir. Örneğin, Işlemci paketini değil, bir hizmet paketi yüklemiş olabilirsiniz. Tüm gerekli ürünleri yükler.
