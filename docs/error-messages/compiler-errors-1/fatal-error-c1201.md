# <a name="fatal-error-c1201"></a>Önemli hata C1201

sınıf şablonu tanımındaki sözdizimi hatasından sonra devam edilemiyor

Sınıf şablonu tanımı ayrıştırılırken beklenmeyen bir hata oluştu.

Diğer hataları giderin ve yeniden derleyin. Bu başarısız olursa, hatanın durumunu unutmayın, sorunu yalıtmak ve tekrarlanabilir bir test çalışması oluşturmak için [Microsoft Ürün Destek Hizmetleri](/visualstudio/ide/talk-to-us)'ne başvurun.
