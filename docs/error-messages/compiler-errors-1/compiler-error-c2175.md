# <a name="compiler-error-c2175"></a>Derleyici hatası C2175

' locale ': geçersiz yerel ayar

Belirtilen yerel ayar geçerli değil. Desteklenen yerel ayarlar için *çalışma zamanı kitaplık başvurusundaki* [dil ve ülke/bölge dizelerine](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) bakın.
