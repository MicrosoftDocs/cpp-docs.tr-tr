# <a name="compiler-error-c2175"></a>Derleyici Hatası C2175

Belirtilen yerel ayarı geçerli değil. Bkz: [dil ve ülke/bölge dizeleri](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) içinde *çalışma zamanı kitaplığı başvurusu* desteklenen yerel ayarlar.
