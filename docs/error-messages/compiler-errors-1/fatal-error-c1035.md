# <a name="fatal-error-c1035"></a>Önemli hata C1035

ifade çok karmaşık; ifadeyi Basitleştir

Derleyici karmaşık bir ifade için kod oluşturamadı. İfadeyi daha basit ifadelerle ayırın ve yeniden derleyin.
