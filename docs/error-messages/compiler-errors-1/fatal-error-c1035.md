# <a name="fatal-error-c1035"></a>Önemli hata C1035

ifade çok karmaşık; ifadesini Basitleştir

Derleyici karmaşık bir ifade için kod üretilemedi. Daha basit ifadeler ifadesine bölme ve yeniden derleyin.
