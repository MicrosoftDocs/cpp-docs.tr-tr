# <a name="compiler-error-c2155"></a>Derleyici hatası C2155

'? ': geçersiz sol işlenen, aritmetik veya işaretçi türü bekleniyordu

`?` sol tarafındaki bir ifade sıfıra karşılaştırılamaz. Sıfıra karşılaştırılabilmeniz için bir aritmetik veya işaretçi ifadesi kullanmanız gerekir.
