# <a name="compiler-error-c2223"></a>Derleyici Hatası C2223
'tanımlayıcısı ->' ın sol struct/union işaret etmesi gerekir  
  
 İşlenen solundaki `->` bir işaretçi bir sınıf, yapı veya birleşim değil.  
  
 Tanımlanmamış bir değişkeni, bir sol işleneni tarafından bu hataya neden olabilir (Bu nedenle yazın `int`).
