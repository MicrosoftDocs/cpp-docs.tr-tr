# <a name="compiler-error-c2223"></a>Derleyici Hatası C2223

'tanımlayıcı '->' ın sol yapıya/birleşime işaret etmelidir

İşlenen solundaki `->` bir sınıf, yapı veya birleşim işaretçisi değil.

Bu hata, tanımlanmamış bir değişkene bir sol işlenen tarafından neden olabilir (Bu nedenle yazın `int`).
