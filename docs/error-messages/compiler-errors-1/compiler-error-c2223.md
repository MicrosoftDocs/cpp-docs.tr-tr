# <a name="compiler-error-c2223"></a>Derleyici hatası C2223

'-> Identifier ' sol tarafı struct/Union öğesine işaret etmelidir

`->` solundaki işlenen, bir sınıf, yapı veya birleşim işaretçisi değil.

Bu hata, tanımsız bir değişken (Bu nedenle `int`) olan bir sol işlenen tarafından neden olmuş olabilir.
