# <a name="compiler-error-c2223"></a>Derleyici hatası C2223

'->Identifier ' sol tarafı struct/Union öğesine işaret etmelidir

Solundaki işlenen, bir `->` sınıfa, yapıya veya birleşime yönelik bir işaretçi değildir.

Bu hata, tanımsız bir değişken (Bu nedenle tür) olan bir sol işlenen nedeniyle oluşabilir **`int`** .
