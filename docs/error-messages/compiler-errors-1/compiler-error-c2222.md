# <a name="compiler-error-c2222"></a>Derleyici hatası C2222

Beklenmeyen tür ' Type ': bir taban sınıf veya üye bekleniyordu

Başlatıcı listesi yalnızca temel sınıfları veya bir türün üyelerini başlatabilir. Bu hatayı onarmak için, başlatıcı listesinde yalnızca temel sınıfların veya türdeki üyelerin başlatıldığını doğrulayın.
