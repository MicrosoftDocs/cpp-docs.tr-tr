# <a name="fatal-error-c1057"></a>Önemli hata C1057

makro genişletilirken beklenmeyen dosya sonu

Büyük olasılıkla makro çağrısında eksik sağ parantez nedeniyle, derleyici, makro çağırma bağımsız değişkenlerini toplarken kaynak dosyanın sonuna ulaştı.
