# <a name="compiler-error-c2307"></a>Derleyici Hatası C2307
Artımlı derleme etkinleştirilirse pragma 'pragma' dışında bir işlev olmalıdır  
  
 Yerleştirmeniz gerekir `data_seg` pragma Artımlı derleme kullanıyorsanız işlevleri arasında.
