# <a name="fatal-error-c1045"></a>Önemli hata C1045

Derleyici sınırı: bağlantı belirtimleri çok derin iç içe

İç içe dışlar, derleyici sınırını aşıyor. İç içe dışlar, `extern` "C++" gibi dış bağlantı türüyle izin verilir. Hatayı çözümlemek için iç içe dışlar sayısını azaltın.
