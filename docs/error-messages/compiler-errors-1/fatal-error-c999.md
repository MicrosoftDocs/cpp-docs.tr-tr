# <a name="fatal-error-c999"></a>Önemli hata C999

BILINMEYEN ILETI lütfen Visual C++ Yardım menüsünde Teknik destek komutunu seçin veya daha fazla bilgi için teknik destek yardım dosyasını açın

Bu hata genellikle derleyicinin farklı sürümlerindeki karışık dosyalarınıza sahip olduğunuz veya derleyici yüklemenizin bozuk olduğu anlamına gelir. Ürünü onarmak ya da yeniden yüklemek için Denetim Masası 'ndaki **Programlar ve Özellikler** uygulamasını kullanın.
