# <a name="fatal-error-c1055"></a>Önemli hata C1055

Derleyici sınırı: anahtar kalmadı

Kaynak dosyası çok fazla sembol içeriyor. Derleyicinin karma anahtarları sembol tablosu için yetersiz.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Aşağıdaki olası çözümleri kullanarak düzeltmek için

1. Kaynak dosya, daha küçük dosyalara bölün.

1. Gereksiz üst bilgi dosyaları kaldırın.

1. Yeni bir tane oluşturmak yerine geçici ve genel değişkenler yeniden kullanın.
