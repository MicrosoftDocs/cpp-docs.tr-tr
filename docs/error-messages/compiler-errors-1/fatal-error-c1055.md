# <a name="fatal-error-c1055"></a>Önemli hata C1055
Derleyici sınırı: anahtarları dışında  
  
 Kaynak dosyası çok fazla simgeler içeriyor. Sembol tablosuna karma tuşları dışında derleyici verdi.  
  
### <a name="to-fix-by-using-the-following-possible-solutions"></a>Aşağıdaki olası çözümlere kullanarak düzeltmek için  
  
1.  Kaynak dosyanın daha küçük dosyalara bölebilirsiniz.  
  
2.  Gereksiz üstbilgi dosyaları ortadan kaldırır.  
  
3.  Yeni bir tane oluşturmak yerine geçici ve genel değişkenler yeniden kullanabilirsiniz.
