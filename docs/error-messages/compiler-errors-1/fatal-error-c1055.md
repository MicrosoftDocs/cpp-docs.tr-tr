# <a name="fatal-error-c1055"></a>Önemli Hata C1055

Derleyici sınırı: anahtar kalmadı

Kaynak dosya çok fazla sembol içeriyor. Derleyicinin sembol tablosu için karma anahtarları kalmadı.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Aşağıdaki olası çözümleri kullanarak düzeltilmesi için

1. Kaynak dosyayı daha küçük dosyalara Böl.

1. Gereksiz üst bilgi dosyalarını kaldırın.

1. Yeni bir tane oluşturmak yerine geçici ve genel değişkenleri yeniden kullanın.
