# <a name="compiler-error-c2028"></a>Derleyici Hatası C2028

Yapı/birleşim üyesi bir yapı/birleşim içinde olmalıdır

Yapı veya birleşim üyeleri, yapı veya birleşim içinde bildirilmelidir.
