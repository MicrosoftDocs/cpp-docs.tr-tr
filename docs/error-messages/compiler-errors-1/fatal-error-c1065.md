# <a name="fatal-error-c1065"></a>Önemli hata C1065
Derleyici sınırı: etiketler dışında  
  
 Kaynak dosya birden fazla 65,523 sınıflar, yapılar, birleşimler, ad alanları veya numaralandırmaları içerir.  
  
 Dosyayı daha küçük dosyalar bölün.
