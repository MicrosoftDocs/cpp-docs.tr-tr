# <a name="fatal-error-c1065"></a>Önemli hata C1065

Derleyici sınırı: Etiket kalmadı

Kaynak dosyası birden fazla 65,523 sınıflar, yapılar, birleşimler, ad alanları veya sabit listeleri içerir.

Dosya, daha küçük dosyalara bölün.
