# <a name="compiler-error-c2301"></a>Derleyici hatası C2301

'-> ~ Identifier ' sol tarafı sınıfa/yapıya/birleşime işaret etmelidir

`->` işlecinin solundaki ifade, bir sınıfa, yapıya veya birleşime yönelik bir işaretçi olarak değerlendirmez.
