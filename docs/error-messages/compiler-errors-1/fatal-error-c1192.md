\#kullanarak 'dosya' üzerinde başarısız oldu  
  
 Yalnızca Microsoft Ara dili (MSIL) biçiminde bir dosya için aktarılabilecek bir [#using](../../preprocessor/hash-using-directive-cpp.md) yönergesi. [/CLR](../../build/reference/clr-common-language-runtime-compilation.md) derleyici seçeneği bir MSIL çıktı dosyası oluşturmanıza olanak sağlar. Diğer Visual Studio dilleri de MSIL dosyaları üretir.
