# <a name="compiler-error-c2066"></a>Derleyici hatası C2066

işlev türüne dönüştürme geçersizdir

ANSI C 'de, bir işlev işaretçisi ve bir veri işaretçisi arasında dönüştürme yapmak geçerli değildir.
