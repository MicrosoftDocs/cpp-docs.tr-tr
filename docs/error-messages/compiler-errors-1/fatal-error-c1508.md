# <a name="fatal-error-c1508"></a>Önemli hata C1508
Derleyici sınırı: 'function': birden fazla 65535 bağımsız değişkeni bayt  
  
 Çalışması biçimsel parametresi 65535 bayt sınırını aşıyor.
