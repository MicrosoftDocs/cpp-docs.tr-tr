# <a name="compiler-error-c2106"></a>Derleyici Hatası C2106
'işleci': sol işleneni, l-değeri olmalıdır  
  
 İşleci, kendi sol işleneni olarak l-değeri olması gerekir.  
  
 Aşağıdaki örnek C2106 oluşturur:  
  
```  
// C2106.cpp  
int main() {  
   int a;  
   1 = a;   // C2106  
   a = 1;   // OK  
}  
```
