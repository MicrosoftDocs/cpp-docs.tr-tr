# <a name="compiler-error-c2212"></a>Derleyici Hatası C2212

'identifier': işlev işaretçileri için __based kullanılamaz

İşlev işaretçileri bildirilemez `__based`. Kod tabanlı veri ihtiyacınız varsa, `__declspec` anahtar sözcüğü veya `data_seg` pragması.
