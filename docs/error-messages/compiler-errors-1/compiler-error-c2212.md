# <a name="compiler-error-c2212"></a>Derleyici Hatası C2212
'tanımlayıcısı': __based işlev işaretçileri için kullanılabilir değil  
  
 İşlev işaretçileri bildirilemez `__based`. Kod tabanlı veri gerekiyorsa kullanın `__declspec` anahtar sözcüğü veya `data_seg` pragması.
