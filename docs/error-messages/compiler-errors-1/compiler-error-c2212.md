# <a name="compiler-error-c2212"></a>Derleyici hatası C2212

' tanımlayıcı ': işlev işaretçileri için __based kullanılamaz

İşlevlere işaretçiler bildirilemez **`__based`** . Kod tabanlı verilere ihtiyacınız varsa, **`__declspec`** anahtar sözcüğünü veya `data_seg` pragmayı kullanın.
