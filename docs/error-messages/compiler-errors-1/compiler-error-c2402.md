# <a name="compiler-error-c2402"></a>Derleyici Hatası C2402

'identifier': yazmaç 'bağlamında' dizin olmalıdır

Dolaylı bellek işleneni kullanılan kaydı bu bağlamda bir dizin yazmacı var olması gerekir.
