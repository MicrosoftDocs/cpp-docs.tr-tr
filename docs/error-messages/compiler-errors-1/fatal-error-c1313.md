# <a name="fatal-error-c1313"></a>Önemli hata C1313

Derleyici sınırı: tür blokları, sayı düzeylerinden daha derin iç içe olamaz

Özel durum işleme (veya yapılandırılmış özel durum işleme) blokları çok derin iç içe geçirildi.  Kodunuzu kolaylaştırın.
