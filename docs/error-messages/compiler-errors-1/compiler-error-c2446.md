# <a name="compiler-error-c2446"></a>Derleyici hatası C2446

' operator ': ' type1 ' öğesinden ' type2 ' öğesine dönüştürme yok

Derleyici `type1` `type2`dönüştüremiyor. Dönüştürme, C/C++ semantiğini ihlal ettiğinden anlamlı olmayabilir.
