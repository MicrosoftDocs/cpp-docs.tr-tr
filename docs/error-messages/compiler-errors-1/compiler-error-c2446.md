# <a name="compiler-error-c2446"></a>Derleyici Hatası C2446
'işleci': 'type1' öğesinden 'type2' dönüştürme  
  
 Derleyici dönüştürülemiyor `type1` için `type2`. C/C++ semantiği ihlal ettiğinden dönüştürme anlamı olmayabilir.
