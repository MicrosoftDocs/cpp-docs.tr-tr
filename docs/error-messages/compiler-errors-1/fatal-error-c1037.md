# <a name="fatal-error-c1037"></a>Önemli hata C1037

nesne dosyası dosya adı açılamıyor

[/Fo](../../build/reference/fo-object-file-name.md) tarafından belirtilen nesne dosyası açılamıyor.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aşağıdaki olası nedenleri denetleyerek onarmak için

1. Geçersiz dosya adı.

1. Dosyayı açmak için yeterli bellek yok.

1. Dosya başka bir işlem tarafından kullanılıyor.

1. Salt okunabilir bir dosya aynı ada sahip.
