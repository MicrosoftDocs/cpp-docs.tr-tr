# <a name="fatal-error-c1210"></a>Önemli hata C1210
/ CLR: pure ve/CLR: safe yüklü çalışma zamanı sürümü tarafından desteklenmiyor  
  
 **/CLR: pure** ve **/CLR: safe** derleyici seçenekleri Visual Studio 2015'te kullanım dışı.  
  
 Geçerli sürümde, ancak bir ortak dil çalışma zamanı önceki bir sürümünden bir derleyici olduğunda C1210 oluşur.  
  
 Bazı işlevler Derleyici önceki bir çalışma zamanı sürümünde çalışmayabilir.  
  
 C1210 gidermek için derleyici kullanmaya yönelik ortak dil çalışma zamanı sürümü yükleyin.
