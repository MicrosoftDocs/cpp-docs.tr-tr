# <a name="fatal-error-c1210"></a>Önemli hata C1210

> /clr: Pure ve/clr: Safe yüklü olan çalışma zamanı sürümü tarafından desteklenmiyor

**/Clr: Pure** ve **/clr: Safe** derleyici seçenekleri Visual Studio 2015 ' de kullanımdan kaldırılmıştır ve Visual Studio 2017 ' de desteklenmez.

C1210, geçerli sürüm için bir derleyicisini, ancak önceki sürümden ortak dil çalışma zamanını kullandığınızda oluşur.

Derleyicinin bazı işlevleri çalışma zamanının önceki bir sürümünde çalışmayabilir.

C1210 çözümlemek için, derleyicinizle birlikte kullanılmak üzere tasarlanan ortak dil çalışma zamanı sürümünü yüklemek için.
