# <a name="compiler-error-c2121"></a>Derleyici Hatası C2121

' # ': geçersiz karakter: olasılıkla bir makro genişletmesinin sonucu

Geçersiz bir # karakteri, dize haline getirme işleci (# #) yerine belirteç yapıştırma işlecini (# #) kullanan yanlış bir makro tarafından eklenmiş olabilir.
