# <a name="compiler-error-c2320"></a>Derleyici Hatası C2320
Beklenen ':' erişim belirticisi 'tanımlayıcısı' izlemek için  
  
 Anahtar sözcüğü `public`, `protected`, veya `private` üste gelmelidir.
