# <a name="compiler-error-c2309"></a>Derleyici Hatası C2309

catch işleyicisi parantez içine alınmış bir özel durum bildirimi bekleniyor.

Bir catch işleyicisi parantez içinde tür yok.

Aşağıdaki örnek, C2309 oluşturur:

```
// C2309.cpp
// compile with: /EHsc
#include <eh.h>
class C {};
int main() {
   try {
      throw "ooops!";
   }
   catch C {}   // C2309
   // try the following line instead
   // catch( C ) {}
}
```
