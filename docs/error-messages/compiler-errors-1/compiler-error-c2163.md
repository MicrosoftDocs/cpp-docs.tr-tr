# <a name="compiler-error-c2163"></a>Derleyici hatası C2163

' function ': iç işlev olarak kullanılamaz

Bir `intrinsic` veya `function` pragma iç formda kullanılamayan bir işlevi listeler. Örneğin,/CLR programlama kullanan bir program derlenirken bazı iç bilgiler kullanılamaz.
