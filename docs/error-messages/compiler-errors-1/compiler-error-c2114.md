# <a name="compiler-error-c2114"></a>Derleyici Hatası C2114

'operator': işaretçi solda; sağda tam sayı değer

Sol işleneni `operator` bir işaretçi olduğu, sağ işlenen bir tamsayı değeri olması gerekir.
