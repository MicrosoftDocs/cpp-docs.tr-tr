# <a name="compiler-error-c2114"></a>Derleyici hatası C2114

' operator ': sol taraftaki işaretçi; sağ tarafta tamsayı değeri gerekiyor

`operator` sol işleneni bir işaretçiydi, bu nedenle sağ işlenen bir tamsayı değeri olmalıdır.
