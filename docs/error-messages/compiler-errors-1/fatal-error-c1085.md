# <a name="fatal-error-c1085"></a>Önemli hata C1085

Dosya türü dosyası yazılamıyor: ' dosya ': ileti

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aşağıdaki olası nedenleri denetleyerek onarmak için

1. Sürücü salt okunurdur.

1. Sürücü dolu.

1. Paylaşım ihlali.

1. İleti "hatalı dosya numarası" ise, arka planda derlenirken dosya ön planda kapanıyor olabilir.
