# <a name="fatal-error-c1208"></a>Önemli hata c1208

Yığında başvuru sınıfları ayırma yüklü olan çalışma zamanı sürümü tarafından desteklenmiyor

C1208, geçerli sürüm için bir derleyicisini, ancak önceki sürümden ortak dil çalışma zamanını kullandığınızda oluşur.

Derleyicinin bazı işlevleri çalışma zamanının önceki bir sürümünde çalışmayabilir.

Derleyicinizle birlikte kullanılması amaçlanan ortak dil çalışma zamanı sürümünü yükler.
