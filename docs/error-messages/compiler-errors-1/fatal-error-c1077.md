# <a name="fatal-error-c1077"></a>Önemli hata C1077
Derleyici sınırı: olamaz sahip birden fazla sayı komut satırı seçenekleri  
  
 Komut satırı seçenekleri sayısı sınırı aşıyor.  
  
 Çok fazla simgeler ile tanımlanmış olması [/D](../../build/reference/d-preprocessor-definitions.md). (Tanımları bunun yerine bir üstbilgi dosyasına yerleştirin.)
