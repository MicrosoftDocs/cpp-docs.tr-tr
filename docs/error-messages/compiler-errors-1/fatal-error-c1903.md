# <a name="fatal-error-c1903"></a>Önemli hata C1903
Önceki hataları kurtarmak eklenemiyor; derleme durdurma  
  
 Derleyici devam etmek için çok fazla hata bulundu. Hataları düzeltin ve yeniden derleyin.
