# <a name="compiler-error-c2185"></a>Derleyici hatası C2185

' tanımlayıcı ': geçersiz tabanlı ayırma

Bir yazmaç değişkeni veya otomatik (yerel) değişken `__based`olarak bildirilmiştir. Yalnızca genel değişkenler `__based`olarak bildirilebilecek.
