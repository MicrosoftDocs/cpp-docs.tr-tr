# <a name="compiler-error-c2185"></a>Derleyici hatası C2185

' tanımlayıcı ': geçersiz tabanlı ayırma

Bir yazmaç değişkeni veya otomatik (yerel) değişken bildirilmiştir **`__based`** . Yalnızca genel değişkenler bildirilebilecek **`__based`** .
