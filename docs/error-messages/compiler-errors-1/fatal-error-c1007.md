# <a name="fatal-error-c1007"></a>Önemli hata C1007

Tanınmayan bayrak dizesinde seçeneği

Komut satırı seçeneğini geçersiz bir dize içeriyor. Denetleme **CL** komut satırı ve hatalar için ortam değişkeni.
