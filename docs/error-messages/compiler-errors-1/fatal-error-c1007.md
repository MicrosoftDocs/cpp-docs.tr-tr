# <a name="fatal-error-c1007"></a>Önemli hata C1007

seçeneğinde Tanınmayan bayrak dizesi

Komut satırı seçeneği geçersiz bir dize içeriyor. Hata için **CL** komut satırını ve ortam değişkenini denetleyin.
