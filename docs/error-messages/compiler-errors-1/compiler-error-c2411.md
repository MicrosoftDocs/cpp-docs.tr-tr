# <a name="compiler-error-c2411"></a>Derleyici hatası C2411

' tanımlayıcı ': ' Context ' içinde geçersiz yapı/birleşim üyesi

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aşağıdaki olası nedenleri denetleyerek onarmak için

1. Tanımlayıcı, bu bağlamda görünür bir yapının veya birleşimin üyesi değil.

1. Tanımlayıcı, üye seçimi (.) işleci ile belirtilen yapının veya birleşimin üyesi değil.
