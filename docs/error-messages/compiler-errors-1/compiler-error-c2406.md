# <a name="compiler-error-c2406"></a>Derleyici Hatası C2406

'identifier': ad 'bağlamında' tanımsız

Tanımlanmamış tanımlayıcı ile kullanılan `SIZE`, `LENGTH`, veya üye seçme (.) işleci.
