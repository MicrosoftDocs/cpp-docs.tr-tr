# <a name="compiler-error-c2151"></a>Derleyici Hatası C2151

birden fazla dil özniteliği

Bir işlevde bir çağırma kuralı belirten birden fazla anahtar sözcük ( **`__cdecl`** , **`__stdcall`** , veya **`__fastcall`** ) vardır.
