# <a name="compiler-error-c2151"></a>Derleyici Hatası C2151

birden fazla dil özniteliği

Birden fazla anahtar sözcüğü bir işleve sahiptir ( `__cdecl`, `__stdcall`, veya `__fastcall`) çağırma kuralını belirten.
