# <a name="compiler-error-c2095"></a>Derleyici Hatası C2095
'function': Gerçek parametre türü 'void' içeriyor: 'numara' parametresi  
  
 İşlevine geçirilen parametre türüdür `void`, izin verilmeyen. Bir işaretçi void kullanın ( `void *`) yerine.  
  
 `number` Hangi parametresi gösterir `void`.
