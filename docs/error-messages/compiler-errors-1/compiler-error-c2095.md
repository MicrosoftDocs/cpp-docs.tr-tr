# <a name="compiler-error-c2095"></a>Derleyici Hatası C2095

' function ': gerçek parametrenin türü ' void ': parametre ' number '

İşleve geçirilen parametre türü **`void`** , buna izin verilmez. Bunun yerine void () işaretçisi kullanın `void *` .

`number`Hangi parametrenin olduğunu gösterir **`void`** .
