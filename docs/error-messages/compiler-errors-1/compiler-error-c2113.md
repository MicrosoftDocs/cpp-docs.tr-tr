# <a name="compiler-error-c2113"></a>Derleyici Hatası C2113
'-': işaretçi yalnızca çıkarılır başka bir işaretçi  
  
 Bir çıkarma işleminde sağ işleneni için bir işaretçi olsa da, sol işleneni değildi.
