# <a name="compiler-error-c2113"></a>Derleyici Hatası C2113

'-': işaretçi yalnızca başka bir işaretçiden çıkarılabilir

Çıkarma işlemi, sağ işlenen bir işaretçi olduğu, ancak sol işlenen değildi.
