# <a name="fatal-error-c1064"></a>Önemli hata C1064

Derleyici sınırı: belirteç iç arabelleği taşırdı

Bir tanımlayıcı tanımlayıcıları için kullanılan iç arabellek uzunluğu aşıyor. Adı kısaltın.
