# <a name="fatal-error-c1350"></a>Önemli hata C1350

dll ' dll ' yüklenirken hata oluştu: dll bulunamadı

Denenen işlemi destekleyen DLL bulunamadı.

Bu, yüklemenizde bir sorun olduğunu gösterir ve DLL 'yi ürün CD 'nizden yeniden yüklemeniz gerekir.
