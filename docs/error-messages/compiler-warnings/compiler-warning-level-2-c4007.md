# <a name="compiler-warning-level-2-c4007"></a>Derleyici Uyarısı (düzey 2) C4007

' function ': ' Attribute ' olmalıdır

Bir işlev için gerekli bir öznitelik açıkça belirtilmemiş. Örneğin, **Main** işlevinin **`__cdecl`** özniteliği olmalıdır. Derleyici, özniteliğini zorlar.
