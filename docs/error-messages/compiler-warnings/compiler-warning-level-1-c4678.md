# <a name="compiler-warning-level-1-c4678"></a>Derleyici Uyarısı (düzey 1) C4678
taban sınıfı 'taban_türü' 'derived_type' az erişilebilir  
  
Ortak tür özel bir türden türetilmiş. Ortak tür başvurulan bütünleştirilmiş kodunda örneği varsa, özel temel türün üyeleri erişilemeyecek.  
  
C4678 eski derleyici seçeneği kullanılarak erişilebilir, yalnızca **/clr:oldSyntax**. Kullanırken bir hata olduğunu **/CLR**, daha az erişilebilir bir taban sınıfı için türetilmiş sınıf.
