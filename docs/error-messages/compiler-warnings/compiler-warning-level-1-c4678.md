# <a name="compiler-warning-level-1-c4678"></a>Derleyici Uyarısı (düzey 1) C4678

' base_type ' temel sınıfı ' derived_type ' öğesinden daha az erişilebilir

Ortak tür özel bir türden türetilir. Ortak tür başvurulan bir derlemede örnekleniyorsa, özel temel türün üyelerine erişilebilir olmayacaktır.

C4678 yalnızca eski derleyici seçeneği **/clr: oldSyntax** kullanılarak erişilebilir. Türetilmiş sınıfına daha az erişilebilir olan bir temel sınıfa sahip olmak için **/clr** kullanılırken bir hatadır.
