# <a name="compiler-warning-level-4-c4132"></a>Derleyici Uyarısı (düzey 4) C4132
'object': const nesne başlatılmalıdır  
  
 Sabit başlatılmadı. İle bir simge değerini **const** özniteliği tanımına sonra değiştirilemez, bir başlatma değeri verilmelidir.
