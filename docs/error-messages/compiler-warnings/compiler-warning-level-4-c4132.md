# <a name="compiler-warning-level-4-c4132"></a>Derleyici Uyarısı (düzey 4) C4132

' Object ': const nesne başlatılmalıdır

Sabit başlatılamadı. Özniteliği olan bir simgenin değeri **`const`** tanımıyla sonra değiştirilemez, bir başlatma değeri verilmelidir.
