# <a name="compiler-warning-level-1-c4409"></a>Derleyici Uyarısı (düzey 1) C4409

Geçersiz yönerge boyutu

Belirtilen boyutta bir form yönergesi yok. En küçük yasal boyutunu kullanıldı.
