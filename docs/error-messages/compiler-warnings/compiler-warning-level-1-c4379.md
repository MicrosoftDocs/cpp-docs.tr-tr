# <a name="compiler-warning-level-1-c4379"></a>Derleyici Uyarısı (düzey 1) C4379

Ortak dil çalışma zamanının sürüm sürümü bu derleyici tarafından desteklenmiyor. Bu sürümün kullanılması beklenmeyen sonuçlara neden olabilir.

Makinenizde ortak dil çalışma zamanının önceki bir sürümüne sahipsiniz, ancak geçerli sürümü değil. C4379 çözümlemek için, derleyicinizle birlikte gelen ortak dil çalışma zamanının sürümünü yüklemelisiniz.
