# <a name="compiler-warning-level-1-c4375"></a>Derleyici Uyarısı (düzey 1) C4375

Genel olmayan ' Method2 ' yöntemi ' Method1 ' öğesini geçersiz kılmıyor

Başka bir türü uygulayan bir tür geçersiz kılma yöntemi tanımladı, ancak geçersiz kılma genel değildi. Bu nedenle, yöntemi temel tür yöntemini geçersiz kılmaz.
