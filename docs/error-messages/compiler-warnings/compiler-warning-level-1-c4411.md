# <a name="compiler-warning-level-1-c4411"></a>Derleyici Uyarısı (düzey 1) C4411

'identifier': simge yazmacını çözümlüyor

Bir öteleme kasaya çözümler ve başka bir simge ile bir işlenen üzerinde kullanılabilir bir yerel sembol tanımlayıcısıdır.
