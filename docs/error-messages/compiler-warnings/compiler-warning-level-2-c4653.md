# <a name="compiler-warning-level-2-c4653"></a>Derleyici Uyarısı (düzey 2) C4653

' Option ' derleyici seçeneği önceden derlenmiş üstbilgiyle tutarsız; geçerli komut satırı seçeneği yoksayıldı

Önceden derlenmiş üst bilgiler kullan ([/yu](../../build/reference/yu-use-precompiled-header-file.md)) seçeneği ile belirtilen seçenek, ön derlenmiş üstbilgi oluşturulduğunda belirtilen seçeneklerle tutarsız. Bu derleme, ön derlenmiş üstbilgi oluşturulduğunda belirtilen seçeneği kullandı.

Bu uyarı, önceden derlenmiş üstbilginin derlenmesi sırasında paket yapıları seçeneği ([/ZP](../../build/reference/zp-struct-member-alignment.md)) için farklı bir değer belirtildiğinde meydana gelebilir.
