# <a name="compiler-warning-level-4-c4960"></a>Derleyici Uyarısı (düzey 4) C4960

' function ' profili oluşturmak için çok büyük

[/LTCG: PGOPTIMIZE](../../build/reference/ltcg-link-time-code-generation.md)kullanılırken, derleyici 65.535 yönergelerinden daha büyük bir giriş modülü algıladı. Bu tür büyük bir işlev, profil temelli iyileştirmeler için kullanılamaz.

Bu uyarıyı çözmek için işlevin boyutunu küçültün.
