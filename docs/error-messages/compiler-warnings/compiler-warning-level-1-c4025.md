# <a name="compiler-warning-level-1-c4025"></a>Derleyici Uyarısı (düzey 1) C4025
'numara': tabanlı işaretçi geçirilen işlevi değişken bağımsız değişken: parametre numarası  
  
 Değişken bağımsız değişkenlere sahip bir işleve göre bir işaretçi geçirme ile öngörülemeyen sonuçlara normalleştirilmiş için işaretçiyi neden olur. Tabanlı işaretçiler işlevleri değişken bağımsız değişkenler ile başarılı.
