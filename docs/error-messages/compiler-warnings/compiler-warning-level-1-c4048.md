# <a name="compiler-warning-level-1-c4048"></a>Derleyici Uyarısı (düzey 1) C4048

bildirilen farklı dizi indisleri: 'ıdentifier1' ve 'identifier2'

Bir ifade farklı boyutta bir dizi işaretçileri içerir. İşaretçileri dönüştürme olmadan kullanılır.

Bu uyarı, açıkça dizileri aynı veya eşdeğer türüne dönüştürme, çözülebilir.
