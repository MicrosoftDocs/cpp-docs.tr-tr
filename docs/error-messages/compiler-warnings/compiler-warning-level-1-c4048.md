# <a name="compiler-warning-level-1-c4048"></a>Derleyici Uyarısı (düzey 1) C4048
farklı bildirilen dizi indisi: 'identifier1' ve 'identifier2'  
  
 Bir ifade farklı boyutu dizileri işaretçiler içerir. İşaretçileri dönüştürme olmadan kullanılır.  
  
 Bu uyarı sabit aynı ya da eşdeğer türüne diziler açıkça cast ise.
