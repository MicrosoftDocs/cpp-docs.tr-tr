# <a name="compiler-warning-level-1-c4048"></a>Derleyici Uyarısı (düzey 1) C4048

farklı tanımlanmış dizi alt simgeleri: ' Identifier1 ' ve ' identifier2 '

Bir ifade, farklı boyuttaki dizilere işaretçiler içerir. İşaretçiler dönüştürme olmadan kullanılır.

Dizileri aynı veya eşdeğer türe açıkça saçmanız durumunda bu uyarı düzeltilebilir.
