# <a name="compiler-warning-level-1-c4445"></a>Derleyici Uyarısı (düzey 1) C4445

' function ': bir WinRT veya yönetilen türde sanal bir yöntem özel olamaz

Bir sanal işlev özel ise, türetilmiş bir tür tarafından erişilemez. Bu hatayı onarmak için, sanal üye işlevinin erişilebilirliğini korumalı veya ortak olarak değiştirin.
