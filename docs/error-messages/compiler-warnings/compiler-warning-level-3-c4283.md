# <a name="compiler-warning-level-3-c4283"></a>Derleyici Uyarısı (Düzey 3) C4283

ve type 'türü aracılığıyla

Bu devamlılık uyarı [C4281](../../error-messages/compiler-warnings/compiler-warning-level-3-c4281.md) gösteren **-> işleci** aracılığıyla kendi kendini çağıran `type`.
