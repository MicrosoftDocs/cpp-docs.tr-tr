# <a name="compiler-warning-level-1-c4473"></a>Derleyici Uyarısı (düzey 1) C4473

> '*işlevi*': yeterli bağımsız değişken geçirilen biçim dizesi için yer tutucular ve bunların parametrelerini beklediğiniz *numarası* değişen sayıda bağımsız değişken, ancak *numarası* sağlandı değişen sayıda bağımsız değişkeni eksik *dizin* biçim dizesi tarafından gerekli '*format_string*' Bu bağımsız değişken, bu bağımsız değişken kullanılan bu bağımsız değişken alan genişliği olarak kullanılan bir dönüştürme tanımlayıcı tarafından kullanılıyor Duyarlık olarak bu bağımsız değişken arabellek boyutu olarak kullanılıyor

Derleyici, biçim dizesindeki yer tutucuları karşılamak adına gerekli bağımsız değişken sayısı ve sağlanan bağımsız değişken sayısı arasında bir uyumsuzluk algılandı. Biçim dizesi tarafından belirtilen sayıda bağımsız değişken sağladığınız değişen sayıda bağımsız değişken işlevleri printf ve scanf ailelerinde doğru kullanımını gerektirir. Bazı yer tutucuları, içerik için bir bağımsız değişken yanı sıra genişlik, kesinlik veya arabellek boyutunu belirtmek için ek bağımsız değişkenler gerektirir. Bir uyumsuzluk genellikle kodunuzda hata olduğu anlamına gelir.

Printf ailesi biçim dizesi yer tutucusunu ilişkili bağımsız değişkenler hakkında daha fazla bilgi için bkz: [biçim belirtim Sözdizimi: printf ve wprintf işlevleri](../../c-runtime-library/format-specification-syntax-printf-and-wprintf-functions.md). İşleve özgü bilgileri belgelerine bakın *işlevi*.

Bu uyarı, Visual Studio 2015'te yeni bir özelliktir.

## <a name="example"></a>Örnek

Bu örnekleri gösterme uyarı C4473 Biçim dizesinden sonuçlanabilir iki yolu ile bağımsız değişkenlerinden uyuşmazlığı ve ayrıca sorunları gidermek nasıl gösterir.

```cpp
// c4473p.cpp
// compile with: cl /c /W4 c4473p.cpp
#include <stdio.h>

void print_func(int a, float f)
{
    printf_s("This takes %*f arguments.\n", f); // C4473, missing * width argument
    printf_s("This takes %*f arguments.\n", a, f); // fix: supply width argument
}
```

İlk hata iletisinde, derleyici, bir bağımsız değişken eksik, ancak eksik değişkeni genişliğini veya içeriği, sağlanan bağımsız değişken varsayar, dolayısıyla genişlik için birinci ve ikinci bağımsız değişkeni eksik olduğu olup olmadığını bildiremez algılar. , tür dönüştürme belirleyici için. Gerçek eksik bağımsız belirlemek için biçim dizesini incelemeniz gerekir.

```cpp
// c4473s.cpp
// compile with: cl /c /W4 c4473s.cpp
#include <stdio.h>

void scan_func(int a, float f)
{
    char name[20];

    scanf_s("%s", &name);                // C4473, missing buffer size argument
    scanf_s("%s", &name, sizeof(name));  // fix: supply buffer size argument
}
```

Bu örnekte, **scanf_s** her yer tutucu bir adresi yazılacak ve ilk boyutu sağlamak için ikinci bir sağlamak için iki bağımsız değişken gerektirir. Gerekli bağımsız bir açıklaması için her kitaplık işlevine yönelik belgelere bakın.
