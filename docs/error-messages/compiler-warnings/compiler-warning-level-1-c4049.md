# <a name="compiler-warning-level-1-c4049"></a>Derleyici Uyarısı (düzey 1) C4049
Derleyici sınırı: satır numarası çıkması sonlandırma  
  
 Birden fazla 16.777.215 dosyayı içeren (2<sup>24</sup>-1) kaynak satırları. Derleme sırasında 16.777.215 numaralandırma durdurur.  
  
 Satırdan 16.777.215 kodu için:  
  
-   Satır numaraları için hata ayıklama bilgisi yok görüntü içerir.  
  
-   Bazı tanılama hatalı satırı numaralarıyla bildirilebilir.  
  
-   .asm listelerini (/ SK'lar) yanlış satır numaralarını olabilir.
