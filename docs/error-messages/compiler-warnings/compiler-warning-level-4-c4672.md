# <a name="compiler-warning-level-4-c4672"></a>Derleyici Uyarısı (düzey 4) C4672

' Identifier1 ': belirsiz. İlk olarak ' identifier2 ' olarak görüldü

Bir blokta oluşturulacak belirtilen nesne **`try`** belirsiz. Bir nesne oluşturulursa, nesne belirsizlenemez.
