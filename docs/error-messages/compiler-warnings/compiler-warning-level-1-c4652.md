# <a name="compiler-warning-level-1-c4652"></a>Derleyici Uyarısı (düzey 1) C4652
'option' önceden derlenmiş üst bilgi ile; tutarsız derleyici seçeneği Geçerli komut satırı seçeneği önceden derlenmiş üst bilgi tanımlanan geçersiz kılar  
  
 Verilen komut satırı seçeneği, önceden derlenmiş üst bilgi (.pch) oluşturulduğunda verilen farklıydı. Geçerli komut satırında belirtilen seçeneği kullanıldı.  
  
 Bu uyarı, verilen komut satırı seçeneği ile önceden derlenmiş üst bilgi yeniden tarafından önlenebilir.
