# <a name="compiler-warning-level-1-c4631"></a>Derleyici Uyarısı (düzey 1) C4631

MSXML veya XPath kullanılamıyor, XML belgesi açıklamaları işlenmeyecek. reason

Ortak dil çalışma zamanı yüklemenizin belge yorumunu işlemeyi desteklemek için gerekli dosyaları yoktu. Ortak dil çalışma zamanını yeniden yükleyin.
