# <a name="compiler-warning-level-1-c4113"></a>Derleyici Uyarısı (düzey 1) C4113

' Identifier1 ', ' identifier2 ' öğesinden parametre listelerine göre farklılık gösterir

Bir işlev işaretçisi başka bir işlev işaretçisine atanır, ancak işlevlerin biçimsel parametre listeleri kabul etmez. Atama, değişiklik yapılmadan derlenir.
