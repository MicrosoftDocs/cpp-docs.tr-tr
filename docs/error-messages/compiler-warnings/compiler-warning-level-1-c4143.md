# <a name="compiler-warning-level-1-c4143"></a>Derleyici Uyarısı (düzey 1) C4143

pragma ' same_seg ' desteklenmiyor; __based ayırmayı kullan

**#Pragma same_seg** artık desteklenmiyor. Bunun yerine [__based](../../cpp/based-pointers-cpp.md) anahtar sözcüğünü kullanın.
