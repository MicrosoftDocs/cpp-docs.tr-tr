# <a name="compiler-warning-level-1-c4264"></a>Derleyici Uyarısı (düzey 1) C4264

'virtual_function': taban 'class'; sanal üye işlev için kullanılabilecek geçersiz kılma yok işlev gizli

C4264 sonra oluşturulan her zaman [C4263](../../error-messages/compiler-warnings/compiler-warning-level-4-c4263.md).

Varsayılan olarak bu uyarıyı kapalıdır. Bkz: [derleyici uyarıları emin olan kapalı varsayılan](../../preprocessor/compiler-warnings-that-are-off-by-default.md) daha fazla bilgi için.
