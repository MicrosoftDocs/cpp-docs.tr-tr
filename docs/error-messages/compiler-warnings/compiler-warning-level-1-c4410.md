# <a name="compiler-warning-level-1-c4410"></a>Derleyici Uyarısı (düzey 1) C4410
işlenen geçersiz boyutu  
  
 İşlenen talimat üzerinde birini boyutu yanlış vardı. En küçük yasal boyutunu işlenen için kullanıldı.
