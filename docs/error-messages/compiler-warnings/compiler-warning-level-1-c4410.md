# <a name="compiler-warning-level-1-c4410"></a>Derleyici Uyarısı (düzey 1) C4410

işlenen için geçersiz boyut

Yönerge işlenenlerde biri hatalı bir boyut vardı. En küçük yasal boyutunu işlenen için kullanıldı.
