# <a name="compiler-warning-level-1-c4226"></a>Derleyici Uyarısı (düzey 1) C4226

> Standart olmayan uzantı kullanıldı: '*anahtar sözcüğü*' eski bir anahtar sözcük

Geçerli görsel C++ sürümü bu anahtar sözcüğü kullanmıyor.

Bu uyarı otomatik olarak bir hataya yükseltilir.
