# <a name="compiler-warning-level-1-c4226"></a>Derleyici Uyarısı (düzey 1) C4226

> Standart olmayan uzantı kullanıldı: '*anahtar sözcüğü*' artık kullanılmayan bir anahtar sözcüğü

Visual C++'ın geçerli sürümü, bu anahtar sözcük kullanmaz.

Bu uyarı, bir hata için otomatik olarak yükseltilir.
