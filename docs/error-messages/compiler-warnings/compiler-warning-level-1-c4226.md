# <a name="compiler-warning-level-1-c4226"></a>Derleyici Uyarısı (düzey 1) C4226

> kullanılan standart olmayan uzantısı: '*anahtar sözcüğü*' geçersiz bir anahtar sözcüğü

Visual C++ geçerli sürümü bu anahtar sözcük kullanmaz.

Bu uyarı için bir hata otomatik olarak yükseltilir.
