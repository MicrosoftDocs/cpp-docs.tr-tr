# <a name="compiler-warning-level-4-c4053"></a>Derleyici Uyarısı (düzey 4) C4053

için bir void işlenen '?:'

`?:` İşleci türündeki bir ifade, verilen `void`. Değerini `void` işlenen tanımsızdır.
