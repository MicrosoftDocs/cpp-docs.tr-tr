# <a name="compiler-warning-level-1-c4403"></a>Derleyici Uyarısı (düzey 1) C4403

Geçersiz PTR işleci

PTR işleci, satır içi assembler kodda uygunsuz bir şekilde kullanılır.

Derleyici, PTR işleci yoksayıldı.
