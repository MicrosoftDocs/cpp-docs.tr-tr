# <a name="compiler-warning-level-4-c4816"></a>Derleyici Uyarısı (düzey 4) C4816
'param': parametre (nesne başvurusu tarafından geçirilen sürece) bu kesilecek sıfır boyutlu bir dizi içeriyor  
  
 Parametre sıfır boyutlu diziye sahip bir nesneye başvuru tarafından geçilemedi. Nesne geçirildiğinde dizi kopyalanmaz.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek C4816 oluşturur:  
  
```  
// C4816.cpp  
// compile with: /W4  
#include <stdio.h>  
  
extern "C" int printf_s(const char *, ...);  
  
#pragma warning(disable : 4200)  
  
struct S1  
{  
    int i;  
    char cArr[];  
};  
  
void TestErr(S1 s1)   // C4816 param with zero-array   
                      // not passed by reference  
{  
    printf_s("%d %c %c\n", s1.i, s1.cArr[0], s1.cArr[1]);  
}  
  
void TestOk(S1 &s1)  
{  
    printf_s("%d %c %c\n", s1.i, s1.cArr[0], s1.cArr[1]);  
}  
  
int main()  
{  
    S1 myS_1 = { 6, 'a', 'b', 'c' };  
    TestErr(myS_1);  
    TestOk(myS_1);  
}  
```
