# <a name="compiler-warning-levels-1-and-4-c4223"></a>Derleyici Uyarısı (düzey 1 ve 4) C4223

Standart olmayan uzantı kullanıldı: lvalue olmayan dizi işaretçiye dönüştürüldü

Standart C 'de, lvalue olmayan bir diziyi işaretçiye dönüştüremezsiniz. Varsayılan Microsoft uzantıları ([/ze](../../build/reference/za-ze-disable-language-extensions.md)) ile kullanabilirsiniz.
