# <a name="compiler-warning-levels-1-and-4-c4223"></a>Derleyici Uyarısı (düzey 1 ve 4) C4223
kullanılan standart olmayan uzantısı: lvalue olmayan dizi dönüştürülen işaretçi  
  
 Standart C'de gösteren bir işaretçi bir lvalue olmayan dizi dönüştürülemiyor. Varsayılan Microsoft Uzantıları ([/Ze](../../build/reference/za-ze-disable-language-extensions.md)), şunları yapabilirsiniz.
