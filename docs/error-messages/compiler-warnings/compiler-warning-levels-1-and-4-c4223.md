# <a name="compiler-warning-levels-1-and-4-c4223"></a>Derleyici Uyarısı (düzey 1 ve 4) C4223

Standart olmayan uzantı kullanıldı: lvalue olmayan dizi işaretçiye dönüştürüldü

Standart C, bir lvalue olmayan dizi bir işaretçiye dönüştürülemiyor. Varsayılan Microsoft uzantılı ([/Ze](../../build/reference/za-ze-disable-language-extensions.md)), şunları yapabilirsiniz.
