# <a name="compiler-warning-level-2-c4051"></a>Derleyici Uyarısı (düzey 2) C4051

tür dönüştürme; olası veri kaybı

Bir ifade, farklı temel türlerine sahip iki veri öğesi içerir. Bir tür dönüştürme, veri öğesinin kesilmesine neden olur.

Veri öğelerini uygun türe dönüştürmek için bu uyarı düzeltilebilir.
