# <a name="compiler-warning-level-1-c4679"></a>Derleyici Uyarısı (düzey 1) C4679

> '*üye*': üye içeri aktarılamadı

Derleyici tarafından desteklenemez ve meta verilerden içeri aktarılabilecek bir yapı ile karşılaşıldı.

Yapıyı kullanmayı denemeyin.
