# <a name="compiler-warning-level-1-c4679"></a>Derleyici Uyarısı (düzey 1) C4679

> '*üye*': üye alınamadı

Derleme meta verileri içeri aktarılamıyor bir desteklemeyen, bir yapısıyla karşılaştı.

Yapı kullanmak çalışmayın.
