# <a name="compiler-warning-level-1-c4406"></a>Derleyici Uyarısı (düzey 1) C4406

yönergedeki işlenen yoksayıldı

Yönerge hiçbir işlenen almaz, ancak bir işlenen belirtildi.

Derleyici verilen işleneni veya işlenenleri yoksaydı.
