# <a name="compiler-warning-level-1-c4179"></a>Derleyici Uyarısı (düzey 1) C4179

> '//* ': '/' ve '/' olarak ayrıştırıldı \* : Standart '//' açıklamalarıyla karıştırılıyor

__//\*__ yanlış bir açıklama sınırlayıcısı. __//__ __/\*__ Bunun yerine veya kullanın.
