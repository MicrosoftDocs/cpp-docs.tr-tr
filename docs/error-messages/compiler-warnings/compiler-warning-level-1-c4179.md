# <a name="compiler-warning-level-1-c4179"></a>Derleyici Uyarısı (düzey 1) C4179

> '//* ': '/' ve '/\*' olarak ayrıştırıldı: Standart '//' açıklamalarıyla karışıklık

__//\*__ yanlış bir açıklama sınırlayıcısı. Bunun yerine __//__ veya __/\*__ kullanın.
