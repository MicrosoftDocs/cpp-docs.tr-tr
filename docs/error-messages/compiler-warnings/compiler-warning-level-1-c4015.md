# <a name="compiler-warning-level-1-c4015"></a>Derleyici Uyarısı (düzey 1) C4015

' tanımlayıcı ': bit alanının türü tamsayı olmalıdır

Bit alanı bir tamsayı türü olarak bildirilmemiş. Derleyici, bit alanının taban türünü işaretsiz olarak varsayar. Bit alanları işaretsiz tamsayı türleri olarak bildirilmelidir.
