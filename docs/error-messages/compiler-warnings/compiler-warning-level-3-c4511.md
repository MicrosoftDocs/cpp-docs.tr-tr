# <a name="compiler-warning-level-3-c4511"></a>Derleyici Uyarısı (Düzey 3) C4511
'class': kopya Oluşturucu oluşturulamadı  
  
 Derleyici bir sınıf için varsayılan copy-constructor oluşturulamadı; bir taban sınıf, özel bir kopya Oluşturucu olabilir.
