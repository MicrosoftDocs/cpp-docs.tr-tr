# <a name="compiler-warning-level-3-c4511"></a>Derleyici Uyarısı (düzey 3) C4511

' class ': kopya Oluşturucusu üretilemedi

Derleyici, bir sınıf için varsayılan bir kopya Oluşturucu oluşturamadı; bir taban sınıfın özel bir kopya Oluşturucusu olabilir.
