# <a name="compiler-warning-level-3-c4511"></a>Derleyici Uyarısı (Düzey 3) C4511

'class': kopya Oluşturucu üretilemedi

Derleyicinin bir varsayılan kopya Oluşturucusu bir sınıf için oluşturulamadı; bir temel sınıf özel bir kopya oluşturucuya sahip olabilir.
