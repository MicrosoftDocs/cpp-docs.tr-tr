# <a name="compiler-warning-level-1-c4088"></a>Derleyici Uyarısı (düzey 1) C4088

' function ': gerçek parametre ' number ', biçimsel parametre ' number ' içinde işaretçi uyumsuzluğu

Karşılık gelen biçimsel ve gerçek parametrelerin farklı bir yöneltme düzeyi vardır. Gerçek parametre değişiklik olmadan geçirilir. Çağrılan işlev, değerini bir işaretçi olarak yorumlar.
