# <a name="compiler-warning-level-1-c4026"></a>Derleyici Uyarısı (düzey 1) C4026

işlev biçimsel parametre listesiyle tanımlandı

İşlev bildiriminde biçimsel parametreler bulunur, ancak işlev tanımı değil. Bu işlev için sonraki çağrılar, işlevin hiçbir parametre aldığını varsayar.
