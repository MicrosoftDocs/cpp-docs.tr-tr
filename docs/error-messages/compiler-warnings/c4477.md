# <a name="compiler-warning-level-1-c4477"></a>Derleyici Uyarısı (düzey 1) C4477

> '*işlevi*': biçim dizesi '*dize*'türünde bir bağımsız değişken gerektirir'*türü*', ancak değişen sayıda bağımsız değişkeni *numarası* türüne sahip '*türü*'

Derleyici bir biçim dizesindeki yer tutucu karşılamak adına gerekli bağımsız değişken türünü ve sağlanan bağımsız değişken türü arasında bir uyumsuzluk algılandı. Biçim dizesi tarafından belirtilen tür bağımsız değişkenleri sağlayın değişen sayıda bağımsız değişken işlevleri printf ve scanf ailelerinde doğru kullanımını gerektirir. Bir uyumsuzluk genellikle kodunuzda hata olduğu anlamına gelir.

Printf ailesi biçim dizesi yer tutucusunu ilişkili bağımsız değişkenler hakkında daha fazla bilgi için bkz: [biçim belirtim Sözdizimi: printf ve wprintf işlevleri](../../c-runtime-library/format-specification-syntax-printf-and-wprintf-functions.md). İşleve özgü bilgileri belgelerine bakın *işlevi*.

Bu uyarı, Visual Studio 2015'te yeni bir özelliktir.

## <a name="example"></a>Örnek

Bu örnek iki bağımsız değişken türü yanlış olarak bulunan ilk printf_s işlevi için iki C4477 uyarılarını gösterir ve ayrıca sorunları gidermek nasıl gösterir.

```cpp
// C4477p.cpp
// compile with: cl /c /W4 C4477p.cpp
#include <stdio.h>

void print_func(int a, float f)
{
    printf_s("This takes int and double arguments %d and %f.\n", f, a); // C4477, expected int then double
    printf_s("This takes int and double arguments %d and %f.\n", a, f); // fix: types in correct order
}
```

İlk hata iletisinde derleyici olduğunu algılarsa bir **çift** bağımsız değişken sağlanmazsa, bir **int** beklenir. A **float** otomatik olarak yükseltilir **çift** değişen sayıda bağımsız değişken listesinde. Beklenen derleyici ikinci hata iletisini gösteren bir **çift** bağımsız değişkeni, ancak bir **int** sağlanmadı. Derleyici, bağımsız değişkenlerin sırası takas bildiremez. Gerçek bağımsız değişken sırası belirlemek ve doğru bağımsız değişken türleri sağlamak için biçim dizesi incelemeniz gerekir.
