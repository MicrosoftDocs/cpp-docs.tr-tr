# <a name="compiler-warning-level-1-c4034"></a>Derleyici Uyarısı (düzey 1) C4034

sizeof 0 döndürüyor

**`sizeof`** İşleci sıfır boyutundaki bir işlenene (boş bir yapı, birleşim, sınıf veya numaralandırılmış tür ya da tür **`void`** ) uygulanır.
