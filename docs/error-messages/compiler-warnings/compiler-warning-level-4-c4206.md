# <a name="compiler-warning-level-4-c4206"></a>Derleyici Uyarısı (düzey 4) C4206

**Standart olmayan uzantı kullanıldı: çeviri birimi boş**

Dosya ön işlemden sonra boştu.

Bu uzantı, kodunuzun diğer derleyicilere taşınabilir olmasını engelleyebilir. ANSI uyumluluğu ([/za](../../build/reference/za-ze-disable-language-extensions.md)) altında bir hata üretir ve yalnızca C kaynak kodu için geçerlidir.
