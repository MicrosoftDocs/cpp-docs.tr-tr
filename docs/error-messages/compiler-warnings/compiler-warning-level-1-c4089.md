# <a name="compiler-warning-level-1-c4089"></a>Derleyici Uyarısı (düzey 1) C4089
'function': Gerçek parametresinde 'number', 'numara' biçimsel parametresi farklı türleri  
  
 Karşılık gelen resmi ve gerçek parametreleri farklı türleri sahiptir. Gerçek parametre değişiklik geçirilir. İşlev işlevi tanımında belirtilen türe gerçek parametresi çevirir.
