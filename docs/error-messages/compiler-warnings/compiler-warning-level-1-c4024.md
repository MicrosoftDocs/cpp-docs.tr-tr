# <a name="compiler-warning-level-1-c4024"></a>Derleyici Uyarısı (düzey 1) C4024
'function': resmi ve gerçek parametresi 'numara' için farklı türleri  
  
 Karşılık gelen resmi ve gerçek parametreleri farklı türleri sahiptir. Derleyici değişiklik olmadan gerçek parametresi geçirir. Alma işlevinin parametre türü beklenen türe dönüştürür.
