# <a name="compiler-warning-level-1-c4024"></a>Derleyici Uyarısı (düzey 1) C4024

' function ': biçimsel ve gerçek parametre ' number ' için farklı türler

Karşılık gelen biçimsel ve gerçek parametreler farklı türlere sahip. Derleyici gerçek parametreyi değişiklik olmadan geçirir. Alma işlevi parametre türünü beklenen türe dönüştürür.
