# <a name="compiler-warning-level-1-c4024"></a>Derleyici Uyarısı (düzey 1) C4024

'function': biçimsel ve fiili 'number' parametresi için farklı türler

Karşılık gelen biçimsel ve fiili parametreleri farklı türlere sahip. Derleyici, değişiklik olmadan gerçek parametre geçirir. Alıcı işlevi parametre türü beklenen türe dönüştürür.
