# <a name="compiler-warning-level-3-c4723"></a>Derleyici Uyarısı (düzey 3) C4723

olası 0 ile bölme

Bir bölme işlemindeki ikinci işlenen derleme zamanında sıfır olarak değerlendirilir ve tanımsız sonuçlar verir.

Bu uyarı yalnızca [/OG](../../build/reference/og-global-optimizations.md) veya/ogkullanan bir iyileştirme seçeneği kullanılırken verilir.

Derleyici sıfır işlenenini oluşturmuş olabilir.
