# <a name="compiler-warning-level-3-c4278"></a>Derleyici Uyarısı (düzey 3) C4278

> '*tanımlayıcı*': '*tlb*' tür kitaplığındaki tanımlayıcı zaten bir makro; ' Rename ' niteleyicisini kullanın

[#İmport](../../preprocessor/hash-import-directive-cpp.md)kullanırken, içeri aktardığınız TypeLib 'teki bir tanımlayıcı tanımlayıcı *tanımlayıcı* bildirmeye çalışıyor. Ancak, bu zaten geçerli bir simgedir.

`#import`Tür kitaplığındaki simgeye bir diğer ad atamak için **Rename** özniteliğini kullanın.
