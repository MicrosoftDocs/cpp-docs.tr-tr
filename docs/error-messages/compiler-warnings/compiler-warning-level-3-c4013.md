# <a name="compiler-warning-level-3-c4013"></a>Derleyici Uyarısı (Düzey 3) C4013

'function' tanımsız; int döndüren extern olduğu varsayılıyor

Derleyici tanımlanmamış bir işlev çağrısı ile karşılaşıldı.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aşağıdaki olası nedenleri kontrol ederek düzeltmek için

1. İşlev adı hatalı yazımını

1. Dış işlevler olarak prototipli `extern`
