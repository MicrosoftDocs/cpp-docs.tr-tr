# <a name="compiler-warning-level-3-c4013"></a>Derleyici Uyarısı (düzey 3) C4013

' function ' tanımsız; int döndüren extern olduğu varsayılıyor

Derleyici tanımsız bir işleve çağrısıyla karşılaştı.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Aşağıdaki olası nedenleri denetleyerek onarmak için

1. İşlev adının yazımı yanlış

1. Dış işlevler prototip `extern`
