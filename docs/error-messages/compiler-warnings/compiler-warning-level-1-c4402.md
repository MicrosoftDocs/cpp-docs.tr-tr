# <a name="compiler-warning-level-1-c4402"></a>Derleyici Uyarısı (düzey 1) C4402

PTR işleci kullanılmalıdır

Bir tür, satır içi derleme kodundaki bir türe başvurulduğunda veya bu türe atama yaparken PTR işleci olmayan bir işlenen üzerinde kullanılır.

Derleyici PTR işlecini kabul eder.
