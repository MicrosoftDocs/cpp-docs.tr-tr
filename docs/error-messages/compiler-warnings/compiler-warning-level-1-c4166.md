# <a name="compiler-warning-level-1-c4166"></a>Derleyici Uyarısı (düzey 1) C4166

**Oluşturucu/yıkıcı için geçersiz çağırma kuralı**

Oluşturucular ve Yıkıcılar platform için varsayılandan farklı çağrı kurallarına sahip olamaz ( **__clrcall** açıkça belirtmeniz dışında).
