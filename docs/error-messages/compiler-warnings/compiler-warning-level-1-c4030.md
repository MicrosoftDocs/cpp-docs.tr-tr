# <a name="compiler-warning-level-1-c4030"></a>Derleyici Uyarısı (düzey 1) C4030

**Birinci biçimsel parametre listesi ikinci listeden uzun**

Bir işlev farklı biçimsel parametrelerle yeniden bildirildi. Derleyici, ilk bildirimde verilen biçimsel parametreleri kullanır.
