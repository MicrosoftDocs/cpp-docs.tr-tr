# <a name="compiler-warning-level-4-c4092"></a>Derleyici Uyarısı (düzey 4) C4092

sizeof ' unsigned long ' döndürüyor

`sizeof` işlecinin işleneni çok büyüktü, bu nedenle `sizeof` işaretsiz bir **Long**döndürdü. Bu uyarı, Microsoft uzantıları ([/ze](../../build/reference/za-ze-disable-language-extensions.md)) altında oluşur. ANSI uyumluluğu (/Za) altında, bunun yerine sonuç kesilir.
