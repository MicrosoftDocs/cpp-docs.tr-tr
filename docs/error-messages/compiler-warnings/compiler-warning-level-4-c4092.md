# <a name="compiler-warning-level-4-c4092"></a>Derleyici Uyarısı (düzey 4) C4092

> sizeof ' unsigned long ' döndürüyor

**`sizeof`** İşlecin işleneni çok büyüktü, bu yüzden **`sizeof`** bir döndürdü **`unsigned long`** . Bu uyarı, Microsoft uzantıları () altında oluşur [`/Ze`](../../build/reference/za-ze-disable-language-extensions.md) . ANSI uyumluluğu ( **`/Za`** ) altında, bunun yerine sonuç kesilir.
