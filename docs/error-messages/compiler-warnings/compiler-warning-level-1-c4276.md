# <a name="compiler-warning-level-1-c4276"></a>Derleyici Uyarısı (düzey 1) C4276

' function ': prototip belirtilmedi; parametre olmadığı varsayıldı

[__Stdcall](../../cpp/stdcall.md) çağırma kuralına sahip bir işlevin adresini aldığınızda, derleyicinin işlevin düzenlenmiş adını oluşturabilmesi için bir prototipi sağlamanız gerekir. *İşlevin* prototipi olmadığından derleyici, düzenlenmiş adı oluştururken, işlevin parametre olmadığını varsayar.
