# <a name="compiler-warning-level-1-c4276"></a>Derleyici Uyarısı (düzey 1) C4276
'function': sağlanan; hiçbir prototip Hiçbir parametre kabul  
  
 Bir işlev adresi gerçekleştireceğiniz zaman [__stdcall](../../cpp/stdcall.md) derleyici işlevin düzenlenmiş adı oluşturabilmesi için çağırma kuralı, bir prototip vermeniz gerekir. Bu yana *işlevi* derleyici hiçbir prototip düzenlenmiş adı oluştururken sahipse, işlevi parametresiz sahip olduğunu varsayar.
