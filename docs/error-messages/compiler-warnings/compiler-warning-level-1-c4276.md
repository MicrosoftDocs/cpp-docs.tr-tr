# <a name="compiler-warning-level-1-c4276"></a>Derleyici Uyarısı (düzey 1) C4276

'function': hiçbir prototip sağlanan; hiç parametre olmadığı varsayıldı

İle bir işlevin adresini almak zaman [__stdcall](../../cpp/stdcall.md) derleyici, işlev düzenlenmiş adı oluşturabilmesi için çağırma kuralı, bir prototip size gerekir. Bu yana *işlevi* düzenlenmiş adı oluşturulurken hiçbir prototip derleyici sahipse, işlev sahiptir parametresiz varsayar.
