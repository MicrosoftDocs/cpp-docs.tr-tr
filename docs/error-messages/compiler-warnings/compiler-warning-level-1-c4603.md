# <a name="compiler-warning-level-1-c4603"></a>Derleyici Uyarısı (düzey 1) C4603

' \<identifier> ': makro tanımlı değil veya tanım ön derlenmiş üstbilgi kullanımı ile farklı

*Tanımlayıcı* yer tutucusu tarafından belirtilen makro farklı ya da precompiler üst bilgisi kullanıldıktan sonra artık tanımlanmamış.
