# <a name="compiler-warning-level-1-c4811"></a>Derleyici Uyarısı (düzey 1) C4811

Pragma uyumlu değeri (forScope, show) = = değer

Bu uyarı, [uyumlu](../../preprocessor/conform.md) pragma 'un **göster** seçeneğini kullandığınızda verilir. *değer* geçerli uyumlu değerdir.
