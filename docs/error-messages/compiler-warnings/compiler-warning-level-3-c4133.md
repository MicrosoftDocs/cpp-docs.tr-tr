# <a name="compiler-warning-level-3-c4133"></a>Derleyici Uyarısı (düzey 3) C4133

' Type ': uyumsuz türler-' type1 ' öğesinden ' type2 ' öğesine

Bu uyarı, farklı türlerin iki işaretçisi çıkartımı kaldırılmaya çalışıldığında oluşabilir.

Bu uyarıyı önlemek için uygun bir tür dönüştürme sağlayın.
