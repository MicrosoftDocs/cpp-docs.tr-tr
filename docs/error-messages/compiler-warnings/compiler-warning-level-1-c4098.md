# <a name="compiler-warning-level-1-c4098"></a>Derleyici Uyarısı (düzey 1) C4098

' function ': void işlevi bir değer döndürüyor

[Void](../../cpp/void-cpp.md) dönüş türü ile belirtilen bir işlevde bir değer döndüren `return` bir ifade vardır. Derleyici, işlevin `int`türünde bir değer döndürdüğünü varsayar.
