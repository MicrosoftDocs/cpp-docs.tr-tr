# <a name="compiler-warning-level-1-c4041"></a>Derleyici Uyarısı (düzey 1) C4041

Derleyici sınırı: tarayıcı çıkışı sonlandırılıyor

Tarayıcı bilgileri derleyici sınırını aştı.

Bu uyarı, [/fr](../../build/reference/fr-fr-create-dot-sbr-file.md) ile derlenmesinin neden olabilir (yerel değişkenler dahil tarayıcı bilgileri).

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Aşağıdaki olası çözümleri kullanarak düzeltilmesi için

1. /Fr ile derleyin (yerel değişkenler olmadan tarayıcı bilgileri).

1. Tarayıcı çıkışını devre dışı bırak (/FR veya/fr olmadan Derle).
