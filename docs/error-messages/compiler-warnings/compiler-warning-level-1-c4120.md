# <a name="compiler-warning-level-1-c4120"></a>Derleyici Uyarısı (düzey 1) C4120

based/Unbased uyuşmazlığı

Derleyici, biri temel olduğundan ve diğeri olmadığından, iki işaretçi arasında dönüştürme yapılamaz.
