# <a name="compiler-warning-level-1-c4120"></a>Derleyici Uyarısı (düzey 1) C4120
Temel ve unbased uyuşmazlığı  
  
 Derleyici iki işaretçileri arasında bir temel alır ve diğer olmadığından dönüştürülemiyor.
