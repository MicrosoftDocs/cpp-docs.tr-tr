# <a name="compiler-warning-level-1-c4711"></a>Derleyici Uyarısı (düzey 1) C4711
'işlevi için satır içi genişletme seçili' işlevi  
  
 İçin işaretlenmedi rağmen derleyici satır içi kullanım verilen işlevi üzerinde gerçekleştirilen satır içi kullanım.  
  
 C4711 etkin olmadığını [/Ob2](../../build/reference/ob-inline-function-expansion.md) belirtilir.  
  
 Satır içi kullanım derleyicinin istediğiniz kadar gerçekleştirilir. Bu uyarı, bilgi amaçlıdır.  
  
 Varsayılan olarak bu uyarı kapalıdır. Bir uyarı etkinleştirmek için [#pragma Uyarısı](../../preprocessor/warning.md). Bkz: [derleyici uyarıları emin olduğunuz kapalı varsayılan](../../preprocessor/compiler-warnings-that-are-off-by-default.md) daha fazla bilgi için.
