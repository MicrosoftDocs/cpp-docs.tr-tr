# <a name="compiler-warning-level-4-c4234"></a>Derleyici Uyarısı (düzey 4) C4234
kullanılan standart olmayan uzantısı: gelecekte kullanılmak üzere ayrılmış 'anahtar sözcüğü' anahtar sözcüğü  
  
 Derleyici kullandığınız anahtar sözcüğü henüz uygulamıyor.  
  
 Bu uyarı için bir hata otomatik olarak yükseltilir. Bu davranışı değiştirmek istiyorsanız, kullanmak [#pragma Uyarısı](../../preprocessor/warning.md). Örneğin, bir düzey 4 uyarı sorunu yaşayıp C4234 yapmak için  
  
```  
#pragma warning(2:4234)  
```  
  
 Kaynak kodu dosyasının içinde.
