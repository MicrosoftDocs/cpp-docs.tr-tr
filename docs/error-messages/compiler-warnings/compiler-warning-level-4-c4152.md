Standart olmayan uzantı, ifadede işlev/veri ptr dönüştürme

Bir işlev işaretçisi için veya veri işaretçisine dönüştürülür. Bu dönüştürme Microsoft Uzatmaları (/Ze) altında ancak ANSI c altında izin verilir
