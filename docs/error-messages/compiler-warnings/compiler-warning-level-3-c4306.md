# <a name="compiler-warning-level-3-c4306"></a>Derleyici Uyarısı (düzey 3) C4306

> '*tanımlayıcı*': '*Type1*' değerinden daha büyük boyuttaki '*type2*' öğesine dönüştürme

Tanımlayıcı, türü daha büyük bir işaretçiye tür dönüştürme. Yeni türün doldurulmamış yüksek bitleri sıfır dolgulu olacaktır.

Bu uyarı istenmeyen bir dönüştürme olduğunu gösteriyor olabilir. Elde edilen işaretçi geçerli olmayabilir.
