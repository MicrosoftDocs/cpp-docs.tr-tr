# <a name="compiler-warning-level-1-c4182"></a>Derleyici Uyarısı (düzey 1) C4182

\#iç içe geçme düzeyi ' number ' derin; olası sonsuz özyineleme

İç içe geçmiş içerme dosyaları nedeniyle derleyicinin yığın üzerinde boş alan kalmadı. Bir içerme dosyası, başka bir içerme dosyasından dahil edildiğinde iç içe.

Bu ileti bilgilendirme amaçlıdır ve hata [C1076](../../error-messages/compiler-errors-1/fatal-error-c1076.md)önce gelir.
