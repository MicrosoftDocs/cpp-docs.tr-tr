# <a name="compiler-warning-level-1-c4031"></a>Derleyici Uyarısı (düzey 1) C4031
İlk liste uzun ikinci biçimsel parametresi listesi  
  
 Bir işlev, farklı resmi parametrelerle yeniden bildirilen. Derleyici ilk bildirimde verilen biçimsel parametresi kullanır.
