# <a name="compiler-warning-level-1-c4031"></a>Derleyici Uyarısı (düzey 1) C4031

İkinci biçimsel parametre listesi birinci listeden uzun

Bir işlev, farklı biçimsel parametrelerle'yeniden tanımlanıyor. Derleyici ilk bildiriminde verilen bir biçimsel parametreler kullanır.
