# <a name="compiler-warning-level-1-c4158"></a>Derleyici Uyarısı (düzey 1) C4158

> #pragma pointers_to_members (full_generality, devralma) varsayılıyor

Bir **#pragma pointers_to_members (** *tek* &#124; *Çoklu* &#124; *sanal* **)** , eşlik eden bir **#pragma pointers_to_members (full_generality)** olmadan verildi.
