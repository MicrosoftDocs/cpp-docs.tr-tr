# <a name="compiler-warning-level-1-c4175"></a>Derleyici Uyarısı (düzey 1) C4175

\#pragma bileşeni (Browser, on): tarayıcı bilgileri başlangıçta komut satırında belirtilmelidir

[Bileşen](../../preprocessor/component.md) pragmasını kullanmak için, derleme sırasında ([/fr](../../build/reference/fr-fr-create-dot-sbr-file.md)) tarama bilgileri oluşturmanız gerekir.
