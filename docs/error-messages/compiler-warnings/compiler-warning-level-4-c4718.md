# <a name="compiler-warning-level-4-c4718"></a>Derleyici Uyarısı (düzey 4) C4718

' işlev çağrısı ': özyinelemeli çağrının yan etkileri yok, siliniyor

Bir işlev özyinelemeli bir çağrı içerir, aksi halde yan etkileri yoktur. Bu işleve yapılan bir çağrı siliniyor. Programın doğruluğu etkilenmez, ancak davranış. İçindeki çağrının bırakılması, çalışma zamanı yığını taşma özel durumuyla sonuçlanacağından, çağrıyı silmenin bu olasılığı ortadan kaldırır.
