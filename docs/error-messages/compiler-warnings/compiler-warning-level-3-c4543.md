# <a name="compiler-warning-level-3-c4543"></a>Derleyici Uyarısı (düzey 3) C4543

Eklenen metin ' no_injected_text ' özniteliğiyle gizlendi

[No_injected_text](../../windows/no-injected-text.md) özniteliği kaynak kodunda görüntülenmediği anlamına gelir. Bu, derleyicinin ekleme kodundaki özniteliklerin engellenmesini engeller.

Bu uyarı, özniteliklerin kod ekleyemeyecek olduğunu hatırlatmak için kullanılır.
