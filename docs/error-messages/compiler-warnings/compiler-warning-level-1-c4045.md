# <a name="compiler-warning-level-1-c4045"></a>Derleyici Uyarısı (düzey 1) C4045

' Array ': dizi sınırları taştı

Dizide çok fazla Başlatıcı yok. Fazla Başlatıcı yok sayılır.

Dizi öğelerinin ve başlatıcıların boyut ve miktarla eşleştiğinden emin olun.
