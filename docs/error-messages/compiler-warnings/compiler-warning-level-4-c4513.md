# <a name="compiler-warning-level-4-c4513"></a>Derleyici Uyarısı (düzey 4) C4513

' class ': yıkıcı üretilemedi

Derleyici verilen sınıf için varsayılan yıkıcıyı oluşturamıyor; yok edicisi oluşturulmadı. Yıkıcı, türetilmiş sınıf tarafından erişilemeyen bir temel sınıfta. Taban sınıfın özel bir yıkıcısı varsa, bunu ortak veya korumalı hale getirin.
