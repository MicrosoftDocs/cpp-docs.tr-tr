# <a name="namedguids"></a>named_guids
**C++ özgü**  
  
Tanımlamak ve formun eski stilde GUID değişkenlerini başlatmak için derleyiciye `LIBID_MyLib`, `CLSID_MyCoClass`, `IID_MyInterface`, ve `DIID_MyDispInterface`.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
named_guids  
```

**END C++ özgü**
