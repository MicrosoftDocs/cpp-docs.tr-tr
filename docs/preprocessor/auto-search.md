# <a name="auto_search-import-attribute"></a>auto_search içeri aktarma özniteliği

**C++ özel**

Bir tür kitaplığına başvuruluyorsa `#import` ve kendisi başka bir tür kitaplığına başvurduğunda, derleyicinin `#import` diğer tür kitaplığı için örtük bir şekilde yapamadığını belirtir.

## <a name="syntax"></a>Syntax

> **#import** *türü-kitaplık* **auto_search**
