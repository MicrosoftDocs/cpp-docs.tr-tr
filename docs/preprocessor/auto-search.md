# <a name="auto_search-import-attribute"></a>auto_search içeri aktarma özniteliği

**C++Belirli**

Bir tür kitaplığına başvuruluyorsa `#import` ve kendisi başka bir tür kitaplığına başvurduğunda, derleyicinin diğer tür kitaplığı için örtük `#import` bir şekilde yapamadığını belirtir.

## <a name="syntax"></a>Sözdizimi

> **#import** *tür kitaplığı* **auto_search**
