# <a name="null-directive"></a>Null Yönergesi
Boş ön işlemci yönergesi bir tek sayı işaretidir (**#**) tek başına bir satırdaki. Hiçbir etkisi yoktur.

## <a name="syntax"></a>Sözdizimi

```
#
```
