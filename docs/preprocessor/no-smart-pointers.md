# <a name="nosmartpointers"></a>no_smart_pointers
**C++ özel**  
  
 Tür Kitaplığı'nda tüm arabirimler için akıllı işaretçiler oluşturulmasını önler.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
no_smart_pointers  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Varsayılan olarak kullandığınızda, `#import`, Tür Kitaplığı'nda tüm arabirimler için bir akıllı işaretçi bildirim alın. Bu akıllı işaretçiler türlerinin [_com_ptr_t sınıfı](../cpp/com-ptr-t-class.md).
