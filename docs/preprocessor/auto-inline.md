# <a name="auto_inline-pragma"></a>auto_inline pragması

**Devre dışı** olarak belirtilen aralıkta tanımlanan işlevleri, otomatik satır içi genişletme için aday olarak kabul edilmeden hariç tutar.

## <a name="syntax"></a>Syntax

> **#pragma auto_inline (** **[{**  |  **off** }] **)**

## <a name="remarks"></a>Açıklamalar

**Auto_inline** pragma 'ı kullanmak için, işlev tanımı içinde değil, önce ve hemen sonra koyun. Pragma, pragma başlatıldıktan sonra ilk işlev tanımı her zaman devreye girer.
