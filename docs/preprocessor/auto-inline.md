# <a name="auto_inline-pragma"></a>auto_inline pragması

**Devre dışı** olarak belirtilen aralıkta tanımlanan işlevleri, otomatik satır içi genişletme için aday olarak kabul edilmeden hariç tutar.

## <a name="syntax"></a>Sözdizimi

> **#pragma auto_inline (** [{ | **off** }] **)**

## <a name="remarks"></a>Açıklamalar

**Auto_inline** pragmasını kullanmak için, bir işlev tanımı içinde değil, daha önce ve hemen sonra yerleştirin. Pragma, pragma başlatıldıktan sonra ilk işlev tanımı her zaman devreye girer.
