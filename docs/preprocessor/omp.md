# <a name="omp"></a>omp
Bir veya daha fazla OpenMP yönergelerini, isteğe bağlı bir yönerge yan tümceleri ile birlikte alır.

## <a name="syntax"></a>Sözdizimi

```
#pragma omp directive
```
