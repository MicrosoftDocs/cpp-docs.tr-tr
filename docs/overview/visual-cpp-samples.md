## <a name="archived-c-samples-on-github"></a>GitHub'da arşivlenmiş C++ örnekleri

Visual Studio önceki sürümlerde C++ örnek kodu içeriyordu. Örnek kod Visual Studio ile yüklendi veya ayrı bir karşıdan yükleme olarak kullanılabilir.
