# <a name="c-in-visual-studio"></a>Visual Studio’da C++

> [!NOTE]
> Bu geliştirici belgeleri Visual Studio 2015 ve sonrası için geçerlidir. Visual Studio'nun tercih ettiğiniz sürümüiçin belgeleri görmek için **Sürüm** seçici denetimini kullanın. Bu sayfadaki içindekiler tablosunun üst kısmında bulunur.
>
> Bir programı çalıştırabilmek için Visual C++ yeniden dağıtılabilir bir paket arıyorsanız, [Microsoft İndirme Merkezi'ne](https://www.microsoft.com/download/) gidin ve arama kutusuna **Visual C++** girin.

Genellikle Visual C++ veya MSVC olarak kısaltılan Microsoft Visual C++, Windows'daki Visual Studio'nun bir parçası olarak kullanılabilen C++, C ve derleme dili geliştirme araçlarının ve kitaplıklarının adıdır. Bu araçlar ve kitaplıklar, Evrensel Windows Platformu (UWP) uygulamaları, yerel Windows masaüstü ve sunucu uygulamaları, Windows, Linux, Android ve iOS'ta çalışan platformlar arası kitaplıklar ve uygulamaların yanı sıra .NET Framework'ü kullanan yönetilen uygulamalar ve kitaplıklar oluşturmanıza izin tanır. Visual C++ basit konsol uygulamalarından Windows masaüstü için en karmaşık ve karmaşık uygulamalara, aygıt sürücülerinden işletim sistemi bileşenlerine, mobil cihazlar için platform ötesi oyunlara ve en küçük IoT aygıtlarından Azure bulutunda çok sunuculu yüksek performanslı bilgisayar kullanımına kadar her şeyi yazmak için kullanabilirsiniz.

Visual Studio 2015, 2017 ve 2019 yan yana kurulabilir. Visual Studio 2015 (v140) ve Visual Studio 2017 (v141) araç setini kullanarak programları düzenlemek ve oluşturmak için Visual Studio 2019 (derleyici araç seti v142) kullanabilirsiniz.
