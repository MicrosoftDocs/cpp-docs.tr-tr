# <a name="system-function"></a>system İşlevi
**ANSI 4.10.4.5** dizesiyle yürütme modu ve içeriği **sistem** işlevi  
  
 **Sistem** işlevi yürüten bir iç işletim sistemi komutu veya bir. EXE, .COM (. Windows NT CMD) veya. .Bat uzantılarını DENER dosyasından C programı içinde yerine komut satırından.  
  
 Sistem işlevi komut yorumlayıcıyı bulur; bu yorumlayıcı, genellikle Windows NT işletim sisteminde CMD.EXE veya Windows'da COMMAND.COM'dur. Daha sonra sistem işlevi, bağımsız değişken dizesini komut yorumlayıcıya geçirir.  
  
 Daha fazla bilgi için bkz: [sistemi, _wsystem](../c-runtime-library/reference/system-wsystem.md).
