# <a name="summary-of-string-literals"></a>Dizi Değişmez Değerleri Özeti

*dize sabit değeri*:<br/>

&nbsp;&nbsp;&nbsp;&nbsp;**L"** *s karakter dizisi*<sub>iyileştirilmiş</sub> **"**

*s karakter dizisi*:<br/>
&nbsp;&nbsp;&nbsp;&nbsp;*s char*<br/>
