# <a name="c-tokens"></a>C Belirteçleri

C kaynak programında, derleyici tarafından tanınan temel öğe "belirteçtir". Belirteç, derleyicinin bileşen öğelerine ayırmadığı bir kaynak programı metnidir.

## <a name="syntax"></a>Sözdizimi

*belirteç*: *anahtar sözcüğü*

*tanımlayıcı*

*Sabit*

*dize sabit değeri*

*operator*

*Noktalama işaretçisi*

> [!NOTE]
>  Giriş bkz[C dili sözdizimi özeti](../c-language/c-language-syntax-summary.md) ANSI sözdizimi kurallarının açıklaması için.

Bu bölümde açıklanan anahtar sözcükler, tanımlayıcılar, sabitler, dize sabit değerleri ve işleçler belirteçlerin örnekleridir. Köşeli ayraçlar gibi noktalama işareti karakterleri (**[]**), küme ayraçları (**{}**), parantezler ( **()** ) ve virgüller (**,**) de belirteçleridir.
