# <a name="c-tokens"></a>C Belirteçleri
C kaynak programında, derleyici tarafından tanınan temel öğe "belirteçtir". Belirteç, derleyicinin bileşen öğelerine ayırmadığı bir kaynak programı metnidir.  
  
## <a name="syntax"></a>Sözdizimi  
 *belirteç*:  
 *Anahtar sözcüğü*  
  
 *Tanımlayıcı*  
  
 *sabiti*  
  
 *dize değişmez değeri*  
  
 *operator*  
  
 *Noktalama işaretçisi*  
  
> [!NOTE]
>  Giriş için bkz[C dili sözdizimi özeti](../c-language/c-language-syntax-summary.md) ANSI sözdizimi kurallarına açıklaması için.  
  
 Bu bölümde açıklanan anahtar sözcükler, tanımlayıcılar, sabitler, dize sabit değerleri ve işleçler belirteçlerin örnekleridir. Noktalama karakterleri köşeli gibi (**[]**), köşeli parantez (**{}**), parantez ( **()** ) ve virgül (**,**) de belirteçleridir.
