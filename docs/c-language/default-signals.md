# <a name="default-signals"></a>Varsayılan Sinyaller

**ANSI 4.7.1.1** Öğesinin eşdeğeri `signal(sig, SIG_DFL)` bir sinyal işleyicisi çağrısından önce yürütülemiyorsa, sinyalin engellenmesi

Bir program çalışmaya başladığında, sinyaller varsayılan durumlarına ayarlanır.
