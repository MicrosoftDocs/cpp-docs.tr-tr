# <a name="signal-function-c"></a>signal İşlevi (C)

**ANSI 4.7.1.1** **Sinyal** işlevi için sinyaller kümesi

**Sinyale** geçirilen ilk bağımsız değişken, **sinyal** Işlevinin *çalışma zamanı kitaplığı başvurusunda* açıklanan sembolik sabitlerden biri olmalıdır. *Çalışma zamanı kitaplık başvurusundaki* bilgiler aynı zamanda her sinyal için işletim modu desteğini de listeler. Sabitler Ayrıca, SIGNAL. H içinde tanımlanır.
