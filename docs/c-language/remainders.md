# <a name="remainders"></a>Kalanlar
**ANSI 3.3.5** tamsayı bölme üzerinde kalanın oturum  
  
 Kalan oturum bölünen oturum ile aynıdır. Örneğin,  
  
```  
 50 / -6 == -8  
 50 % -6 ==  2  
-50 /  6 == -8  
-50 %  6 == -2  
```
