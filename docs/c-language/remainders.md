# <a name="remainders"></a>Kalanlar

**ANSI 3.3.5** kalan tamsayı bölme üzerinde oturum açma

Kalan oturum bölünen işareti ile aynıdır. Örneğin,

```
50 / -6 == -8
50 % -6 ==  2
-50 /  6 == -8
-50 %  6 == -2
```
