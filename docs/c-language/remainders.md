# <a name="remainders"></a>Kalanlar

**ANSI 3.3.5** Kalan tamsayı bölümünün işareti

Kalanın işareti, bölünme işaretiyle aynıdır. Örneğin,

```
50 / -6 == -8
50 % -6 ==  2
-50 /  6 == -8
-50 %  6 == -2
```
