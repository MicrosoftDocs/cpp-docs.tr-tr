# <a name="type-for-string-literals"></a>Dize Sabit Değerleri Türü

Dize sabit değerleri türü Array türündedir **`char`** (yani, **Char []**). (Geniş karakter dizeleri tür dizisine **`wchar_t`** sahiptir (yani, **wchar_t []**).) Bu, bir dizenin türünde öğeler içeren bir dizi olduğu anlamına gelir **`char`** . Dizideki öğe sayısı, dizedeki karakter sayısına ve Sonlandırıcı null karakteri için bir sayıya eşittir.
