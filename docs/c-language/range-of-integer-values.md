# <a name="range-of-integer-values"></a>Tamsayı Değerleri Aralığı
**ANSI 3.1.2.5** tamsayılar çeşitli türlerde değer kümesini ve temsili  
  
 Tamsayıları 32 bit (dört bayt) içerir. İşaretli tam sayılar ikiye tamamlayıcı biçimde temsil edilir. En anlamlı bit işareti tutar: negatif için 1, pozitif ve sıfır için 0. Değerleri, aşağıda listelenmiştir:  
  
|Tür|Minimum ve maksimum|  
|----------|-------------------------|  
|**İmzasız short**|0 ile 65535|  
|`signed short`|-32768-32767|  
|`unsigned long`|0-4294967295|  
|**uzun imzalı**|-2147483648 ile 2147483647 arasındaki|
