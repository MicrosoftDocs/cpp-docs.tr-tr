# <a name="tn049-mfcole-mbcs-to-unicode-translation-layer-mfcans32"></a>TN049: MFC/OLE MBCS'den Unicode Çeviri Katmanına (MFCANS32)

Bu Not, başlangıçta nasıl açıklanan MFCANS32. DLL sağlar ANSI arabirimlerde öncelikle 32-bit OLE Unicode dünya. Bu DLL, MFC tarafından artık kullanılmamaktadır.
