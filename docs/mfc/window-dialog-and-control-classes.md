# <a name="window-dialog-and-control-classes"></a>Pencere, İletişim Kutusu ve Denetim Sınıfları

Sınıfı `CWnd` ve türetilmiş sınıfları bir `HWND` Windows penceresine yönelik bir tutamacı kapsülleyin. `CWnd` , yeni sınıfları türetmede kendisi veya temel olarak kullanılabilir. Sınıf kitaplığı tarafından sağlanan türetilmiş sınıflar çeşitli pencere türlerini temsil eder.

[CWnd](../mfc/reference/cwnd-class.md)<br/>
Tüm pencereler için temel sınıf. İçinden türetilmiş sınıflardan birini kullanabilir `CWnd` veya doğrudan bu sınıftan türetebilirsiniz.
