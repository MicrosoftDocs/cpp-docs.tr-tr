# <a name="ole-db-classes"></a>OLE DB Sınıfları

MFC 'deki OLE DB desteği şu anda, [COLEDBRecordView](../mfc/reference/coledbrecordview-class.md)sınıfından oluşur. `COleDBRecordView`, bir [CRowset](../data/oledb/crowset-class.md) nesnesine doğrudan bağlı form görünümü aracılığıyla denetimlerde veritabanı kayıtlarını görüntüler. OLE DB tüketici şablonları hakkında daha fazla bilgi için bkz. [OLE DB tüketici şablonlarının listesi](../data/oledb/ole-db-consumer-templates-reference.md).
