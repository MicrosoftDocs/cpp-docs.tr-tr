# <a name="drawing-and-printing-classes"></a>Sınıf Çizme ve Boyama

Windows 'da, tüm grafik çıkışları, cihaz bağlamı (DC) olarak adlandırılan bir sanal çizim alanında çizilir. MFC, çeşitli DC türlerini kapsüllemek için sınıflar sağlar ve bitmapler, fırçalar, paletler ve kalemler gibi Windows çizim araçları için encapsulations.
