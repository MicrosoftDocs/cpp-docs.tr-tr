# <a name="automation-manager-mfc"></a>Otomasyon Yöneticisi (MFC)
AUTMGR32. EXE uzaktan Otomasyon nesneleri sağlamak için planladığı her makinede Windows sistem dizinine kopyalanmalıdır. Windows 95 ve Windows 98, bu genellikle C:\WINDOWS\SYSTEM dizindir. Windows NT ve Windows 2000, genellikle C:\WINNT\SYSTEM32 içindir.  
  
 Geri aramalar sunucudan istemciye etkinleştirmek istiyorsanız, bu yürütülebilir dosya Ayrıca, her bir istemci makinenin sistem dizinine kopyalanmalıdır.  
  
 Otomasyon Yöneticisi çalışırken kısa durum bilgisi içeren sunucu makinesinde küçük bir pencere görüntüler. Gizlemek istiyorsanız, Q138067 Microsoft Bilgi Bankası makalesine bakın.
