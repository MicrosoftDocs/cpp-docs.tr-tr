# <a name="windows-sockets-classes"></a>Windows Yuvaları: Sınıflar

Windows Yuvaları iki bilgisayar arasında iletişim kurmanın ağ protokolüne bağımsız bir yolunu sağlar. Bu yuvalar zaman uyumlu olabilir (programınız iletişim tamamlanana kadar bekler) veya zaman uyumsuz (iletişim devam ederken programınız çalışmaya devam eder).

[CAsyncSocket 'ini kullanma](../mfc/reference/casyncsocket-class.md)<br/>
Windows Sockets API 'sini ince bir sarmalayıcı içinde kapsüller.

[CSocket](../mfc/reference/csocket-class.md)<br/>
`CAsyncSocket`türetilen üst düzey soyutlama. Zaman uyumlu olarak çalışır.

[CSocketFile](../mfc/reference/csocketfile-class.md)<br/>
Bir Windows yuvasına `CFile` arabirimi sağlar.
