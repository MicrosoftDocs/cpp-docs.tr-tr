# <a name="commonly-added-member-functions"></a>Yaygın Olarak Eklenen Üye İşlevleri

İletişim kutusu, Tamam veya Iptal dışında bir pushbutton içeriyorsa, oluşturdukları denetim bildirim iletilerine yanıt vermek için diyalog sınıfınıza ileti işleyici üye işlevleri yazmanız gerekir. Bir örnek için bkz. [karalama](../overview/visual-cpp-samples.md) örnek programı. Ayrıca, iletişim kutusundaki diğer denetimlerden denetim bildirim iletilerini de işleyebilirsiniz.
