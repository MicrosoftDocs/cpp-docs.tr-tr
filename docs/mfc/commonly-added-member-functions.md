# <a name="commonly-added-member-functions"></a>Yaygın Olarak Eklenen Üye İşlevleri

İletişim kutusunda Tamam'ı veya iptal dışında pushbuttons içeriyorsa, iletişim sınıfınızdaki hazırlanmasının denetim bildirimi iletileri cevaplayin üye işlevleri ileti işleyicisi yazmanız gerekir. Bir örnek için bkz. [karalama](../overview/visual-cpp-samples.md) örnek program. Ayrıca, sizin iletişim kutunuzda denetim bildirimi iletileri diğer denetimlerden başa çıkabilir.
