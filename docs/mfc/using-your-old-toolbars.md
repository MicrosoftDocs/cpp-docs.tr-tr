# <a name="using-your-old-toolbars"></a>Eski Araç Çubuklarınızı Kullanma

Özelleştirilmiş araç çubukları, sınıfın yeni uygulaması oluşturmak için Visual C++'ın önceki sürümlerini kullandıysanız [CToolBar](../mfc/reference/ctoolbar-class.md) sorunlara neden olabilir. Böylece yeni işlevselliği kullanmak için eski araç çubuklarınızı vermek zorunda değilsiniz, eski uygulama hala desteklenmektedir.

Eski stil araç çubuklarını, yalnızca yeni stil araç çubukları DOCKTOOL örnek kullanmaz.

Eski stil araç çubukları araç çubuğu Kaynak Düzenleyicisi ile düzenleyemezsiniz.
