# <a name="using-your-old-toolbars"></a>Eski Araç Çubuklarınızı Kullanma

Özelleştirilmiş araç çubukları oluşturmak için Visual C++ önceki sürümlerini kullandıysanız, [CToolBar](../mfc/reference/ctoolbar-class.md) sınıfının yeni uygulanması sorun oluşmasına neden olabilir. Yeni işlevselliği kullanmak için eski araç çubuklarınızı vermeniz gerekmiyorsa, eski uygulama hala desteklenmektedir.

DOCKTOOL örneği eski stil araç çubuklarını, yalnızca yeni stil araç çubuklarını kullanmaz.

Araç çubuğu kaynak Düzenleyicisi ile eski stil araç çubuklarını düzenleyemezsiniz.
