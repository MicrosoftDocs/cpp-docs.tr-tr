# <a name="mfc-technical-notes"></a>MFC Teknik Notları

 Her Teknik Not bir sorun veya MFC belgeleri kalan kapsamı dışındadır özelliğini açıklar. Sağlanan teknik notlar MFC geliştiriciler İleri düzey kullanıcılar isteyeceksiniz düşündüğünüz özel bilgi yanı sıra, kullanıcılardan bilgi isteklerinde yansıtır.
