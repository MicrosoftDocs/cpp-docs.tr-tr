```
    CPane* pBar,
    CPane* pLeftOf);
```

### <a name="parameters"></a>Parametreler

*pBar*<br/>
'ndaki Dock bölmesine yönelik bir işaretçi.

*pLeftOf*<br/>
'ndaki Kaynak olarak hizmet veren bölmeye yönelik bir işaretçi.
