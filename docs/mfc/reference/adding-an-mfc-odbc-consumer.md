# <a name="adding-an-mfc-odbc-consumer"></a>MFC ODBC Tüketicisi Ekleme
MFC ODBC Tüketicisi bir ODBC kayıt kümesi sınıf ve veri bağlamalarını bir veri kaynağına erişmek için gerekli oluşur.  
  
### <a name="to-add-an-mfc-odbc-consumer"></a>MFC ODBC Tüketicisi Ekleme  
  
1.  Sınıf Görünümü'nde projeye sağ tıklayın. Kısayol menüsünde **Ekle** ve ardından **sınıfı Ekle**.  
  
2.  Visual C++ klasörünü çift tıklatarak **MFC ODBC tüketici** simgesini seçin veya tıklatın **açık**.  
  
     MFC ODBC Tüketici Sihirbazı açılır.  
  
3.  Bölümünde açıklandığı gibi ayarlarını tanımla [MFC ODBC Tüketici Sihirbazı](../../mfc/reference/mfc-odbc-consumer-wizard.md).  
  
4.  Tıklatın **son** yeni oluşturulan ODBC tüketici kodunu projenize ekler sihirbazı kapatın.
