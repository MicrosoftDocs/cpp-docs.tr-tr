# <a name="adding-an-mfc-odbc-consumer"></a>MFC ODBC Tüketicisi Ekleme

MFC ODBC Tüketicisi bir ODBC kayıt sınıf ve veri bağlamalarını bir veri kaynağına erişmek için gerekli oluşur.

### <a name="to-add-an-mfc-odbc-consumer"></a>MFC ODBC Tüketicisi Ekleme

1. Sınıf Görünümü'nde projeye sağ tıklayın. Kısayol menüsünde **Ekle** ve ardından **sınıfı Ekle**.

1. Visual C++ klasörünü çift tıklatarak **MFC ODBC Tüketicisi** simgesi veya seçin ve **açık**.

   MFC ODBC Tüketici Sihirbazı açılır.

1. Bölümünde anlatıldığı gibi ayarları tanımlama [MFC ODBC Tüketicisi Sihirbazı](../../mfc/reference/mfc-odbc-consumer-wizard.md).

1. Tıklayın **son** yeni oluşturulan ODBC tüketici kodunu projenize ekler sihirbazı kapatın.
