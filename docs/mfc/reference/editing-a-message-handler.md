# <a name="editing-a-message-handler"></a>İleti İşleyicisini Düzenleme
Bir ileti işleyicisini tanımladıktan sonra eklemek veya kod değiştirmek için üye işlevin tanımını gidebilirsiniz.  
  
 Üye işlevi tanımından atlamak için [iletişim kutusu Düzenleyicisi](../../windows/dialog-editor.md), kendisi için bir işleyici zaten tanımlanmış bir denetimi çift tıklatın. Seçili denetimin ileti işleyicisi tanımlandığı dosyasına gider.  
  
 Üye işlevi tanımından atlamak için [sınıf görünümü](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925), Sınıf Görünümü'nde işlevi adına çift tıklayın.
