# <a name="cmfctoolbarimages-class"></a>CMFCToolBarImages sınıfı

Daha fazla ayrıntı için, Visual Studio yüklemenizin **VC\\atlmfc\\src\\MFC** klasöründe bulunan kaynak koduna bakın.
