## <a name="remarks"></a>Açıklamalar  
 Rebar denetimiyle bulunduğu uygulama rebar bant rebar denetimi tarafından bulunan alt pencere atar. Genellikle başka bir ortak denetimi alt penceredir.  
  
 Rebar denetimleri bir veya daha fazla bantları içerir. Her bant bir Mandal çubuğu, bir bit eşlem, bir metin etiketi ve alt pencere bir birleşimini içerebilir. Bant bu öğelerin her biri yalnızca birini içerebilir.  
  
 Rebar denetimi alt pencere belirtilen arka plan bit eşlemi görüntüleyebilirsiniz. Tüm rebar denetimi bantlar, kullananlar dışında yeniden boyutlandırılabilir **RBBS_FIXEDSIZE** stili. Yeniden konumlandırmak ya da bir rebar denetimi bandı yeniden boyutlandırma gibi rebar denetimi boyutunu ve konumunu, bant atanan alt penceresinin yönetir. Yeniden boyutlandırma veya Denetim içindeki bantları sırasını değiştirmek için tıklatın ve bir bant 's Mandal çubuğu sürükleyin.  
  
 Aşağıdaki çizim üç bantları sahip rebar denetimiyle göstermektedir:  
  
-   Bant 0 düz, saydam araç çubuğu denetimi içerir.  
  
-   Bant 1 hem saydam standart ve saydam açılır düğmeler içerir.  
  
-   Birleşik giriş kutusu ve dört standart düğmeler bant 2 içerir.

##  <a name="create"></a>CReBarCtrl::Create  
 Rebar denetimi oluşturur ve ona ekler `CReBarCtrl` nesnesi.  
  
```  
virtual BOOL Create(
    DWORD dwStyle,  
    const RECT& rect,  
    CWnd* pParentWnd,  
    UINT nID);
```  
  
### <a name="parameters"></a>Parametreler  
 `dwStyle`  
 Rebar denetimi stilleri denetime uygulanan bileşimini belirtir. Bkz: [Rebar denetimi stilleri](http://msdn.microsoft.com/library/windows/desktop/bb774377) desteklenen stilleri listesi için Windows SDK.  
  
 `rect`  
 Bir başvuru bir [CRect](../../atl-mfc-shared/reference/crect-class.md) nesne veya [RECT](http://msdn.microsoft.com/library/windows/desktop/dd162897) konum ve boyutlarının rebar denetimi yapısı.  
  
 `pParentWnd`  
 Bir işaretçi bir [CWnd](../../mfc/reference/cwnd-class.md) rebar denetimi üst pencere nesnesi. Değil olmalıdır **NULL**.  
  
 `nID`  
 Rebar denetimin denetim kimliğini belirtir.  
  
### <a name="return-value"></a>Dönüş Değeri  
 Nesne başarıyla oluşturuldu, sıfır olmayan; Aksi takdirde 0.  
  
### <a name="remarks"></a>Açıklamalar  
 Rebar denetimiyle iki adımda oluşturun:  
  
1.  Çağrı [CReBarCtrl](#crebarctrl) oluşturmak için bir `CReBarCtrl` nesnesi.  
  
2.  Windows rebar denetimi oluşturur ve ekler bu üye işlevini çağırın `CReBarCtrl` nesnesi.  
  
 Çağırdığınızda **oluşturma**, ortak denetimleri başlatılır.

##  <a name="setpalette"></a>CReBarCtrl::SetPalette  
 Win32 ileti davranışını uygulayan [RB_SETPALETTE](http://msdn.microsoft.com/library/windows/desktop/bb774520), Windows SDK'ın açıklandığı gibi.  
  
```  
CPalette* SetPalette(HPALETTE hPal);
```  
  
### <a name="parameters"></a>Parametreler  
 *hPal*  
 Bir `HPALETTE` rebar denetimi kullanacağı yeni paletini belirtir.  
  
### <a name="return-value"></a>Dönüş Değeri  
 Bir işaretçi bir [CPalette](../../mfc/reference/cpalette-class.md) rebar denetimin önceki palet belirtme nesnesi.  
  
### <a name="remarks"></a>Açıklamalar  
 Bu üye fonksiyonu kullanır Not bir `CPalette` nesnesi onun dönüş değeri olarak yerine bir `HPALETTE`.  
  
##  <a name="settextcolor"></a>CReBarCtrl::SetTextColor  
 Win32 ileti davranışını uygulayan [RB_SETTEXTCOLOR](http://msdn.microsoft.com/library/windows/desktop/bb774524), Windows SDK'ın açıklandığı gibi.  
  
```  
COLORREF SetTextColor(COLORREF clr);
```  
  
### <a name="parameters"></a>Parametreler  
 `clr`  
 A **COLORREF** yeni metin temsil eden değer renk `CReBarCtrl` nesnesi.  
  
### <a name="return-value"></a>Dönüş Değeri  
 [COLORREF](http://msdn.microsoft.com/library/windows/desktop/dd183449) önceki metin rengi temsil eden değer ilişkili `CReBarCtrl` nesnesi.  
  
### <a name="remarks"></a>Açıklamalar  
 Bir rebar denetiminde metin rengi esneklik desteklemek için sağlanır.

```  
void SetToolTips(CToolTipCtrl* pToolTip);
```  

 Rebar denetimiyle visual stilini ayarlar.  
  
```  
HRESULT SetWindowTheme(LPCWSTR pszSubAppName);
```  
  
### <a name="parameters"></a>Parametreler  
 `pszSubAppName`  
 Ayarlanacak rebar görsel stil içeren bir Unicode dize için bir işaretçi.  
  
### <a name="return-value"></a>Dönüş Değeri

### <a name="remarks"></a>Açıklamalar

##  <a name="showband"></a>CReBarCtrl::ShowBand  
 Win32 ileti davranışını uygulayan [RB_SHOWBAND](http://msdn.microsoft.com/library/windows/desktop/bb774532), Windows SDK'ın açıklandığı gibi.  
  
```  
BOOL ShowBand(
    UINT uBand,  
    BOOL fShow = TRUE);
```  
  
### <a name="parameters"></a>Parametreler  
 `uBand`  
 Rebar denetiminde bir bant sıfır tabanlı dizini.  
  
 *fShow*  
 Bant gösterileceğini veya gizleneceğini olmadığını gösterir. Bu değer ise **doğru**, bant gösterilir. Aksi takdirde, bant gizlenir.  
  
### <a name="return-value"></a>Dönüş Değeri  
 Başarılıysa sıfır olmayan; Aksi takdirde sıfır.

```  
BOOL SizeToRect(CRect& rect);
```  
  
### <a name="parameters"></a>Parametreler

### <a name="return-value"></a>Dönüş Değeri

### <a name="remarks"></a>Açıklamalar
