
## <a name="remarks"></a>Açıklamalar

WebBrowser denetimi, kullanıcının World Wide Web'deki sitelerin yanı sıra yerel dosya sistemindeki ve ağdaki klasörlere göz atabileceği bir penceredir. WebBrowser denetimi, hiperbağlantı, Tek Düzen Kaynak Bulucu (URL) gezintisini destekler ve bir geçmiş listesi tutar.
