## <a name="remarks"></a>Açıklamalar

WebBrowser denetimi, kullanıcının World Wide Web sitelere ve yerel dosya sisteminde ve bir ağdaki klasörlere gözatabileceği bir penceredir. WebBrowser denetimi, hiper bağlamayı, Tekdüzen Kaynak Konumlandırıcı (URL) gezintisini destekler ve bir geçmiş listesini tutar.

*pbChecked*<br/>
Komutun ( *dwCommandID* tarafından tanımlanır) işaretli olup OLMADıĞıNı belirten bool için bir işaretçi. TRUE ise, komut desteklenir; Aksi halde yanlış.

### <a name="return-value"></a>Dönüş Değeri

Standart bir HRESULT değeri. Olası değerlerin tamamen listesi için, bkz. Windows SDK [IOleCommandTarget:: QueryStatus](/windows/win32/api/docobj/nf-docobj-iolecommandtarget-querystatus) .
