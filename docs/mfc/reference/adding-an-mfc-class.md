# <a name="adding-an-mfc-class"></a>MFC Sınıfı Ekleme
Projeniz için Microsoft Foundation Class (MFC) kitaplığı sınıflarından türetilmiş sınıfları eklemek için kullanın **sınıfı Ekle** komutu kullanılabilir [sınıf görünümü](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925). Yeni sınıf adını belirtin, temel sınıfı seçin ve (varsa) ilişkili olduğu iletişim kutusunun kimliği seçin. Kod Sihirbazı'nı bir üstbilgi dosyası ve bir uygulama dosyası oluşturur ve bunları projenize ekler.  
  
> [!NOTE]
>  MFC sınıfları, ATL COM uygulamaya ekleyebilirsiniz, başlangıçta [uygulama MFC desteği ile oluşturulan](../../atl/reference/mfc-support-in-atl-projects.md). MFC desteği için Win32 projeleri MFC sınıfları de ekleyebilirsiniz.  
  
### <a name="to-add-an-mfc-class-to-your-project"></a>Bir MFC sınıfı projenize eklemek için  
  
1.  Sınıf görünümünden proje adına sağ tıklayın. Tıklatın **Ekle** ve ardından **sınıfı Ekle** açmak için [sınıfı Ekle](../../ide/add-class-dialog-box.md) iletişim kutusu.  
  
2.  Şablonlar bölmesinde seçin **MFC sınıfı** ve basın **Ekle** düğmesi.  
  
3.  Yeni sınıfta için ayarları tanımlayın [MFC Sınıf Sihirbazı](../../mfc/reference/mfc-add-class-wizard.md) iletişim kutusu.  
  
4.  Tıklatın **son** Sihirbazı kapatmak ve yeni sınıf Sınıf Görünümü'nde görüntülemek için. Sihirbazı tarafından oluşturulan dosyalar da görüntüleyebilirsiniz **Çözüm Gezgini**.
