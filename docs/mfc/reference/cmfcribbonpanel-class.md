## <a name="remarks"></a>Açıklamalar  
 Şerit paneller Şerit kategorilerde oluşturduğunuz ilgili görevlerin mantıksal gruplandırmaları olan. Şerit değişiklikleri boyutu olarak paneli düzeni mümkün olduğu kadar öğeleri görüntülemek için otomatik olarak ayarlar.  
  
 Şerit çağırarak bir Şerit kategoride yer alan paneller alabilirsiniz [CMFCRibbonCategory::GetPanel](../../mfc/reference/cmfcribboncategory-class.md#getpanel) yöntemi.

##  <a name="addtoolbar"></a>  CMFCRibbonPanel::AddToolBar  
 Araç şeridi paneline ekler.  
  
```  
CMFCRibbonButtonsGroup* AddToolBar(
UINT uiToolbarResID,  
UINT uiColdResID = 0,  
UINT uiHotResID = 0,  
UINT uiDisabledResID = 0);
```  
  
### <a name="parameters"></a>Parametreler  
 [in] `uiToolbarResID`  
 Eklemek için araç kaynak Kimliğini belirtir.  
  
 [in] `uiColdResID`  
 Araç çubuğunun soğuk görüntüleri kaynak Kimliğini belirtir.  
  
 [in] `uiHotResID`  
 Araç çubuğunun etkin görüntüleri kaynak Kimliğini belirtir.  
  
 [in] `uiDisabledResID`  
 Araç çubuğunun devre dışı görüntüleri kaynak Kimliğini belirtir.  
  
### <a name="return-value"></a>Dönüş Değeri  
 Araç şeridi paneline eklemek için bu yöntemi çağırın. Araç çubuğu önceki çağrısı tarafından eklenen Şerit öğesi yanındaki eklenen [CMFCRibbonPanel::Add](#add).  
  
### <a name="remarks"></a>Açıklamalar  
 Araç çubukları, sık kullanılan görüntüleri, soğuk görüntüler ve devre dışı görüntüleri hakkında daha fazla bilgi için bkz: [CMFCToolBar sınıfı](../../mfc/reference/cmfctoolbar-class.md).

 Şerit paneline sağlanan çalışma zamanı sınıf bilgileri tarafından belirtilen bir Şerit öğesi ekler.  
  
```  
CMFCRibbonBaseElement* SetElementRTCByID(
UINT uiCmdID,  
CRuntimeClass* pRTC);
```  
  
### <a name="parameters"></a>Parametreler  
 [in] `uiCmdID`  
 Eklemek için Şerit öğesi komut Kimliğini belirtir.  
  
 [in] [out] `pRTC`  
 Şerit Masası'na eklenen Şerit öğesi ile ilişkili çalışma zamanı sınıf bilgileri için bir işaretçi.  
  
### <a name="return-value"></a>Dönüş Değeri  
 Belirtilen çalışma zamanı sınıf bilgileri kullanılarak oluşturulmuş Şerit öğesi.  
  
### <a name="remarks"></a>Açıklamalar  
 Özel bir öğe (örneğin, bir renk düğmesi) Şerit paneline eklemek istiyorsanız, özel öğenin çalışma zamanı sınıf bilgileri belirtmeniz gerekir. Şerit bu bilgileri depolar, özel öğesi oluşturur ve belirtilen komut kimliği tarafından bulunan var olan öğenin yerini alır Ardından bir işaretçi yeni oluşturulan öğeyi döndürür.  
  
### <a name="example"></a>Örnek  
 Aşağıdaki örnekte nasıl kullanılacağını gösterir `SetElementRTCByID` yöntemi:  
  
```  
 
// Load and add toolbar with standard buttons. This toolbar  
// should display a custom color button with id ID_CHAR_COLOR:  
 
pPanel->AddToolBar(IDR_MAINFRAME,
    IDB_MAINFRAME256);

CMFCRibbonColorButton* pColorButton = 
(CMFCRibbonColorButton*)pPanel->SetElementRTCByID(
ID_CHAR_COLOR,
    RUNTIME_CLASS (CMFCRibbonColorButton));

 
// SetElementRTCByID sets runtime class and returns a pointer  
// to the newly created custom button,
    which can be set up immediately:  
pColorButton->EnableAutomaticButton(_T("Automatic"),
    RGB (0,
    0,
    0));  
```  
  
##  <a name="setjustifycolumns"></a>  CMFCRibbonPanel::SetJustifyColumns  
 Etkinleştirir veya Şerit öğeleri aynı sütunda genişliğini ayarlama devre dışı bırakır.  
  
```  
void SetJustifyColumns(BOOL bSet = TRUE);
```  
  
### <a name="parameters"></a>Parametreler  
 [in] `bSet`  
 `TRUE` sütundaki en büyük Şerit öğesi genişliğini aynı sütuna Şerit öğeleri genişliğini ayarlamak için; `FALSE` bu genişliği ayarlama devre dışı bırakmak için.  
  
### <a name="remarks"></a>Açıklamalar  
 Bu özellik bir Şerit panelinde etkinleştirildiğinde, Şerit öğeleri aynı sütuna genişlikler aynı sütundaki en büyük Şerit öğesi genişliğine ayarlanır.
