# <a name="adding-atl-support-to-your-mfc-project"></a>MFC Projenize ATL Desteği Ekleme
MFC tabanlı bir uygulama zaten oluşturduysanız, ardından destek Etkin Şablon kitaplığı (ATL) için kolayca MFC projesi Sihirbazı eklemek ATL desteği'i çalıştırarak ekleyebilirsiniz.  
  
> [!NOTE]
>  ATL ve MFC genellikle Visual Studio Express sürümleri desteklenmez.  
  
> [!NOTE]
>  Bu destek eklenmiş bir MFC yürütülebilir veya DLL projesi yalnızca basit COM nesnelerine uygulanır. MFC projeleri için (ActiveX denetimlerini dahil) diğer COM nesneleri ekleyebilirsiniz, ancak nesnelerin beklendiği gibi çalışmayabilir.  
  
### <a name="to-add-atl-support-to-your-mfc-project"></a>MFC projenize ATL desteği eklemek için  
  
1.  Çözüm Gezgini'nde ATL desteği eklemek istediğiniz projesine sağ tıklayın.  
  
2.  Kısayol menüsünde **Ekle**ve ardından **sınıfı Ekle**.  
  
3.  Seçin **MFC projesinde ATL desteği ekleme** simgesi.  
  
    > [!NOTE]
    >  Bu simge ATL klasöründe bulunan **kategorileri** bölmesi.  
  
4.  İstendiğinde, tıklatın **Evet** ATL desteği eklemek için.  
  
 ATL desteği ekleme MFC projenizin kodu nasıl değiştiğini hakkında daha fazla bilgi için bkz: [ayrıntıları, ATL desteği eklenen ATL Sihirbazı tarafından](../../mfc/reference/details-of-atl-support-added-by-the-atl-wizard.md)
