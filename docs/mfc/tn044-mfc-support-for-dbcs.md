# <a name="tn044-mfc-support-for-dbcs"></a>TN044: DBCS için MFC Desteği

Bu teknik notta, "çift baytlı karakter kümeleri" veya DBCS için MFC desteği açıklanmaktadır. Bu bilgilerin yanı sıra, MFC 'nin UNICODE desteğiyle ilgili bilgiler de *sınıf kitaplığı başvurusunda* bulunabilir.
