# <a name="tn044-mfc-support-for-dbcs"></a>TN044: DBCS için MFC desteği

Bu teknik Not "çift baytlı karakter kümeleri" veya DBCS için MFC desteği açıklanmaktadır. MFC'nin bilgi UNICODE kullanıma sunulduğunu desteğinin yanı sıra bu bilgileri *sınıf kitaplığı başvurusu*.
