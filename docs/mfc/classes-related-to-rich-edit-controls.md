# <a name="classes-related-to-rich-edit-controls"></a>Zengin Düzenleme Denetimleriyle İlgili Sınıflar

[CRichEditView](reference/cricheditview-class.md), [CRichEditDoc](reference/cricheditdoc-class.md)ve [Cricheditcntridıtem](reference/cricheditcntritem-class.md) sınıfları, MFC 'nin belge/görünüm mimarisi bağlamı Içinde zengin düzenleme denetimi ([CRichEditCtrl](reference/cricheditctrl-class.md)) işlevlerini sağlar. `CRichEditView` metnin metin ve biçimlendirme özelliklerini korur. `CRichEditDoc` görünümdeki OLE istemci öğelerinin listesini tutar. `CRichEditCntrItem` OLE istemci öğesine kapsayıcı tarafı erişimi sağlar. Uygulamasının içeriğini değiştirmek için `CRichEditView` , temeldeki zengin düzenleme denetimine erişmek Için [CRichEditView:: GetRichEditCtrl](reference/cricheditview-class.md#getricheditctrl) kullanın.
