# <a name="dragging-and-dropping-files-in-a-frame-window"></a>Bir Çerçeve Penceresinde Dosyaları Sürükleme ve Bırakma

Çerçeve penceresi, dosya Gezgini veya dosya yöneticisiyle bir ilişkiyi yönetir.

`CWinApp` `InitInstance` [CWinApp: uygulama sınıfı](cwinapp-the-application-class.md)' nda açıklandığı gibi, üye işlevinin geçersiz kılmasına birkaç başlatma çağrısı ekleyerek, çerçeve pencerenizin dolaylı olarak dosya Gezgini 'nden veya dosya yöneticisinden sürüklenen dosyaları sürüklemiş ve çerçeve penceresine bırakılmış olarak açılmasını sağlayabilirsiniz. Bkz. [Dosya Yöneticisi sürükleme ve bırakma](special-cwinapp-services.md).
