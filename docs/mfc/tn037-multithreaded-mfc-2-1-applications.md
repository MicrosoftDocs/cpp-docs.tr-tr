# <a name="tn037-multithreaded-mfc-21-applications"></a>TN037: Çoklu İş Parçacığı Kullanan MFC 2.1 Uygulamaları

Bu teknik notta ilk olarak Windows NT için Visual C++ 1,0 ile birlikte sunulan MFC 2,1 ile çok iş parçacıklı programların sınırlamaları açıklanmamaktadır. MFC 3,0, çoklu iş parçacığı oluşturmayı doğrudan destekler ve belgelenmiştir. Daha fazla bilgi için bu başvuruya bakın.
