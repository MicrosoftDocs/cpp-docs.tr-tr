# <a name="tn037-multithreaded-mfc-21-applications"></a>TN037: Çoklu İş Parçacığı Kullanan MFC 2.1 Uygulamaları

Bu teknik Not başlangıçta ilk olarak Windows NT için Visual C++ 1.0 ile sağlanan MFC 2.1 ile birden çok iş parçacıklı programlar sınırlamaları açıklanmaktadır. Çoklu iş parçacığı kullanımı MFC 3.0 destekleyen doğrudan ve belgelenmiştir. Daha fazla bilgi için bkz.
