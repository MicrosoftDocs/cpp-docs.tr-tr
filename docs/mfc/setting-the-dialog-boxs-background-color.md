# <a name="setting-the-dialog-boxs-background-color"></a>İletişim Kutusunun Arka Plan Rengini Ayarlama
İletişim kutuları arka plan rengini iletişim kutusu penceresinin WM_CTLCOLOR iletilerini işleme göre ayarlayabilirsiniz. Ayarladığınız rengi yalnızca belirtilen iletişim kutusu için kullanılır.
