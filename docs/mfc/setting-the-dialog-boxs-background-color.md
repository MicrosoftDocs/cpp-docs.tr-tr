# <a name="setting-the-dialog-boxs-background-color"></a>İletişim Kutusunun Arka Plan Rengini Ayarlama

İletişim kutusu penceresi için WM_CTLCOLOR iletileri işleyerek iletişim kutularınızın arka plan rengini ayarlayabilirsiniz. Belirlediğiniz renk yalnızca belirtilen iletişim kutusu için kullanılır.

Örnek için [codexpert bloguna](https://codexpert.ro/blog/2013/03/13/painting-the-dialog-backround/) bakın.
