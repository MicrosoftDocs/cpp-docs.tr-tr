# <a name="file-and-database-classes"></a>Dosya ve Veritabanı Sınıfları

Bu sınıflar, bir veritabanına veya bir disk dosyası bilgileri depolamak izin verin. Üç veritabanı sınıfları kümesi vardır; OLE DB, ODBC ve DAO — benzer işlevselliği sağlar. OLE DB Grup OLE DB kullanılarak uygulanır ve OLE DB tüketici şablonlarıyla birlikte çalışır, DAO grubuna veri erişim nesnesi kullanılarak uygulanır ve ODBC grubu açık veritabanı bağlantısı kullanılarak uygulanır. Vardır ayrıca standart dosyaları, etkin akışları ve HTML akışlarını düzenlemek için sınıf kümesi.

Veri kalıcılığı sınıfları aşağıdaki kategorileri destekler.

- [Dosya g/ç sınıfları](../mfc/file-i-o-classes.md)

- [OLE DB Sınıfları](../mfc/ole-db-classes.md)

- [DAO Sınıfları](../mfc/dao-classes.md)

- [ODBC Sınıfları](../mfc/odbc-classes.md)
