- [Türetilen ileti eşlemeleri](../mfc/derived-message-maps.md)

- [İleti, komut kimlikleri veya Denetim kimliklerinin aralıkları için bir işleyici eşlemesi](../mfc/handlers-for-message-map-ranges.md)
