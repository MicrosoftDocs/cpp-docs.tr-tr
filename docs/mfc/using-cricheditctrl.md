# <a name="using-cricheditctrl"></a>CRichEditCtrl Kullanma

Zengin düzenleme denetimi, kullanıcının metin girebileceği ve düzenleyebileceği bir penceredir. Metne karakter ve paragraf biçimlendirme atanabilir ve katıştırılmış OLE nesneleri dahil edilebilir. Zengin düzenleme denetimi, MFC 'de [CRichEditCtrl](../mfc/reference/cricheditctrl-class.md) sınıfı tarafından temsil edilir.
