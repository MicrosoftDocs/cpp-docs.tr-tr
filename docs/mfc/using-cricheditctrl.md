# <a name="using-cricheditctrl"></a>CRichEditCtrl Kullanma

Bir zengin düzenleme denetimi kullanıcı girin ve metin düzenleme bir penceredir. Metnin karakteri ve paragraf biçimlendirme atanabilir ve katıştırılmış OLE nesnelerine içerebilir. Zengin Düzenleme denetiminin, MFC tarafından temsil edilen [CRichEditCtrl](../mfc/reference/cricheditctrl-class.md) sınıfı.
