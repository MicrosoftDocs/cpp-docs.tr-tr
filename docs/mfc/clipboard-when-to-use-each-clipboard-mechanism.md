# <a name="clipboard-when-to-use-each-clipboard-mechanism"></a>Pano: Her Pano mekanizmasının ne zaman kullanılacağı

Pano'yu kullanarak, aşağıdaki yönergeleri izleyin:

- Gelecekte yeni özellikleri etkinleştirmek için artık OLE Pano mekanizmasını kullanın. Standart panoya API'si korunur, ancak OLE mekanizmasını veri aktarımı geleceğidir.

- OLE uygulama yazıyorsanız, OLE özelliklerinden herhangi birini istediğiniz gibi sürükleyip veya OLE Pano mekanizmasını kullanın.

- OLE biçimleri sağlıyorsanız OLE Pano mekanizmasını kullanın.
