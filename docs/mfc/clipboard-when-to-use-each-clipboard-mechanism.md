# <a name="clipboard-when-to-use-each-clipboard-mechanism"></a>Pano: Her Pano Mekanizmasının Ne Zaman Kullanılacağı

Panoyu kullanarak şu yönergeleri izleyin:

- Gelecekte yeni özellikleri etkinleştirmek için şimdi OLE panosu mekanizmasını kullanın. Standart Pano API 'SI korunurken, OLE mekanizması veri aktarımının geleceği olacaktır.

- OLE uygulaması yazıyorsanız veya sürükle ve bırak gibi OLE özelliklerinden herhangi birini istiyorsanız OLE panosu mekanizmasını kullanın.

- OLE biçimleri sağlıyorsanız OLE panosu mekanizmasını kullanın.
