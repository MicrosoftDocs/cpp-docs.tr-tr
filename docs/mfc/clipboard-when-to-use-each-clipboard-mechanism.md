# <a name="clipboard-when-to-use-each-clipboard-mechanism"></a>Pano: Her Pano Mekanizmasının Ne Zaman Kullanılacağı
Pano kullanarak aşağıdaki yönergeleri izleyin:  
  
-   OLE Pano mekanizmasını şimdi yeni özellikler gelecekte etkinleştirmek için kullanın. Standart Pano API korunur, ancak OLE veri aktarımı geleceği mekanizmadır.  
  
-   OLE uygulama yazıyorsanız veya, OLE özelliklerinden herhangi birini istediğiniz, gibi sürükle ve bırak OLE Pano mekanizmasını kullanın.  
  
-   OLE biçimleri sağlıyorsanız OLE Pano mekanizmasını kullanın.
