# <a name="benefits-of-character-set-portability"></a>Karakter Kümesi Taşınabilirliğinin Yararları

Uygulamanızı Uluslararası hale getirmek için geçerli olmasa bile MFC ve C çalışma zamanı taşınabilirlik özelliklerini kullanmayı yararlı hale getirebilirsiniz:

- Kodlanması, kod tabanınızı esnek hale getirir. Daha sonra kolayca Unicode veya MBCS 'ye taşıyabilirsiniz.

- Unicode kullanmak, uygulamalarınızı Windows için daha verimli hale getirir. Windows Unicode kullandığından, işletim sisteminden ve bu bilgisayardan geçirilen Unicode olmayan dizeler çevrilmelidir, bu da ek yük.
