# <a name="benefits-of-character-set-portability"></a>Karakter Kümesi Taşınabilirliğinin Yararları

Şu anda uygulamanızı amaçlamasanız istemediğiniz bile MFC ve C çalışma zamanı taşınabilirlik özelliklerini kullanarak avantaj elde:

- Temelinizi kodlama kodunuzu temel esnek hale getirir. Daha sonra bunu kolayca Unicode veya MBCS taşıyabilirsiniz.

- Unicode kullanarak Windows için uygulamalarınızı daha verimli hale getirir. Windows Unicode kullandığından, hangi ek yüke neden olur ve işletim sisteminden Unicode olmayan dizeleri çevrilmelidir.
