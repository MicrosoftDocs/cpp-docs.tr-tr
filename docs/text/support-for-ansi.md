# <a name="support-for-ansi"></a>ANSI Desteği
MFC çerçevesi bir bütün olarak yalnızca Unicode desteklemeye doğru kademeli olarak gelişen rağmen çoğu MFC sınıflar ve yöntemler ANSI karakter kümesini destekler karakter kümesi. Sürekli geliştirmeler nedeniyle [!INCLUDE[windowsver](../build/reference/includes/windowsver_md.md)] ve Windows ortak denetimleri sürüm 6.1, birkaç ANSI sınıflar ve yöntemler desteği kullanım dışıdır.  Daha fazla bilgi için bkz: [kullanım dışı ANSI API'ları](../mfc/deprecated-ansi-apis.md) ve [Unicode desteği](../text/support-for-unicode.md).
