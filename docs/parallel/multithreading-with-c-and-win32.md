# <a name="multithreading-with-c-and-win32"></a>C ve Win32 ile Çoklu İş Parçacığı Kullanımı

Microsoft Visual C++, çoklu iş parçacığı kullanan uygulamalar oluşturmak için destek sağlar. Uygulamanızın yanıt veremez duruma gelmesine kullanıcı arabirimi neden olan pahalı işlemler gerçekleştirmek gerekiyorsa birden fazla iş parçacığı kullanmanız gerekir.
