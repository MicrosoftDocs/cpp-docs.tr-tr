# <a name="ompgetwtick"></a>omp_get_wtick

```
double omp_get_wtick( );
```

Daha fazla bilgi için [3.3.2 omp_get_wtick işlevi](../../../parallel/openmp/3-3-2-omp-get-wtick-function.md).
