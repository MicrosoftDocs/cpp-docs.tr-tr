# <a name="ompgetthreadnum"></a>omp_get_thread_num
Kendi iş parçacığı ekibi içinde iş parçacığının iş parçacığı sayısını döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
int omp_get_thread_num( );  
```

 Daha fazla bilgi için bkz: [3.1.4 omp_get_thread_num işlevi](../../../parallel/openmp/3-1-4-omp-get-thread-num-function.md).

 Bkz: [paralel](../../../parallel/openmp/reference/parallel.md) kullanma örneği için `omp_get_thread_num`.
