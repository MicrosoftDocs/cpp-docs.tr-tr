# <a name="barrier"></a>barrier
Bir takım tüm iş parçacıkları eşitler; tüm iş parçacıklarının engel yürütme kadar tüm iş parçacıklarının engel duraklatın.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
#pragma omp barrier  
```  
  
## <a name="remarks"></a>Açıklamalar  
 `barrier` Yönergesi yok OpenMP yan tümceleri destekler.  
  
 Daha fazla bilgi için bkz: [2.6.3 barrier yönergesi](../../../parallel/openmp/2-6-3-barrier-directive.md).
