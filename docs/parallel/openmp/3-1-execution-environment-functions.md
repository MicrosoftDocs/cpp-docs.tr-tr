# <a name="31-execution-environment-functions"></a>3.1 Yürütme Ortamı İşlevleri

Bu bölümde açıklanan işlevleri etkiler ve iş parçacıkları, işlemci ve paralel ortam izleyin:

- **omp_set_num_threads** işlevi.

- **omp_get_num_threads** işlevi.

- **omp_get_max_threads** işlevi.

- **omp_get_thread_num** işlevi.

- **omp_get_num_procs** işlevi.

- **omp_in_parallel** işlevi.

- **omp_set_dynamic** işlevi.

- **omp_get_dynamic** işlevi.

- **omp_set_nested** işlevi.

- **omp_get_nested** işlevi.
