# <a name="concurrencygraphicsdirect3d-namespace"></a>Concurrency::graphics::direct3d Ad Alanı
Sağlar [get_texture](concurrency-graphics-direct3d-namespace-functions.md#get_texture) ve [make_texture](concurrency-graphics-direct3d-namespace-functions.md#make_texture) yöntemleri.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
namespace direct3d;  
```  
  
## <a name="members"></a>Üyeler  
  
### <a name="functions"></a>İşlevler  
  
|Ad<br /><br /> Açıklama|  
|--------------------------|  
|[get_sampler](concurrency-graphics-direct3d-namespace-functions.md#get_sampler)<br /><br /> Get verilen Hızlandırıcı Direct3D örneği durumu arabirimde belirtilen örneği nesne bu temsil görüntüleyin.|  
|[get_texture](concurrency-graphics-direct3d-namespace-functions.md#get_texture)<br /><br /> Belirtilen temel Direct3D Doku arabirimi alır [doku](texture-class.md) nesnesi.|  
|[make_sampler](concurrency-graphics-direct3d-namespace-functions.md#make_sampler)<br /><br /> Bir örneği Direct3D örneği durumu arabirimi işaretçi oluşturun.|  
|[make_texture](concurrency-graphics-direct3d-namespace-functions.md#make_texture)<br /><br /> Oluşturur bir [doku](texture-class.md) belirtilen parametreleri kullanarak nesne.|  
|[msad4](concurrency-graphics-direct3d-namespace-functions.md#msad4)<br /><br /> Bir 4-bayt başvurusu ve 8-bayt kaynak değeri karşılaştırır ve 4 SUM'ları vektörü birikir.|
