## <a name="syntax"></a>Sözdizimi  
  
```cpp  
  
namespace Platform {  namespace Collections {    namespace Details {}}}  
```

 Bu ad alanı üyeleri collection.h içinde tanımlanır ve Nesne Tarayıcısı'nda görüntülenmez.
