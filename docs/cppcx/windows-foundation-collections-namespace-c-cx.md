# <a name="windowsfoundationcollections-namespace-ccx"></a>Windows::Foundation::Collections Ad Alanı (C++/CX)

C++/CX, vektör, Vektörtorview, Map ve MapView koleksiyon sınıflarını kullanmayı kolaylaştıran işlevlerle Windows:: Foundation:: Collections ad alanını tamamlar.

## <a name="syntax"></a>Syntax

```

namespace Windows {
    namespace Foundation {
        namespace Collections;
    }
}
```

### <a name="functions"></a>İşlevler

|Ad|Açıklama|
|----------|-----------------|
|[back_inserter Işlevi](../cppcx/back-inserter-function.md)|Bir koleksiyonun sonuna bir değer eklemek için kullanılabilecek bir yineleyici döndürür.|
|[Begin Işlevi](../cppcx/begin-function.md)|Bir koleksiyonun başlangıcına işaret eden bir yineleyici döndürür.|
|[End Işlevi](../cppcx/end-function.md)|Bir koleksiyonun sonunun ötesinde işaret eden bir yineleyici döndürür.|
|[to_vector Işlevi](../cppcx/to-vector-function.md)|Bir koleksiyonu std:: vector olarak döndürür.|

### <a name="requirements"></a>Gereksinimler

**Üstbilgi:** Collection. h

**Ad alanı:** Windows:: Foundation:: Collections
