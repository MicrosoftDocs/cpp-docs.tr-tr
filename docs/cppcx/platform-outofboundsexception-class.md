# <a name="platformoutofboundsexception-class"></a>Platform::OutOfBoundsException sınıfı

## <a name="syntax"></a>Sözdizimi  
  
```cpp  
public ref class OutOfBoundsException : COMException,    IException,    IPrintable,    IEquatable  
```
