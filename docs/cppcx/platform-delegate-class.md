# <a name="platformdelegate-class"></a>Platform::D elegate sınıfı

Bir işlev nesnesini temsil eder.

## <a name="syntax"></a>Syntax

```cpp
public delegate void delegate_name();
```

### <a name="members"></a>Üyeler

Delegate sınıfı, [Platform:: Object sınıfından](../cppcx/platform-object-class.md)türetilmiş Equals (), GetHashCode () ve ToString () yöntemlerine sahiptir.

### <a name="remarks"></a>Açıklamalar

Temsilciler oluşturmak için [Delegate](../extensions/delegate-cpp-component-extensions.md) anahtar sözcüğünü kullanın; Platform::D elegate 'i açık olarak kullanmayın. Daha fazla bilgi için bkz. [Temsilciler](../cppcx/delegates-c-cx.md). Bir temsilciyi oluşturma ve kullanma hakkında bir örnek için bkz. [C++ ' de Windows çalışma zamanı bileşenleri oluşturma](/windows/uwp/winrt-components/creating-windows-runtime-components-in-cpp).
