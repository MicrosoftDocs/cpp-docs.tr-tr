# <a name="defaulttypenametostring-method"></a>Varsayılan:: (type_name):: ToString yöntemi
Geçerli türünü temsil eden bir dize döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp  
  
String^ ToString()  
```  
  
### <a name="return-value"></a>Dönüş Değeri
