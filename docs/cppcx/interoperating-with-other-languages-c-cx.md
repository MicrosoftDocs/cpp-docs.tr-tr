# <a name="interoperating-with-other-languages-ccx"></a>Diğer Dillerle Birlikte Çalışma (C++/CX)

Belgelerinin bu bölümü, C++/CX 'in diğer programlama dilleri ve kitaplıkları tarafından tüketilen Windows Çalışma Zamanı bileşenleri yazmak için nasıl kullanılacağını açıklar. Ayrıca, tüm diller tarafından tüketilen bileşenleri de yazabilirsiniz. Bu bölümde, C++/CX uygulamanız JavaScript, .NET Framework yönetilen bir dil veya Windows Çalışma Zamanı C++ Şablon kitaplığı kullanılarak yazılmış bileşenlerle birlikte çalışırken göz önünde bulundurmanız gereken farklı yönleri açıklar.
