# <a name="crt-functions-not-supported-in-universal-windows-platform-apps"></a>Evrensel Windows Platformu uygulamalarında desteklenmeyen CRT işlevleri

Evrensel Windows Platformu (UWP) uygulamaları oluşturduğunuzda birçok C çalışma süresi (CRT) işlevi kullanılamaz. Bazen geçici geçici işler kullanılabilir— örneğin, Windows Runtime veya Win32 API'lerini kullanabilirsiniz. Diğer durumlarda, ilgili özellikler veya destekleyici API'ler UWP uygulamaları için geçerli olmadığından CRT işlevleri yasaklanmıştır. Windows Runtime için desteklenen alternatif bir yöntem aramak için [UWP uygulamalarında Windows API'larına Alternatifler'e](/uwp/win32-and-com/alternatives-to-windows-apis-uwp)bakın.

Aşağıdaki tabloda, UWP uygulamaları oluşturduğunuzda kullanılamayan CRT işlevleri listelenir. Geçerli olan geçici işleri gösterir.
