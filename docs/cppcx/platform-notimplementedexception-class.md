# <a name="platformnotimplementedexception-class"></a>Platform::NotImplementedException sınıfı
Bir arabirim üyesi türetilmiş bir tür uygulanmadı zaman oluşturulur.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp  
public ref class NotImplementedException : COMException,    IException,    IPrintable,    IEquatable  
```
