# <a name="platformnotimplementedexception-class"></a>Platform::NotImplementedException sınıfı

Bir arabirim üyesi türetilmiş bir tür uygulanmadı zaman oluşturulur.

## <a name="syntax"></a>Sözdizimi

```cpp
public ref class NotImplementedException : COMException,    IException,    IPrintable,    IEquatable
```

### <a name="remarks"></a>Açıklamalar

Daha fazla bilgi için [COMException](../cppcx/platform-comexception-class.md) sınıfı.

### <a name="requirements"></a>Gereksinimler

**En düşük desteklenen istemci:** Windows 8

**Sunucu desteklenen en düşük:** Windows Server 2012

**Namespace:** Platform

**Meta veri:** platform.winmd
