# <a name="namespaces-reference-ccx"></a>Ad alanı başvurusuC++(/CX)

Belgelerinin bu bölümündeki makaleler/Cxiçin C++derleyicisini destekleyen ad alanlarını anlatmaktadır.

## <a name="compiler-supplied-namespaces"></a>Derleyicinin sağladığı ad alanları

Windows Çalışma Zamanı hedefi olan programların kodlamasını basitleştirmek için, C++/CX derleyicisi ve destekleyici üst bilgi dosyaları, çok sayıda tür tanımlayan ad alanları sağlar. Ad alanları yerleşik sayısal türleri tanımlar; dizeler, diziler ve koleksiyonlar; C++ Windows çalışma zamanı hataları temsil eden özel durumlar; ve standart Windows Çalışma Zamanı türlerindeki dile özgü geliştirmeler.
