# <a name="namespaces-reference-ccx"></a>Ad alanları başvurusu (C + +/ CX)
Bu bölümdeki makaleleri belgelerin derleyici C + Destek ad alanlarını açıklayan +/ CX.  
  
## <a name="compiler-supplied-namespaces"></a>Derleyici tarafından sağlanan ad alanları  
 Windows çalışma zamanı, C + hedefleyen programları kodlama basitleştirmek için +/ CX derleyici ve Tamamlayıcı üstbilgi dosyaları sağlayan çok çeşitli türleri tanımlayan ad alanları. Ad alanları yerleşik sayısal türler tanımlayın; dizeleri, dizileri ve koleksiyonları; Windows çalışma zamanı hataları temsil eder, visual C++ özel durumlarını; ve standart Windows çalışma zamanı türler dile özgü geliştirmeler.
