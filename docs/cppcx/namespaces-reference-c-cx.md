# <a name="namespaces-reference-ccx"></a>Ad Alanları Başvurusu (C++/CX)

Belgelerinin bu bölümündeki makaleler, C++/CXIÇIN derleyicisini destekleyen ad alanlarını anlatmaktadır.

## <a name="compiler-supplied-namespaces"></a>Derleyicinin sağladığı ad alanları

Windows Çalışma Zamanı hedefi olan programların kodlamasını basitleştirmek için, C++/CX derleyicisi ve destekleyici üst bilgi dosyaları, çok sayıda tür tanımlayan ad alanları sağlar. Ad alanları yerleşik sayısal türleri tanımlar; dizeler, diziler ve koleksiyonlar; Windows Çalışma Zamanı hatalarını temsil eden C++ özel durumları; ve standart Windows Çalışma Zamanı türlerindeki dile özgü geliştirmeler.
