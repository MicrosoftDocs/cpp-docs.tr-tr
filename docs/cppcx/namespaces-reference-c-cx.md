# <a name="namespaces-reference-ccx"></a>Ad alanları başvurusu (C + +/ CX)
Belgelerin bu bölümdeki makaleleri, derleyicinin C + destekleyen ad alanlarını açıklayan +/ CX.  
  
## <a name="compiler-supplied-namespaces"></a>Derleyici tarafından sağlanan ad alanları  
 Windows çalışma zamanı, C + hedefleyen programlarını kodlamanızı basitleştirmek için +/ CX derleyici ve Tamamlayıcı üst bilgi dosyaları çok çeşitli türlerini tanımlayan bir ad sağlayın. Ad alanları, yerleşik sayısal türler tanımlayın. dizeleri, diziler ve Koleksiyonlar; Windows çalışma zamanı hataları temsil eden, visual C++ özel durumlarını; ve standart Windows çalışma zamanı türleri için dile özgü geliştirmeleri.
