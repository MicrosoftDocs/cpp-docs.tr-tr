# <a name="defaulttype_namegettype-method"></a>Varsayılan::(type_name):: GetType yöntemi

Geçerli türü temsil eden bir platform:: Type ^ döndürür.

## <a name="syntax"></a>Syntax

```cpp
Platform::Type^ GetType();
```

### <a name="return-value"></a>Dönüş Değeri

Geçerli nesneyi temsil eden bir [Platform:: Type](../cppcx/platform-type-class.md)^ nesnesi.

### <a name="requirements"></a>Gereksinimler

**Desteklenen en düşük istemci:** Windows 8

**Desteklenen en düşük sunucu:** Windows Server 2012

**Ad alanı:** varsayılan

**Üstbilgi:** vccorlib. h
