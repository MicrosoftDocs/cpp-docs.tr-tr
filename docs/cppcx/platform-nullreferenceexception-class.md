# <a name="platformnullreferenceexception-class"></a>Platform::NullReferenceException sınıfı

## <a name="syntax"></a>Sözdizimi  
  
```cpp  
public ref class NullReferenceException : COMException,    IException,    IPrintable,    IEquatable  
```
