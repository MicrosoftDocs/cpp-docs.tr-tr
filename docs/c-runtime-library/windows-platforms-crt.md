# <a name="windows-platforms-crt"></a>Windows Platformları (CRT)

Visual Studio için C çalışma zamanı kitaplıkları, geçerli Windows ve Windows Server, Windows 8, Windows Server 2012, Windows 7, Windows Server 2008 ve Windows Vista sürümlerini destekler ve isteğe bağlı olarak x86, Windows XP hizmeti için Windows XP Service Pack 3 (SP3) desteğini destekler X64 için Pack 2 (SP2) ve x86 ve x64 için Windows Server 2003 Service Pack 2 (SP2). Bu işletim sistemlerinin hepsi Windows Masaüstü API 'sini (Win32) destekler ve Unicode desteği sağlar. Ayrıca, herhangi bir Win32 uygulaması çok baytlı bir karakter kümesi (MBCS) kullanabilir.

> [!NOTE]
> Visual Studio 2017 ' de iş yüküne **sahip C++ masaüstü geliştirme** varsayılan yüklemesi, Windows XP ve Windows Server 2003 geliştirme desteğini içermez. Windows XP platformu araç takımını etkinleştirmek için **Windows XP desteği C++ için** isteğe bağlı bileşenini yüklemelisiniz.
