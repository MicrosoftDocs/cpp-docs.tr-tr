# <a name="windows-platforms-crt"></a>Windows Platformları (CRT)
C çalışma zamanı kitaplıkları için [!INCLUDE[vs2012_upd01](../c-runtime-library/includes/vs2012_upd01_md.md)] Destek [!INCLUDE[win8](../build/reference/includes/win8_md.md)], [!INCLUDE[winserver8](../build/reference/includes/winserver8_md.md)], [!INCLUDE[win7](../build/includes/win7_md.md)], [!INCLUDE[winsvr08](../build/reference/includes/winsvr08_md.md)], Windows Vista, [!INCLUDE[winxp](../build/includes/winxp_md.md)] x86, Service Pack 3 (SP3) [!INCLUDE[winxp](../build/includes/winxp_md.md)] x64, Service Pack 2 (SP2) ve [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)] Hem x86 hem x64 için Service Pack 2 (SP2). Tüm bu işletim sistemlerini destekleyen Windows uygulama programlama arabirimi (API) ve Unicode desteği sağlar. Ayrıca, herhangi bir Win32 uygulaması birden çok baytlı karakter kümesi (MBCS) kullanabilirsiniz.  
  
> [!NOTE]
>  [!INCLUDE[vs_dev11_long](../build/includes/vs_dev11_long_md.md)]desteklemediği [!INCLUDE[winxp](../build/includes/winxp_md.md)] ve [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)] geliştirme. Yüklemelisiniz [!INCLUDE[vs2012_upd01](../c-runtime-library/includes/vs2012_upd01_md.md)] desteği eklemek için [!INCLUDE[winxp](../build/includes/winxp_md.md)] ve [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)].
