# <a name="windows-platforms-crt"></a>Windows Platformları (CRT)

Visual Studio için C çalışma zamanı kitaplıkları, geçerli Windows ve Windows Server, Windows 8, Windows Server 2012, Windows 7, Windows Server 2008 ve Windows Vista sürümlerini destekler ve isteğe bağlı olarak x86 ve x64 için Windows Server 2003 Service Pack 2 (SP2) için Windows XP Service Pack 3 (SP3), x64 için Windows XP Service Pack 2 (SP2) destekler. Bu işletim sistemlerinin hepsi Windows Masaüstü API 'sini (Win32) destekler ve Unicode desteği sağlar. Ayrıca, herhangi bir Win32 uygulaması çok baytlı bir karakter kümesi (MBCS) kullanabilir.

> [!NOTE]
> Visual Studio 2017 ' de **C++ iş yüküne sahip varsayılan masaüstü geliştirme** yüklemesi, Windows XP ve windows Server 2003 geliştirme desteğini içermez. Windows XP platformu araç takımını etkinleştirmek için, **C++ Için WINDOWS XP desteği** isteğe bağlı bileşenini yüklemelisiniz.
