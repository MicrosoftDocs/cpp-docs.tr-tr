# <a name="cisqrt"></a>_CIsqrt
Yığındaki üst değer karekökünü hesaplar.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
void __cdecl _CIsqrt();  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bu sürümü `sqrt` işlevi derleyici özelliğini algılayan bir özelleştirilmiş çağırma vardır. Oluşturulmasını önler kopyaları önlediği için yürütmeyi hızlandırır ve YAZMAÇ ayırma ile yardımcı olur.  
  
 Sonuç değeri yığının en üst gönderilir.
