# <a name="ciexp"></a>_CIexp
Üstel yığında üst değerinin hesaplar.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
void __cdecl _CIexp();  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bu sürümü `exp` işlevi derleyici özelliğini algılayan bir özelleştirilmiş çağırma vardır. Oluşturulmasını önler kopyaları önlediği için yürütmeyi hızlandırır ve YAZMAÇ ayırma ile yardımcı olur.  
  
 Sonuç değeri yığının en üst gönderilir.  
  
## <a name="requirements"></a>Gereksinimler  
 **Platform:** x86
