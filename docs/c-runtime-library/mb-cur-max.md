# <a name="mb_cur_max"></a>MB_CUR_MAX

Geçerli yerel ayar için çok baytlı bir karakterin en fazla bayt sayısını belirten bir makro.

## <a name="syntax"></a>Syntax

```
#include <stdlib.h>
```

## <a name="remarks"></a>Açıklamalar

Bağlam: ANSI çok baytlı ve geniş karakterli dönüştürme işlevleri

Değeri, `MB_CUR_MAX` geçerli yerel ayar için çok baytlı bir karakter olan en fazla bayt sayısıdır.
