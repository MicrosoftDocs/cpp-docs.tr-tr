# <a name="cicos"></a>_CIcos
Yığındaki üst değer kosinüsünü hesaplar.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
void __cdecl _CIcos();  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bu sürümü `cos` işlevi derleyici özelliğini algılayan bir özelleştirilmiş çağırma vardır. Oluşturulmasını önler kopyaları önlediği için yürütmeyi hızlandırır ve YAZMAÇ ayırma ile yardımcı olur.  
  
 Sonuç değeri yığının en üst gönderilir.  
  
## <a name="requirements"></a>Gereksinimler  
 **Platform:** x86
