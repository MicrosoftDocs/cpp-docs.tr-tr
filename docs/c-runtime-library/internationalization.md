# <a name="internationalization"></a>Uluslararası duruma getirme

Microsoft çalışma zamanı kitaplığı, uluslararası pazarlar için bir programın farklı sürümlerini oluşturmak üzere yararlı olan çok sayıda yordam sağlar. Bu, [yerel ayara ilişkin yordamları](../c-runtime-library/locale.md), geniş karakter yordamlarını, çok baytlı karakter yordamlarını ve genel metin yordamlarını içerir. Kolaylık sağlaması için, çoğu yerel ayarla ilgili yordam bu başvuruda gerçekleştirdikleri işlemlere göre kategorize edilir. Bu bölümde ve alfabetik bir başvuruda, çok baytlı karakter yordamları ve geniş karakter yordamları mevcut oldukları tek baytlı karakter karşılıklarıyla açıklanır.

Ayrıca, ISO646 işleci alternatifleri de içerilir.
