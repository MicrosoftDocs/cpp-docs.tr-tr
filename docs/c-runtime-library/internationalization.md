# <a name="internationalization"></a>Uluslararası duruma getirme

Microsoft çalışma zamanı kitaplığı, uygulamanızı şu gibi uluslararası pazarlar için özelleştirmek üzere kullanabileceğiniz birçok yordam sağlar:
- [Yerel ayar ile ilgili yordamlar](../c-runtime-library/locale.md)
- geniş karakter yordamları
- çok baytlı karakter yordamları
- Genel metin yordamları

Kolaylık olması için, yerel ayar ile ilgili yordamların çoğu, ne yapacaklarına göre de kategorilere ayrılır.

Çok baytlı yordamlar ve geniş karakter yordamları, mevcut olduklarında tek baytlık karakter karşılıklarıyla birlikte açıklanmıştır.

ISO646 işleç alternatifleri de eklenmiştir.
