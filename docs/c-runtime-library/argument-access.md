# <a name="argument-access"></a>Bağımsız değişken erişimi

**Va_arg**, **va_end**, ve **va_start** makrolar, değişken sayıda bağımsız değişken olduğunda, işlev bağımsız değişkenleri erişim sağlar. Bu makrolar tanımlanan \<stdarg.h > ANSI/ISO C uyumluluğu ve \<XENIX > UNIX sistem V uyumluluğu için.

## <a name="argument-access-macros"></a>Bağımsız değişken erişimi makroları

|Makrosu|Bir yönetim grubuna bağlanmak veya bağlı bir yönetim grubunun özelliklerini düzenlemek için Yönetim çalışma alanında|
|-----------|-------------------------------|
|[va_arg](../c-runtime-library/reference/va-arg-va-copy-va-end-va-start.md)|Bağımsız değişken listesinden alınamıyor|
|[va_end](../c-runtime-library/reference/va-arg-va-copy-va-end-va-start.md)|İşaretçiyi Sıfırla|
|[va_start](../c-runtime-library/reference/va-arg-va-copy-va-end-va-start.md)|Bağımsız değişken listesinin başından kümesi işaretçisi|
