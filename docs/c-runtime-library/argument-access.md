# <a name="argument-access"></a>Bağımsız değişken erişimi

**Va_arg**, **va_end** ve **va_start** makroları, bağımsız değişkenlerin sayısı değişken olduğunda işlev bağımsız değişkenlerine erişim sağlar. Bu makrolar, \<stdarg.h> ANSI/ISO C uyumluluğu ve ' de \<varargs.h> UNIX Sistem V ile uyumluluk için ' de tanımlanmıştır.

## <a name="argument-access-macros"></a>Bağımsız değişken erişimi makroları

|Makroya|Kullanın|
|-----------|-------------------------------|
|[va_arg](../c-runtime-library/reference/va-arg-va-copy-va-end-va-start.md)|Bağımsız değişkeni listeden Al|
|[va_end](../c-runtime-library/reference/va-arg-va-copy-va-end-va-start.md)|İşaretçiyi Sıfırla|
|[va_start](../c-runtime-library/reference/va-arg-va-copy-va-end-va-start.md)|Bağımsız değişken listesinin başına işaretçiyi ayarla|
