# <a name="iob"></a>_iob

Stdio denetim yapıları dizisi.

## <a name="syntax"></a>Sözdizimi

```
FILE _iob[_IOB_ENTRIES];
```

## <a name="remarks"></a>Açıklamalar

`IOB_ENTRIES` 20 stdio.h'olarak tanımlanır.
