# <a name="_iob"></a>_iob

Stdio denetim yapılarından oluşan dizi.

## <a name="syntax"></a>Sözdizimi

```
FILE _iob[_IOB_ENTRIES];
```

## <a name="remarks"></a>Açıklamalar

`IOB_ENTRIES`, stdio. h içinde 20 olarak tanımlanır.
