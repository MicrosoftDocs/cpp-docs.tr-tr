# <a name="_iob"></a>_iob

## <a name="syntax"></a>Syntax

```
FILE _iob[_IOB_ENTRIES];
```

## <a name="remarks"></a>Açıklamalar

`IOB_ENTRIES` , stdio. h içinde 20 olarak tanımlanır.
