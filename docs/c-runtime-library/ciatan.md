# <a name="_ciatan"></a>_CIatan

Yığındaki en üstteki değerin ark tanjantını hesaplar.

## <a name="syntax"></a>Sözdizimi

```
void __cdecl _CIatan();
```

## <a name="remarks"></a>Açıklamalar

`atan` İşlevin bu sürümünde derleyicinin anladığı özelleştirilmiş bir çağırma kuralı vardır. Kopyaların oluşturulmasını ve YAZMAÇ ayrılmasına yardımcı olmasını önlediği için yürütmeyi hızlandırır.

Elde edilen değer yığının en üstüne gönderilir.

## <a name="requirements"></a>Gereksinimler

**Platform:** x86
