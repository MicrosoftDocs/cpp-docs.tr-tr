# <a name="ciatan"></a>_CIatan
Yığında üst değer tanjantını hesaplar.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
void __cdecl _CIatan();  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bu sürümü `atan` işlevi derleyici özelliğini algılayan bir özelleştirilmiş çağırma vardır. Oluşturulmasını önler kopyaları önlediği için yürütmeyi hızlandırır ve YAZMAÇ ayırma ile yardımcı olur.  
  
 Sonuç değeri yığının en üst gönderilir.  
  
## <a name="requirements"></a>Gereksinimler  
 **Platform:** x86
