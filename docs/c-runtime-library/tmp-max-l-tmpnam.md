
## <a name="remarks"></a>Açıklamalar

`TMP_MAX` , işlevin oluşturabileceği en fazla benzersiz dosya adı sayısıdır `tmpnam` . `L_tmpnam` , tarafından oluşturulan geçici dosya adlarının uzunluğudur `tmpnam` .
