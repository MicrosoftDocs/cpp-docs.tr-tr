## <a name="syntax"></a>Sözdizimi

```

```

## <a name="remarks"></a>Açıklamalar

Sabit `RAND_MAX` tarafından döndürülen en yüksek değer `rand` işlevi. `RAND_MAX` 0x7fff değeri tanımlanır.
