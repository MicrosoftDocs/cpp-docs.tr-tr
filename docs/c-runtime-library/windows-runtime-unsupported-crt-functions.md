# <a name="windows-runtime-unsupported-crt-functions"></a>Windows Çalışma Zamanı Desteklenmeyen CRT İşlevleri

Birçok C çalışma zamanı (CRT) API 'si, Windows Çalışma Zamanı yürütülen Evrensel Windows Platformu (UWP) uygulamalarında kullanılamaz. Bu uygulamalar/ZW derleyici bayrağı kullanılarak oluşturulmuştur. Desteklenmeyen CRT işlevlerinin bir listesi için bkz. [Evrensel Windows platformu uygulamalarda desteklenmeyen crt işlevleri](../cppcx/crt-functions-not-supported-in-universal-windows-platform-apps.md).

Tüm CRT API 'Leri, belgelerinin [alfabetik Işlev Başvurusu](../c-runtime-library/reference/crt-alphabetical-function-reference.md) bölümünde açıklanmaktadır.
