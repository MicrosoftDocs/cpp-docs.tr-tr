# <a name="atanh-atanhf-atanhl"></a>atanh, atanhf, atanhl
Ters hiperbolik tanjant hesaplar.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
double atanh(  
   double x   
);  
float atanh(  
   float x   
);  // C++ only  
long double atanh(  
   long double x  
);  // C++ only  
float atanhf(  
   float x   
);  
long double atanhl(  
   long double x  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `x`  
 Kayan nokta değeri.  
  
## <a name="return-value"></a>Dönüş Değeri  
 `atanh` İşlevler (Ark hiperbolik tanjantı) ters hyberbolic tanjantını döndürür `x`. Varsa `x` 1 veya daha az -1'den, büyük `errno` ayarlanır `EDOM` ve sessiz NaN sonucudur. Varsa `x` olan 1 veya -1 eşit, pozitif veya negatif sonsuz, sırasıyla döndürülür ve `errno` ayarlanır `ERANGE`.  
  
|Giriş|SEH özel durumu|`Matherr`Özel durumu|  
|-----------|-------------------|-------------------------|  
|± QNAN, UL|yok|yok|  
|`X` ≥ 1; `x` ≤ -1|yok|yok|  
  
## <a name="remarks"></a>Açıklamalar  
 Aşırı yükleme C++ izin verdiğinden, aşırı çağırabilirsiniz `atanh` alın ve dönüş `float` veya `long double` değerleri. Bir C programı `atanh` her zaman alan ve döndüren `double`.

## <a name="example"></a>Örnek  
  
```  
// crt_atanh.c  
// This program displays the hyperbolic tangent of pi / 4  
// and the arc hyperbolic tangent of the result.  
//  
  
#include <math.h>  
#include <stdio.h>  
  
int main( void )  
{  
   double pi = 3.1415926535;  
   double x, y;  
  
   x = tanh( pi / 4 );  
   y = atanh( x );  
   printf( "tanh( %f ) = %f\n", pi/4, x );  
   printf( "atanh( %f ) = %f\n", x, y );  
}  
```  
  
```Output  
tanh( 0.785398 ) = 0.655794  
atanh( 0.655794 ) = 0.785398  
```
