# <a name="y0-y1-yn"></a>y0, y1, yn

`y0`, `y1`ve `yn` Microsoft tarafından uygulanan POSIX işlev adları [_y0, _y1 ve _Yn](bessel-functions-j0-j1-jn-y0-y1-yn.md) işlevleri için kullanım dışı diğer adlardır. Varsayılan olarak, [Derleyici Uyarısı (düzey 3) oluşturur C4996](../../error-messages/compiler-warnings/compiler-warning-level-3-c4996.md). Adlar, uygulamaya özgü adlarla ilgili standart C kurallarını izlemediğinden kullanım dışı bırakılmıştır. Ancak, işlevleri hala desteklenmektedir.

Bunun yerine [_y0, _y1 ve _Yn](bessel-functions-j0-j1-jn-y0-y1-yn.md) kullanmanızı öneririz. Ya da bu işlev adlarını kullanmaya devam edebilir ve uyarıyı devre dışı bırakabilirsiniz. Daha fazla bilgi için bkz. [Uyarı](../../error-messages/compiler-warnings/compiler-warning-level-3-c4996.md#turn-off-the-warning) ve [POSIX işlev adlarını](../../error-messages/compiler-warnings/compiler-warning-level-3-c4996.md#posix-function-names)kapatma.
