# <a name="ismbbprint-ismbbprintl"></a>_ismbbprint, _ismbbprint_l
Belirtilen birden çok baytlı karakter yazdırma karakter olup olmadığını belirler.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
int _ismbbprint(  
   unsigned int c   
);  
int _ismbbprint_l(  
   unsigned int c,  
   _locale_t locale  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `c`  
 Sınanacak tamsayı.  
  
 `locale`  
 Kullanılacak yerel ayar.  
  
## <a name="return-value"></a>Dönüş Değeri  
 `_ismbbprint`sıfır olmayan bir değer döndürür ifade:  
  
```  
isprint || _ismbbkprint  
```  
  
 için sıfır olmayan bir değer olan `c`, veya değilse 0. `_ismbbprint`Geçerli yerel ayar için tüm yerel ayara bağımlı davranışı kullanır. `_ismbbprint_l`Bunun yerine geçirilen yerel ayar kullanır ancak bu aynıdır. Daha fazla bilgi için bkz: [yerel ayar](../../c-runtime-library/locale.md).
