# <a name="ceil-ceilf-ceill"></a>ceil, ceilf, ceill

Bir değer değişkenin tavanını hesaplar.

## <a name="syntax"></a>Sözdizimi

```C
double ceil(
   double x
);
float ceil(
   float x
);  // C++ only
long double ceil(
   long double x
);  // C++ only
float ceilf(
   float x
);
long double ceill(
   long double x
);
```

### <a name="parameters"></a>Parametreler

*x*<br/>
Kayan nokta değeri.

## <a name="return-value"></a>Dönüş Değeri

**Ceil** işlevler büyük veya eşit en küçük tamsayıyı temsil eden bir kayan nokta değeri döndürür *x*. Döndürülen hata yok.

|Giriş|SEH özel durumu|Matherr özel durumu|
|-----------|-------------------|-----------------------|
|± **QNAN**, **ONRAKİNİ BUL**|yok|**_ETKİ ALANI**|

**ceil** , Streaming SIMD Extensions 2 (SSE2) kullanan bir uygulamaya sahiptir. Bilgi ve SSE2 uygulamasını kullanmayla ilgili kısıtlamalar için bkz. [_set_SSE2_enable](set-sse2-enable.md).

## <a name="remarks"></a>Açıklamalar

C++ aşırı yüklemeye izin verdiğinden, aşırı yüklemesini çağırabilirsiniz **ceil** Süren **float** veya **uzun** **çift** türleri. C programında **ceil** her zaman alan ve döndüren bir **çift**.
