# <a name="xor"></a>xor
^ işleci alternatifi.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
#define xor ^  
  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Makro, ^ işlecini verir.  
  
## <a name="example"></a>Örnek  
  
```  
// iso646_xor.cpp  
// compile with: /EHsc  
#include <iostream>  
#include <iso646.h>  
  
int main( )  
{  
   using namespace std;  
   int a = 3, b = 2, result;  
  
   result= a ^ b;  
   cout << result << endl;  
  
   result= a xor_eq b;  
   cout << result << endl;  
}  
```  
  
```Output  
1  
1  
```
