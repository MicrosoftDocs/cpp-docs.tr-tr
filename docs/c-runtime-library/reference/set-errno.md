# <a name="_set_errno"></a>_set_errno

**errno** global değişkeninin değerini ayarlayın.

## <a name="syntax"></a>Sözdizimi

```C
errno_t _set_errno( int error_value );
```

### <a name="parameters"></a>Parametreler

*error_value*<br/>
**Errno'nun**yeni değeri.

## <a name="return-value"></a>Dönüş Değeri

Başarılı olursa sıfır döndürür.

## <a name="remarks"></a>Açıklamalar

Olası değerler Errno.h'de tanımlanır. Ayrıca, [errno Sabitleri](../../c-runtime-library/errno-constants.md)bakın.

Varsayılan olarak, bu işlevin genel durumu uygulamaya kapsamlıdır. Bunu değiştirmek için [CRT'deki Genel duruma](../global-state.md)bakın.

## <a name="example"></a>Örnek

```C
// crt_set_errno.c
#include <stdio.h>
#include <errno.h>

int main()
{
   _set_errno( EILSEQ );
   perror( "Oops" );
}
```

```Output
Oops: Illegal byte sequence
```
