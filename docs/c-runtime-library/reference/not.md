# <a name="not"></a>not
Alternatif! işleç.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
#define not !  
  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Makro, ! işlecini verir.  
  
## <a name="example"></a>Örnek  
  
```  
// iso646_not.cpp  
// compile with: /EHsc  
#include <iostream>  
#include <iso646.h>  
  
int main( )  
{  
   using namespace std;  
   int a = 0;  
  
   if (!a)  
      cout << "a is zero" << endl;  
  
   if (not(a))  
      cout << "a is zero" << endl;  
}  
```  
  
```Output  
a is zero  
a is zero  
```
