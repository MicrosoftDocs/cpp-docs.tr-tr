# <a name="_kbhit"></a>_kbhit

Klavye girişi için konsolu denetler.

> [!IMPORTANT]
> Bu API, Windows Runtime'da çalışan uygulamalarda kullanılamaz. Daha fazla bilgi için Evrensel [Windows Platformu uygulamalarında desteklenmeyen CRT işlevlerine](../../cppcx/crt-functions-not-supported-in-universal-windows-platform-apps.md)bakın.

## <a name="syntax"></a>Sözdizimi

```C

int _kbhit( void );
```

## <a name="return-value"></a>Dönüş Değeri

**_kbhit** tuşuna basıldığında sıfır olmayan bir değer verir. Aksi takdirde, 0 döndürür.

## <a name="remarks"></a>Açıklamalar

**_kbhit** işlevi konsolu son tuş vuruşu için denetler. İşlev sıfır olmayan bir değer döndürürse, arabellekte bir tuş vuruşu bekliyor. Program daha sonra tuş vuruşunu almak için **_getch** veya **_getche** arayabilir.

Varsayılan olarak, bu işlevin genel durumu uygulamaya kapsamlıdır. Bunu değiştirmek için [CRT'deki Genel duruma](../global-state.md)bakın.

## <a name="example"></a>Örnek

```C
// crt_kbhit.c
// compile with: /c
/* This program loops until the user
* presses a key. If _kbhit returns nonzero, a
* keystroke is waiting in the buffer. The program
* can call _getch or _getche to get the keystroke.
*/

#include <conio.h>
#include <stdio.h>

int main( void )
{
   /* Display message until key is pressed. */
   while( !_kbhit() )
      _cputs( "Hit me!! " );

   /* Use _getch to throw key away. */
   printf( "\nKey struck was '%c'\n", _getch() );
}
```

### <a name="sample-output"></a>Örnek Çıktı

```Output
Hit me!! Hit me!! Hit me!! Hit me!! Hit me!! Hit me!! Hit me!!
Key struck was 'q'
```
