# <a name="chgsign-chgsignf-chgsignl"></a>_chgsign, _chgsignf, _chgsignl
Kayan nokta bağımsız değişkeni oturum tersine çevirir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
double _chgsign(   
   double x   
);  
float _chgsignf(  
   float x   
);  
long double _chgsignl(   
   long double x   
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `x`  
 Değiştirilecek kayan nokta değeri.  
  
## <a name="return-value"></a>Dönüş Değeri  
 `_chgsign` İşlevler kayan nokta bağımsız değişkenine eşit bir değer döndürür `x`, ancak tersine, oturum. Döndürülen hata yoktur.

|Yordam|Gerekli başlık|  
|-------------|---------------------|  
|`_chgsign`|\<float.h >|  
|`_chgsignf`, `_chgsignl`|\<Math.h >|
