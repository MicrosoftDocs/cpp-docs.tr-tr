# <a name="strrev-wcsrev"></a>strrev, wcsrev

Microsoft 'a özgü işlev adları `strrev` ve `wcsrev` [_strrev ve _wcsrev](strrev-wcsrev-mbsrev-mbsrev-l.md) işlevleri için kullanım dışı diğer adlar. Varsayılan olarak, [Derleyici Uyarısı (düzey 3) oluşturur C4996](../../error-messages/compiler-warnings/compiler-warning-level-3-c4996.md). Adlar, uygulamaya özgü adlarla ilgili standart C kurallarını izlemediğinden kullanım dışı bırakılmıştır. Ancak, işlevleri hala desteklenmektedir.

Bunun yerine [_strrev ve _wcsrev](strrev-wcsrev-mbsrev-mbsrev-l.md) kullanmanızı öneririz. Ya da bu işlev adlarını kullanmaya devam edebilir ve uyarıyı devre dışı bırakabilirsiniz. Daha fazla bilgi için bkz. [Uyarı](../../error-messages/compiler-warnings/compiler-warning-level-3-c4996.md#turn-off-the-warning) ve [POSIX işlev adlarını](../../error-messages/compiler-warnings/compiler-warning-level-3-c4996.md#posix-function-names)kapatma.
