# <a name="character-classification"></a>Karakter Sınıflaması

Bu yordamların her biri, bir koşulun memnuniyet açısından belirtilen tek baytlık bir karakter, geniş karakter veya çok baytlı bir karakter sınar. (Tanım olarak, 0 ve 127 arasındaki ASCII karakter kümesi tüm çok baytlı karakter kümelerinin bir alt kümesidir. Örneğin, Japonca katakana ASCII ve ASCII olmayan karakterler de içerir.)

Test koşulları, yerel ayarın **LC_CTYPE** kategori ayarı ayarından etkilenir; daha fazla bilgi için bkz. [setlocale](../c-runtime-library/reference/setlocale-wsetlocale.md) . **_L** soneki olmayan bu işlevlerin sürümleri, yerel ayara bağımlı davranış için geçerli yerel ayarı kullanır; **_l** sonekine sahip sürümler, bunun yerine geçirilen yerel ayar parametresini kullanmaları dışında aynıdır.

Genellikle bu yordamlar, yazmak isteyebileceğiniz testlerden daha hızlı yürütülür ve çok daha fazla kırmızı olmalıdır. Örneğin, aşağıdaki kod öğesine yapılan çağrıdan daha yavaş yürütülür `isalpha(c)` :

```C
if ((c >= 'A') && (c <= 'Z')) || ((c >= 'a') && (c <= 'z'))
    return TRUE;
```
