# <a name="clocks_per_sec-clk_tck"></a>CLOCKS_PER_SEC, CLK_TCK

## <a name="syntax"></a>Syntax

```
#include <time.h>
```

## <a name="remarks"></a>Açıklamalar

Saniye cinsinden süre, işlev tarafından döndürülen değerdir `clock` , tarafından bölünür `CLOCKS_PER_SEC` . `CLK_TCK` eşdeğerdir, ancak artık kullanılmıyor olarak kabul edilir.
