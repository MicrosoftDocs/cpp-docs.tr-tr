# <a name="clockspersec-clktck"></a>CLOCKS_PER_SEC, CLK_TCK

## <a name="syntax"></a>Sözdizimi

```

#include <time.h>
```

## <a name="remarks"></a>Açıklamalar

Tarafından döndürülen değeri saniye cinsinden süredir `clock` işlevi, bölü `CLOCKS_PER_SEC`. `CLK_TCK` ancak eski olarak kabul, eşdeğerdir.
