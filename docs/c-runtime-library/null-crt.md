# <a name="null-crt"></a>NULL (CRT)

**Null** , çok sayıda işaretçi işlemi ve işlevi ile kullanılan null işaretçi değeridir. 0 ile eşdeğerdir. Şu üst bilgi dosyalarında **null** tanımlanmış: CRTDBG. H, YEREL AYAR. H, STDDEF. H, STDIO. H, STDLIB. H, DIZE. H, TCHAR. H, SAAT. H ve WCHAR. H.
