# <a name="cilog"></a>_CIlog
Yığındaki üst değer doğal logaritmasını hesaplar.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
void __cdecl _CIlog();  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bu sürümü `log` işlevi derleyici özelliğini algılayan bir özelleştirilmiş çağırma vardır. Oluşturulmasını önler kopyaları önlediği için yürütmeyi hızlandırır ve YAZMAÇ ayırma ile yardımcı olur.  
  
 Sonuç değeri yığının en üst gönderilir.  
  
## <a name="requirements"></a>Gereksinimler  
 **Platform:** x86
