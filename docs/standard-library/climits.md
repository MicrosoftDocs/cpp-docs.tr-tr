# <a name="ltclimitsgt"></a>&lt;climits&gt;

C Standart Kitaplığı üst bilgisi \<lımıts.h > ve ilişkili adlar ekler `std` ad alanı. Bu üstbilginin dahil sağlar C Standart Kitaplığı üstbilgisinde harici bağlantı kullanılarak bildirilen adların bildirilir `std` ad alanı.

## <a name="syntax"></a>Sözdizimi

```cpp
#include <climits>
```

## <a name="macros"></a>Makrolar

```cpp
#define CHAR_BIT
#define SCHAR_MIN
#define SCHAR_MAX
#define UCHAR_MAX
#define CHAR_MIN
#define CHAR_MAX
#define MB_LEN_MAX
#define SHRT_MIN
#define SHRT_MAX
#define USHRT_MAX
#define INT_MIN
#define INT_MAX
#define UINT_MAX
#define LONG_MIN
#define LONG_MAX
#define ULONG_MAX
#define LLONG_MIN
#define LLONG_MAX
#define ULLONG_MAX
```
