# <a name="ltcsetjmpgt"></a>&lt;csetjmp&gt;
Standart C Kitaplığı üstbilgi içeren \<setjmp.h'ı > ve ilişkili adlarıyla ekler `std` ad alanı.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
#include <csetjmp>  
  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bu üst dahil olmak üzere sağlar standart C Kitaplığı üstbilgisinde dış bağlantı kullanılarak bildirilen adları bildirilir `std` ad alanı.
