# <a name="ltbitsetgt"></a>&lt;bitset&gt;
Şablon sınıfı bitset ve temsil eden ve sabit boyutlu dizileri bit düzenleme için iki destekleyen şablon işlevleri tanımlar.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
#include <bitset>  
  
```  
  
### <a name="operators"></a>İşleçler  
  
|||  
|-|-|  
|[operator &](../standard-library/bitset-operators.md#op_amp)|Bit düzeyinde AND iki bit kümeleri arasında gerçekleştirir.|  
|[operator <\<](../standard-library/bitset-operators.md#op_lt_lt)|Bit sırası metin gösterimi standart çıktı akışı ekler.|  
|[İşleç >>](../standard-library/bitset-operators.md#op_gt_gt)|Bit sırası metin gösterimi standart giriş akışı ekler.|  
|[operator ^](../standard-library/bitset-operators.md#op_xor)|Bit tabanlı gerçekleştirir hariç veya iki bit kümeleri arasında.|  
|[operator &#124;](../standard-library/bitset-operators.md#op_or)|Bit düzeyinde OR iki bit kümeleri arasında gerçekleştirir.|  
  
### <a name="classes"></a>Sınıflar  
  
|||  
|-|-|  
|[bitset sınıfı](../standard-library/bitset-class.md)|Şablon sınıfı bayrakları öğeler veya koşullar kümesi için koruma altında tutmada kısa bir yol sağlamak BITS sabit sayıda oluşan bir dizi depolar nesnesi türünü açıklar.|
