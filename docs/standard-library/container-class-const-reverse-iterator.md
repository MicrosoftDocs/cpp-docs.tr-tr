# <a name="container-classconstreverseiterator"></a>Kapsayıcı Sınıfı::const_reverse_iterator

> [!NOTE]
> Bu konu Microsoft olan C++ kullanılan kapsayıcıları işlevsiz bir örnek olarak belgeleri C++ standart kitaplığı. Daha fazla bilgi için [C++ Standart Kitaplığı kapsayıcıları](../standard-library/stl-containers.md).

Denetlenen dizi için bir sabit ters yineleyici olarak hizmet verebilen bir nesneyi tanımlar.

## <a name="syntax"></a>Sözdizimi

```

typedef T8 const_reverse_iterator;
```

## <a name="remarks"></a>Açıklamalar

Açıklanmıştır belirtilmeyen türünün eşanlamlısı olarak burada `T8` (genellikle [reverse_iterator](../standard-library/container-class-reverse-iterator.md) <[const_iterator](../standard-library/container-class-const-iterator.md)`>`).
