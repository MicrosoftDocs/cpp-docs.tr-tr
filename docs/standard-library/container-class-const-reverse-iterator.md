# <a name="container-classconstreverseiterator"></a>Kapsayıcı Sınıfı::const_reverse_iterator

> [!NOTE]
> Bu konu, Visual C++ belge C++ Standart Kitaplığı'nda kullanılan kapsayıcıları işlevsel bir örnek olarak kullanılıyor. Daha fazla bilgi için bkz: [C++ Standart Kitaplığı kapsayıcıları](../standard-library/stl-containers.md).

Denetimli sırası için sabit bir ters yineleyici olarak hizmet verebilir nesneyi açıklar.

## <a name="syntax"></a>Sözdizimi

```

typedef T8 const_reverse_iterator;
```

## <a name="remarks"></a>Açıklamalar

Bunu açıklanan burada belirtilmemiş türler eşanlamlısı olarak **T8** (genellikle [reverse_iterator](../standard-library/container-class-reverse-iterator.md) <[const_iterator](../standard-library/container-class-const-iterator.md)`>`).
