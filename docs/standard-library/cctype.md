# <a name="ltcctypegt"></a>&lt;cctype&gt;

Standart C Kitaplığı üst bilgisini içerir \<ctype.h> ve ilgili adları `std` ad alanına ekler.

## <a name="requirements"></a>Gereksinimler

**Üst bilgi:**\<cctype>

**Ad alanı:** std

## <a name="remarks"></a>Açıklamalar

Bu üstbilginin dahil edilmesi, standart C Kitaplığı üstbilgisinde harici bağlantı kullanılarak belirtilen adların `std` ad alanında bildirilmesini sağlar.

## <a name="functions"></a>İşlevler

```cpp
int isalnum(int c);
int isalpha(int c);
int isblank(int c);
int iscntrl(int c);
int isdigit(int c);
int isgraph(int c);
int islower(int c);
int isprint(int c);
int ispunct(int c);
int isspace(int c);
int isupper(int c);
int isxdigit(int c);
int tolower(int c);
int toupper(int c);
```
