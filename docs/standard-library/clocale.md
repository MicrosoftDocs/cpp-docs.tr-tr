# <a name="ltclocalegt"></a>&lt;clocale&gt;
Standart C Kitaplığı üstbilgi içeren \<locale.h > ve ilişkili adlarıyla ekler `std` ad alanı.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
#include <clocale>  
  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bu üst dahil olmak üzere sağlar standart C Kitaplığı üstbilgisinde dış bağlantı kullanılarak bildirilen adları bildirilir `std` ad alanı.
