# <a name="container-classsizetype"></a>Kapsayıcı Sınıfı::size_type

> [!NOTE]
> Bu konu, Visual C++ belge C++ Standart Kitaplığı'nda kullanılan kapsayıcıları işlevsel bir örnek olarak kullanılıyor. Daha fazla bilgi için bkz: [C++ Standart Kitaplığı kapsayıcıları](../standard-library/stl-containers.md).

Denetlenen herhangi sırası uzunluğu temsil eden bir nesne açıklar.

## <a name="syntax"></a>Sözdizimi

```

typedef T0 size_type;
```

## <a name="remarks"></a>Açıklamalar

Bunu açıklanan burada belirtilmemiş türler eşanlamlısı olarak **T0** (genellikle **Alloc::size_type**).
