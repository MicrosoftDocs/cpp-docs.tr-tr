# <a name="regex_traitsltwchar_tgt-class"></a>regex_traits &lt;wchar_t &gt; sınıfı

**Wchar_t**için `regex_traits` özelleştirmesi.

## <a name="syntax"></a>Sözdizimi

```cpp
template <>
class regex_traits<wchar_t>
```

## <a name="remarks"></a>Açıklamalar

Sınıfı, **wchar_t** türü öğeler için [regex_traits](../standard-library/regex-traits-class.md) sınıfının açık bir özelleştirmesi olur (Bu sayede, bu türden nesneleri işleyen kitaplık işlevlerinden yararlanabilir).

## <a name="requirements"></a>Gereksinimler

**Üst bilgi:** \<regex >

**Ad alanı:** std
