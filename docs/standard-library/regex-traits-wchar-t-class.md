# <a name="regextraitsltwchartgt-class"></a>regex_traits&lt;wchar_t&gt; sınıfı

Regex_traits wchar_t için alt uzmanlaşması.

## <a name="syntax"></a>Sözdizimi

```cpp
template <>
class regex_traits<wchar_t>
```
