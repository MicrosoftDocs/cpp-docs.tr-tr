# <a name="regex_traitsltchargt-class"></a>regex_traits &lt;char &gt; sınıfı

**Char**için `regex_traits` özelleştirmesi.

## <a name="syntax"></a>Sözdizimi

```cpp
template <>
class regex_traits<char>
```

## <a name="remarks"></a>Açıklamalar

Sınıfı, **char** türündeki öğeler için [regex_traits](../standard-library/regex-traits-class.md) sınıfının açık bir özelleştirmesi olur (Bu sayede, bu türden nesneleri işleyen kitaplık işlevlerinden yararlanabilir).

## <a name="requirements"></a>Gereksinimler

**Üst bilgi:** \<regex >

**Ad alanı:** std
