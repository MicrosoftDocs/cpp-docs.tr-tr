# <a name="container-classdifferencetype"></a>Kapsayıcı Sınıfı::difference_type
> [!NOTE]
>  Bu konu, Visual C++ belge C++ Standart Kitaplığı'nda kullanılan kapsayıcıları işlevsel bir örnek olarak kullanılıyor. Daha fazla bilgi için bkz: [C++ Standart Kitaplığı kapsayıcıları](../standard-library/stl-containers.md).  
  
 Denetlenen sıradaki herhangi iki öğe adreslerini arasındaki farkı temsil eden bir nesne açıklar.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
 
typedef T1 difference_type;  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bunu açıklanan burada belirtilmemiş türler eşanlamlısı olarak **T1** (genellikle **Alloc::difference_type**).
