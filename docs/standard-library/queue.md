# <a name="ltqueuegt"></a>&lt;Sıradaki&gt;

Sınıf şablonları priority_queue ve sırası ve birkaç destekleyici şablonu tanımlar.

## <a name="requirements"></a>Gereksinimler

**Üst bilgi:**\<queue>

**Ad alanı:** std

> [!NOTE]
> \<queue>Kitaplık, ifadesini de kullanır `#include <initializer_list>` .
