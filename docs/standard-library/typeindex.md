# <a name="lttypeindexgt"></a>&lt;typeındex&gt;

[Type_info](../cpp/type-info-class.md)sınıfının Dizin oluşturmayı \<destekleyen bir sınıf ve işlev tanımlamak için standart üst bilgi typeındex > ekleyin.

## <a name="syntax"></a>Sözdizimi

```cpp
#include <typeindex>
```

## <a name="remarks"></a>Açıklamalar

[Karma yapısı](../standard-library/hash-structure.md) , type_index türündeki `hash function` değerleri dizin değerlerinin dağıtımına uygun bir şekilde tanımlar [](../standard-library/type-index-class.md) .

Sınıfı `type_index` , dizin oluşturmaya yardımcı olmak için `type_info` bir nesneye bir işaretçi sarmalar.
