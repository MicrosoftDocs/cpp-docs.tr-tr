# <a name="ltstackgt"></a>&lt;yığın&gt;

Sınıf şablonu yığınını ve iki destekleyici şablonu tanımlar.

## <a name="requirements"></a>Gereksinimler

**Üst bilgi:**\<stack>

**Ad alanı:** std

> [!NOTE]
> \<stack>Kitaplık, ifadesini de kullanır `#include <initializer_list>` .
