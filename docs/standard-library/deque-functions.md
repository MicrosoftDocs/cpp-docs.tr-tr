##  <a name="swap">Değiştirme</a>  
 İki deques öğelerini değiş tokuş eder.  
  
```
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```  
  
### <a name="parameters"></a>Parametreler  
 `left`  
 Türünde bir nesne `deque`.  
  
 `right`  
 Türünde bir nesne `deque`.  
  
### <a name="example"></a>Örnek
