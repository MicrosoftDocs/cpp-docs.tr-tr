# <a name="ltcvtwstringgt"></a>&lt;CVT/wstring&gt;

Visual Studio `<cvt/wstring>` 'nun önceki sürümlerindeki üst bilgi, stdext:: CVT ad alanındaki Template Class [wstring_convert sınıfını](../standard-library/wstring-convert-class.md) tanımladı. Üst bilgi geriye dönük uyumluluk için korunur. Yeni kod, `std` ad alanındaki [ \<> yerel ayarda](../standard-library/locale.md) tanımlanan sınıfın sürümünü kullanmalıdır

## <a name="syntax"></a>Sözdizimi

```cpp
#include <cvt/wstring>
```
