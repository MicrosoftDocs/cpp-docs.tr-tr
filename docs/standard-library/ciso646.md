# <a name="ltciso646gt"></a>&lt;ciso646&gt;

Standart C Kitaplığı üstbilgisi içerir \<iso646.h > ve ilişkili adlar ekler `std` ad alanı.

## <a name="syntax"></a>Sözdizimi

```cpp
#include <ciso646>
```

## <a name="remarks"></a>Açıklamalar

Bu üstbilginin dahil sağlar, standart C Kitaplığı üstbilgisinde harici bağlantı kullanılarak bildirilen adların bildirilir `std` ad alanı.
