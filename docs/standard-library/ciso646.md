# <a name="ltciso646gt"></a>&lt;ciso646&gt;

C standart kitaplığı üst bilgisini içerir \<iso646.h> ve ilgili adları `std` ad alanına ekler.

> [!NOTE]
> Bu uyumluluk üst bilgisi C++ ' ta anahtar sözcük olan adları tanımladığından, bu da bir etkiye sahip değildir. \<iso646.h>Üst bilgi C++ ' da kullanımdan kaldırılmıştır. \<ciso646>Üst bilgi, taslak c++ 20 standardına göre kaldırılır.

## <a name="requirements"></a>Gereksinimler

**Üst bilgi:**\<ciso646>

**Ad alanı:** std

## <a name="remarks"></a>Açıklamalar

Bu üst bilgiyi dahil etmek, C standart kitaplığı üstbilgisinde harici bağlantı kullanılarak belirtilen adların `std` ad alanında bildirilmesini sağlar.
