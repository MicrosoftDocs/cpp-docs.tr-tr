# <a name="ltciso646gt"></a>&lt;ciso646&gt;

C standart kitaplığı üst bilgisini \<iso646. h > içerir ve ilişkili adları `std` ad alanına ekler.

> [!NOTE]
> Bu uyumluluk üst bilgisi içinde C++anahtar sözcük olan adları tanımladığından, bu da dahil değildir. İso646. h > üstbilgisi ' de C++kullanım dışıdır. \< \<Ciso646 > üstbilgisi taslak c++ 20 standardına göre kaldırılır.

## <a name="requirements"></a>Gereksinimler

**Üst bilgi:** \<ciso646 >

**Ad alanı:** std

## <a name="remarks"></a>Açıklamalar

Bu üst bilgiyi dahil etmek, C standart kitaplığı üstbilgisinde harici bağlantı kullanılarak belirtilen adların `std` ad alanında bildirilmesini sağlar.
