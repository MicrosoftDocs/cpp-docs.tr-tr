# <a name="ltciso646gt"></a>&lt;ciso646&gt;
Standart C Kitaplığı üstbilgi içeren \<iso646.h > ve ilişkili adlarıyla ekler `std` ad alanı.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
#include <ciso646>  
  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bu üst dahil olmak üzere sağlar standart C Kitaplığı üstbilgisinde dış bağlantı kullanılarak bildirilen adları bildirilir `std` ad alanı.
