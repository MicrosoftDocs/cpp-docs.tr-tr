# <a name="trytolockt-structure"></a>try_to_lock_t Yapı

Bir [try_to_lock](../standard-library/mutex-functions.md#try_to_lock) nesnesini tanımlayan bir **struct** temsil eder. [Unique_lock](../standard-library/unique-lock-class.md)'ın aşırı yüklenmiş oluşturucularından birini seçmek için kullanılır.

## <a name="syntax"></a>Sözdizimi

```cpp
struct try_to_lock_t;
```

## <a name="requirements"></a>Gereksinimler

**Üst bilgi:** \<mutex >

**Ad alanı:** std
