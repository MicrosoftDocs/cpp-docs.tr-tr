# <a name="ltlistgt"></a>&lt;list&gt;

Kapsayıcı sınıfı şablon listesini ve çeşitli destekleyici şablonları tanımlar.

## <a name="syntax"></a>Sözdizimi

```cpp
#include <list>
```

> [!NOTE]
> @No__t_0list > kitaplığı `#include <initializer_list>` ifadesini de kullanır.
