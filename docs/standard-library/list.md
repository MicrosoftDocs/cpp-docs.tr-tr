# <a name="ltlistgt"></a>&lt;list&gt;

Kapsayıcı sınıfı şablon listesini ve çeşitli destekleyici şablonları tanımlar.

## <a name="syntax"></a>Sözdizimi

```cpp
#include <list>
```

> [!NOTE]
> \<listesi > kitaplığı `#include <initializer_list>` ifadesini de kullanır.
