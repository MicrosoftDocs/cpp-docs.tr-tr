# <a name="ltlistgt"></a>&lt;list&gt;

Kapsayıcı Şablon sınıfı listesini ve çeşitli destek şablonları tanımlar.

## <a name="syntax"></a>Sözdizimi

```cpp
#include <list>
```

> [!NOTE]
> \<Listesi > Kitaplığı'nı da kullanan `#include <initializer_list>` deyimi.
