# <a name="ltlistgt"></a>&lt;list&gt;

Kapsayıcı şablonu sınıf listesini ve çeşitli destekleyici şablonları tanımlar.

## <a name="syntax"></a>Sözdizimi

```cpp
#include <list>
```

> [!NOTE]
> Liste > kitaplığı, `#include <initializer_list>` ifadesini de kullanır. \<
