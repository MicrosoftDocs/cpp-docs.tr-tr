# <a name="ltdequegt"></a>&lt;deque&gt;

Kapsayıcı Şablon sınıfı deque ve çeşitli destek şablonları tanımlar.

## <a name="requirements"></a>Gereksinimler

**Üst bilgi**: \<deque >

> [!NOTE]
> \<Deque > Kitaplığı'nı da kullanan `#include <initializer_list>` deyimi.
