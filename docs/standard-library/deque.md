# <a name="ltdequegt"></a>&lt;deque&gt;

Kapsayıcı Şablon sınıfı deque ve çeşitli destek şablonları tanımlar.

## <a name="syntax"></a>Sözdizimi

```cpp
#include <deque>

```

### <a name="operators"></a>İşleçler

|İşleç|Açıklama|
|-|-|
|[operator!=](../standard-library/deque-operators.md#op_neq)|İşlecin sol tarafındaki deque nesne işlecin sağ tarafındaki deque nesneye eşit olup olmadığını sınar.|
|[işleç <](../standard-library/deque-operators.md#op_lt)|İşlecin sol tarafındaki deque nesnesinin işlecin sağ tarafındaki deque nesneden küçük olup olmadığını sınar.|
|[İşleci\<=](../standard-library/deque-operators.md#op_gt_eq)|Deque işlecinin sol tarafında nesne küçük olup olmadığını sınar deque nesnesine eşit veya işlecin sağ tarafındaki.|
|[operator==](../standard-library/deque-operators.md#op_eq_eq)|İşlecin sol tarafındaki deque nesnesinin işlecin sağ tarafındaki deque nesneye eşit olup olmadığını sınar.|
|[operator >](../standard-library/deque-operators.md#op_gt)|İşlecin sol tarafındaki deque nesnesinin işlecin sağ tarafındaki deque nesneden büyük olup olmadığını sınar.|
|[operator>=](../standard-library/deque-operators.md#op_gt_eq)|İşlecin sol tarafındaki deque nesnesinin değerinden büyük veya işlecin sağ tarafındaki deque nesneye eşit olup olmadığını sınar.|
