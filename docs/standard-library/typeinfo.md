# <a name="lttypeinfogt"></a>&lt;TypeInfo&gt;

\<typeinfo>Hem statik hem de dinamik türler hakkında bilgi veren tür tanımlama operatörü [TypeId işleciyle](../cpp/typeid-operator.md)ilişkili çeşitli türleri tanımlamak için standart üstbilgiyi ekleyin.

## <a name="syntax"></a>Syntax

```cpp
#include <typeinfo>
```
