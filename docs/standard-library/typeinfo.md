# <a name="lttypeinfogt"></a>&lt;typeinfo&gt;

Standart üst bilgiyi dahil \<typeinfo > türü kimlik işleci ile ilgili çeşitli türleri tanımlamak için [typeid işleci](../cpp/typeid-operator.md), hem statik hem de dinamik türleri hakkında bilgi verir.

## <a name="syntax"></a>Sözdizimi

```cpp
#include <typeinfo>

```

- [bad_typeid Özel Durumu](../cpp/bad-typeid-exception.md)
