# <a name="ltcuchargt"></a>&lt;cuchar&gt;

C standart kitaplığı üst bilgisini içerir \<uchar.h> ve ilgili adları `std` ad alanına ekler.

## <a name="requirements"></a>Gereksinimler

**Üst bilgi:**\<cuchar>

**Ad alanı:** std

## <a name="remarks"></a>Açıklamalar

Bu üst bilgi, C standart kitaplık türlerini `mbstate_t` ve `size_t` ve [c16rtomb](../c-runtime-library/reference/c16rtomb-c32rtomb1.md), [c32rtomb](../c-runtime-library/reference/c16rtomb-c32rtomb1.md), [mbrtoc16](../c-runtime-library/reference/mbrtoc16-mbrtoc323.md)ve [mbrtoc32](../c-runtime-library/reference/mbrtoc16-mbrtoc323.md) işlevlerini `std` ad alanına ekler.
