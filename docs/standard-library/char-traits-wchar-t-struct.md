# <a name="chartraitsltwchartgt-struct"></a>char_traits&lt;wchar_t&gt; yapısı

Şablon yapı özelleştirmesi bir sınıfı **char_traits\<CharType >** türünde bir öğe için **wchar_t**.

## <a name="syntax"></a>Sözdizimi

```cpp
template <>
struct char_traits<wchar_t>;
```

## <a name="remarks"></a>Açıklamalar

Bu tür nesneler düzenleme kitaplığı işlevleri yararlanmak yapı özelleştirmesi sağlar **wchar_t**.

## <a name="requirements"></a>Gereksinimler

**Başlık:** \<dizesi >

**Namespace:** std
