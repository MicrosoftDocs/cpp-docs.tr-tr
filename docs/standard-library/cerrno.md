# <a name="ltcerrnogt"></a>&lt;cerrno&gt;

C standart kitaplığı üst bilgisini içerir \<errno.h> ve ilgili adları `std` ad alanına ekler. Bu üst bilgiyi dahil etmek, C standart kitaplığı üstbilgisinde harici bağlantı kullanılarak belirtilen adların `std` ad alanında bildirilmesini sağlar.

## <a name="syntax"></a>Syntax

```cpp
#include <cerrno>
```
