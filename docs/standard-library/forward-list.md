# <a name="ltforward_listgt"></a>&lt;forward_list&gt;

Forward_list kapsayıcı sınıfı şablonunu ve çeşitli destekleyici şablonları tanımlar.

## <a name="requirements"></a>Gereksinimler

**Üst bilgi:**\<forward_list>

**Ad alanı:** std

> [!NOTE]
> \<forward_list>Kitaplık, ifadesini de kullanır `#include <initializer_list>` .
