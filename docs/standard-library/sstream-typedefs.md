## <a name="istringstream"></a>istringstream

Bir **char** Şablon parametresinde özelleştirilmiş `basic_istringstream` bir tür oluşturur.

```cpp
typedef basic_istringstream<char> istringstream;
```

### <a name="remarks"></a>Açıklamalar

Tür, **char**türündeki öğeler için özelleştirilmiş [basic_istringstream](../standard-library/basic-istringstream-class.md)sınıf şablonu için bir eş anladır.
