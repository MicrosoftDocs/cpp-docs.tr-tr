##  <a name="istringstream"></a>istringstream  
 Bir türü oluşturur `basic_istringstream` üzerinde özel bir `char` şablon parametresi.  
  
```  
typedef basic_istringstream<char> istringstream;  
```  
  
### <a name="remarks"></a>Açıklamalar  
 Şablon sınıfı için eş anlamlı türüdür [basic_istringstream](../standard-library/basic-istringstream-class.md), türündeki öğeler için özelleştirilmiş `char`.
