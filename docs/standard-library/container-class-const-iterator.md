# <a name="container-classconstiterator"></a>Kapsayıcı Sınıfı::const_iterator

> [!NOTE]
> Bu konu Microsoft olan C++ kullanılan kapsayıcıları işlevsiz bir örnek olarak belgeleri C++ standart kitaplığı. Daha fazla bilgi için [C++ Standart Kitaplığı kapsayıcıları](../standard-library/stl-containers.md).

Denetlenen dizi için sabit bir yineleyici olarak hizmet verebilen bir nesneyi tanımlar.

## <a name="syntax"></a>Sözdizimi

```

typedef T6 const_iterator;
```

## <a name="remarks"></a>Açıklamalar

Açıklanmıştır belirtilmeyen türünün eşanlamlısı olarak burada `T6`.
