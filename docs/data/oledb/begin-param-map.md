# <a name="beginparammap"></a>BEGIN_PARAM_MAP
Parametre eşleme girdilerini başlangıcını işaretler.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp
BEGIN_PARAM_MAP(x)  
  
```  
  
#### <a name="parameters"></a>Parametreler  
 *x*  
 [in] Kullanıcı kayıt sınıfı adı.  
  
## <a name="remarks"></a>Açıklamalar  
 Tarafından kullanılan parametreleri [komutları](https://msdn.microsoft.com/en-us/library/ms724608.aspx).  
  
## <a name="example"></a>Örnek  
 Örneğin bkz [BEGIN_COLUMN_MAP](../../data/oledb/begin-column-map.md) makrosu.
