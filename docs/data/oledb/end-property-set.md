# <a name="endpropertyset"></a>END_PROPERTY_SET
Bir özellik kümesi sonunu işaretler.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
END_PROPERTY_SET(  
guid   
)  
  
```  
  
#### <a name="parameters"></a>Parametreler  
 `guid`  
 [in] Özellik GUID.
