# <a name="endcolumnmap"></a>END_COLUMN_MAP
Sütun eşlemesi girişleri sonunu işaretler.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp
END_COLUMN_MAP()  
  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bir satır kümesi üzerinde tek bir erişimcisi ile kullanılır. `BEGIN_COLUMN_MAP` Makrosu ile tamamlandı `END_COLUMN_MAP` makrosu.
