# <a name="beginschemamap"></a>BEGIN_SCHEMA_MAP
Şema eşleme başlangıcını gösterir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp
      BEGIN_SCHEMA_MAP(SchemaClass);  
```  
  
#### <a name="parameters"></a>Parametreler  
 *SchemaClass*  
 Harita içeren sınıf. Genellikle bu oturumu sınıfı olacaktır.  
  
## <a name="remarks"></a>Açıklamalar  
 Bkz: [IDBSchemaRowset](https://msdn.microsoft.com/en-us/library/ms713686.aspx) şema satır kümeleri hakkında daha fazla bilgi için Windows SDK'sındaki.
