# <a name="cdynamicparameteraccessorgetparamcount"></a>CDynamicParameterAccessor::GetParamCount

## <a name="syntax"></a>Sözdizimi  
  
```  
  
DB_UPARAMS GetParamCount( ) const throw( );  
  
```  
  
## <a name="return-value"></a>Dönüş Değeri  
 Parametre sayısı.
