# <a name="beginpropertyset"></a>BEGIN_PROPERTY_SET
Bir özelliğin başına bir özellik Ayarla işaretleri eşlemesi ayarlayın.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp
BEGIN_PROPERTY_SET(guid)  
  
```  
  
#### <a name="parameters"></a>Parametreler  
 `guid`  
 [in] Özellik GUID.
