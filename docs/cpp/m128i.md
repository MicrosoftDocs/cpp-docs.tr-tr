# <a name="__m128i"></a>__m128i

**Microsoft 'a özgü**

Streaming SIMD Extensions 2 (SSE2) yönergeleri iç bilgileri ile kullanılmak üzere **__m128i** veri türü \<emmintrin. h > içinde tanımlanmıştır.

```cpp
// data_types__m128i.cpp
#include <emmintrin.h>
int main() {
   __m128i x;
}
```

## <a name="remarks"></a>Açıklamalar

**__M128i** alanlarına doğrudan erişmemelisiniz. Ancak, hata ayıklayıcıda bu türlere bakabilirsiniz. **__M128i** bir değişken, XMM [0-7] Yazmaçları ile eşlenir.

**_M128i** türündeki değişkenler otomatik olarak 16 baytlık sınırlarda hizalanır.

> [!NOTE]
>  **__M128i** türündeki değişkenlerin KULLANıLMASı derleyicinin SSE2 `movdqa` yönergesini oluşturmasına neden olur. Bu yönerge, Pentium III işlemcilerde hatalara neden olmaz, ancak, `movdqa` her türlü yönerge, Pentium III işlemcilerde ' a dönüştürdüğünde olası yan etkileri olan sessiz hata oluşmasına neden olur.

**__M128i** VERI türü ARM işlemcilerde desteklenmez.
