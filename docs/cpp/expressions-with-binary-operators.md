-   [Çarpma işleçleri](../cpp/multiplicative-operators-and-the-modulus-operator.md)  
  
    -   Çarpma (*)  
  
    -   Bölme (/)  
  
    -   Mod (%)  
  
-   [Toplama işleçleri](../cpp/additive-operators-plus-and.md)

    -   Çıkarma (-)
