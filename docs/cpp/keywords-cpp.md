## <a name="microsoft-specific"></a>Microsoft'a Özgü  
 Microsoft C++'da iki başında alt çizgi ile tanımlayıcıları derleyici uygulamaları için ayrılmıştır. Bu nedenle, Microsoft çift alt çizgi ile Microsoft'a özgü anahtar sözcükler öncesinde kuraldır. Bu sözcükleri tanımlayıcı adları olarak kullanılamaz.  
  
 Microsoft uzantıları varsayılan olarak etkindir. Programlarınızı tümüyle taşınabilir olduğundan emin olmak için Microsoft uzantıları ANSI uyumlu belirterek devre dışı bırakabilirsiniz [/Za](../build/reference/za-ze-disable-language-extensions.md) derleme sırasında komut satırı seçeneği (derleme ANSI uyumluluğu için). Bunu yaptığınızda, Microsoft'a özgü anahtar sözcükler devre dışı bırakılır.  
  
 Microsoft uzantıları etkinleştirildiğinde, Microsoft'a özgü anahtar sözcükler, programlarında kullanabilirsiniz. ANSI uyumluluğu için bu anahtar sözcükler çift çizgiyle başlar. Geriye dönük uyumluluk, tüm çift alt çizgisi olan anahtar dışında tek alt çizgi sürümleri için `__except`, `__finally`, `__leave`, ve `__try` desteklenir. Ayrıca, `__cdecl` başında hiçbir alt çizgi ile kullanılabilir.  
  
 `__asm` Anahtar sözcüğü değiştirir C++ `asm` sözdizimi. `asm` diğer C++ uygulamaları ile uyumluluk için ayrılmış, ancak uygulanmadı. Kullanım `__asm`.  
  
 `__based` Anahtar sözcüğü kullanan 32 bit ve 64 bit hedef derlemelerine sınırlı.
