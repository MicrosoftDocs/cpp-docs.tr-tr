# <a name="w64"></a>__w64

**Microsoft özel** 

Bu anahtar sözcük kullanılmıyor. Böylece ile derleme zaman Visual Studio 2013'ten önceki Visual Studio sürümlerinde bu değişkenleri, işaretlemek sağlar [/Wp64](../build/reference/wp64-detect-64-bit-portability-issues.md) derleyici ile 64 bit derleyici derleme durumunda bildirilen tüm uyarılar bildirir.  
  
## <a name="syntax"></a>Sözdizimi  
  
> *tür* **__w64** *tanımlayıcısı*  
 
  
### <a name="parameters"></a>Parametreler  

*Türü*  
 Bir 32 bit bir 64 bit derleyici bağlantı noktası kurulmuş kodda sorunlara neden olabilecek üç tür: `int`, `long`, veya bir işaretçi.  
  
*Tanımlayıcı*  
Oluşturmakta olduğunuz değişkeni tanımlayıcısı.  
  
## <a name="remarks"></a>Açıklamalar  
  
> [!IMPORTANT]
>  [/Wp64](../build/reference/wp64-detect-64-bit-portability-issues.md) derleyici seçeneği ve `__w64` anahtar sözcüğü Visual Studio 2010 ve Visual Studio 2013'ün de kullanım dışıdır ve Visual Studio 2013'te başlayarak kaldırıldı. Kullanırsanız `/Wp64` derleyici seçeneği komut satırında derleyici sorunları [komut satırı uyarısı D9002](http://msdn.microsoft.com/en-us/c58b405b-0f26-434e-b57f-4f05e1ca81e6). `__w64` Anahtar sözcüğü sessizce göz ardı edilir. Bu seçenek ve anahtar sözcüğü 64 bit taşınabilirlik sorunlarını algılamak için kullanmak yerine, bir 64-bit platformu hedefleyen Visual C++ Derleyici kullanın. Daha fazla bilgi için bkz: [yapılandırma Visual C++ 64-bit, x64 için hedefleri](../build/configuring-programs-for-64-bit-visual-cpp.md).  
  
 Sahip typedef `__w64` üzerinde bu x86 üzerinde 32 bit ve 64 bit x64 üzerinde olmalıdır.  
  
 Visual C++ Derleyici Visual Studio 2010'den önceki sürümleri kullanılarak taşınabilirlik sorunlarını Algıla için `__w64` anahtar sözcüğü, 32 bit ve 64 bit platformları arasında boyutunu değiştirme herhangi bir tür tanımları üzerinde belirtilmelidir. Böyle herhangi bir tür için `__w64` yalnızca TypeDef 32-bit tanımını görünmesi gerekir.  
  
 `__w64` Anahtar sözcüğü derleme kullanmıyorsa yoksayılır `/Wp64`.  
  
 64 bit için bağlantı noktası oluşturma hakkında daha fazla bilgi için aşağıdaki konulara bakın:  
  
-   [Derleyici Seçenekleri](../build/reference/compiler-options.md)  
  
-   [32 Bit kodu 64 Bit koda bağlantı noktası oluşturma](../build/common-visual-cpp-64-bit-migration-issues.md)  
  
-   [Visual C++’ı 64 bit, x64 hedefler için yapılandırma](../build/configuring-programs-for-64-bit-visual-cpp.md)  
  
## <a name="example"></a>Örnek  
  
```cpp  
// __w64.cpp  
// compile with: /W3 /Wp64  
typedef int Int_32;  
#ifdef _WIN64  
typedef __int64 Int_Native;  
#else  
typedef int __w64 Int_Native;  
#endif  
  
int main() {  
   Int_32 i0 = 5;  
   Int_Native i1 = 10;  
   i0 = i1;   // C4244 64-bit int assigned to 32-bit int  
  
   // char __w64 c;  error, cannot use __w64 on char  
}  
```
