# <a name="w64"></a>__w64

**Microsoft'a özgü** 

Bu anahtar sözcüğü artık kullanılmıyor. İle derleme yaparken, böylece Visual Studio 2013'den önceki Visual Studio sürümlerinde bu değişkenleri işaretlemek sayesinde [/Wp64](../build/reference/wp64-detect-64-bit-portability-issues.md) derleyici bir 64 bit derleyici ile derleme bildirilebilecek uyarıları bildirir.  
  
## <a name="syntax"></a>Sözdizimi  
  
> *tür* **__w64** *tanımlayıcısı*  
 
### <a name="parameters"></a>Parametreler  

*Türü*  
 Bir 64 bit derleyici için bir 32-bit Taşınmakta kodda sorunlara neden olabilecek üç türlerden biri: **int**, **uzun**, veya bir işaretçi.  
  
*tanımlayıcı*  
Oluşturmakta olduğunuz değişken tanımlayıcısı.  
  
## <a name="remarks"></a>Açıklamalar  
  
> [!IMPORTANT]
>  [/Wp64](../build/reference/wp64-detect-64-bit-portability-issues.md) derleyici seçeneği ve **__w64** anahtar sözcüğü, Visual Studio 2010 ve Visual Studio 2013'ün kullanım dışı ve Visual Studio 2013 itibariyle kaldırıldı. Kullanırsanız `/Wp64` derleyici seçeneği komut satırında derleyici sorunları [komut satırı uyarısı D9002](https://msdn.microsoft.com/c58b405b-0f26-434e-b57f-4f05e1ca81e6). **__W64** anahtar sözcüğü sessizce yoksayılır. Bu seçenek ve anahtar sözcüğü 64-bit taşınabilirlik sorunlarını algılamak için kullanmak yerine 64-bit platformu hedefleyen bir Visual C++ Derleyici kullanın. Daha fazla bilgi için [yapılandırma Visual C++ 64 bit x64 için hedefleri](../build/configuring-programs-for-64-bit-visual-cpp.md).  
  
 Sahip herhangi bir typedef **__w64** x86 32 bit ve 64 bit x64 açık olmalıdır.  
  
 Visual Studio 2010'dan önceki Visual C++ Derleyici sürümleri kullanılarak taşınabilirlik sorunlarını algılamak için **__w64** anahtar sözcüğü, 32 bit ve 64 bit platformları arasında boyutunu değiştirme herhangi bir tür tanımları üzerinde belirtilmelidir. Bu tür herhangi bir tür için **__w64** yalnızca typedef, 32-bit tanımını yer almalıdır.  
  
 **__W64** derleme kullanmıyorsa anahtar sözcüğünü sayılır `/Wp64`.  
  
 64-bit için bağlantı noktası oluşturma hakkında daha fazla bilgi için aşağıdaki konulara bakın:  
  
-   [Derleyici Seçenekleri](../build/reference/compiler-options.md)  
  
-   [32-Bit kodu 64-Bit kod bağlantı noktası oluşturma](../build/common-visual-cpp-64-bit-migration-issues.md)  
  
-   [Visual C++’ı 64 bit, x64 hedefler için yapılandırma](../build/configuring-programs-for-64-bit-visual-cpp.md)  
  
## <a name="example"></a>Örnek  
  
```cpp  
// __w64.cpp  
// compile with: /W3 /Wp64  
typedef int Int_32;  
#ifdef _WIN64  
typedef __int64 Int_Native;  
#else  
typedef int __w64 Int_Native;  
#endif  
  
int main() {  
   Int_32 i0 = 5;  
   Int_Native i1 = 10;  
   i0 = i1;   // C4244 64-bit int assigned to 32-bit int  
  
   // char __w64 c;  error, cannot use __w64 on char  
}  
```
