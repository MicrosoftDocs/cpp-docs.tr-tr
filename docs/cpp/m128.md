# <a name="m128"></a>__m128
## <a name="microsoft-specific"></a>Microsoft'a Özgü  
 **__M128 data** , Streaming SIMD Extensions 2 yönergeleri yapı içleri ve Akış SIMD uzantıları kullanmak için türü tanımlanmış \<xmmintrin.h >.  
  
```cpp 
// data_types__m128.cpp  
#include <xmmintrin.h>  
int main() {  
   __m128 x;  
}  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Değil erişmeli **__m128** alanlarına doğrudan erişmemelisiniz. Ancak, bu türleri hata ayıklayıcıda görebilirsiniz. Türünde bir değişken **__m128** XMM [0-7] kayıtları eşlenir.  
  
 Türü değişkenlerindeki **_m128** otomatik olarak 16 baytlık sınırlara hizalanır.  
  
 **__M128** veri türü ARM işlemciler üzerinde desteklenmiyor.  
  
**END Microsoft özgü**
