# <a name="m128"></a>__m128
## <a name="microsoft-specific"></a>Microsoft'a Özgü  
 **__M128 veri** türü Streaming SIMD Extensions 2 yönergeleri iç bilgileri ve Streaming SIMD uzantıları ile kullanmak için tanımlanmış \<xmmintrin.h >.  
  
```  
// data_types__m128.cpp  
#include <xmmintrin.h>  
int main() {  
   __m128 x;  
}  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Değil erişmelisiniz `__m128` doğrudan alanları. Ancak, bu tür hata ayıklayıcısında görebilirsiniz. Türünde bir değişken `__m128` XMM [0-7] kayıtlarını eşlenir.  
  
 Türündeki değişkenler **_m128** 16 bayt sınırları otomatik olarak hizalanır.  
  
 `__m128` ARM işlemci veri türü desteklenmiyor.
