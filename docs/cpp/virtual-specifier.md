# <a name="virtual-specifier"></a>virtual Tanımlayıcısı

[Sanal](../cpp/virtual-cpp.md) anahtar sözcüğü yalnızca statik olmayan sınıf üyesi işlevlerine uygulanabilir. Çağrıların işleve bağlanmasının, çalışma zamanına kadar ertelendiğini belirtir. Daha fazla bilgi için bkz. [sanal işlevler](../cpp/virtual-functions.md).
