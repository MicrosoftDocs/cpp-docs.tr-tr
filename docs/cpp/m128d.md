# <a name="m128d"></a>__m128d
## <a name="microsoft-specific"></a>Microsoft'a Özgü  
 **__M128d** , Streaming SIMD Extensions 2 yönergeleri yapı içleri ile kullanılacak veri türü tanımlanmış \<emmintrin.h >.  
  
```cpp 
// data_types__m128d.cpp  
#include <emmintrin.h>  
int main() {  
   __m128d x;  
}  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Değil erişmeli **__m128d** alanlarına doğrudan erişmemelisiniz. Ancak, bu türleri hata ayıklayıcıda görebilirsiniz. Türünde bir değişken **__m128** XMM [0-7] kayıtları eşlenir.  
  
 Türü değişkenlerindeki **_m128d** otomatik olarak 16 baytlık sınırlara hizalanır.  
  
 **__M128d** veri türü ARM işlemciler üzerinde desteklenmiyor.  
  
**END Microsoft özgü**
