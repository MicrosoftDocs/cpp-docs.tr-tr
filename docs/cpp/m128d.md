# <a name="m128d"></a>__m128d
## <a name="microsoft-specific"></a>Microsoft'a Özgü  
 `__m128d` Streaming SIMD Extensions 2 yönergeleri iç bilgileri ile kullanmak için veri türü tanımlanmış \<emmintrin.h >.  
  
```  
// data_types__m128d.cpp  
#include <emmintrin.h>  
int main() {  
   __m128d x;  
}  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Değil erişmelisiniz `__m128d` doğrudan alanları. Ancak, bu tür hata ayıklayıcısında görebilirsiniz. Türünde bir değişken `__m128` XMM [0-7] kayıtlarını eşlenir.  
  
 Türündeki değişkenler **_m128d** 16 bayt sınırları otomatik olarak hizalanır.  
  
 `__m128d` ARM işlemci veri türü desteklenmiyor.
