# <a name="jump-statements-c"></a>Atlama Deyimleri (C++)
C++ atlama deyimi, denetimin anlık bir yerel aktarmasını gerçekleştirir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
