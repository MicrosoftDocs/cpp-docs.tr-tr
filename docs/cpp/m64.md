# <a name="m64"></a>__m64

## <a name="microsoft-specific"></a>Microsoft'a Özgü

**__M64** veri türü olan 3DNow ve MMX ile kullanım için! Yapı ve tanımlanan \<xmmintrin.h >.

```cpp
// data_types__m64.cpp
#include <xmmintrin.h>
int main()
{
   __m64 x;
}
```

## <a name="remarks"></a>Açıklamalar

Değil erişmeli **__m64** alanlarına doğrudan erişmemelisiniz. Ancak, bu türleri hata ayıklayıcıda görebilirsiniz. Türünde bir değişken **__m64** MM [0-7] kayıtları eşlenir.

Türü değişkenlerindeki **_m64** otomatik olarak 8 baytlık sınırlara hizalanır.

**__M64** veri türü üzerinde x64 desteklenmiyor işlemci. Eşdeğer SSE ve SSE2 yapı içleri kullanılacak __m64 MMX ön tanımlı bir parçası olarak kullanan uygulamaların yeniden yazılması gerekir.
