# <a name="m64"></a>__m64
## <a name="microsoft-specific"></a>Microsoft'a Özgü  
 `__m64` Veri türü olan MMX ve 3DNow ile kullanım için! İç bilgiler ve xmmintrin.h tanımlanır.  
  
```  
// data_types__m64.cpp  
#include <xmmintrin.h>  
int main()  
{  
   __m64 x;  
}  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Değil erişmelisiniz `__m64` doğrudan alanları. Ancak, bu tür hata ayıklayıcısında görebilirsiniz. Türünde bir değişken `__m64` MM [0-7] kayıtlarını eşlenir.  
  
 Türündeki değişkenler **_m64** 8-bayt sınırları otomatik olarak hizalanır.  
  
 `__m64` Veri türü desteklenmiyor [!INCLUDE[vcprx64](../assembler/inline/includes/vcprx64_md.md)] işlemci. __M64 MMX ön tanımlı bir parçası olarak kullanan uygulamalar eşdeğer SSE ve SSE2 iç bilgileri kullanmak için yazılması gerekir.
