# <a name="mul128"></a>__mul128
**Microsoft özel**  
  
 İlk iki bağımsız değişken olarak geçirilen iki 64-bit tamsayı çarpar ve ürünün yüksek 64 bit gösterdiği 64-bit tamsayı koyan `HighProduct` ve ürünün düşük 64 bit döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
__int64 _mul128(   
   __int64 Multiplier,   
   __int64 Multiplicand,   
   __int64 *HighProduct   
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 [in]`Multiplier`  
 Çarpılacağı ilk 64-bit tam sayı.  
  
 [in]`Multiplicand`  
 Çarpılacağı ikinci 64-bit tam sayı.  
  
 [out]`HighProduct`  
 Ürünün yüksek 64 bit.  
  
## <a name="return-value"></a>Dönüş Değeri  
 Ürünün düşük 64 bit.

 **Üstbilgi dosyası** \<intrin.h >  
  
## <a name="example"></a>Örnek  
  
```  
// mul128.c  
// processor: x64  
#include <stdio.h>  
#include <intrin.h>  
  
#pragma intrinsic(_mul128)  
  
int main()  
{  
    __int64 a = 0x0fffffffffffffffI64;  
    __int64 b = 0xf0000000I64;  
    __int64 c, d;  
  
    d = _mul128(a, b, &c);  
  
    printf_s("%#I64x * %#I64x = %#I64x%I64x\n", a, b, c, d);  
}  
```  
  
```Output  
0xfffffffffffffff * 0xf0000000 = 0xeffffffffffffff10000000  
```
