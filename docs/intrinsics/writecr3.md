# <a name="writecr3"></a>__writecr3
**Microsoft'a özgü**  
  
 Değeri Yazar `Data` CR3 kaydı.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
void writecr3(   
   unsigned __int64 Data   
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 [in] `Data`  
 CR3 kasaya yazılacak değer.  
  
## <a name="requirements"></a>Gereksinimler  
  
|İç|Mimari|  
|---------------|------------------|  
|`__writecr3`|x86, x64|
