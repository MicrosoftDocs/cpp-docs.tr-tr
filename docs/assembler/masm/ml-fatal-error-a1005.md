# <a name="ml-fatal-error-a1005"></a>ML Önemli Hatası A1005

**Derleyici sınırı: Makro parametre adı tablo dolu**

Bir makro için çok fazla parametre, Yereller veya makro etiketi tanımlandı. Makro adı tablosunda daha fazla yer yoktu.

Daha kısa veya daha az ad tanımlayın veya gereksiz makroları kaldırın.
