# <a name="operator-highword"></a>HIGHWORD işleci
Düşük 32 bit sayısı yüksek 16 bit döndürür *ifade*. MASM ifadeleri 64-bit değerlerdir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
HIGHWORD expression  
```
