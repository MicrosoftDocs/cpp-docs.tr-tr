# <a name="ml-fatal-error-a1017"></a>ML Önemli Hatası A1017

**eksik kaynak dosya adı**

ML, birleştirmek veya bağlayıcıya geçirilecek bir dosya bulamadı.

Bu hata, üzerinde işlem yapmak için bir dosya adı belirtmeden ML komut satırı seçenekleri verdiğinizde oluşturulur. . Asm uzantısı olmayan dosyaları birleştirmek için **/ta** komut satırı seçeneğini kullanın.

Bu hata, ML ortam değişkeni komut satırı seçeneklerini içeriyorsa, ML 'nin parametresi olmadan çağırarak da oluşturulabilir.
