# <a name="ml-fatal-error-a1017"></a>ML Önemli Hatası A1017
**eksik kaynak dosya adı**  
  
 ML birleştirin veya bağlayıcıya geçirme için bir dosya bulunamadı.  
  
 Bu hata, görev için bir dosya adı belirtmeden ML komut satırı seçenekleri verdiğinizde oluşturulur. Bir .asm uzantısı olmayan dosyaları birleştirmek için kullanın **/Ta** komut satırı seçeneği.  
  
 Bu hata, ML ortam değişkeni komut satırı seçenekleri içeriyorsa, ML parametresiz harekete geçirerek de oluşturulabilir.
