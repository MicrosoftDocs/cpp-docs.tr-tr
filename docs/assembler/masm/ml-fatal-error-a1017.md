# <a name="ml-fatal-error-a1017"></a>ML Önemli Hatası A1017

**Kaynak dosya adı eksik**

ML birleştirin veya bağlayıcıya geçirme için bir dosya bulunamadı.

ML komut satırı seçenekleri alacak bir dosya adı belirtmeden verdiğinizde, bu hata oluşturulur. .Asm uzantısı olmayan dosyaları birleştirmek için kullanın **/Ta** komut satırı seçeneği.

Bu hata, ML ortam değişkeni komut satırı seçenekleri içeriyorsa, ML hiçbir parametre olmadan harekete geçirerek de oluşturulabilir.
