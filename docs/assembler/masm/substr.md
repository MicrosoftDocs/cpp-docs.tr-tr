# <a name="substr"></a>SUBSTR

*Konumundan* başlayarak *textıtem*'ın alt dizesini döndürür. *Textıtem* bir sabit dize, önünde bir sabit `%` veya bir makro işlevi tarafından döndürülen dize olabilir.

## <a name="syntax"></a>Syntax

> *ad* **substr** *textıtem*__,__ *konum* ⟦__,__ *length*⟧
