# <a name="operator-lt"></a>LT işleci

(-1) true döndürür *İfade1* olduğu küçüktür *expression2*, ya da, değilse false (0) döndürür.

## <a name="syntax"></a>Sözdizimi

> İfade1 LT İfade2
