# <a name="operator-lt"></a>LT işleci
True değeri döndürür (-1) olmadığını *İfade1* olan değerinden *İfade2*, veya değilse false (0) döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
expression1 LT expression2  
```
