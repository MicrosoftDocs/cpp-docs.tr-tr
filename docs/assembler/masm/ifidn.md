# <a name="ifidn-ifidni"></a>IFIDN, IFIDNI

Metin öğelerini aynıysa derleme verir.

## <a name="syntax"></a>Sözdizimi

> **IFIDN**\[**miyim**] *textitem1* __,__ *textitem2*

## <a name="remarks"></a>Açıklamalar

Varsa **IFIDNI** olduğunu göz önünde bulundurulduğunda, karşılaştırma büyük/küçük harf ve büyük harflere duyarlı değildir. Bkz: [IF](../../assembler/masm/if-masm.md) tam söz dizimi.
