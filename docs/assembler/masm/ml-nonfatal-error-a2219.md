# <a name="ml-nonfatal-error-a2219"></a>ML Önemli Olmayan Hatası A2219

> Bırakma kodundaki uzaklığa yönelik hatalı hizalama

## <a name="remarks"></a>Açıklamalar

[&period;ALLOCSTACK](../../assembler/masm/dot-allocstack.md) ve [&period;savereg](../../assembler/masm/dot-savereg.md) işleneni, 8 ' in katı olmalıdır.  [&period;SAVEXMM128](../../assembler/masm/dot-savexmm128.md) ve [&period;SETFRAME](../../assembler/masm/dot-setframe.md) işleneni 16 ' dan fazla olmalıdır.
