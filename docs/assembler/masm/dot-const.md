# <a name="const-32-bit-masm"></a>. CONST (32-bit masa)

İle kullanıldığında [. MODEL](dot-model.md), sabit bir veri segmenti başlatır (kesim adı **const** ile).

## <a name="syntax"></a>Syntax

> **. SABIT**\
> ⟦ *segmentItem* ⟧...

## <a name="remarks"></a>Açıklamalar

Bu segmentin salt okunurdur özniteliği vardır.
