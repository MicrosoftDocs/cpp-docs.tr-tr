# <a name="group"></a>GROUP

(yalnızca 32-bit masa.) Belirtilen *kesimleri* *adı*adlı gruba ekleyin.

## <a name="syntax"></a>Sözdizimi

> *ad* **grubu** *segmenti* ⟦ __,__ *segment* ... ⟧
