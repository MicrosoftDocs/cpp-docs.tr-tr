# <a name="386"></a>.386

Derleme 80386 işlemci için ayrıcalıksız yönerge sağlar; derleme yönergeleri sonraki işlemcilere sahip tanıtılan devre dışı bırakır.

## <a name="syntax"></a>Sözdizimi

> .386

## <a name="remarks"></a>Açıklamalar

Ayrıca 80387 yönergeleri sağlar.
