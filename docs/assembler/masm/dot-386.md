# <a name="386-32-bit-masm"></a>.386 (32-bit masa)

80386 işlemcisi için ayrıcalıksız yönergelerin derlemesini sunar; sonraki işlemcilerle sunulan yönergelerin derlemesini devre dışı bırakır. (yalnızca 32-bit masa.)

## <a name="syntax"></a>Syntax

> **.386**

## <a name="remarks"></a>Açıklamalar

Ayrıca 80387 yönergeleri de etkinleştirilir.
