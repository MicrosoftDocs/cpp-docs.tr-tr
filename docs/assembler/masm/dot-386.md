# <a name="386"></a>.386
Derleme 80386 işlemci ayrıcalıksız yönergeleri sağlar; sonraki işlemcilerle sunulan yönergeleri derlemenin devre dışı bırakır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
.386  
  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Ayrıca 80387 yönergeleri sağlar.
