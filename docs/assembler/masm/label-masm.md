# <a name="label-masm"></a>LABEL (MASM)
Geçerli konumu sayacı değerini atayarak yeni bir etiket oluşturur ve belirtilen `type` için *adı*.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
      name LABEL type  
name LABEL [[NEAR | FAR | PROC]] PTR [[type]]   
```
