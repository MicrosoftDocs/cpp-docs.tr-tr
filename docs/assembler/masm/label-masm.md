# <a name="label-masm"></a>LABEL (MASM)

Geçerli konum sayacı değerini atayarak yeni bir etiket oluşturur ve verilen *türü* için *adı*.

## <a name="syntax"></a>Sözdizimi

> *adı* etiket *türü*

> *adı* etiket [[yakın | ŞU ANA KADAR | PTR PROC]] [[*türü*]]
