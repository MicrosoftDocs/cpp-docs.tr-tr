## <a name="syntax"></a>Syntax

> *ad* **etiketi** *qualifiedtype*
>
> *ad* **etiketi** ⟦**yakınındaki**  |  **uzak**  |  **proc**⟧ **PTR** ⟦*qualifiedtype*⟧
