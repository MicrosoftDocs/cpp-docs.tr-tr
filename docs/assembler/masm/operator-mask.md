# <a name="operator-mask"></a>MASK işleci
Döndürür bir bit maskesi içinde bitleri *recordfieldname* veya *kaydı* ayarlanır ve diğer tüm bitler temizlenir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
MASK {recordfieldname | record}  
```
