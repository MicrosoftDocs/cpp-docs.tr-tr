# <a name="operator-mask"></a>MASK işleci

Döndürür bir bit maskesi içinde bitler *recordfieldname* veya *kayıt* ayarlanır ve diğer tüm bitleri temizlenir.

## <a name="syntax"></a>Sözdizimi

> MASKE {recordfieldname | kayıt}
