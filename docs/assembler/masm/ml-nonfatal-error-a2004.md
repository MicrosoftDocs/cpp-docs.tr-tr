# <a name="ml-nonfatal-error-a2004"></a>ML Önemli Olmayan Hatası A2004

**sembol türü çakışması: tanımlayıcı**

[Externdef](externdef.md) veya [Label](label-masm.md) yönergesi, aynı modülde tanımlanmış, ancak farklı bir tür ile tanımlanmış bir değişken, sembol, veri yapısı veya etiket üzerinde kullanıldı.
