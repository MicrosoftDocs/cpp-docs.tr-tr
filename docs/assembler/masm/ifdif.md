# <a name="ifdif"></a>IFDIF]

Metin öğelerini farklıysa derleme verir.

## <a name="syntax"></a>Sözdizimi

> IFDIF [[ı]] textitem1, textitem2

## <a name="remarks"></a>Açıklamalar

Varsa **miyim** olduğunu göz önünde bulundurulduğunda, karşılaştırma büyük/küçük harf ve büyük harflere duyarlı değildir. Bkz: [IF](../../assembler/masm/if-masm.md) tam söz dizimi.
