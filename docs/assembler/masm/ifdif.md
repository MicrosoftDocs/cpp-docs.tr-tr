# <a name="ifdif-ifdifi"></a>IFDIF, IFDIFI

Metin öğelerini farklıysa derleme verir.

## <a name="syntax"></a>Sözdizimi

> **IFDIF**\[**miyim**] *textitem1* __,__ *textitem2*

## <a name="remarks"></a>Açıklamalar

Varsa **IFDIFI** olduğunu göz önünde bulundurulduğunda, karşılaştırma büyük/küçük harf ve büyük harflere duyarlı değildir. Bkz: [IF](../../assembler/masm/if-masm.md) tam söz dizimi.
