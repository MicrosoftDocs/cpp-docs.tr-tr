# <a name="ifdif"></a>IFDIF]
Metin öğelerini farklıysa derleme verir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
IFDIF[[I]] textitem1, textitem2  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Varsa **ı** olan verildiğinde, karşılaştırma büyük küçük harfe duyarlı. Bkz: [IF](../../assembler/masm/if-masm.md) tam sözdizimi için.
