# <a name="operator-ge"></a>GE işleci
True değeri döndürür (-1) olmadığını *İfade1* büyük veya eşit *İfade2*, veya değilse false (0) döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
expression1 GE expression2  
```
