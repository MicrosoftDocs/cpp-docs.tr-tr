# <a name="ml-fatal-error-a1008"></a>ML Önemli Hatası A1008

**eşleşmeyen makrosu iç içe geçme**

Ya da bir makro dosya ya da Sonlandırıcı yönergenin bitiminden önce sonlandırıldı değil [ENDM](../../assembler/masm/endm.md) dışında makrosu blok bulundu.

Bu hatanın bir nedeni olduğundan noktanın önce atlama [. Yineleme](../../assembler/masm/dot-repeat.md) veya [. SIRADA](../../assembler/masm/dot-while.md).
