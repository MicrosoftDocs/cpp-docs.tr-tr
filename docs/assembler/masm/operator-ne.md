# <a name="operator-ne"></a>NE işleci

*İfade1* , *İfade2* değerine eşit değilse, true (-1) döndürür veya varsa false (0) döndürür.

## <a name="syntax"></a>Syntax

> *İfade1* **ne** *İfade2*
