# <a name="operator-sectionrel"></a>SECTIONREL işleci
Son yürütülebilir dosyayı hedef içeren bölüm göre ifade bölüm göreli uzaklığını döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
SECTIONREL expression  
```  
  
## <a name="remarks"></a>Açıklamalar  
 SECTIONREL yalnızca COFF nesne çıkması ile kullanılabilir.
