# <a name="operator-"></a>;; işleci

*Metni* yalnızca makro tanımında görüntülenen bir makroda açıklama olarak değerlendirir. Liste, makronun genişletildiği *metni* göstermez.

## <a name="syntax"></a>Sözdizimi

> **;;** *metin*
