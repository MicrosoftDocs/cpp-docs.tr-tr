# <a name="operator-"></a>;; işleci
İşler *metin* yalnızca makro tanımı'nda görüntülenen makro açıklama olarak. Liste gösterme *metin* makrosu burada genişletilir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
;;  
text  
  
```
