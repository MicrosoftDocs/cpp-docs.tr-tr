# <a name="version"></a>\@Sürüm

Masd 'nin ana ve alt sürümü tek bir sayı (Metin makrosu) olarak komut satırında raporlanır. Örneğin MASı sürüm 14.23.28107.0 "1423" üretir.

## <a name="syntax"></a>Syntax

> **\@Sürüm**
