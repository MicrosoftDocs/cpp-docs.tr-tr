# <a name="operator-"></a>|| işleci

Mantıksal `OR`. Yalnızca kullanılan [. Eğer](../../assembler/masm/dot-if.md), [. SIRADA](../../assembler/masm/dot-while.md), veya [. Yineleme](../../assembler/masm/dot-repeat.md) engeller ve derleme zamanında çalışma zamanında değerlendirilir.

## <a name="syntax"></a>Sözdizimi

> expression1 || expression2
