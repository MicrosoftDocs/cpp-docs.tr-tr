# <a name="word"></a>WORD
Ayırır ve isteğe bağlı olarak depolama bir sözcük (2 bayt) her biri için başlatır `initializer`. Tür tanımlayıcısı da kullanılabilir herhangi bir yasal türüdür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
[[name]] WORD initializer [[, initializer]]...  
```
