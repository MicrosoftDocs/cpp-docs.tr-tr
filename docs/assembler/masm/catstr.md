# <a name="catstr"></a>CATSTR
Metin öğelerini art arda ekler.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
name CATSTR [[textitem1 [[, textitem2]] ...]]  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Her metin öğesi öncesinde bir sabit hazır bir dize olabilir bir `%`, veya bir makro işlev tarafından döndürülen dize.
