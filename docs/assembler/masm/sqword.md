# <a name="sqword"></a>SQWORD
Ayırır ve isteğe bağlı olarak depolama imzalı 8 bayt her biri için başlatır `initializer`. Tür tanımlayıcısı da kullanılabilir herhangi bir yasal türüdür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
name SQWORD initializer [[, initializer]]...  
```
