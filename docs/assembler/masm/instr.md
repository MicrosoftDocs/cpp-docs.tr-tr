# <a name="instr"></a>INSTR

*Textitem1* içinde ilk *textitem2* oluşumunu bulur.

## <a name="syntax"></a>Syntax

> *Name* **InStr** ⟦*Position*__,__⟧ *textitem1*__,__ *textitem2*

## <a name="remarks"></a>Açıklamalar

Başlangıç *konumu* isteğe bağlıdır. Her metin öğesi bir sabit dize, önünde bir sabit **%** veya bir makro işlevi tarafından döndürülen dize olabilir.
