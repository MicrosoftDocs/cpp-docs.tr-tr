# <a name="instr"></a>INSTR

İlk oluşumunu bulur *textitem2* içinde *textitem1*.

## <a name="syntax"></a>Sözdizimi

> *adı* INSTR [[*konumu*,]] *textitem1*, *textitem2*

## <a name="remarks"></a>Açıklamalar

Başlangıç *konumu* isteğe bağlıdır. Her metin öğesi önünde bir sabit bir sabit dize olabilir bir `%`, veya bir makro işlevi tarafından döndürülen dize.
