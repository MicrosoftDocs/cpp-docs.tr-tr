# <a name="byte-masm"></a>BYTE (MASM)
Ayırır ve isteğe bağlı olarak depolama baytını her biri için başlatır `initializer`.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
[[name]] BYTEinitializer [[, initializer]] ...  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Tür tanımlayıcısı da kullanılabilir herhangi bir yasal türüdür.
