# <a name="ml-nonfatal-error-a2063"></a>ML Önemli Olmayan Hatası A2063

**yalnızca 2 ' nin kuvvetinin HIZALANMASıNA izin verebilir: ifadesi**

[ALIGN](../../assembler/masm/align-masm.md) yönergesi ile belirtilen ifade geçersizdi.

**Hizalama** ifadesi 2 ila 256 arasında 2 ' nin üssü olmalıdır ve geçerli segment, yapı veya birleşimin hizalamasına eşit veya ondan küçük olmalıdır.
