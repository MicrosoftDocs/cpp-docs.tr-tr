# <a name="sword"></a>SWORD
Ayırır ve isteğe bağlı olarak depolama imzalı word (2 bayt) her biri için başlatır `initializer`. Tür tanımlayıcısı da kullanılabilir herhangi bir yasal türüdür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
name SWORD initializer [[, initializer]]...  
```
