# <a name="ml-nonfatal-error-a2107"></a>ML Önemli Olmayan Hatası A2107

**yakın etikete örtük olarak atlanamaz veya çağrı yapılamaz**

Başka bir kesimdeki yakın bir etikete örtük bir şekilde geç veya çağrı yapma girişiminde bulunuldu.
