# <a name="ml-nonfatal-error-a2047"></a>ML Önemli Olmayan Hatası A2047
**(boş) boş bir dize**  
  
 Tırnak işaretleri içindeki herhangi bir karakter ve sınırlandırma çiftinin bir dize içermektedir.  
  
 Bir dize geçerli olması 1-255 karakter içermelidir.
