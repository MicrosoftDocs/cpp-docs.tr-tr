# <a name="operator-short"></a>SHORT işleci
Türünü ayarlar *etiket* kısa için. Tüm atlar *etiket* kısa olması gerekir (için + 127 bayt -128 atlama yönergesi gelen aralıkta *etiket*).  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
SHORT label  
```
