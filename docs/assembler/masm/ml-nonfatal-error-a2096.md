# <a name="ml-nonfatal-error-a2096"></a>ML Önemli Olmayan Hatası A2096

**Segment, Grup veya segment kaydı bekleniyor**

Bir segment veya grup bekleniyordu, ancak bulunamadı.

Aşağıdakilerden biri oluştu:

- Segment geçersiz kılma işleci ( **:** ) ile belirtilen sol işlenen bir segment kaydı (CS, DS, SS, es, FS veya GS), Grup adı, segment adı veya segment ifadesi değildi.

- [Varsay](../../assembler/masm/assume.md) yönergesine, geçerli bir segment adresi, segment kaydı, Grup veya özel **düz** grup olmadan bir segment kaydı verilmıştı.
