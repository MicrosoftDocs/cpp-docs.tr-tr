# <a name="stack-32-bit-masm"></a>. YıĞıN (32-bit masa)

İle kullanıldığında [. MODEL](dot-model.md), yığın segmenti tanımlar (kesim adı **yığını**ile). İsteğe bağlı *Boyut* , yığın için bayt sayısını belirtir (varsayılan 1.024). **. STACK** yönergesi otomatik olarak yığın ifadesini kapatır. (yalnızca 32-bit masa.)

## <a name="syntax"></a>Sözdizimi

> **. STACK** ⟦*boyutu*⟧
