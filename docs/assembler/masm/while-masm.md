# <a name="while"></a>WHILE

*ConstantExpression* doğru kaldığı sürece blok *deyimlerinin* derlemesini yineler.

## <a name="syntax"></a>Sözdizimi

> *ConstantExpression*\
> *deyimler*\
> **ENDM**
