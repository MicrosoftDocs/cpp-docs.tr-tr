# <a name="while-masm"></a>WHILE (MASM)

Bütünleştirilmiş kod bloğunun yinelenen *deyimleri* sürece *ifade* doğrudur.

## <a name="syntax"></a>Sözdizimi

> SIRADA *ifadesi*<br/>
> *Deyimleri*<br/>
> ENDM
