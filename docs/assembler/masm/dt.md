# <a name="dt"></a>DT
Ayırır ve isteğe bağlı olarak 10 bayt depolama her biri için başlatır `initializer`. Tür tanımlayıcısı da kullanılabilir herhangi bir yasal türüdür. `DT`bir eş anlamlı olduğundan [TBYTE](../../assembler/masm/tbyte.md).  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
[[name]] DT initializer [[, initializer]]...  
```
