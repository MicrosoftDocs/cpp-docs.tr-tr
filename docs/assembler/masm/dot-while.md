# <a name="while"></a>.WHILE
Bloğunu yürütür kod oluşturur `statements` sırada `condition` doğrudur.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
   .WHILE condition  
statements  
.ENDW  
```
