# <a name="real8"></a>REAL8

Her *Başlatıcı*için bir çift duyarlıklı (8 baytlık) kayan noktalı sayı ayırır ve isteğe bağlı olarak başlatır.

## <a name="syntax"></a>Sözdizimi

> *ad* **REAL8** *Başlatıcı* ⟦ __,__ *Başlatıcı* ... ⟧
