# <a name="ml-nonfatal-error-a2031"></a>ML Önemli Olmayan Hatası A2031
**Dizin veya temel kayıt olması gerekir**  
  
 Bir temel veya dizin kayıt bir bellek ifadesinde değildi bir kayıt kullanmak için girişimde bulunuldu.  
  
 Örneğin, aşağıdaki ifadeler bu hataya neden:  
  
```  
[ax]  
[bl]  
```
