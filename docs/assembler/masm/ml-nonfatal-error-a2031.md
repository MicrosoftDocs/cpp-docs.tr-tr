# <a name="ml-nonfatal-error-a2031"></a>ML Önemli Olmayan Hatası A2031

**Dizin veya temel kayıt olmalıdır**

Bir bellek ifadesinde temel veya dizin kaydı olmayan bir kayıt kullanılmaya çalışıldı.

Örneğin, aşağıdaki ifadeler bu hataya neden olur:

```asm
[ax]
[bl]
```
