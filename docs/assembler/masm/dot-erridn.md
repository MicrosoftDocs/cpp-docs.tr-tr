# <a name="erridn"></a>.ERRIDN]

Metin öğelerini aynıysa, bir hata oluşturur.

## <a name="syntax"></a>Sözdizimi

> . [[I]] ERRIDN *textitem1*, *textitem2* [[, *ileti*]]

## <a name="remarks"></a>Açıklamalar

Varsa **miyim** olduğunu göz önünde bulundurulduğunda, karşılaştırma büyük/küçük harf ve büyük harflere duyarlı değildir.
