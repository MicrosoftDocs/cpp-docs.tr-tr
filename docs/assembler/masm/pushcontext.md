# <a name="pushcontext"></a>PUSHCONTEXT

Geçerli *bağlamın* parçasını veya tümünü kaydeder: segment yazmacı varsayılmaktadır, taban değeri, listeleme ve cref bayrakları veya işlemci/Coprocessor değerleri. *Bağlam* (yalnızca 32-bit ması), **Radix**, **listeleme**, **CPU** (yalnızca 32-bit ması) veya **Tümü** olarak **kabul** edilebilir.

## <a name="syntax"></a>Syntax

> **PushContext** *bağlamı*
