# <a name="ml-nonfatal-error-a2085"></a>ML Önemli Olmayan Hatası A2085

**yönerge veya kayıt geçerli CPU modunda kabul edilmedi**

Geçerli işlemci modu için geçerli olmayan bir yönerge, kayıt veya anahtar sözcük kullanma girişiminde bulunuldu.

Örneğin, 32 bitlik Yazmaçları için [. 386](dot-386.md) veya üzeri gerekir. CR0 gibi denetim kayıtları, ayrıcalıklı mod gerektirir [. 386p](dot-386p.md) veya üzeri. Bu hata ayrıca, gereken **NEAR32**, **FAR32** ve **düz** anahtar kelimeler için de oluşturulacaktır. **386** veya üzeri.
