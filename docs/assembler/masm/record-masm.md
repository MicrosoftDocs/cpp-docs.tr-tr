# <a name="record-masm"></a>RECORD (MASM)
Belirtilen alanları içeren bir kayıt türü bildirir. *FieldName* alan adları *genişliği* bit sayısını belirtir ve *ifade* ilk değerini verir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
   recordname RECORD fieldname:width [[= expression]]   
[[, fieldname:width [[= expression]]]]...  
```
