# <a name="typedef"></a>TYPEDEF

*Tür* veya *protodefinition*ile eşdeğer olan, *Name*adlı yeni bir tür veya **proto** tanımlar.

## <a name="syntax"></a>Sözdizimi

> *ad* **typedef** *türü* | **proto** *protodefinition*
