# <a name="386p"></a>.386P

Tüm yönergeleri (Ayrıcalıklı dahil) 80386 işlemcisi için derleme sağlar; derleme yönergeleri sonraki işlemcilere sahip tanıtılan devre dışı bırakır.

## <a name="syntax"></a>Sözdizimi

> .386P

## <a name="remarks"></a>Açıklamalar

Ayrıca 80387 yönergeleri sağlar.
