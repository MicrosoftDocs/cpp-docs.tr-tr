# <a name="386p"></a>.386P
Derleme (Ayrıcalıklı dahil) tüm yönerge 80386 işlemci sağlar; sonraki işlemcilerle sunulan yönergeleri derlemenin devre dışı bırakır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
.386P  
  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Ayrıca 80387 yönergeleri sağlar.
