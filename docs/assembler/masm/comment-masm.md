# <a name="comment"></a>COMMENT

Bir yorum olarak sınırlayıcılarla aynı satırdaki tüm *metni* değerlendirir.

## <a name="syntax"></a>Syntax

> **Açıklama** *sınırlayıcısı* ⟦*metin*⟧ \
> ⟦*Text*⟧ \
> ⟦*Text*⟧ *sınırlayıcısı* ⟦*Text*⟧
