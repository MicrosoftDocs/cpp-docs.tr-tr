# <a name="operator-high"></a>HIGH işleci

Düşük 16 bitten oluşan 16 bit *ifadenin*üst 8 bitini döndürür. Masd ifadeleri 64 bitlik değerlerdir.

## <a name="syntax"></a>Sözdizimi
