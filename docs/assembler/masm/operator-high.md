# <a name="operator-high"></a>HIGH işleci
Düşük 16 bit düzeyi yüksek 8 bit döndürür *ifade*. MASM ifadeleri 64-bit değerlerdir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
HIGH expression  
```
