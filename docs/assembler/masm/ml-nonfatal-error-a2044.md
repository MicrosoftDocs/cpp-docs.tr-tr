# <a name="ml-nonfatal-error-a2044"></a>ML Önemli Olmayan Hatası A2044

**dosya geçersiz karakter**

Kaynak dosyasının bir karakter dizesi, bir açıklama dışında yer alan veya sabit değer, bir işleç veya diğer yasal karakter olarak tanınmadı.
