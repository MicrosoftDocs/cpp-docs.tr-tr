# <a name="operator-"></a>. işleci
İlk işleci döndürür *ifade* uzaklığı artı *alan* yapısı veya birleşim içinde. İkinci işleci gösterdiği konumda değeri döndürür *kaydetmek* uzaklığı artı *alan* yapısı veya birleşim içinde.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
      expression  
      . field [[. field]]...  
[register]. field [[. field]]...  
```
