# <a name="ml-warning-a4012"></a>ML Uyarısı A4012
**Satır numarası bilgilerini kesimin sınıfı 'CODE' olmadan**  
  
 "CODE" ile biten bir sınıf adı olmayan bir segment yönergeleri vardı Derleyici CodeView bilgi için bu yönergeleri oluşturmadı.  
  
 CodeView "CODE" ile bitmeyen sınıf adları olan kodda modüllerle işleyemiyor
