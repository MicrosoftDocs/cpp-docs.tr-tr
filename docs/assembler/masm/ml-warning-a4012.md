# <a name="ml-warning-a4012"></a>ML Uyarısı A4012

**kesimin sınıfı 'CODE' olmadan satır numarası bilgisi**

"CODE" ile biten bir sınıf adı olmadığı bir segment yönergeleri vardı. Assembler CodeView bilgisi için bu yönergeleri oluşturmadı.

CodeView kesimlerinde "CODE" ile bitmeyen sınıf adları ile kod modülleriyle işlenemiyor
