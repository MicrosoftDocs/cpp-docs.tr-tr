# <a name="ml-warning-a4012"></a>ML Uyarısı A4012

**' CODE ' sınıfı olmadan segment için satır numarası bilgisi**

"CODE" ile biten bir sınıf adı olmayan bir kesimde yönergeler vardı. Assembler bu yönergeler için CodeView bilgisi oluşturmadı.

CodeView, "CODE" ile bitolmayan sınıf adlarıyla kesimlerdeki kod ile modülleri işleyemez.
