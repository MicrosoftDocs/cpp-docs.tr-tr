# <a name="errdifi"></a>. ERRDIF [[I]]

Metin öğelerini farklı olması durumunda bir hata oluşturur.

## <a name="syntax"></a>Sözdizimi

> . [[I]] ERRDIF <textitem1>, <textitem2> [[, ileti]]

## <a name="remarks"></a>Açıklamalar

Varsa **miyim** olduğunu göz önünde bulundurulduğunda, karşılaştırma büyük/küçük harf ve büyük harflere duyarlı değildir.
