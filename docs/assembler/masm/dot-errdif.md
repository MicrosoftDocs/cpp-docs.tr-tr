# <a name="errdifi"></a>. [[I]] ERRDIF
Metin öğelerini farklı olması durumunda bir hata oluşturur.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
.ERRDIF[[I]] <textitem1>, <textitem2> [[, message]]  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Varsa **ı** olan verildiğinde, karşılaştırma büyük küçük harfe duyarlı.
