# <a name="perioderrdif-perioderrdifi"></a>&period;ERRDIF, &period;ERRDIFI

Metin öğelerini farklı olması durumunda bir hata oluşturur.

## <a name="syntax"></a>Sözdizimi

> **.ERRDIF**\[**I**] *textitem1* __,__ *textitem2* \[ __,__ *message*]

## <a name="remarks"></a>Açıklamalar

Varsa  **&period;ERRDIFI** olduğunu göz önünde bulundurulduğunda, karşılaştırma büyük/küçük harf ve büyük harflere duyarlı değildir.
