# <a name="ml-warning-a4014"></a>ML Uyarısı A4014
yönergeler ve başlatılmış veri BSS segmentlerinde desteklenmiyor  
  
 BSS bölümü içindeki başlatılmış verileri tanımlamak için girişimde bulunuldu.  BSS bölüm BSS adı olan bir sınıf olarak tanımlanır.  Bu Basitleştirilmiş segmenti içerir `.data?`.
