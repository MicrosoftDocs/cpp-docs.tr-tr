# <a name="ml-warning-a4014"></a>ML Uyarısı A4014

yönergeler ve başlatılmış veriler BSS segmentlerinde desteklenmiyor

Başlatılmış veriler BSS bölümü içindeki tanımlamak için girişimde bulunuldu.  BSS bölüm BSS adı olan bir sınıf olarak tanımlanır.  Bu Basitleştirilmiş segmenti içerir `.data?`.
