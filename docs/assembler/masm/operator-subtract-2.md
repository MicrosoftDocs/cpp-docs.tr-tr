# <a name="operator--"></a>\- işleci

*İfade1* eksi *İfade2*döndürür. İkinci işleç, *ifadesinin*işaretini tersine çevirir.

## <a name="syntax"></a>Sözdizimi

> *ifade1* **-** *İfade2*
>
> __-__ *ifadesi*
