# <a name="real4"></a>REAL4

Her *Başlatıcı* için tek duyarlıklı (4 baytlık) kayan noktalı sayı ayırır ve isteğe bağlı olarak başlatır.

## <a name="syntax"></a>Syntax

> *ad*  **REAL4** *Başlatıcı* ⟦__,__ *Başlatıcı* ... ⟧
