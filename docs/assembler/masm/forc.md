# <a name="forc"></a>FORC

Her bir tekrardaki *parametreyi* değiştirirken geçerli karakterle birlikte *dizedeki*her karakter için bir kez tekrarlanmış bir blok işaretler.

## <a name="syntax"></a>Sözdizimi

> **Forc** *parametresi*, __\<__ *dize* __>__ \
> *deyimler*\
> **ENDM**
