# <a name="forc"></a>FORC
Her karakteri için bir kez yineleneceğini bir blok işaretler `string`, geçerli değiştirme karakteri ile `parameter` her tekrar üzerinde.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
   FORC  
parameter, <string> statements  
ENDM  
```
