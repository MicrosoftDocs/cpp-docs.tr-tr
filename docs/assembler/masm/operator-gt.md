# <a name="operator-gt"></a>GT işleci
True değeri döndürür (-1) olmadığını *İfade1* değerinden daha büyük *İfade2*, veya değilse false (0) döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
expression1 GT expression2  
```
