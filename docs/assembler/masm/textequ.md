# <a name="textequ"></a>TEXTEQU
Atar *textitem* için *adı*. *Textitem* öncesinde bir sabit hazır bir dize olabilir bir `%`, veya bir makro işlev tarafından döndürülen dize.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
name TEXTEQU [[textitem]]  
```
