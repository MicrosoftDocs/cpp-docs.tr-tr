# <a name="textequ"></a>TEXTEQU

*Name*'e *textıtem* atar. *Textıtem* bir sabit dize, önünde bir `%`veya bir makro işlevi tarafından döndürülen dize olabilir.

## <a name="syntax"></a>Sözdizimi

> *ad* **TEXTEQU** ⟦*textıtem*⟧
