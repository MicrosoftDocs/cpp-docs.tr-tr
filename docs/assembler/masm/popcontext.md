# <a name="popcontext"></a>POPCONTEXT
Bir kısmı veya tümü geçerli yükler `context` (kaydeden [PUSHCONTEXT](../../assembler/masm/pushcontext.md) yönergesi). `context` Olabilir **ASSUMES**, `RADIX`, **LİSTELEME**, **CPU**, veya **tüm**.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
POPCONTEXT context  
```
