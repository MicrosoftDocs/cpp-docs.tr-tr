# <a name="popcontext"></a>POPCONTEXT

Bölümü veya tümü geçerli yükler `context` (kaydeden [PUSHCONTEXT](../../assembler/masm/pushcontext.md) yönergesi). `context` Olabilir **ASSUMES**, `RADIX`, **LİSTELEME**, **CPU**, veya **tüm**.
