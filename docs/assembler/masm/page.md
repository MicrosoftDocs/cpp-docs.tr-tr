# <a name="page"></a>PAGE

İlk yönerge satır ayarlar *uzunluğu* ve karakter *genişliği* program liste. Bağımsız değişken olmadan verilmişse, sayfa sonu oluşturur. İkinci yönerge bölüm numarasını artırır ve sayfa sayısı 1 olarak sıfırlar.

## <a name="syntax"></a>Sözdizimi

> Sayfa [[*uzunluğu*]] [[, *genişliği*]]<br/><br/>
> SAYFA +
