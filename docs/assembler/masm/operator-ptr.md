# <a name="operator-ptr"></a>PTR işleci

İlk işleç zorlar *ifade* belirtilen sahip olarak kabul edilmesi için *türü*. İkinci işleci bir işaretçiye belirtir *türü*.

## <a name="syntax"></a>Sözdizimi

> *tür* PTR *ifadesi*
>
> \[*uzaklık*] PTR *türü*
