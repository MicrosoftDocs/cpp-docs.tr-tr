# <a name="operator-ptr"></a>PTR işleci
İlk işleci zorlar *ifade* belirtilen sahip olarak kabul edilmesi için `type`. İkinci işleci için bir işaretçi belirtir `type`.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
