# <a name="operator-ptr"></a>PTR işleci

İlk operatör, *ifadeyi* belirtilen *türe*sahip olacak şekilde kabul etmeye zorlar. İkinci işleç, *türüne*yönelik bir işaretçi belirtir.

## <a name="syntax"></a>Sözdizimi

> *tür* **PTR** *ifadesi*
>
> \[*uzaklık*] **PTR** *türü*
