# <a name="ml-nonfatal-error-a2074"></a>ML Önemli Olmayan Hatası A2074

**Bölüm Yazmaçları aracılığıyla etikete erişilemiyor**

Kesimine veya grubuna Varsayılmayan bir segment kaydı aracılığıyla bir etikete erişme girişiminde bulunuldu.
