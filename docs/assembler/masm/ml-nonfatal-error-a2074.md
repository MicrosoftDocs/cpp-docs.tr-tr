# <a name="ml-nonfatal-error-a2074"></a>ML Önemli Olmayan Hatası A2074

**Etiket segment kayıtları erişilemiyor**

Bir etiket kendi kesim veya gruba varsayıldı değil bir segment kayıt üzerinden erişmek için girişimde bulunuldu.
