# <a name="operator-lowword"></a>LOWWORD işleci
Düşük 16 bit döndürür *ifade*. MASM ifadeleri 64-bit değerlerdir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
LOWWORD expression  
```
