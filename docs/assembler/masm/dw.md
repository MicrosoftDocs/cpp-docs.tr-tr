# <a name="dw"></a>DW
Ayırır ve isteğe bağlı olarak depolama bir sözcük (2 bayt) her biri için başlatır `initializer`. Tür tanımlayıcısı da kullanılabilir herhangi bir yasal türüdür. `DW` bir eş anlamlı olduğundan [WORD](../../assembler/masm/word.md).  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
[[name]] DW initializer [[, initializer]]...  
```
