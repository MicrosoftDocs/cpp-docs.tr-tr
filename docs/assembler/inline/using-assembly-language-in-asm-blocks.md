# <a name="using-assembly-language-in-__asm-blocks"></a>__asm Bloklarında Derleme Dili Kullanma

**Microsoft'a Özgü**

Satır içi assembler, diğer birleştiriciler ile çok yaygın olarak bulunur. Örneğin, Masd 'de geçerli olan herhangi bir ifadeyi kabul eder. Bu bölümde, blok içindeki derleme dili özelliklerinin kullanımı açıklanmaktadır **`__asm`** .
