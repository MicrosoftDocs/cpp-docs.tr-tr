# <a name="viewing-a-menu-as-a-pop-up-menu"></a>Menüyü Açılır Menü Olarak Görüntüleme
Normalde, Menü Düzenleyici'de çalışırken menü kaynağı bir menü çubuğu görüntülenir. Ancak, program çalışırken, uygulamanın menü çubuğu'na eklenmesini menü kaynaklarınız olabilir.  
  
### <a name="to-view-a-menu-resource-as-a-pop-up-menu"></a>Menü kaynağı açılır menü olarak görüntülemek için  
  
1.  Menüyü sağ tıklatın ve seçin **Görünüm açılan olarak** kısayol menüsünden.  
  
     Bu yalnızca bir görüntüleme tercihi ve menünüze değiştirmez.  
  
    > [!NOTE]
    >  Menü çubuğu görünümüne geçmek için tıklatın **Görünüm açılan olarak** yeniden (hangi onay işaretini kaldırır ve menü çubuğu görünümünüzü döndürür).  
  

  
 **Gereksinimler**  
  
 Win32
