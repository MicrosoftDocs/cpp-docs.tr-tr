# <a name="hstringdetach-method"></a>HString::Detach Yöntemi
Belirtilen ayırır **Hstrıng** nesnesini temel değerinden.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HSTRING Detach() throw()  
```

 Arka plandaki **Hstrıng** değerden önce ayırma işlemi başlatıldı.
