# <a name="creating-an-empty-windows-desktop-application"></a>Boş Windows masaüstü uygulaması oluşturma
### <a name="to-create-an-empty-windows-desktop-application"></a>Boş bir Windows masaüstü uygulaması oluşturmak için  
  
1.  Gelen **dosya** menüsünde, select **yeni, proje**.  
  
2.  Sol bölmesinde **yeni proje** iletişim kutusunda, tıklatıldığında **Win32** ve Orta bölmede seçin **Win32 konsol uygulaması**.  
  
3.  Yeni projeniz, proje dizinine bir yol için bir ad girin ve ardından **Tamam**.  
  
4.  İçinde [Win32 Uygulama Sihirbazı'nı](../windows/win32-application-wizard.md), tıklatın **uygulama ayarları** sayfası. Seçin **uygulama türü** kaynak kodu dosyanızda ile oluşturun ve ardından istediğiniz **boş proje** altındaki onay kutusunu **ek seçenekler**.  
  
5.  **Tamam**'ı tıklatın.  
  
 Proje görünür **Çözüm Gezgini** ile üç dizinler kaynak dosyaları, üst bilgi dosyaları ve kaynak dosyaları içerir.  
  
 Ardından, şunları yapabilirsiniz [dosyaları boş Visual C++ projenize eklemek](../windows/adding-files-to-an-empty-win32-applications.md).
