# <a name="opening-the-resource-editor-for-a-given-symbol"></a>Belirli Bir Sembol İçin Kaynak Düzenleyicisini Açma
Ne zaman gözatarken sembolleri [kaynak sembolleri iletişim kutusu](../windows/resource-symbols-dialog-box.md), belirli bir sembolle nasıl kullanıldığı hakkında daha fazla bilgi isteyebilirsiniz. **Görünüm kullanım** düğmesi, bu bilgileri almak için hızlı bir yol sağlar.  
  
### <a name="to-move-to-the-resource-editor-where-a-symbol-is-being-used"></a>Kaynak Düzenleyici bir simge burada taşımak için kullanılır  
  
1.  Bir simge seçin **adı** kutusunun **kaynak semboller** iletişim kutusu.  
  
2.  İçinde **kullanılan tarafından** kutusunda, ilgilendiğiniz kaynak türü seçin.  
  
3.  Tıklatın **görünüm kullanım** düğmesi.  
  
     Kaynak uygun Düzenleyicisi penceresinde görünür.  
  
 Kaynakları yönetilen projelerine ekleme hakkında daha fazla bilgi için lütfen bkz [masaüstü uygulamalarında kaynakları](/dotnet/framework/resources/index) içinde *.NET Framework Geliştirici Kılavuzu.*  
  
 Gereksinimler  
  
 Win32
