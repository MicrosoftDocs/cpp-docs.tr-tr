# <a name="deleting-a-control"></a>Denetimi Silme
### <a name="to-delete-a-control"></a>Bir denetim silmek için  
  
1.  Denetim ve tuşuna basın iletişim kutusunda seçin **silmek** anahtarı.  
  
     \-veya -  
  
2.  Üzerinde **Düzenle** menüsünde tıklatın **silmek**.  
  
    > [!TIP]
    >  Çoğu durumda iletişim kutusu Düzenleyicisi ' ni kullanırken, sık kullanılan komutlar kısayol menüsünü görüntülemek için sağ fare düğmesini tıklatabilirsiniz.
