# <a name="inserting-a-new-menu-between-existing-menus"></a>Varolan Menülerin Arasına Yeni Menü Ekleme
### <a name="to-insert-a-new-menu-between-existing-menus"></a>Varolan menülerin arasına yeni menü eklemek için  
  
1.  Var olan bir menü adı ve tuşuna seçin **Ekle** anahtarı. **Yeni öğe** kutusunda, seçili öğe önce eklenir.  
  
     \-veya -  
  
2.  Menü çubuğunda sağ tıklatın ve seçin **yeni Ekle** kısayol menüsünden.  
  
 Kaynakları yönetilen projelerine ekleme hakkında daha fazla bilgi için lütfen bkz [masaüstü uygulamalarında kaynakları](/dotnet/framework/resources/index) içinde *.NET Framework Geliştirici Kılavuzu.*  
  
 Gereksinimler  
  
 Win32
