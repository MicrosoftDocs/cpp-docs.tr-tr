# <a name="array-attributes"></a>Dizi Öznitelikleri
Diziler için aşağıdaki öznitelikler uygulamak veya **SAFEARRAY**s.  
  
|Öznitelik|Açıklama|  
|---------------|-----------------|  
|[library_block](../windows/library-block.md)|.İdl dosyanın kitaplığı bloktaki bir yapı yerleştirir.|  
|[satype](../windows/satype.md)|SAFEARRAY yapısı veri türünü belirtir.|  
|[dize](../windows/string-cpp.md)|Belirten tek boyutlu `char`, `wchar_t`, **bayt** (veya eşdeğer) dizi ya da böyle bir dizi işaretçisine gerekir kabul olarak bir dize.|
