# <a name="chaininterfacescasttounknown-method"></a>ChainInterfaces::CastToUnknown Yöntemi

## <a name="syntax"></a>Sözdizimi  
  
```  
__forceinline IUnknown* CastToUnknown();  
```  
  
## <a name="return-value"></a>Dönüş Değeri  
 IUnknown işaretçisi.
