Bu modülü tarafından yönetilen nesne sayısını alır.

## <a name="syntax"></a>Sözdizimi

```cpp
virtual long GetObjectCount() const;
```

## <a name="return-value"></a>Dönüş Değeri

Bu modülü tarafından yönetilen nesnelere geçerli sayısı.
