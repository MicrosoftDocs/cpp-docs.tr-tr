# <a name="aligning-groups-of-controls"></a>Denetim Gruplarını Hizalama
Aşağıdaki yordamda denetim gruplarını hizalama gösterilmiştir.  
  
### <a name="to-align-groups-of-controls"></a>Denetim gruplarını hizalama için  
  
1.  [Denetimleri seçme](../windows/selecting-multiple-controls.md) hizalamak istediğiniz. Baskın denetimi ilk olmasını istediğiniz denetim seçtiğinizden emin olun veya hizalama yürütme ya da komut boyutlandırma önce baskın denetimi olacak şekilde ayarlayın.  
  
     Denetim grubu son konumunu baskın denetimi konumuna bağlıdır. Baskın denetimi seçme konusunda daha fazla bilgi için bkz: [baskın denetimi belirtme](../windows/specifying-the-dominant-control.md).  
  
2.  Gelen **biçimi** menüsünde seçin **Hizala**ve ardından aşağıdaki hizalamaları birini seçin:  
  
    -   **Sola**: kendi sol kenarı boyunca Seçili denetimleri hizalar.  
  
    -   **Merkezleri**: kendi merkez nokta boyunca yatay olarak seçilen denetimleri hizalar.  
  
    -   **Hakları**: kendi sağ tarafa boyunca Seçili denetimleri hizalar.  
  
    -   **Dows masaüstleri**: üst kenarları boyunca Seçili denetimleri hizalar.  
  
    -   **Ortaları**: dikey olarak kendi orta nokta boyunca Seçili denetimleri hizalar.  
  
    -   **Alta**: Seçili denetimleri alt kenarları boyunca hizalar.
