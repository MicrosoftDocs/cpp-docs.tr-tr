# <a name="evening-the-spacing-between-controls"></a>Denetimler Arasındaki Boşlukları Eşitleme
İletişim kutusu Düzenleyicisi eşit olarak seçilen en dıştaki denetimler arasındaki boşluk denetimlere sağlar.  
  
### <a name="to-even-the-spacing-between-controls"></a>Denetimler arasındaki boşlukları bile için  
  
1.  Yeniden düzenlemek istediğiniz denetimleri seçin.  
  
2.  Gelen **biçimi** menüsünde seçin **eşit**ve ardından aşağıdaki aralık hizalamaları birini seçin:  
  
    -   **Üzerinden**: alanları, soldaki ve en sağdaki denetimi seçili arasında eşit olarak denetimleri.  
  
    -   **Aşağı**: eşit olarak en üstteki ve alttaki denetimi seçili arasında denetimler alanları.
