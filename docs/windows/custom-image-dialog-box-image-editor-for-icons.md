# <a name="custom-image-dialog-box-c-image-editor-for-icons"></a>Özel görüntü iletişim kutusu (simgeler için görüntü Düzenleyicisi) (C++)

Özel boyut ve renk sayısı ile yeni bir görüntü oluşturmanıza olanak sağlar.

- **Genişlik**

   Özel görüntü genişliğini piksel cinsinden (1-512 sınırını 2048) girmeniz için bir alan sağlar.

- **Yükseklik**

   Özel görüntü (1-512 sınırını 2048) piksel cinsinden yüksekliği girmek bir alan sağlar.

- **Renkler**

   Bir alan için özel görüntü için renk sayısını seçmenize olanak sağlar: 2, 16 veya 256.

## <a name="requirements"></a>Gereksinimler

Yok.
