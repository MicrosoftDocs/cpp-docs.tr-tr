# <a name="hstringgetaddressof-method"></a>HString::GetAddressOf Metodu
Temel alınan HSTRING işlemek için bir işaretçi alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HSTRING* GetAddressOf() throw()  
```  
  
## <a name="return-value"></a>Dönüş Değeri  
 Temel alınan HSTRING işlemek için bir işaretçi.  
  
## <a name="remarks"></a>Açıklamalar  
 Bu işlemi daha sonra temel alınan HSTRING tanıtıcının dize değeri yok.  
  
## <a name="requirements"></a>Gereksinimler  
 **Başlık:** corewrappers.h  
  
 **Namespace:** Microsoft::wrl:: Wrappers
