# <a name="usesgetlasterror"></a>usesgetlasterror
Bu işlev çağrılırken bir hata varsa sonra çağıran sonra arayabileceğiniz çağıran söyler `GetLastError` hata kodu alınamadı.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
[usesgetlasterror]  
  
```  
  
## <a name="remarks"></a>Açıklamalar  
 **Usesgetlasterror** C++ özniteliğine sahip ile aynı işlevselliği [usesgetlasterror](http://msdn.microsoft.com/library/windows/desktop/aa367297) MIDL özniteliği.  
  
## <a name="example"></a>Örnek  
 Bkz: [idl_module](../windows/idl-module.md) nasıl kullanılacağını gösteren örnek bir örnek için **usesgetlasterror**.
