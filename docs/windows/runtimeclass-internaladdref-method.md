## <a name="syntax"></a>Sözdizimi  
  
```  
ULONG InternalAddRef();  
```  
  
## <a name="return-value"></a>Dönüş Değeri  
 Sonuçta elde edilen başvuru sayısı.

 **Namespace:** Microsoft::WRL
