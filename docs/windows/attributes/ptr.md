# <a name="ptr"></a>ptr

Bir işaretçiyi tam bir işaretçi olarak belirler.

## <a name="syntax"></a>Syntax

```cpp
[ptr]
```

## <a name="remarks"></a>Açıklamalar

**PTR** C++ özniteliği, [PTR](/windows/win32/Midl/ptr) MIDL özniteliğiyle aynı işlevselliğe sahiptir.
