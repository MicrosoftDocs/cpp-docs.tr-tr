# <a name="attributes-by-group"></a>Gruplara Göre Öznitelikler

C++ öznitelikleri aşağıdaki işlevsel gruplar halinde düzenlenir.

|Öznitelik|Açıklama|
|---------------|-----------------|
|[COM öznitelikleri](com-attributes.md)|COM geliştirmenin çok sayıda alanını desteklemek ve ortak dil çalışma zamanı geliştirmeyi .NET Framework için kod ekleme.|
|[IDL öznitelikleri](idl-attributes.md)|Bir sihirbaz kullanmadan ve bu dosyanın yapısı ve söz dizimi hakkında bilgi sahibi olmadan. IDL dosyasının değiştirilmesini bir kaynak kodu dosyasından etkinleştirin.|
|[OLE DB tüketici öznitelikleri](ole-db-consumer-attributes.md)|Tablo açma, komutları yürütme ve verilere erişme gibi görevleri gerçekleştiren bir çalışan OLE DB tüketicisini oluşturmak için [OLE DB tüketici şablonlarına](../../data/oledb/ole-db-consumer-templates-reference.md)göre kod ekleme.|
|[Derleyici öznitelikleri](compiler-attributes.md)|Microsoft C++ derleyicisi tarafından sağlanır.|

.NET Framework uygulamanız için özel özniteliklerin nasıl tanımlanacağı hakkında daha fazla bilgi için, bkz. [Kullanıcı tanımlı öznitelikler](../../extensions/user-defined-attributes-cpp-component-extensions.md).
