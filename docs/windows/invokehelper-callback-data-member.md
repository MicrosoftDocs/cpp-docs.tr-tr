# <a name="invokehelpercallback-data-member"></a>InvokeHelper::callback_ Veri Üyesi

## <a name="syntax"></a>Sözdizimi  
  
```  
TCallback callback_;  
```

 `TCallback` Şablon parametresi olay işleyicisi türünü belirtir.
