# <a name="using-the-256-color-palette-image-editor-for-icons"></a>256-Renk Paletini Kullanma (Simgeler İçin Görüntü Düzenleyicisi)
256-renk paletinden bir seçimi çizmek için renk paletindeki renkleri seçin gereksinim [renkler penceresi](../windows/colors-window-image-editor-for-icons.md).  
  
### <a name="to-choose-a-color-from-the-256-color-palette-for-large-icons"></a>256-renk paletini büyük simgeler için renk Seçilecek  
  
1.  Büyük simgesi veya imleci seçin veya yeni büyük simgesi veya imleci oluşturma.  
  
2.  Görüntülenen 256 renkten bir renk seçin **renkleri** palette **renkleri** penceresi.  
  
     Seçili renk renk paleti geçerli rengi olur **renkleri** penceresi.  
  
    > [!NOTE]
    >  256-renk görüntüleri için kullanılan ilk palet CreateHalftonePalette Windows API tarafından döndürülen palet eşleşir. Windows Kabuğu için hedeflenen tüm simgeleri palet gerçekleştirme sırasında titreşimi önlemek için bu paletin kullanmanız gerekir.
