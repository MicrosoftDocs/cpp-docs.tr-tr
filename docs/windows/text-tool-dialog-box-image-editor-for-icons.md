# <a name="text-tool-dialog-box-image-editor-for-icons"></a>Metin Aracı İletişim Kutusu (Simgeler İçin Görüntü Düzenleyicisi)
Kullanım **metin aracı** metin imleç, bit eşlem veya simge kaynak eklemek için iletişim kutusu.  
  
 Bu iletişim kutusunu erişmek için açık [görüntü Düzenleyicisi](../windows/window-panes-image-editor-for-icons.md). Seçin **Araçları** gelen **görüntü** menüsüne ve ardından **metin aracı** komutu.  
  
 **Yazı tipi düğmesi**  
 Açılır [metin aracı yazı tipi iletişim kutusu](../windows/text-tool-font-dialog-box-image-editor-for-icons.md), hangi değiştirebileceğiniz yazı tipini, stili veya imleç yazı tipi boyutu. Metin alanında görüntülenen metni yapılan değişikliklerin uygulanması.  
  
 **Metin alanı**  
 Kaynak bir parçası olarak görüntülenen metni görüntüler. Başlangıçta bu alanı boş olur.  
  
> [!NOTE]
>  Varsa **saydam arka plan** , yalnızca metin görüntüsüne yerleştirilir ayarlanmadı. Varsa **donuk arka plan** ayarlanır ile doldurulan bir sınırlayıcı dikdörtgenini [arka plan rengi](../windows/selecting-foreground-or-background-colors-image-editor-for-icons.md), metnin arkasına yerleştirilecek. Daha fazla bilgi için bkz: [seçme donuk ve saydam arka planlar](../windows/choosing-a-transparent-or-opaque-background-image-editor-for-icons.md).  
  
 Üzerinde sağ **metin aracı** standart Windows komutların listesini içeren bir varsayılan kısayol menüsüne erişmek için iletişim kutusu.
