# <a name="open-ltdevicegt-image-dialog-box-image-editor-for-icons"></a>Açık &lt;aygıt&gt; görüntü iletişim kutusu (simgeler için görüntü Düzenleyicisi)
Cihaz görüntüleri açmak için bu iletişim kutusunu kullanın. Geçerli kaynak (geçerli kaynak parçası olan görüntüler) var olan cihaz görüntülerinde listeler.  
  
 **Geçerli görüntüleri**  
 Kaynağında bulunan görüntüleri listeler. Açmak istediğiniz resim türünü seçin.  
  
## <a name="requirements"></a>Gereksinimler  
 Yok.
