# <a name="open-ltdevicegt-image-dialog-box-c-image-editor-for-icons"></a>Açık &lt;cihaz&gt; görüntü iletişim kutusu (simgeler için görüntü Düzenleyicisi) (C++)

Cihaz görüntülerinde C++ projelerinde açmak için bu iletişim kutusunu kullanın. Geçerli kaynak (geçerli kaynak bir parçası olan görüntüleri) var olan cihaz görüntülerinde listeler.

### <a name="current-images"></a>Mevcut görüntülerin

Kaynakta bulunan görüntüleri listeler. Açmak istediğiniz görüntü türü seçin.

## <a name="requirements"></a>Gereksinimler

Yok.
