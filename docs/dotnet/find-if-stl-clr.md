# <a name="findif-stlclr"></a>find_if (STL/CLR)
Bir öğenin belirli bir koşulu karşıladığı aralıktaki ilk geçtiği konumu bulur.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
template<class _InIt, class _Pr> inline  
    _InIt find_if(_InIt _First, _InIt _Last, _Pr _Pred);  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bu işlev C++ Standart Kitaplığı işlevi ile aynı şekilde davranır `find_if`. Daha fazla bilgi için bkz: [find_if](../standard-library/algorithm-functions.md#find_if).  
  
## <a name="requirements"></a>Gereksinimler  
 **Başlık:** \<cliext/algoritması >  
  
 **Namespace:** cliext
