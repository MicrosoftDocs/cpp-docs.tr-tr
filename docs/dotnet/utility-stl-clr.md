# <a name="utility-stlclr"></a>yardımcı program (STL/CLR)
STL/CLR üstbilgisini `<cliext/utility>` Şablon sınıfı tanımlamak için `pair` ve çeşitli destekleyici şablon işlevleri.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
#include <utility>  
```
