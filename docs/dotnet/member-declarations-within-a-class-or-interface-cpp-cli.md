# <a name="member-declarations-within-a-class-or-interface-ccli"></a>Sınıfta veya Arabirimde Üye Bildirimleri (C++/CLI)

Özellikleri ve işleçleri bildirimi kapsamlı bir şekilde Yönetilen Uzantılar'dan C++ için Yönetilen Uzantılar tasarımında ifşa edildi temel uygulama ayrıntılarını gizleme Visual C++ için yeniden oluşturulmuştur. Olay bildirimleri de değiştirilmiş.

Değişiklik kategorisi altında sahip hiçbir Yönetilen Uzantılar'desteği, statik oluşturucular artık olmasını tanımlanan çıkış (bunlar Yönetilen Uzantılar içinde'tanımlanan satır içi gerekiyordu) satır sonu ve temsilci oluşturucusu kavramı tanıtılmıştır.
