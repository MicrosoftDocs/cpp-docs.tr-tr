# <a name="net-programming-with-ccli-visual-c"></a>.NET programlama ile C + +/ CLI (Visual C++)
Aşağıdaki konularda, Visual C++ .NET platformu nasıl hedefleyebilirsiniz açıklanmaktadır.  
  
 Varsayılan olarak, Visual Studio 2015 ile oluşturulan CLR projeleri .NET Framework 4.5.2 hedefleyin. Hedef, yeni bir proje oluşturduğunuzda, .NET Framework 4.6 **yeni proje** iletişim kutusunda, açılır iletişim kutusunun üst orta adresindeki hedef çerçevesini değiştirebilir. Var olan bir proje hedef çerçevesi değiştirmek için projeyi kapatın, proje dosyası (.vcxproj) düzenlemek ve hedef Framework sürümü 4.6 için değiştirin. Proje sonraki açışınızda ayarlar geçerli olur.
