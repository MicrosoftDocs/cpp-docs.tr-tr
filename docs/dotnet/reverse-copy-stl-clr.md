# <a name="reversecopy-stlclr"></a>reverse_copy (STL/CLR)
Bir hedef aralığına kopyalarken bir kaynak aralıkta öğelerin sırasını tersine çevirir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
template<class _BidIt, class _OutIt> inline  
    _OutIt reverse_copy(_BidIt _First, _BidIt _Last, _OutIt _Dest);  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bu işlev C++ Standart Kitaplığı işlevi ile aynı şekilde davranır `reverse_copy`. Daha fazla bilgi için bkz: [reverse_copy](../standard-library/algorithm-functions.md#reverse_copy).

 **Namespace:** cliext
