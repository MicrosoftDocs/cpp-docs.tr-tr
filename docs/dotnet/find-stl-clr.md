# <a name="find-stlclr"></a>bul (STL/CLR)
Bir öğenin belirli bir değere sahip olan aralıktaki ilk geçtiği konumu bulur.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
template<class _InIt, class _Ty> inline  
    _InIt find(_InIt _First, _InIt _Last, const _Ty% _Val);  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bu işlev C++ Standart Kitaplığı işlevi ile aynı şekilde davranır `find`. Daha fazla bilgi için bkz: [Bul](../standard-library/algorithm-functions.md#find).
