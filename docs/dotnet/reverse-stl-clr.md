# <a name="reverse-stlclr"></a>tersine döndür (STL/CLR)
Bir aralık içindeki öğelerin sırasını tersine çevirir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
template<class _BidIt> inline  
    void reverse(_BidIt _First, _BidIt _Last);  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bu işlev C++ Standart Kitaplığı işlevi ile aynı şekilde davranır `reverse`. Daha fazla bilgi için bkz: [ters](../standard-library/algorithm-functions.md#reverse).

 **Namespace:** cliext
