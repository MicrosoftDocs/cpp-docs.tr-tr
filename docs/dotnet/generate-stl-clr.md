# <a name="generate-stlclr"></a>oluştur (STL/CLR)
Bir işlev nesnesi tarafından oluşturulan değerleri bir aralıktaki her öğeye atar.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
template<class _FwdIt, class _Fn0> inline  
    void generate(_FwdIt _First, _FwdIt _Last, _Fn0 _Func);  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bu işlev C++ Standart Kitaplığı işlevi ile aynı şekilde davranır `generate`. Daha fazla bilgi için bkz: [oluşturmak](../standard-library/algorithm-functions.md#generate).  
  
## <a name="requirements"></a>Gereksinimler  
 **Başlık:** \<cliext/algoritması >  
  
 **Namespace:** cliext
