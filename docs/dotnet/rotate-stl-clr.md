# <a name="rotate-stlclr"></a>döndür (STL/CLR)
İki bitişik aralık içindeki öğeleri birbiriyle değiştirir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
template<class _FwdIt> inline  
    void rotate(_FwdIt _First, _FwdIt _Mid, _FwdIt _Last);  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bu işlev C++ Standart Kitaplığı işlevi ile aynı şekilde davranır `rotate`. Daha fazla bilgi için bkz: [döndürme](../standard-library/algorithm-functions.md#rotate).

 **Namespace:** cliext
