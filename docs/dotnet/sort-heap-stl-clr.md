# <a name="sortheap-stlclr"></a>sort_heap (STL/CLR)

```  
template<class _RanIt> inline  
    void sort_heap(_RanIt _First, _RanIt _Last);

    void sort_heap(_RanIt _First, _RanIt _Last, _Pr _Pred);
```  
