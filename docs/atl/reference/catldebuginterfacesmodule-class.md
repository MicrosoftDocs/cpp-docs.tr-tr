# <a name="catldebuginterfacesmodule-class"></a>CAtlDebugInterfacesModule sınıfı

Bu sınıf, hata ayıklama arabirimleri için destek sağlar.

## <a name="syntax"></a>Syntax

```cpp
class CAtlDebugInterfacesModule
```

## <a name="remarks"></a>Açıklamalar

`CAtlDebugInterfacesModule` , arabirimlerde hata ayıklama için gereken desteği sağlar. [_ATL_DEBUG_QI](debugging-and-error-reporting-macros.md#_atl_debug_qi)sembolünü tanımlayan herhangi bir projeye dahildir.

## <a name="requirements"></a>Gereksinimler

**Üstbilgi:** atlbase. h
