# <a name="default-atl-project-configurations"></a>Varsayılan ATL Proje yapılandırması

ATL Proje Sihirbazı varsayılan olarak iki proje yapılandırması oluşturur:

|Yapılandırma|Karakter kümesi|ATL kullanımı|
|-------------------|-------------------|----------------|
|Yayınla|MBCS|DLL|
|Hata ayıklama|MBCS|DLL|

**Karakter kümesi**, **ATL kullanımı** ve **genel** sekmesi altındaki **proje ayarları** iletişim kutusunda tümü değiştirilebilir. Ayrıca, Configuration Manager kullanarak kendi yapılandırmalarınızı ekleyebilirsiniz. Ayrıntılar için bkz. [Derleme yapılandırması](/visualstudio/ide/understanding-build-configurations).
