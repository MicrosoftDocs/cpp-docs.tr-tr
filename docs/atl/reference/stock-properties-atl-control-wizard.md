# <a name="stock-properties-atl-control-wizard"></a>Hisse senedi özellikleri, ATL Denetim Sihirbazı

Sihirbazın bu sayfası, denetim için desteklenen stok özelliklerini tanımlar. Varsayılan olarak, hiçbir özellik tanımlanmaz.

- **Desteklenmiyor**

   Denetim için şu anda desteklenmeyen kullanılabilir özellikleri gösterir.

- **Desteklenir**

   Denetim için şu anda desteklenen özellikleri gösterir.

|Aktar düğmesi|Açıklama|
|---------------------|-----------------|
|**>**|**Desteklenen listeye,** **Desteklenmeyen** listede şu anda seçili olan özellik adını ekler.|
|**>>**|Desteklenen **listede,** **Desteklenmeyen** listede bulunan tüm özellik adlarını ekler.|
|**\<**|**Desteklenen** listede şu anda seçili olan özellik adını kaldırır.|
|**\<\<**|**Desteklenen** listede şu anda listelenen tüm özellik adlarını kaldırır.|
