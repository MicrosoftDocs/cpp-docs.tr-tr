# <a name="strings-atl-property-page-wizard"></a>Dizeleri, ATL Özellik Sayfası Sihirbazı
Özellik sayfası ile ilişkili metin sağlar.  
  
 **Başlık**  
 Özellik Sayfası sekmesinde görüntülenen metni ayarlar.  
  
 **Belge dize**  
 Sayfa açıklayan bir metin dizesini ayarlar. Bu dize özellik sayfası iletişim kutusunda görüntülenebilir. Özellik çerçeve açıklama bir durum satırı veya araç ipucunda kullanabilirsiniz. Standart özellik çerçeve şu anda bu dize kullanmaz.  
  
 **Yardım dosyası**  
 Özellik sayfası kullanmayı açıklar Yardım dosyasının adını ayarlar. Bu ad, bir yol içermemelidir. Kullanıcı bastığında **yardımcı**, çerçeve kendi CLSID altında özellik sayfası kayıt defteri girdileri HelpDir anahtarının değerini adlı dizinde Yardım dosyasını açar.
