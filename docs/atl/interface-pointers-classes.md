# <a name="interface-pointers-classes"></a>Arabirim işaretçileri sınıfları
Aşağıdaki sınıflar verilen arabirim işaretçisi yönetin:  
  
-   [CComPtr](../atl/reference/ccomptr-class.md) otomatik başvuru sayım gerçekleştirir.  
  
-   [CComQIPtr](../atl/reference/ccomqiptr-class.md) benzer `CComPtr`, ancak aynı zamanda otomatik arabirimleri sorgulama yapar.  
  
-   [CInterfaceArray](../atl/reference/cinterfacearray-class.md) arabirim işaretçileri sağlar yöntemleri COM dizisi oluşturma olduğu durumlarda faydalıdır.  
  
-   [CInterfaceList](../atl/reference/cinterfacelist-class.md) arabirim işaretçileri sağlar yöntemleri COM listesini oluşturma olduğu durumlarda faydalıdır.  
  
-   [CComGITPtr](../atl/reference/ccomgitptr-class.md) arabirim işaretçileri ve genel arabirim tablosu (GIT) ile ilgilenen için yöntemleri sağlar.
