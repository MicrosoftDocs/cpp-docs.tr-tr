# <a name="class-factories-classes"></a>Sınıf fabrikası sınıfları

Aşağıdaki sınıflar bir sınıf fabrikası uygular veya destekler:

- [CComClassFactory](../atl/reference/ccomclassfactory-class.md) Nesne oluşturma için varsayılan bir sınıf fabrikası sağlar.

- [CComClassFactory2](../atl/reference/ccomclassfactory2-class.md) Bir lisans aracılığıyla nesne oluşturmayı denetler.

- [CComClassFactoryAutoThread](../atl/reference/ccomclassfactoryautothread-class.md) Nesnelerin birden çok iş parçacığı havuza alınmış apartmanlar içinde oluşturulmasına izin verir.

- [CComClassFactorySingleton](../atl/reference/ccomclassfactorysingleton-class.md) Tek bir nesnesi oluşturur.

- [CComCoClass](../atl/reference/ccomcoclass-class.md) Nesne için sınıf fabrikasını tanımlar.
