# <a name="data-transfer-classes"></a>Veri Aktarımı sınıfları

Aşağıdaki sınıflar çeşitli veri aktarımı türlerini destekler:

- [IDataObjectImpl](../atl/reference/idataobjectimpl-class.md) Verileri almak ve ayarlamak için standart biçimleri kullanarak Tekdüzen Veri Aktarımı destekler. Öneri havuzları bağlantılarını yöneterek veri değişikliği bildirimlerini işler.

- [CBindStatusCallback](../atl/reference/cbindstatuscallback-class.md) Zaman uyumsuz bir bilinen adın, nesnenizin zaman uyumsuz veri aktarımı hakkındaki bilgileri göndermesini ve almasını sağlar.
