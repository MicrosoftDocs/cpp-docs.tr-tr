# <a name="site-information-classes"></a>Site bilgisi sınıfları

-   [IObjectWithSiteImpl](../atl/reference/iobjectwithsiteimpl-class.md) alır ve bir işaretçi bir nesnenin siteye ayarlar. Denetimleri olmayan nesneler için kullanılır.
