# <a name="composite-controls-classes"></a>Bileşik denetimler sınıfları
Aşağıdaki sınıf bileşik denetimler oluşturmak için destek sağlar.  
  
-   [CComCompositeControl](../atl/reference/ccomcompositecontrol-class.md) türetilmiş ActiveX denetimlerini `CComCompositeControl` standart iletişim kutusu tarafından barındırılır. Bu tür denetimler, diğer denetimler (yerel Windows denetimleri ve ActiveX denetimlerini) barındırabilen olduklarından bileşik denetimler adı verilir.
