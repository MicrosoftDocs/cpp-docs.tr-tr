# <a name="composite-controls-classes"></a>Bileşik denetim sınıfları

Aşağıdaki sınıf bileşik denetimler oluşturma desteği sağlar

- [CComCompositeControl](../atl/reference/ccomcompositecontrol-class.md) İçinden türetilmiş ActiveX denetimleri `CComCompositeControl` , standart iletişim kutusuyla barındırılır.
