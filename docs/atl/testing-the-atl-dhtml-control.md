# <a name="testing-the-atl-dhtml-control"></a>ATL DHTML denetimini test etme

Projeniz oluşturulduktan sonra oluşturun ve örnek denetim test edebilirsiniz. Bunu yapmadan önce sınıf görünümü ve Çözüm Gezgini proje incelemek için kullanın. Proje öğelerini daha ayrıntılı olarak açıklanan [DHTML denetim projesinin öğelerini tanımlama](../atl/identifying-the-elements-of-the-dhtml-control-project.md).

#### <a name="to-build-and-test-the-atl-dhtml-control"></a>Derleme ve ATL DHTML denetimini test etmek için

1. Projeyi oluşturun. Gelen **derleme** menüsünde tıklatın **Çözümü Derle**.

2. Derleme tamamlandığında Test kapsayıcıyı açın. Bkz: [Test kapsayıcısı ile test etme özellikleri ve olayları](../mfc/testing-properties-and-events-with-test-container.md) Test kapsayıcısı erişim hakkında daha fazla bilgi için.

3. Test kapsayıcısında, gelen **Düzenle** menüsünde tıklatın **yeni denetimi Ekle**.

4. İçinde **ekler denetimi** iletişim kutusunda, liste kutusundan denetiminizi seçin. Unutmayın, ATL denetimi Sihirbazı'nda gösterilen kısa ad adını temel alır. **Tamam**'ı tıklatın.

5. Denetim inceleyin. Bir kaydırma çubuğuna sahip olduğunu unutmayın. Kaydırma çubuğunu etkinleştirmek için yeniden boyutlandırma denetim tutamaçlarını kullanın.

6. Denetimin düğmeleri test edin. Düğme tarafından belirtilen renk arka plan rengi değişir.

7. Test kapsayıcısını kapatın.

Ardından, deneyin [DHTML denetimini değiştirme](../atl/modifying-the-atl-dhtml-control.md).
