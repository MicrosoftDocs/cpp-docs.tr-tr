# <a name="testing-the-atl-dhtml-control"></a>ATL DHTML denetimi test etme
Projenizi oluşturduktan sonra derleme ve örnek denetimi test. Bunu önce sınıf görünümü ve Çözüm Gezgini proje incelemek için kullanın. Projenizin öğeleri daha ayrıntılı olarak açıklanmıştır [DHTML denetimi proje öğelerini tanımlama](../atl/identifying-the-elements-of-the-dhtml-control-project.md).  
  
#### <a name="to-build-and-test-the-atl-dhtml-control"></a>Derleme ve ATL DHTML denetimi sınamak için  
  
1.  Projeyi oluşturun. Gelen **yapı** menüsünde tıklatın **yapı çözümü**.  
  
2.  Yapı tamamlandığında Test kapsayıcısı açın. Bkz: [test özellikleri ve olayları Test kapsayıcısı ile](../mfc/testing-properties-and-events-with-test-container.md) Test kapsayıcısı erişim hakkında bilgi için.  
  
3.  Test kapsayıcısı içinde gelen **Düzenle** menüsünde tıklatın **yeni denetimi Ekle**.  
  
4.  İçinde **ekler denetimi** iletişim kutusunda, liste kutusundan denetiminizi seçin. ATL Denetim Sihirbazı'nda gösterilen kısa ad adını dayalı unutmayın. **Tamam**'ı tıklatın.  
  
5.  Denetim inceleyin. Bir kaydırma çubuğunun olduğuna dikkat edin. Scrollbar etkinleştirmek için denetimin yeniden boyutlandırmak için denetimin tanıtıcıları kullanın.  
  
6.  Denetimin düğmeleri sınayın. Düğme tarafından belirtilen renk için arka plan rengi değişir.  
  
7.  Test kapsayıcısı kapatın.  
  
 Ardından, deneyin [DHTML denetimi değiştirme](../atl/modifying-the-atl-dhtml-control.md).
