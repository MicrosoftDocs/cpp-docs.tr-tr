# <a name="atl-registry-component-registrar"></a>ATL Kayıt Defteri Bileşeni (Kaydedici)

> [!NOTE]
>  ATL Kaydedicisi için kaynak kodunu atlmfc\include\atliface.h içinde bulunabilir.
