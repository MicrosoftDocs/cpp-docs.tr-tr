# <a name="creating-registrar-scripts"></a>Kaydedici betikleri oluşturma
Kaydedici betik sistem kayıt defteri API temelli yerine verilere erişim sağlar. Yalnızca bir veya iki satır bir anahtarı kayıt defterine eklemek için komut sürdüğünü beri verilere erişim genellikle daha verimli olur.  
  
 [ATL Denetim Sihirbazı](../atl/reference/atl-control-wizard.md) Kaydedici betik COM sunucunuz için otomatik olarak oluşturur. Bu komut, nesneyle ilişkili .rgs dosyasında bulabilirsiniz.  
  
 ATL kayıt şirketinin betik altyapısı kayıt betiğinizin çalışma zamanında işler. ATL betik altyapısı server kurulumu sırasında otomatik olarak çağırır.
