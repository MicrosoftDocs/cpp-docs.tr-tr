# <a name="scope-of-atl"></a>ATL'nin kapsamı

ATL COM nesneleri, otomasyon sunucuları ve ActiveX denetimlerini kolayca oluşturmanıza olanak sağlar. ATL temel COM arabirimlerinin çoğu için yerleşik destek sağlar.

ATL uygulamanızda içeren kaynak kodu olarak sevk edilir. ATL ayrıca bileşenler genelinde paylaşılabilir kod içeren bir DLL kullanılabilir (atl90.dll) sağlar. Ancak, bu DLL gerekli değildir.
