# <a name="creating-an-atl-dhtml-control"></a>ATL DHTML denetimi oluşturma
ATL Denetim Sihirbazı'nı DHTML denetimi oluşturma işlemini otomatikleştirir. Örnek kodunu içeren bir HTML dosyası da dahil, tüm gerekli kaynak dosyaları oluşturur.  
  
#### <a name="to-create-an-atl-dhtml-control"></a>ATL DHTML denetimi oluşturmak için  
  
1.  Adımları [bir ATL projesi oluşturma](../atl/reference/creating-an-atl-project.md).  
  
2.  Sınıf Görünümü'nde, proje düğümüne sağ tıklayın, fareyle **Ekle**, tıklatıp **sınıfı Ekle** kısayol menüsünden. İçinde **sınıfı Ekle** iletişim kutusu, çift [ATL Denetim Sihirbazı](../atl/reference/atl-control-wizard.md). ATL Denetim Sihirbazı'nda tıklatın **seçenekleri** sekmesinde ve seçin **DHTML denetimi**.  
  
 Artık [varsayılan denetim test](../atl/testing-the-atl-dhtml-control.md).
