# <a name="using-task-manager"></a>Görev Yöneticisini kullanma

Bir hizmette hata ayıklamanın en basit yöntemlerinden biri, görev yöneticisinin kullanımı ile aynıdır. Hizmet çalışırken, Görev Yöneticisi 'Ni başlatın ve **süreçler** sekmesine tıklayın. EXE adına sağ tıklayın ve ardından **Hata Ayıkla**' ya tıklayın. Bu, çalışan işleme eklenen Visual C++ başlatır. Şimdi, kodunuzda kesme noktaları ayarlamanıza olanak tanımak için **hata ayıklama** menüsünde **Kes** ' e tıklayın. Seçili kesme noktalarınız için çalıştırmak üzere **Çalıştır** ' a tıklayın.
