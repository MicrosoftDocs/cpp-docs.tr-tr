# <a name="atl-collection-and-enumerator-classes"></a>ATL koleksiyon ve Numaralandırıcı sınıfları
ATL koleksiyonları ve numaralandırmalar uygulamanıza yardımcı olmak için aşağıdaki sınıflar sağlar.  
  
|örneği|Açıklama|  
|-----------|-----------------|  
|[ICollectionOnSTLImpl](../atl/reference/icollectiononstlimpl-class.md)|Koleksiyon arabirim uygulaması|  
|[IEnumOnSTLImpl](../atl/reference/ienumonstlimpl-class.md)|Numaralandırıcı arabirim uygulaması (C++ Standart kitaplığı ile uyumlu kapsayıcısında depolanan verileri varsayar)|  
|[CComEnumImpl](../atl/reference/ccomenumimpl-class.md)|Numaralandırıcı arabirim uygulaması (bir dizisinde depolanan verileri varsayar)|  
|[CComEnumOnSTL](../atl/reference/ccomenumonstl-class.md)|Numaralandırıcı nesne uygulaması (kullanan `IEnumOnSTLImpl`)|  
|[CComEnum](../atl/reference/ccomenum-class.md)|Numaralandırıcı nesne uygulaması (kullanan `CComEnumImpl`)|  
|[_Copy](../atl/atl-copy-policy-classes.md)|Kopya ilke sınıfı|  
|[_CopyInterface](../atl/atl-copy-policy-classes.md)|Kopya ilke sınıfı|  
|[CAdapt](../atl/reference/cadapt-class.md)|Bağdaştırıcı sınıfı (gizler **işleci &** izin vererek `CComPtr`, `CComQIPtr`, ve `CComBSTR` C++ Standart Kitaplığı kapsayıcılarında depolanması için)|
