# <a name="c28289"></a>C28289

> Uyarı C28289: işlev Için: ReadableTo veya WritableTo parametre olarak bir limit-spec içermiyordu

Kod çözümleme aracı, işlev/parametre ek açıklaması Not edildiğinde yanlış kodlandığı zaman bu uyarıyı bildirir.
