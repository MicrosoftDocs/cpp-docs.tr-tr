# <a name="c28283"></a>C28283

> Uyarı C28283: sembol Için, belirtilen boyut belirtimi henüz desteklenmiyor

Uyarı belirtimi "Sentinel" ek açıklamanın sıfırdan farklı bir değer aldığını gösterir. Temelde, çağıran dizenin ikili sıfır dışında bir karakterle sonlandırıldığını söylemeye çalıştı.
