# <a name="c28227"></a>C28227

> Uyarı C28227: ek açıklama aynı zamanda bir PrimOp olamaz (önceki bildirime bakın)

Bu uyarı, adlandırılmış ek açıklamanın sıradan bir ek açıklama olarak bildirildiği ve daha önce bir PrimOp olarak bildirildiği anlamına gelir. Bu, standart makrolar ek açıklamalar için kullanılıyorsa mümkün olmamalıdır; Bu uyarının tipik kullanımda görülebilmesi beklenmez.
