# <a name="c28206"></a>C28206

> Uyarı C28206: \<expression> : Left işleneni bir yapıya işaret ediyor, kullanma `->`

Bu uyarı, yapı işaretçisi başvuru gösterimi `->` beklendiğinde bildirilir.
