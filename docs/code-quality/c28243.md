# <a name="c28243"></a>C28243

> Uyarı C28243: parametre üzerindeki işlev için ek açıklama, ek açıklama eklenmiş olan gerçek türden daha fazla başvuru gerektiriyor

`__deref`Ek açıklamada bulunan işleç sayısı, parametre türü tarafından tanımlanan işaretçinin seviyelerinin sayısından daha fazla. Ek açıklamanın numarasını ya da başvurulan parametrenin işaretçi düzeylerini değiştirerek bu uyarıyı düzeltin.
