# <a name="c28022"></a>C28022

> uyarı C28022: bu işlevdeki işlev sınıfları, \<classlist1> \<classlist2> tanımlamak için kullanılan typedef üzerindeki işlev sınıfları (es) ile eşleşmiyor.

Bu uyarı, ek açıklamalarda bir hata olduğunda bildirilir. Hem typedef hem de işlevin kendisi `_Function_class_` ek açıklamalar vardır, ancak eşleşmez. Her ikisi de kullanılırsa eşleşmelidir.
