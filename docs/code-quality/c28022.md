# <a name="c28022"></a>C28022

> uyarı C28022: bu işlevdeki işlev sınıfı (es) \<classlist1 >, tanımlamak için kullanılan typedef üzerinde \<classlist2 > işlev sınıfları ile eşleşmiyor.

Bu uyarı, ek açıklamalarda bir hata olduğunda bildirilir. Hem typedef hem de işlevin kendisi `_Function_class_` ek açıklamalarına sahiptir, ancak eşleşmez. Her ikisi de kullanılırsa eşleşmelidir.
