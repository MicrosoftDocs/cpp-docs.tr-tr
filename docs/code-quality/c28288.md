# <a name="c28288"></a>C28288

> Uyarı C28288: işlev Için, \_ at \_ () ek açıklamasında söz dizimi hatası (geçersiz parametre adı)

`SAL_at`( `__drv_at` ) Ek açıklaması kullanıldığında ve parametre ifadesi geçerli bağlamda yorumlanırken, Kod Analizi Aracı bu uyarıyı bildirir. Bu, yanlış yazılmış bir parametre veya üye adı ya da "return" veya "This" anahtar sözcüklerinden oluşan hatalı bir işlem içerebilir.
