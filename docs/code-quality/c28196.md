# <a name="c28196"></a>C28196

> Uyarı C28196: gereksinim karşılanmadı. (İfade true olarak değerlendirilmez.)

Bu uyarı, çözümlenmekte olan işlevin işlev prototipinin bir `__notnull` `__null` veya bir `__drv_valueIs` `_Out_` parametre ya da dönüş değeri olduğunu gösterir, ancak döndürülen değer bu ek açıklama ile tutarsız olur.
