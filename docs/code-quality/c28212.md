# <a name="c28212"></a>C28212

> Uyarı C28212: ek açıklama için beklenen Işaretçi ifadesi

Bu uyarı, bir ek açıklamanın (açıklanmakta olan işlevin değil) bir işaretçi veya dizi türü olması beklendiğini, ancak başka bir tür ile karşılaşıldığını gösterir. Ek açıklamanın düzeltilmesi gerekiyor.
