# <a name="c26477-use_nullptr_not_constant"></a>C26477 USE_NULLPTR_NOT_CONSTANT

`nullptr` , `nullptr_t` özel null işleme ile aşırı yüklemelerin kullanılmasına izin veren özel bir tür içerir. Yerine `0` veya yerine kullanılması `NULL` `nullptr` , tarafından sağlanan tür güvenliğini ve kesintiyi atlar `nullptr` .
