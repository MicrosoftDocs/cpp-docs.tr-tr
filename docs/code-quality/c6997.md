# <a name="c6997"></a>C6997

> Uyarı C6997: Bu konumdaki ek açıklamalar anlamsız ve yok sayılacak.

Ek açıklamalar öğesine uygulanamaz `extern "C" {...}` . Ek açıklamaları belirli bir nesneye uygulayın.
