# <a name="c28290"></a>C28290

> Uyarı C28290: işlev için ek açıklama, gerçek parametre sayısından daha fazla dışlar içeriyor

Kod Analizi Aracı, işlevin ek açıklaması gerçek parametre sayısından daha fazla dışlar içerdiğinde bu uyarıyı bildirir.
