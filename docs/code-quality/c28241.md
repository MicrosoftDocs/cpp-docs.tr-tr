# <a name="c28241"></a>C28241

> Uyarı C28241: parametre üzerindeki işlev için ek açıklama tanınmıyor

Tanınmayan bir ek açıklama adı kullanıldı. Bu, standart makrolar ek açıklamalar için kullanılıyorsa mümkün olmamalıdır; Bu uyarının tipik kullanımda görülebilmesi beklenmez.
