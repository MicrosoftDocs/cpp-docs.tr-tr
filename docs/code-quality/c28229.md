# <a name="c28229"></a>C28229

> uyarı C28229: ek açıklama parametreleri desteklemiyor

Bu uyarı, ek açıklamanın parametre olmadan bildirildiği bir parametreyle birlikte kullanıldığını gösterir. Bu, standart makrolar ek açıklamalar için kullanılıyorsa mümkün olmamalıdır; Bu uyarının tipik kullanımda görülebilmesi beklenmez.
