# <a name="c28304"></a>C28304

> uyarı C28304: C++ başvuru parametresi Için <parameter_name>, uygulanmış uygun bir `_Notref_` işleç bulundu \<token> .

`_Notref_`İşleci yalnızca C++ başvuru parametreleri ve yalnızca sistem tarafından belirtilen makrolarda bulunan özel koşullarda kullanılmalıdır. Hemen arkasından bir `_Deref_` operatör veya işlevsel ek açıklama gelmelidir.
