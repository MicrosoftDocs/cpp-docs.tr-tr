# <a name="c28304"></a>C28304

> uyarı C28304: başvuru C++ parametresi < Parameter_name > için, \<belirteci > uygulanmış, yanlış yerleştirilmiş `_Notref_` işleci bulundu.

`_Notref_` işleci yalnızca başvuru parametreleri ve sistem tarafından belirtilen makrolarla ilgili C++ özel koşullarda kullanılmalıdır. Hemen arkasından bir `_Deref_` işleci veya işlevsel ek açıklama gelmelidir.
