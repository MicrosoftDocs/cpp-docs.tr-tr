# <a name="c28204"></a>C28204

> Uyarı C28204: \<function> : Bu aşırı yükten yalnızca biri \<filename> ( \<line> ) şu şekilde \<paramname> açıklanmalıdır: her ikisi de veya hiçbirine açıklama eklenmelidir.

Bu uyarı, ek açıklamalarda bir hata olduğunda bildirilir.
