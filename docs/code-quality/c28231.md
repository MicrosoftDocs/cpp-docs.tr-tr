# <a name="c28231"></a>C28231

> Uyarı C28231: ek açıklama yalnızca dizide geçerlidir

Bu uyarı, bir ek açıklamanın bağımsız değişkeninin bir dizi olması gerektiğini ve başka bir tür ile karşılaşıldığını gösterir.
