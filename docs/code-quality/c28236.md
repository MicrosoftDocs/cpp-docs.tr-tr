# <a name="c28236"></a>C28236

> Uyarı C28236: ek açıklama bir ifadede kullanılamaz

Bu uyarı, bir ifade bağlamında ek açıklama olarak belirtilen bir işlevi kullanmak için bir girişimde bulunuldu olduğunu gösterir. Bu, standart makrolar ek açıklamalar için kullanılıyorsa mümkün olmamalıdır; Bu uyarının tipik kullanımda görülebilmesi beklenmez.
