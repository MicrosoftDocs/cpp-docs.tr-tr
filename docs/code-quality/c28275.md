# <a name="c28275"></a>C28275

> Uyarı C28275: \_makro\_değer\_ parametresi null

Bu uyarı, analiz edilen kodda değil, model dosyasında bir iç hata olduğunu gösterir. *Makrodeğeri* işlevi parametre olmadan çağrıldı.
