# <a name="c26464-use_const_pointer_for_elements"></a>C26464 USE_CONST_POINTER_FOR_ELEMENTS

'% Array% ' dizisinin öğeleri tarafından işaret edilen değerler yalnızca bir kez atanır, öğeleri `const` (con. 4) işaretçisi olarak işaretleyin.

## <a name="see-also"></a>Ayrıca bkz.

[C++ temel yönergeleri Con. 4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).

## <a name="remarks"></a>Açıklamalar

- Bu kural şu anda CppCoreCheck içinde uygulanmıyor.
