# <a name="c26496-use_const_for_variable"></a>C26496 USE_CONST_FOR_VARIABLE

> '*Variable*' değişkeni yalnızca bir kez atanır, olarak işaretleyin `const` . 

## <a name="see-also"></a>Ayrıca bkz.
[C++ temel yönergeleri Con. 4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).

## <a name="example"></a>Örnek
```cpp
int GetTheNumber();
int GiveMeTheNumber(int);

void function1()
{
    int theNumber = GetTheNumber(); // C26496, 'theNumber' is never assigned to again, so it can be marked as const
    std::cout << theNumber << '\n';
    
    GiveMeTheNumber(theNumber);
    // ...
}
```
