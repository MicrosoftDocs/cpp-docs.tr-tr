# <a name="c28160"></a>C28160

> Uyarı C28160: hata ek açıklaması

Bu uyarı, bir `__drv_error` ek açıklama ile karşılaşıldığında bildirilir. Bu ek açıklama, düzeltilmesi gereken kodlama uygulamalarını işaretlemek için kullanılır ve `__drv_when` belirli parametre birleşimlerini göstermek için bir ek açıklamayla birlikte kullanılabilir.
