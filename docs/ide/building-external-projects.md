# <a name="building-external-projects"></a>Harici Projeler Derleme
Harici bir proje, derleme görevleri dosyası veya (yabancı veya dış) dışındaki diğer tesis kullanan bir Visual C++ proje olan Visual C++ geliştirme ortamı.  
  
 Visual C++ geliştirme ortamında yapı, bir derleme görevleri dosyası projesi oluşturun ve projenizin belirtmek istediğiniz harici projenin (örneğin, bir derleme görevleri dosyası projesi) varsa, komut ve çıktı derleme görevleri dosyası Uygulama Sihirbazı'nda oluşturun. Daha fazla bilgi için bkz: [derleme görevleri dosyası projesi oluşturma](../ide/creating-a-makefile-project.md).  
  
 Visual C++ artık etkin proje için bir derleme görevleri dosyası geliştirme ortamından verme olanağı desteklediğini unutmayın. Kullanım [Devenv komut satırı anahtarları](/visualstudio/ide/reference/devenv-command-line-switches) komut satırında Visual Studio projeleri oluşturmak üzere.
