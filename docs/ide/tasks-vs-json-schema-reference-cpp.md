# <a name="tasksvsjson-c"></a>Tasks.vs.json (C++)

A `tasks.vs.json` dosyası, herhangi bir rastgele görev tanımlamak ve ondan çağırmak için bir klasör Aç projesine eklenebilir **Çözüm Gezgini** bağlam menüsü. CMake projeleri genellikle kullanmayın bu dosya, tüm derleme komutları belirtilmediğinden `CMakeLists.txt`. CMake dışındaki derleme sistemlerinde, burada belirtin, derleme komutları ve derleme betikleri çağırma budur. Tasks.vs.json kullanma hakkında genel bilgi için bkz. [yapı özelleştirme ve hata ayıklama görevleri için "Klasör Aç" geliştirme](/visualstudio/ide/customize-build-and-debug-tasks-in-visual-studio).
