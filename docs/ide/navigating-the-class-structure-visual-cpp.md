# <a name="navigating-the-class-structure-visual-c"></a>Sınıf Yapısında Gezinme (Visual C++)
İşlevler, devralma ve arabirimler kullanarak da dahil olmak üzere bir proje sınıf yapısını gidebilirsiniz **sınıf görünümü** veya **Nesne Tarayıcısı**. Daha fazla bilgi için [kod yapısını görüntüleme](/visualstudio/ide/viewing-the-structure-of-code).
