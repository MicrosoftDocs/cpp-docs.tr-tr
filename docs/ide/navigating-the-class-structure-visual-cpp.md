# <a name="navigate-the-class-structure"></a>Sınıf yapısında gezinme

Bir proje sınıf yapısını kullanarak gidebilirsiniz **sınıf görünümü** veya **Nesne Tarayıcısı**. Proje sınıfı yapısını, İşlevler, kalıtım ve arabirimler içerebilir. Daha fazla bilgi için [kod yapısını görüntüleme](/visualstudio/ide/viewing-the-structure-of-code).

Bir işlev bildirimi için Git, Sınıf Görünümü'nde işlev adına çift tıklayın. Bir sınıf tanımına gitmek için Sınıf Görünümü'nde bir sınıf adına çift tıklayın.
